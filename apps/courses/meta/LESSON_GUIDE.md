# Руководство по созданию уроков
**Glottos Matrix — структурный курс**

---

## Философия

> 5% — понять правило. 95% — натренировать речевой аппарат.
> Язык — это спорт, не наука. Рот должен работать, не голова.

Каждый урок — это тренировка, не лекция. Минимум теории, максимум повторений вслух. Ученик должен закрыть урок с ощущением «я это **могу сказать**», а не «я это **понял**».

---

## Тон и стиль

### Голос
- **Прямой, дерзкий, без воды.** Как тренер, не как профессор.
- **Обращение на «ты»**, императив: «Прочитай», «Повтори», «Не думай — говори».
- **Короткие предложения.** Одна мысль = одно предложение.

### Приёмы

| Приём | Пример | Когда использовать |
|-------|--------|-------------------|
| **Метафора** | «Язык — это спорт» | Мотивация, объяснение подхода |
| **Ловушка!** | «fragen = Akk, antworten = Dat — не путай!» | Частые ошибки, контрасты |
| **Якорь** | «Wem? → M → deM. Это твой якорь.» | Мнемоники, хаки запоминания |
| **Два по цене одного** | «Мужской и средний в Dativ одинаковые» | Упрощение, снятие страха |
| **Провокация** | «Немцы ленивые — им лень произносить лишние буквы» | Юмор, запоминание исключений |

### Чего НЕ делать
- Не писать длинные объяснения — если абзац больше 3 строк, разбей его
- Не использовать лингвистические термины без перевода на простой язык
- Не копировать чужие примеры — всегда свои, аналогичные
- Не добавлять эмодзи
- Не писать «в этом уроке мы рассмотрим...» — сразу к делу

## Структура урока

### Шапка

```markdown
# Урок N: [Грамматическая тема]
**Словарный запас: [Тема лексики]**

---
```

### Блок 1: Как работать с уроком

Инструкция из 3–4 шагов + мотивационная цитата в blockquote.

**Уроки 1–8 (Блок «Выживание»):**
```markdown
1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — медленно, осознанно, анализируя каждый звук
3. **Ускоряйся** — повторяй, пока фразы не будут вылетать сами
```

**Уроки 9–50 (Блоки с грамматикой):**
```markdown
1. **Прочитай** правило — пойми логику (5 минут)
2. **Переведи** упражнения письменно — проверь по ключу
3. **Проговори вслух** — медленно, осознанно, анализируя каждый артикль/окончание
4. **Ускоряйся** — гоняй матрицу, пока не будет вылетать на автомате
```

Затем — блокцитата, связывающая с предыдущим материалом:
```markdown
> К этому моменту ты уже знаешь [предыдущая тема]. Теперь добавляем [новая тема].
> [Снятие страха или мотивация].
```

### Блок 2: Теория (грамматика)

**Структура:**
1. **Объяснение концепции** — 2–5 предложений простым языком с примером на русском
2. **Главный хак / мнемоника** — в отдельном подразделе с blockquote-якорем
3. **Таблица** — компактная, с выделением паттернов жирным шрифтом
4. **3 ключевых вывода** — пронумерованный список после таблицы
5. **Ловушки** — blockquote с контрастными примерами

**Формат таблиц грамматики:**
```markdown
|  | Мужской (der) | Средний (das) | Женский (die) | Множ. число |
|---|---|---|---|---|
| **Форма** | de**m** | de**m** | der | den + **(-n)** |
```

Жирным выделяй окончания-паттерны, которые нужно запомнить.

**Формат ловушек:**
```markdown
> **Ловушка!** [Частая ошибка]. [Контрастный пример].
> [Правильная логика / мнемоника].
```

### Блок 3: Словарь урока

Таблица с новыми словами, привязанными к теме лексики из curriculum.md.

**Для существительных (уроки 9+) — всегда указывай род:**
```markdown
| Немецкий | Род | Русский |
|----------|-----|---------|
| der Bahnhof | m | вокзал |
| die Brücke | f | мост |
| das Rathaus | n | ратуша |
```

**Для глаголов/прилагательных/фраз — без рода:**
```markdown
| Немецкий | Русский | Заметка |
|----------|---------|---------|
| Guten Morgen | Доброе утро | до ~10:00 |
```

**Количество:** 15–25 слов/фраз на урок. Не больше — иначе не запомнят.

### Блок 4: Языковые гаммы (уроки 3+)

Гаммы — это прогон одной фразы через все лица (ich → du → er → sie → wir → ihr → Sie/sie). Формат — code block:

```markdown
### Гамма N — [глагол/конструкция] + [что отрабатываем]

‍```
Ich [фраза]
Du [фраза]
Er [фраза]
Sie [фраза]
Wir [фраза]
Ihr [фраза]
Sie [фраза]
‍```
```

**Правила:**
- 2–3 гаммы на урок
- Каждая гамма отрабатывает **одну** конструкцию
- Фраза должна быть содержательной (не «Ich lerne», а «Ich lerne jeden Tag neue Wörter»)
- После гаммы — примечание если глагол нерегулярный

### Блок 5: Языковая матрица (уроки 3+)

Матрица — Q&A дриллы в трёх сценариях. Формат — code block:

```markdown
### Матрица 1 — «Ты ↔ Я»
‍```
[Вопрос на du]? – [Ответ на ich]
[Следующий вопрос]? – [Ответ]
‍```

### Матрица 2 — «Третьи лица» (он/она/они)
‍```
[Вопрос про третье лицо]? – [Ответ]
‍```

### Матрица 3 — Ситуативная (привязана к теме словаря)
‍```
[Вопрос из жизненной ситуации]? – [Ответ]
‍```
```

**Правила:**
- 5–7 пар в каждой матрице
- Используй «принцип зеркала» — ответ строится из вопроса
- Матрица 3 всегда привязана к теме словаря (город, врач, магазин и т.д.)
- В ранних уроках (1–2) матрица отсутствует — только диалоги

### Блок 6: Упражнения

4–5 упражнений с нарастающей сложностью. У каждого — задание + пункты + ключ в `<details>`.

#### Типы упражнений (выбирай 4–5 из списка):

| Тип | Описание | Когда использовать |
|-----|----------|-------------------|
| **Чтение вслух** | Прочитай слова/фразы, применяя правила | Уроки 1–2 (произношение) |
| **Подстановка** | Вставь правильный артикль/окончание/форму | Уроки с грамматикой падежей, склонений |
| **Определи падеж/форму** | Поставь (A), (D), (N) над артиклями | Уроки 9–18 (падежи) |
| **Перевод RU→DE** | Переведи 8–10 предложений | Каждый урок |
| **Контраст** | Переведи пары, подчёркивающие разницу | Когда есть путаница (nicht/kein, Akk/Dat, wenn/als) |
| **Диалог-перевод** | Переведи мини-диалог (2–3 реплики) | Каждый урок — привязка к теме словаря |
| **Матрица наизусть** | Закрой ответы, отвечай вслух мгновенно | Уроки 3+ (закрепление матрицы) |
| **Трансформация** | Перестрой предложение (актив→пассив, наст.→прош.) | Продвинутые уроки (27+) |
| **Исправь ошибку** | Найди и исправь ошибку в предложении | Любой урок для разнообразия |

**Формат упражнения:**
```markdown
### Упражнение N. [Краткое задание]

[Инструкция одним предложением.]

1. [Пункт]
2. [Пункт]
...

<details>
<summary>Ключ</summary>

1. *[Ответ на немецком курсивом]* (пояснение если нужно)
2. *[Ответ]*
...

</details>
```

**Формат диалога-упражнения:**
```markdown
**Диалог N:**
— [Реплика на русском]
— [Ответная реплика]

<details>
<summary>Ключ</summary>

**Диалог N:**
— *[Реплика на немецком]*
— *[Ответная реплика]*

</details>
```

### Блок 7: Памятка урока

Компактная шпаргалка в code block — всё главное на одном экране:

```markdown
## Памятка урока

‍```
[ПРАВИЛО в одну строку]

[Таблица/схема самого главного]

[Список ключевых слов/глаголов/исключений]
‍```
```

### Блок 8: Анонс следующего урока

```markdown
> **Следующий шаг:** Урок N+1 — [Тема]. [Интригующая фраза — почему это важно или интересно].
```

## Адаптация по уровням

### Блок 1: Выживание (уроки 1–8)

| Компонент | Особенности |
|-----------|------------|
| Теория | Минимальная. Правила через примеры, не таблицы |
| Словарь | Без указания рода (ещё не знают падежей) |
| Гаммы | Появляются с урока 3 (haben/sein) |
| Матрица | Появляется с урока 3. Простая: только «ты ↔ я» |
| Упражнения | Чтение вслух, перевод фраз, диалоги |
| Падежи | Не упоминаются |

### Блок 2: Фундамент падежей (уроки 9–18)

| Компонент | Особенности |
|-----------|------------|
| Теория | Таблицы артиклей. Контрасты падежей. Мнемоники |
| Словарь | С указанием рода (m/f/n) |
| Гаммы | 2–3 штуки, прогон через все лица |
| Матрица | Полная: ты↔я + третьи лица + ситуативная |
| Упражнения | Подстановка, определение падежа, контраст, перевод, диалоги |
| Акцент | Осознанный выбор артикля: «задай вопрос → определи падеж → выбери артикль» |

### Блок 3: Расширение (уроки 19–25)

| Компонент | Особенности |
|-----------|------------|
| Теория | Средняя сложность. Новые конструкции (отделяемые приставки, sich, werden) |
| Словарь | С родом. Тематические блоки (распорядок дня, природа, погода) |
| Гаммы | 2–3 штуки с усложнением (возвратные глаголы, приставки) |
| Матрица | Полная + более длинные фразы |
| Упражнения | Перевод, трансформация, диалоги из жизни |

### Блок 4: Прошедшее время (уроки 26–35)

| Компонент | Особенности |
|-----------|------------|
| Теория | Три формы глаголов. Группировка неправильных по паттернам |
| Словарь | С родом. Тематика: работа, образование, технологии |
| Гаммы | Прогон через времена (Präsens → Perfekt → Präteritum) |
| Матрица | С переключением времён в Q&A |
| Упражнения | Трансформация времён, перевод, контраст Perfekt/Präteritum |

### Блок 5: Продвинутая грамматика (уроки 36–46)

| Компонент | Особенности |
|-----------|------------|
| Теория | Genitiv, склонение прилагательных, Passiv, Konjunktiv II |
| Словарь | Абстрактная лексика, идиомы, деловой немецкий |
| Гаммы | Сложные конструкции (пассив, сослагательное) |
| Матрица | Дискуссионная: мнения, аргументы, гипотезы |
| Упражнения | Трансформация (актив→пассив), перевод сложных структур |

### Блок 6: Мастерство (уроки 47–50)

| Компонент | Особенности |
|-----------|------------|
| Теория | Plusquamperfekt, нюансы модальных, Konjunktiv I |
| Словарь | Публичные выступления, журналистика, частицы |
| Гаммы | Переключение между временами и наклонениями |
| Матрица | Пересказ, косвенная речь, дебаты |
| Упражнения | Трансформация речи, стилистические контрасты |

## Чеклист перед сдачей урока

- [ ] Шапка: номер, грамматическая тема, тема словаря
- [ ] Блок «Как работать» с 3–4 шагами
- [ ] Мотивационная цитата + связка с предыдущим уроком
- [ ] Теория: объяснение + хак/мнемоника + таблица + ловушки
- [ ] Словарь: 15–25 слов, род указан (для уроков 9+)
- [ ] Гаммы: 2–3 штуки (для уроков 3+)
- [ ] Матрица: 3 сценария (для уроков 3+)
- [ ] Упражнения: 4–5 штук с `<details>` ключами
- [ ] Памятка урока в code block
- [ ] Анонс следующего урока
- [ ] Все примеры оригинальные
- [ ] Тон: прямой, «ты», императив, без воды
- [ ] Ловушки и контрасты выделены
