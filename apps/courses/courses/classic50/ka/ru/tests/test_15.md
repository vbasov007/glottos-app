# Тест к уроку 15: Закрепление всех 7 падежей

**Как сказать по-грузински?**
Переведи фразы на грузинский язык (мхедрули). Используй грамматику и лексику урока. В каждом предложении один или несколько именных аргументов стоят в одном из семи падежей — определи нужный падеж и его суффикс.

---

1. Мужчина идёт. (номинатив)
2. Я вижу мужчину. (датив)
3. Дом мужчины красивый. (генитив)
4. Я пишу ручкой. (инструменталис)
5. Я работаю учителем. (адвербиалис)
6. Мама, ты где? (вокатив)
7. Ребёнок, иди сюда! (вокатив)
8. На столе лежит книга. (датив + -ze)
9. Я был в доме. (датив + -shi)
10. У ребёнка есть собака (dzaghli). (датив подлеж. + akvs/hyavs)
11. Мать ребёнка красивая. (генитив)
12. Для отца я купил книгу. (генитив + -tvis)
13. Перед городом — древний (udzveli) храм (tadzari). (генитив + ts'in)
14. За домом — сад (baghi). (генитив + uk'an)
15. Я еду на машине. (инструменталис)
16. К вечеру придём. (адвербиалис)
17. Утром я был у друга (megobari). (инструменталис времени + датив + -tan)
18. Я учу язык по книге. (датив + инструменталис)
19. Ребёнок любит маму. (датив подлеж. + uqvars)
20. Я иду к городу. (генитив + -k'en)
21. Из дома я вышел (gamovedi) рано (adre). (-dan через основу)
22. Боже (gmerti), помоги мне! (вокатив + дат. косв.)
23. Книга мамы лежит на стуле (sk'ami). (генитив + датив + -ze)
24. Отец работает в селе. (адвербиалис не нужен — «в селе» = дат. + -shi)
25. Я пишу ручкой о городе. (инстр. + ген. + shesakheb)
26. Машина отца стоит (dgas) перед домом. (генитив × 2 + ts'in)
27. Ночью ребёнок спит (dzinavs) в доме. (инстр. + ном. + дат. + -shi)
28. У мужчины есть мать и ребёнок. (дат. подлеж. + ном. × 2)
29. Мама, для тебя я купил подарок (sachukari) в городе. (вок. + ген.+-tvis + дат.+-shi)
30. Древний город красив, и мы едем к нему на машине. (ном. + ген.+-k'en + инстр.)

---

<summary>Ключи</summary>

1. *კაცი მოდის.* (k'aci modis.) — **ном.**
2. *(მე) ვხედავ კაცს.* (vkhedav k'acs.) — **дат.** прям. дополн.
3. *კაცის სახლი ლამაზია.* (k'acis sakhli lamazia.) — **ген.** принадлеж.
4. *კალამით ვწერ.* (k'alamit vts'er.) — **инстр.** орудие.
5. *მასწავლებლად ვმუშაობ.* (masts'avleblad vmushaob.) — **адв.** «в качестве».
6. *დედავ, სად ხარ?* (dedav, sad khar?) — **вок.** после гласной → -v.
7. *ბავშვო, მოდი აქ!* (bavshvo, modi ak!) — **вок.** после согл. → -o.
8. *მაგიდაზე წიგნი დევს.* (magidaze ts'igni devs.) — **дат. + -ze**.
9. *სახლში ვიყავი.* (sakhlshi viqavi.) — **дат. + -shi**.
10. *ბავშვს ჰყავს ძაღლი.* (bavshvs hyavs dzaghli.) — подлеж. в **дативе** с *hyavs*.
11. *ბავშვის დედა ლამაზია.* (bavshvis deda lamazia.) — **ген.**
12. *მამისთვის ვიყიდე წიგნი.* (mamistvis viqide ts'igni.) — **ген. + -tvis**.
13. *ქალაქის წინ უძველი ტაძარია.* (kalakis ts'in udzveli tadzaria.) — **ген. + ts'in**.
14. *სახლის უკან ბაღია.* (sakhlis uk'an baghia.) — **ген. + uk'an**.
15. *მანქანით მივდივარ.* (mankanit mivdivar.) — **инстр.** средство.
16. *საღამოდ მოვალთ.* (saghamod movalt.) — **адв.** «к моменту».
17. *დილით ვიყავი მეგობართან.* (dilit viqavi megobartan.) — **инстр.** время + **дат. + -tan**.
18. *ენას ვსწავლობ წიგნით.* (enas vsts'avlob ts'ignit.) — **дат.** прям. дополн. + **инстр.** орудие.
19. *ბავშვს უყვარს დედა.* (bavshvs uqvars deda.) — подлеж. в **дативе** с *uqvars*.
20. *ქალაქისკენ მივდივარ.* (kalakisk'en mivdivar.) — **ген. + -k'en**.
21. *სახლიდან ადრე გამოვედი.* (sakhlidan adre gamovedi.) — основа + **-dan**.
22. *ღმერთო, მომიშველე!* (gmerto, momishvele!) — **вок.** + косв. дополн.
23. *დედის წიგნი სკამზე დევს.* (dedis ts'igni sk'amze devs.) — **ген.** + **дат. + -ze**.
24. *მამა სოფელში მუშაობს.* (mama soplshi mushaobs.) — **дат. + -shi** (с усечением: *sopeli → sopl-*).
25. *კალამით ვწერ ქალაქის შესახებ.* (k'alamit vts'er kalakis shesakheb.) — **инстр.** + **ген. + shesakheb**.
26. *მამის მანქანა სახლის წინ დგას.* (mamis mankana sakhlis ts'in dgas.) — **ген.** × 2 + **ts'in**.
27. *ღამით ბავშვი სახლში სძინავს.* (ghamit bavshvi sakhlshi sdzinavs.) — **инстр.** время + **ном.** + **дат. + -shi**.
28. *კაცს ჰყავს დედა და ბავშვი.* (k'acs hyavs deda da bavshvi.) — **дат.** подлеж. + **ном.** × 2 (два «владения» — *hyavs*, для одушевл.).
29. *დედავ, შენთვის ვიყიდე საჩუქარი ქალაქში.* (dedav, shentvis viqide sachukari kalakshi.) — **вок.** + ген.местоим.+*-tvis* + **дат. + -shi**.
30. *უძველი ქალაქი ლამაზია და ჩვენ მისკენ მივდივართ მანქანით.* (udzveli kalaki lamazia da chven misk'en mivdivart mankanit.) — **ном.** + ген.местоим.+*-k'en* + **инстр.**
