# Тест к уроку 39: Причастия и отглагольные прилагательные

**Как сказать по-грузински?**
Переведи фразы на грузинский язык (мхедрули). Используй все четыре типа причастий: страдательное прошлое (-il-i / -ul-i), действительное настоящее (m-...-eli), будущее (sa-...-i), отрицательное (u-...-eli).
