# Тест к уроку 12: Родительный падеж (-is) и притяжательные

**Как сказать по-грузински?**
Переведи фразы на грузинский язык (мхедрули). Помни главное правило: владелец **стоит перед** главным словом и принимает окончание **-is** (или **-s** для основ на гласный).
