# Тест к уроку 11: Дательный падеж. Глаголы «иметь» (*makvs* и *myavs*)

**Как сказать по-грузински?**
Переведи фразы на грузинский язык (мхедрули). Выбирай *makvs* для неодушевлённых (вещи, абстракции, части тела, растения) и *myavs* для одушевлённых (люди и животные).

---

1. У меня есть дом.
2. У тебя есть собака?
3. У него есть машина.
4. У нас есть кошка.
5. У вас есть время?
6. У них есть деньги.
7. У меня есть брат.
8. У тебя есть сестра?
9. У него есть ребёнок.
10. У нас есть лошадь.
11. У меня нет работы.
12. У тебя нет проблемы.
13. У него нет жены.
14. У нас нет мужа... (то есть: у нас его нет — *chven ar gvyavs kmari*)
15. У мамы есть подарок.
16. У ребёнка есть книга.
17. У Гиоргия есть друг.
18. У Нино есть кот и собака.
19. У бабушки есть внуки. (внук/ребёнок — *shvili*; здесь: *shvilebi* — дети/внуки)
20. У учителя много книг.
21. У меня есть голова и глаза.
22. У него болит сердце. (используй: *makvs gulis t'k'ivili* — «у меня есть сердечная боль»)
23. У ребёнка большие уши.
24. У собаки есть хозяин. (хозяин — *p'at'roni*; собака — подлежащее в дативе!)
25. У меня есть один вопрос.
26. У них нет детей.
27. У тебя есть телефон?
28. У нас есть курица и свинья.
29. У мамы есть дочь и сын.
30. У меня есть друг в Тбилиси.

---

<details>
<summary>Ключи</summary>

1. *მე მაქვს სახლი.* (me makvs sakhli.) — дом — вещь → *makvs*.
2. *შენ გყავს ძაღლი?* (shen gyavs dzaghli?) — собака — живая → *gyavs*.
3. *მას აქვს მანქანა.* (mas akvs mankana.) — машина — вещь; 3 л. ед. *akvs*.
4. *ჩვენ გვყავს კატა.* (chven gvyavs k'ata.) — кошка — живая → *gvyavs*.
5. *თქვენ გაქვთ დრო?* (tkven gakvt dro?) — время — абстракция → *gakvt*.
6. *მათ აქვთ ფული.* (mat akvt puli.) — деньги — вещь; 3 л. мн. *akvt*.
7. *მე მყავს ძმა.* (me myavs dzma.) — брат — живой → *myavs*.
8. *შენ გყავს და?* (shen gyavs da?) — сестра — живая → *gyavs*.
9. *მას ჰყავს შვილი.* (mas hyavs shvili.) — ребёнок — живой; 3 л. ед. *hyavs* (с *h-*!).
10. *ჩვენ გვყავს ცხენი.* (chven gvyavs tskheni.) — лошадь — живая → *gvyavs*.
11. *მე არ მაქვს სამსახური.* (me ar makvs samsakhuri.) — работа — абстракция.
12. *შენ არ გაქვს პრობლემა.* (shen ar gakvs p'roblema.) — проблема — абстракция.
13. *მას არ ჰყავს ცოლი.* (mas ar hyavs tsoli.) — жена — живая → *hyavs*.
14. *ჩვენ არ გვყავს ქმარი.* (chven ar gvyavs kmari.) — муж — живой → *gvyavs*.
15. *დედას აქვს საჩუქარი.* (dedas akvs sachukari.) — *deda → dedas* (датив подлежащего!). Подарок — вещь → *akvs*.
16. *ბავშვს აქვს წიგნი.* (bavshvs akvs ts'igni.) — *bavshvi → bavshvs* (датив подлежащего). Книга — вещь.
17. *გიორგის ჰყავს მეგობარი.* (giorgis hyavs megobari.) — *giorgi → giorgis*. Друг — живой → *hyavs*.
18. *ნინოს ჰყავს კატა და ძაღლი.* (ninos hyavs k'ata da dzaghli.) — *nino → ninos*. Кот и собака — живые → *hyavs*.
19. *ბებიას ჰყავს შვილები.* (bebias hyavs shvilebi.) — *bebia → bebias*. Дети — живые → *hyavs*. *shvili → shvilebi* (мн. ч., L3).
20. *მასწავლებელს აქვს ბევრი წიგნი.* (masts'avlebels akvs bevri ts'igni.) — *masts'avlebeli → masts'avlebels*. Книги — вещи → *akvs*. После *bevri* — ед. ч. (L7).
21. *მე მაქვს თავი და თვალები.* (me makvs tavi da tvalebi.) — части тела — *makvs*! *tvali → tvalebi* для подчёркивания пары.
22. *მას აქვს გულის ტკივილი.* (mas akvs gulis t'k'ivili.) — «сердечная боль» — абстракция → *akvs*. *gulis* — генитив (L12).
23. *ბავშვს აქვს დიდი ყურები.* (bavshvs akvs didi qurebi.) — части тела — *akvs*! *didi* перед существительным (L6).
24. *ძაღლს ჰყავს პატრონი.* (dzaghls hyavs p'at'roni.) — *dzaghli → dzaghls*: датив подлежащего на животном! Хозяин — человек → *hyavs*.
25. *მე მაქვს ერთი კითხვა.* (me makvs erti k'itkhva.) — вопрос — абстракция → *makvs*. После *erti* — ед. ч.
26. *მათ არ ჰყავთ შვილები.* (mat ar hyavt shvilebi.) — дети — живые; 3 л. мн. *hyavt* (с *h-*!).
27. *შენ გაქვს ტელეფონი?* (shen gakvs t'eleponi?) — телефон — вещь → *gakvs*.
28. *ჩვენ გვყავს ქათამი და ღორი.* (chven gvyavs katami da ghori.) — курица и свинья — животные → *gvyavs*.
29. *დედას ჰყავს ქალიშვილი და ვაჟი.* (dedas hyavs kalishvili da vazhi.) — *deda → dedas*. Дочь и сын — живые → *hyavs*.
30. *მე მყავს მეგობარი თბილისში.* (me myavs megobari tbilisshi.) — друг — живой → *myavs*. *tbilisshi* — послелог *-shi* (L13).

</details>
