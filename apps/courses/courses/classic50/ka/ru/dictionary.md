# Словарь курса грузинского языка
**50 уроков + 150 текстов для аудирования (A1 → B2)**

> Сортировка по грузинскому слову в алфавитном порядке мхедрули (ა, ბ, გ, დ, ე, ვ, ზ, თ, ი, კ, ლ, მ, ნ, ო, პ, ჟ, რ, ს, ტ, უ, ფ, ქ, ღ, ყ, შ, ჩ, ც, ძ, წ, ჭ, ხ, ჯ, ჰ — 33 буквы).
>
> Колонка «Класс» зарезервирована для пометок класса глагола (переходный, непереходный, медиальный, индиректный — см. L16) и для возможных уточнений вроде «послелог + ген.». Для существительных, прилагательных и наречий ранних уроков колонка пуста: в грузинском нет рода и нет артиклей.
