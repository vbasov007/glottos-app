# Test bloku 5 (Lekcje 41–50): Precyzja i styl

**Jak powiedzieć to po francusku?**
Przetłumacz każdą frazę na francuski. Ten test obejmuje gramatykę i słownictwo wszystkich dziesięciu lekcji piątego bloku: zaawansowane zaimki względne (lequel, auquel, duquel, ce qui / ce que), czasy literackie (rozpoznawanie passé simple), zaawansowane przeczenie (ne … que, ne … aucun, ne explétif), czasowniki z obowiązkowym przyimkiem, łączniki tekstowe, sufiksy i przedrostki, emfaza (c'est … qui/que), rejestry (soutenu / courant / familier), niuanse subjonctif i conditionnel, idiomatyka i wariacja frankofońska.
