# Program kursu języka francuskiego (50 lekcji)
**Glottos Matrix — strukturalny kurs nauki języka dla osób polskojęzycznych**

> Zasada: 5% to zrozumieć regułę, 95% to wytrenować aparat mowy. Każdy temat przerabiamy przez matrycę: je → tu → il/elle → nous → vous → ils/elles.

## Jak pracować z każdą lekcją

1. **Teoria** — obejrzyj wideo / przeczytaj regułę (5 minut)
2. **Tłumaczenie** — przetłumacz frazy z polskiego na francuski, sprawdź klucz
3. **Gamy językowe** — przegnaj nową regułę przez wszystkie osoby (je → tu → il → nous → vous → ils)
4. **Matryca** — przepracuj pytania-odpowiedzi w trzech scenariuszach: ty (sg) → wy → osoby trzecie
5. **Zasada lustra** — odpowiedź już siedzi w pytaniu, nie buduj od zera

## Triki na zapamiętywanie

- **Rodzaj**: 70% wynika z końcówki (-tion → la, -ment → le, -age → le, -té → la). Resztę ucz się przez kolor: niebieski = le, czerwony = la. Nie wyciągaj rodzaju z polskiego — *le soleil* (m), nie n
- **Liczba mnoga**: ucz się przez kontrast „Un... → Des...". Końcowe -s nieme — w mowie tylko rodzajnik mówi, że mnoga
- **Czasowniki nieregularne**: grupuj wzorcami (prendre / comprendre / apprendre). Ucz się od razu z czasownikiem posiłkowym: *Je suis allé*
- **Liaison**: po les, des, mes, mon, un, deux, trois itd. cicha końcowa spółgłoska budzi się przed samogłoską: *les_amis* [lę-zami]
- **Francuskie R**: gardłowe, jak miękkie chrapnięcie. Polskie drżące „r" brzmi obco — lepiej pomiń, niż wymawiaj po polsku
- **Nosówki an/en/in/on/un**: polskie ą i ę są tu Twoim przyjacielem. *Bon* ≈ „bą", *vin* ≈ „wę"
