# Tekst do Lekcji 41: Zaawansowane zaimki względne

**Temat: Opis formalny/abstrakcyjny**

---

## Tekst do słuchania

1. Voici le livre auquel je pense souvent.
2. C'est le sujet duquel nous avons parlé hier.
3. La personne avec laquelle je travaille est très compétente.
4. Les amis chez lesquels nous habitons sont français.
5. Je ne sais pas ce qui se passe ici.
6. Dis-moi ce que tu veux vraiment.
7. Voilà ce dont j'ai besoin pour finir le projet.
8. C'est exactement ce à quoi je pensais.
9. Le problème auquel nous faisons face est complexe.
10. La raison pour laquelle je suis ici est simple.
11. Mon frère, qui habite à Paris, est avocat.
12. Cette idée, que tout le monde apprécie, est excellente.
13. Le pays dont je rêve est le Japon.
14. Voici la solution à laquelle je n'avais pas pensé.
15. Ce sont les outils sans lesquels rien n'est possible.
16. Tout ce qui brille n'est pas or.
17. Ce que tu dis est très important.
18. Ce dont elle parle est compliqué.
19. Ce à quoi il s'intéresse, c'est la philosophie.
20. La maison dans laquelle nous vivons est ancienne.
21. Le but pour lequel il travaille est noble.
22. Les conditions sous lesquelles il accepte sont claires.
23. Voici l'élément grâce auquel tout fonctionne.
24. C'est une question à laquelle je ne peux pas répondre.
25. Le sens dans lequel j'emploie ce mot est figuré.
26. Ce qui m'intéresse, c'est la vérité.
27. Ce que nous cherchons, c'est la justice.
28. Ce dont il a peur, c'est l'échec.
29. Tout ce à quoi je tiens, c'est ma famille.
30. Le moment auquel cela est arrivé reste flou.
