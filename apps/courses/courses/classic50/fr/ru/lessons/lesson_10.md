# Урок 10: Глаголы на -RE. Faire, prendre, venir. Venir de + infinitif
**Словарный запас: бытовые сцены — faire les courses, prendre le métro, venir de**

---

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Прогони через гамму** — каждый глагол через je → tu → il → nous → vous → ils
3. **Матрица** — отработай через вопрос-ответ, пока не вылетает само

> Этот урок закрывает настоящее время. После него ты умеешь спрягать любой глагол всех трёх групп плюс главные неправильные. Это фундамент — без него не двинешься.

---

## Часть 1: Третья группа — глаголы на -RE

После -ER (1-я группа, урок 7) и -IR (2-я группа, урок 9) — третья группа. Самая маленькая и самая неоднородная, но «правильное ядро» у неё есть: глаголы типа **vendre**, **attendre**, **répondre**.

**Главное правило:** убираешь `-re`, добавляешь окончания. В единственном числе третье лицо — **ничего не добавляешь** (никакого -t, просто основа!).

### Vendre (продавать) — образец

| Лицо | Форма | Окончание | Произношение |
|------|-------|-----------|--------------|
| je | <say>vends</say> | -s | «ван» (s, d немые) |
| tu | <say>vends</say> | -s | «ван» |
| il / elle | <say>vend</say> | — (ничего!) | «ван» |
| nous | <say>vendons</say> | -ons | «ван-дон» |
| vous | <say>vendez</say> | -ez | «ван-де» |
| ils / elles | <say>vendent</say> | -ent | «ванд» (ent немое, **d читается**) |

> **Ловушка!** В *<say>il vend</say>* нет никакого -t на конце. Это уникальная черта группы -RE. У -ER и -IR в третьем лице — `-e` (*<say>il parle</say>*) или `-it` (*<say>il finit</say>*). У -RE — голая основа: *<say>il vend</say>*, *<say>il attend</say>*, *<say>il répond</say>*.

> **Слуховая ловушка:** *<say>je vends</say>* и *<say>ils vendent</say>* звучат по-разному! В единственном числе `d` немое («ван»), во множественном `d` читается («ванд»). Это разделитель ед./мн.

### Глаголы того же типа

| Глагол | Перевод | Заметка |
|--------|---------|---------|
| **<say>vendre</say>** | продавать | |
| **<say>attendre</say>** | ждать | прямое дополнение, без предлога: *<say>j'attends le bus</say>* |
| **<say>répondre</say>** | отвечать | с предлогом à: *<say>je réponds à la question</say>* |
| **<say>entendre</say>** | слышать | не путать с *<say>écouter</say>* (слушать) |
| **<say>descendre</say>** | спускаться, выходить (из транспорта) | прошедшее с *être*! |
| **<say>rendre</say>** | возвращать, отдавать | *<say>rendre un livre</say>* |
| **<say>perdre</say>** | терять | |

Все они спрягаются как *vendre*. Прогони *attendre*: *<say>j'attends</say>, <say>tu attends</say>, <say>il attend</say>, <say>nous attendons</say>, <say>vous attendez</say>, <say>ils attendent</say>*.

## Часть 2: Faire — «делать», но не только

**Faire** — самый частотный глагол после *être* и *avoir*. Полностью неправильный, учи целиком.

| Лицо | Форма | Произношение |
|------|-------|--------------|
| je | <say>fais</say> | «фэ» |
| tu | <say>fais</say> | «фэ» |
| il / elle | <say>fait</say> | «фэ» |
| nous | <say>faisons</say> | «фё-зон» (внимание: ai = «ё»!) |
| vous | <say>faites</say> | «фэт» |
| ils / elles | <say>font</say> | «фон» |

> **Три ловушки faire:**
> 1. *<say>nous faisons</say>* читается «фё-зон», а не «фэ-зон». Единственное место, где `ai` = «ё».
> 2. *<say>vous faites</say>* — окончание **-tes**, а не -ez. Только три глагола во всём французском так делают: *<say>vous êtes</say>, <say>vous faites</say>, <say>vous dites</say>*.
> 3. *<say>ils font</say>* — короткое, как *<say>ils sont</say>* (être), *<say>ils ont</say>* (avoir), *<say>ils vont</say>* (aller). Запомни четвёрку: **<say>font</say>, <say>sont</say>, <say>ont</say>, <say>vont</say>**.

### Что значит faire

Базово «делать», но в идиомах — куда больше. Faire — это «заниматься», «совершать», «играть в».

| Выражение | Перевод |
|-----------|---------|
| <say>faire les courses</say> | делать покупки (продукты) |
| <say>faire le ménage</say> | делать уборку |
| <say>faire la cuisine</say> | готовить |
| <say>faire la vaisselle</say> | мыть посуду |
| <say>faire ses devoirs</say> | делать домашнее задание |
| <say>faire du sport</say> | заниматься спортом |
| <say>faire du vélo</say> | кататься на велосипеде |
| <say>faire la queue</say> | стоять в очереди |
| <say>faire un voyage</say> | путешествовать |
| <say>faire attention</say> | быть внимательным |
| <say>Il fait beau</say> / <say>Il fait froid</say> / <say>Il fait chaud</say> | Погода хорошая / холодно / жарко |

## Часть 3: Prendre — «брать», «есть», «ехать на»

**Prendre** — второй чемпион по частотности. Запомни его, и ещё бесплатно получишь *comprendre* (понимать) и *apprendre* (учить) — они спрягаются точно так же.

| Лицо | Форма | Произношение |
|------|-------|--------------|
| je | <say>prends</say> | «пран» |
| tu | <say>prends</say> | «пран» |
| il / elle | <say>prend</say> | «пран» |
| nous | <say>prenons</say> | «прё-нон» (одна n!) |
| vous | <say>prenez</say> | «прё-не» (одна n!) |
| ils / elles | <say>prennent</say> | «прэн» (две n, носового нет!) |

> **Главная ловушка:** во множественном числе **меняется основа**. В *nous/vous* — одна `n` и звук «прё». В *<say>ils prennent</say>* — две `nn`, и носовой `en` исчезает, остаётся открытое «прэн».

### Что значит prendre

| Выражение | Перевод |
|-----------|---------|
| <say>prendre le métro</say> / <say>prendre le bus</say> / <say>prendre le train</say> | ехать на метро / автобусе / поезде |
| <say>prendre un taxi</say> | взять такси |
| <say>prendre le petit-déjeuner</say> | завтракать |
| <say>prendre un café</say> | выпить кофе |
| <say>prendre une douche</say> | принять душ |
| <say>prendre une photo</say> | сделать фото |
| <say>prendre rendez-vous</say> | записаться (на приём) |

> Французы не «едут на метро» — они **берут** метро: *<say>Je prends le métro</say>*. Не «делают фото» — **берут** фото. Логика: «беру = выбираю и пользуюсь».

### Бонус: comprendre, apprendre

| | comprendre | apprendre |
|---|------------|-----------|
| je | <say>comprends</say> | <say>apprends</say> |
| tu | <say>comprends</say> | <say>apprends</say> |
| il | <say>comprend</say> | <say>apprend</say> |
| nous | <say>comprenons</say> | <say>apprenons</say> |
| vous | <say>comprenez</say> | <say>apprenez</say> |
| ils | <say>comprennent</say> | <say>apprennent</say> |

## Часть 4: Venir — «приходить», «приезжать»

| Лицо | Форма | Произношение |
|------|-------|--------------|
| je | <say>viens</say> | «вьен» (носовое) |
| tu | <say>viens</say> | «вьен» |
| il / elle | <say>vient</say> | «вьен» |
| nous | <say>venons</say> | «вё-нон» |
| vous | <say>venez</say> | «вё-не» |
| ils / elles | <say>viennent</say> | «вьен» (двойная n, носовой исчез) |

Та же логика, что у *prendre*: в *nous/vous* — основа «vén-», в остальных лицах — «vien-». В *ils* — двойная `nn`, носовой пропадает.

Так же спрягаются: **<say>tenir</say>** (держать), **<say>devenir</say>** (становиться), **<say>revenir</say>** (возвращаться), **<say>se souvenir</say>** (помнить).

### Venir de + страна / город

Откуда ты? Используй *venir de*. Согласуй предлог как с *aller* (урок 9), только зеркально:

| Страна | venir de … | aller … |
|--------|-----------|---------|
| <say>la France</say> (ж.р.) | <say>de France</say> | <say>en France</say> |
| <say>le Canada</say> (м.р.) | <say>du Canada</say> | <say>au Canada</say> |
| <say>les États-Unis</say> (мн.ч.) | <say>des États-Unis</say> | <say>aux États-Unis</say> |
| <say>Paris</say> (город) | <say>de Paris</say> | <say>à Paris</say> |

- *<say>Je viens de France.</say>* — Я из Франции.
- *<say>Tu viens d'où ?</say>* — Ты откуда?
- *<say>Elle vient du Japon.</say>* — Она из Японии.

## Часть 5: Venir de + infinitif — «только что …»

Это не движение, это **время**. Конструкция «*venir de* + инфинитив» = «только что сделать что-то». Зеркало к *aller + infinitif* (futur proche, урок 9).

| Конструкция | Значение | Пример |
|-------------|----------|--------|
| **aller + infinitif** | сейчас сделаю (futur proche) | *<say>Je vais manger.</say>* — Я сейчас поем. |
| **venir de + infinitif** | только что сделал (passé récent) | *<say>Je viens de manger.</say>* — Я только что поел. |

Это «недавнее прошедшее» — passé récent. Удобно: настоящее время глагола *venir*, а смысл — прошедший.

| Пример | Перевод |
|--------|---------|
| <say>Je viens d'arriver.</say> | Я только что приехал. |
| <say>Tu viens de finir ?</say> | Ты только что закончил? |
| <say>Il vient de partir.</say> | Он только что ушёл. |
| <say>Nous venons de manger.</say> | Мы только что поели. |
| <say>Vous venez d'appeler ?</say> | Вы только что звонили? |
| <say>Ils viennent de rentrer.</say> | Они только что вернулись. |

> **Не путай!** *<say>Je viens de Paris</say>* = «Я **из** Парижа» (откуда). *<say>Je viens de manger</say>* = «Я **только что** поел» (когда). Если после *de* стоит **глагол в инфинитиве** — это passé récent.

## Языковая гамма

Прогоняй каждый глагол вслух полностью. Сначала медленно, потом на скорость.

### Гамма 1: attendre le bus

> <say>J'attends le bus.</say> — <say>Tu attends le bus.</say> — <say>Il attend le bus.</say> — <say>Nous attendons le bus.</say> — <say>Vous attendez le bus.</say> — <say>Ils attendent le bus.</say>

### Гамма 2: faire les courses

> <say>Je fais les courses.</say> — <say>Tu fais les courses.</say> — <say>Elle fait les courses.</say> — <say>Nous faisons les courses.</say> — <say>Vous faites les courses.</say> — <say>Ils font les courses.</say>

### Гамма 3: prendre le métro

> <say>Je prends le métro.</say> — <say>Tu prends le métro.</say> — <say>Il prend le métro.</say> — <say>Nous prenons le métro.</say> — <say>Vous prenez le métro.</say> — <say>Elles prennent le métro.</say>

### Гамма 4: venir de France

> <say>Je viens de France.</say> — <say>Tu viens de France.</say> — <say>Il vient de France.</say> — <say>Nous venons de France.</say> — <say>Vous venez de France.</say> — <say>Elles viennent de France.</say>

### Гамма 5: venir de + infinitif (passé récent)

> <say>Je viens d'arriver.</say> — <say>Tu viens d'arriver.</say> — <say>Il vient d'arriver.</say> — <say>Nous venons d'arriver.</say> — <say>Vous venez d'arriver.</say> — <say>Ils viennent d'arriver.</say>

### Гамма 6: отрицание (повтор урока 8)

> <say>Je ne fais pas les courses.</say> — <say>Tu ne fais pas les courses.</say> — <say>Il ne fait pas les courses.</say> — <say>Nous ne faisons pas les courses.</say> — <say>Vous ne faites pas les courses.</say> — <say>Ils ne font pas les courses.</say>

## Языковая матрица

Отвечай зеркально: ответ уже в вопросе. Сначала про себя, потом про «нас», потом про третье лицо.

### Матрица 1: faire / prendre / venir

| <say>Tu attends le bus ?</say> (ответь: да) | <say>Oui, j'attends le bus.</say> | <say>Oui, nous attendons le bus.</say> | <say>Oui, ils attendent le bus.</say> |
| <say>Tu comprends ?</say> (ответь: да) | <say>Oui, je comprends.</say> | <say>Oui, nous comprenons.</say> | <say>Oui, ils comprennent.</say> |

## Упражнения

### Упражнение 1. Проспрягай глаголы на -RE

Поставь глагол в правильную форму:

1. J'\_\_\_\_ (attendre) le train.
2. Tu \_\_\_\_ (vendre) ta voiture ?
3. Elle \_\_\_\_ (répondre) à la question.
4. Nous \_\_\_\_ (perdre) du temps.
5. Vous \_\_\_\_ (entendre) la musique ?
6. Ils \_\_\_\_ (descendre) à la prochaine station.

<details>
<summary>Ключ</summary>

1. *J'attends le train.*
2. *Tu vends ta voiture ?*
3. *Elle répond à la question.*
4. *Nous perdons du temps.*
5. *Vous entendez la musique ?*
6. *Ils descendent à la prochaine station.*

</details>

### Упражнение 2. Faire, prendre, venir — выбери и проспрягай

1. Le matin, je \_\_\_\_ une douche et je \_\_\_\_ le petit-déjeuner.
2. Nous \_\_\_\_ les courses au supermarché.
3. Tu \_\_\_\_ d'où ? — Je \_\_\_\_ d'Italie.
4. Ils \_\_\_\_ du sport tous les jours.
5. Vous \_\_\_\_ le métro ou le bus ?
6. Elle \_\_\_\_ chez nous ce soir.

<details>
<summary>Ключ</summary>

1. *Le matin, je **prends** une douche et je **prends** le petit-déjeuner.*
2. *Nous **faisons** les courses au supermarché.*
3. *Tu **viens** d'où ? — Je **viens** d'Italie.*
4. *Ils **font** du sport tous les jours.*
5. *Vous **prenez** le métro ou le bus ?*
6. *Elle **vient** chez nous ce soir.*

</details>

### Упражнение 3. Passé récent — «только что …»

Переделай в конструкцию *venir de + infinitif*:

1. (manger / je) — Я только что поел.
2. (arriver / nous) — Мы только что приехали.
3. (finir / elle) — Она только что закончила.
4. (partir / ils) — Они только что ушли.
5. (téléphoner / tu) — Ты только что звонил.
6. (rentrer / vous) — Вы только что вернулись.

<details>
<summary>Ключ</summary>

1. *Je viens de manger.*
2. *Nous venons d'arriver.*
3. *Elle vient de finir.*
4. *Ils viennent de partir.*
5. *Tu viens de téléphoner.*
6. *Vous venez de rentrer.*

</details>

### Упражнение 4. Контраст: futur proche vs passé récent

Переведи, обращая внимание на «сейчас сделаю» vs «только что сделал». (Повтор futur proche — урок 9.)

1. Я сейчас поеду на метро.
2. Я только что приехал на метро.
3. Они сейчас будут ужинать.
4. Они только что поужинали.
5. Ты сейчас будешь делать уборку?
6. Ты только что сделал уборку?

<details>
<summary>Ключ</summary>

1. *Je vais prendre le métro.*
2. *Je viens de prendre le métro.*
3. *Ils vont dîner.* (или: *Ils vont prendre le dîner.*)
4. *Ils viennent de dîner.*
5. *Tu vas faire le ménage ?*
6. *Tu viens de faire le ménage ?*

</details>

### Упражнение 5. Отрицание и вопросы (повтор урока 8)

Поставь фразы в отрицание, потом задай вопрос через *est-ce que*:

1. Je comprends la question.
2. Nous prenons le bus.
3. Elle attend Marc.
4. Ils font la vaisselle.
5. Vous venez de Paris.

<details>
<summary>Ключ</summary>

1. *Je ne comprends pas la question.* / *Est-ce que tu comprends la question ?*
2. *Nous ne prenons pas le bus.* / *Est-ce que vous prenez le bus ?*
3. *Elle n'attend pas Marc.* / *Est-ce qu'elle attend Marc ?*
4. *Ils ne font pas la vaisselle.* / *Est-ce qu'ils font la vaisselle ?*
5. *Vous ne venez pas de Paris.* / *Est-ce que vous venez de Paris ?*

</details>

### Упражнение 6. Переведи с русского

1. Что ты делаешь? — Я мою посуду.
2. Они едут на поезде в Лион. *(повтор: предлоги направления, урок 9)*
3. Я только что сделал покупки.
4. Мы стоим в очереди уже двадцать минут. *(подсказка: «уже» = déjà)*
5. Извините, я не понимаю. Вы говорите слишком быстро. *(повтор: vite — урок 7)*
6. Мой брат изучает французский. Он очень умный. *(повтор: прилагательное, урок 6 — intelligent)*
7. Откуда вы приехали? — Мы из Канады.
8. Что вы хотите взять? — Маленький кофе, пожалуйста. *(повтор: BAGS-прилагательное petit, урок 6)*

<details>
<summary>Ключ</summary>

1. *Qu'est-ce que tu fais ? — Je fais la vaisselle.*
2. *Ils prennent le train pour Lyon.* (или: *Ils vont à Lyon en train.*)
3. *Je viens de faire les courses.*
4. *Nous faisons la queue depuis vingt minutes déjà.* (или просто: *Nous attendons déjà depuis vingt minutes.*)
5. *Pardon, je ne comprends pas. Vous parlez trop vite.* (или: *Excusez-moi…*)
6. *Mon frère apprend le français. Il est très intelligent.*
7. *D'où venez-vous ? — Nous venons du Canada.* (или: *Vous venez d'où ?*)
8. *Qu'est-ce que vous voulez prendre ? — Un petit café, s'il vous plaît.*

</details>

## Памятка урока

```
ТРЕТЬЯ ГРУППА -RE (vendre, attendre, répondre):
  je    vend-s        nous  vend-ons
  tu    vend-s        vous  vend-ez
  il    vend  (!)     ils   vend-ent
  → il vend — БЕЗ окончания. Голая основа.
  → je vends «ван», ils vendent «ванд» (d читается!).

FAIRE (делать):
  je fais       nous faisons  (читается «фё-зон»!)
  tu fais       vous faites   (-tes, как êtes, dites)
  il fait       ils font
  ИДИОМЫ: faire les courses / le ménage / la cuisine / la vaisselle
          faire ses devoirs / du sport / du vélo / la queue
          faire un voyage / attention
          il fait beau / froid / chaud

PRENDRE (брать, есть, ехать на):
  je prends     nous prenons   (одна n, «прё-нон»)
  tu prends     vous prenez
  il prend      ils prennent   (двойная nn, носовой исчезает!)
  → так же: comprendre, apprendre
  ИДИОМЫ: prendre le métro / le bus / un taxi
          prendre le petit-déjeuner / un café
          prendre une douche / une photo / rendez-vous

VENIR (приходить):
  je viens      nous venons    (одна n, «вё-нон»)
  tu viens      vous venez
  il vient      ils viennent   (двойная nn, носовой исчезает!)
  → так же: revenir, devenir, tenir, se souvenir

VENIR DE + СТРАНА (откуда):
  de France / d'Italie / du Canada / des États-Unis / de Paris
  (зеркало к aller en/au/aux — урок 9)

VENIR DE + ИНФИНИТИВ (только что сделал):
  Je viens de manger. — Я только что поел.
  ↔ Je vais manger. — Я сейчас поем. (futur proche, урок 9)

КОРОТКАЯ ЧЕТВЁРКА в 3-м лице мн. ч.:
  ils SONT (être) / ils ONT (avoir) / ils FONT (faire) / ils VONT (aller)

НАСТОЯЩЕЕ ВРЕМЯ — ИТОГ ТРЁХ ГРУПП:
  -ER (parler):  -e, -es, -e,  -ons, -ez, -ent
  -IR (finir):   -is, -is, -it, -issons, -issez, -issent
  -RE (vendre):  -s, -s, —,    -ons, -ez, -ent
```

> **Следующий шаг:** Блок 1 закрыт. Тест «Apprenti / Apprentie» — проверь, что хребет языка стоит ровно. Затем урок 11: модальные неправильные — pouvoir, vouloir, devoir, savoir. После них ты сможешь сказать «я хочу», «я могу», «я должен», «я умею» — и начнётся настоящая речь.
