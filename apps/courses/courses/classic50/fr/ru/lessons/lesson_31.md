# Урок 31: Présent du subjonctif: формы

**Словарный запас: 30 высокочастотных глаголов в сослагательном — все группы**

---

## Как работать с этим уроком

1. **Прочитай** правило про основу «ils» — это ключ ко всему (5 минут)
2. **Прогоняй гаммы** — спрягай вслух, пока окончания не станут автоматическими
3. **Не учи примеры использования** — это Урок 32. Здесь только формы.

> Subjonctif = 90% механики + 10% магии. Сначала натренируй механику.
> Француз не «думает» сослагательное — он его слышит. Твоя задача — натренировать рот.

---

## Часть 1: Что такое subjonctif (в двух словах)

Subjonctif — это **наклонение субъективности**. Indicatif говорит о фактах («он приходит»), subjonctif — о том, что **окрашено волей, эмоцией, сомнением, необходимостью**: «я хочу, чтобы он пришёл», «надо, чтобы он пришёл», «я боюсь, что он придёт».

**В этом уроке мы НЕ учим, когда его использовать** — это уроки 32–35. Здесь только **формы**. Запоминай по триггеру:

> **«Il faut que…» → дальше всегда subjonctif.**

Это твой тренировочный пинок. *Il faut que je parle.* *Il faut que tu finisses.* *Il faut qu'il vienne.*

---

## Часть 2: Главная формула — основа «ils» + окончания

Subjonctif в 4 из 6 форм строится **из формы `ils` настоящего времени**.

> **Алгоритм:** взять `ils` в présent → отбросить **-ent** → добавить окончания subjonctif.

**Окончания (учи как мантру):**

| Лицо | Окончание |
|------|-----------|
| que je | **-e** |
| que tu | **-es** |
| qu'il / qu'elle | **-e** |
| que nous | **-ions** |
| que vous | **-iez** |
| qu'ils / qu'elles | **-ent** |

**Пример 1 — `parler` (группа -ER):**

`ils parlent` → основа **parl-**

| | |
|---|---|
| que je **parle** | que nous **parlions** |
| que tu **parles** | que vous **parliez** |
| qu'il **parle** | qu'ils **parlent** |

> Заметь: у глаголов на **-ER** формы je/tu/il/ils в subjonctif **совпадают с indicatif**. Это не баг — это фича. Француз слышит разницу только по контексту триггера («que…»).

**Пример 2 — `finir` (группа -IR с -iss-):**

`ils finissent` → основа **finiss-**

| | |
|---|---|
| que je **finisse** | que nous **finissions** |
| que tu **finisses** | que vous **finissiez** |
| qu'il **finisse** | qu'ils **finissent** |

**Пример 3 — `écrire`:**

`ils écrivent` → основа **écriv-**

| | |
|---|---|
| que j'**écrive** | que nous **écrivions** |
| que tu **écrives** | que vous **écriviez** |
| qu'il **écrive** | qu'ils **écrivent** |

## Часть 3: Подвох — «сапог», nous/vous заимствуют imparfait

> **🔁 Возврат к Уроку 11.** Помнишь «глаголы-сапоги» — `prendre`, `venir`, `boire`, `recevoir`, `vouloir`, `devoir`? У них **две основы**: одна для `je/tu/il/ils`, другая для `nous/vous`. Тот же сапог работает в subjonctif!

**Принцип:**
- `que je / tu / il / ils` → основа **«ils»** настоящего времени
- `que nous / vous` → основа **«nous»** настоящего времени (= основа imparfait!)

**Пример — `prendre`:**

- `ils prennent` → **prenn-**
- `nous prenons` → **pren-**

| | |
|---|---|
| que je **prenne** | que nous **prenions** |
| que tu **prennes** | que vous **preniez** |
| qu'il **prenne** | qu'ils **prennent** |

**Пример — `boire`:**

- `ils boivent` → **boiv-**
- `nous buvons` → **buv-**

| | |
|---|---|
| que je **boive** | que nous **buvions** |
| que tu **boives** | que vous **buviez** |
| qu'il **boive** | qu'ils **boivent** |

**Пример — `venir`:**

| que je **vienne** | que nous **venions** |
|---|---|
| que tu **viennes** | que vous **veniez** |
| qu'il **vienne** | qu'ils **viennent** |

**Глаголы-сапоги, которые надо знать:** prendre (+ comprendre, apprendre), venir (+ tenir, devenir), boire, recevoir, devoir, croire, voir, mourir, vouloir, aller.

> **Ловушка для русского уха!** *Que nous prenions* звучит «прёнь-он», а не «прё-нь-онь». Это просто `pren-` + `-ions`, как в imparfait `nous prenions` (мы брали). Они **омофоны**. Различает только триггер `que`.

## Часть 4: Семь главных неправильных — учи наизусть

Эти не подчиняются «правилу ils». Их **просто заучивают**. Это самые частые глаголы — их обязательно знать в лицо.

### 1. être

| | |
|---|---|
| que je **sois** | que nous **soyons** |
| que tu **sois** | que vous **soyez** |
| qu'il **soit** | qu'ils **soient** |

### 2. avoir

| | |
|---|---|
| que j'**aie** | que nous **ayons** |
| que tu **aies** | que vous **ayez** |
| qu'il **ait** | qu'ils **aient** |

> *Aie / aies / ait / aient* — все читаются «э». Различает только письмо.

### 3. aller (сапог!)

| | |
|---|---|
| que j'**aille** | que nous **allions** |
| que tu **ailles** | que vous **alliez** |
| qu'il **aille** | qu'ils **aillent** |

### 4. faire

| | |
|---|---|
| que je **fasse** | que nous **fassions** |
| que tu **fasses** | que vous **fassiez** |
| qu'il **fasse** | qu'ils **fassent** |

### 5. pouvoir

| | |
|---|---|
| que je **puisse** | que nous **puissions** |
| que tu **puisses** | que vous **puissiez** |
| qu'il **puisse** | qu'ils **puissent** |

### 6. savoir

| | |
|---|---|
| que je **sache** | que nous **sachions** |
| que tu **saches** | que vous **sachiez** |
| qu'il **sache** | qu'ils **sachent** |

### 7. vouloir (сапог!)

| | |
|---|---|
| que je **veuille** | que nous **voulions** |
| que tu **veuilles** | que vous **vouliez** |
| qu'il **veuille** | qu'ils **veuillent** |

**Бонус — безличные:**
- `falloir` → **qu'il faille** (только 3-е лицо ед.)
- `pleuvoir` → **qu'il pleuve**
- `valoir` → que je **vaille**, que nous **valions**

## Часть 5: Триггеры (чтобы было что тренировать)

Полный разбор — уроки 32–34. Здесь — минимальный набор для упражнений:

| Триггер | Перевод |
|---------|---------|
| **il faut que** | надо, чтобы |
| **je veux que** | я хочу, чтобы |
| **je suis content(e) que** | я рад(а), что |
| **bien que** | хотя |
| **pour que** | чтобы (цель) |
| **avant que** | прежде чем |

После любого из них — subjonctif. Без исключений.

> **🔁 Возврат к Уроку 30 (relatives) и Уроку 29 (conditionnel passé).** Subjonctif дружит со сложными придаточными: *Je cherche quelqu'un **qui sache** réparer ça* (l'antécédent неопределённый → qui + subjonctif). *J'aurais aimé **qu'il vienne*** (сожаление через conditionnel passé + que + subjonctif). Эти конструкции — на горизонте, держи в голове.

## Языковая гамма

Прогон через все 6 лиц с триггером `il faut que…`.

### Гамма 1 — `parler` (правильный -ER, основа одна)

> Il faut que je **parle**. Il faut que tu **parles**. Il faut qu'il **parle**.
> Il faut que nous **parlions**. Il faut que vous **parliez**. Il faut qu'ils **parlent**.

### Гамма 2 — `finir` (правильный -IR с -iss-, основа одна)

> Il faut que je **finisse**. Il faut que tu **finisses**. Il faut qu'elle **finisse**.
> Il faut que nous **finissions**. Il faut que vous **finissiez**. Il faut qu'elles **finissent**.

### Гамма 3 — `prendre` (сапог!)

> Il faut que je **prenne**. Il faut que tu **prennes**. Il faut qu'il **prenne**.
> Il faut que nous **prenions**. Il faut que vous **preniez**. Il faut qu'ils **prennent**.

### Гамма 4 — `être` (полностью неправильный)

> Il faut que je **sois**. Il faut que tu **sois**. Il faut qu'il **soit**.
> Il faut que nous **soyons**. Il faut que vous **soyez**. Il faut qu'ils **soient**.

### Гамма 5 — `aller` (неправильный + сапог)

> Il faut que j'**aille**. Il faut que tu **ailles**. Il faut qu'elle **aille**.
> Il faut que nous **allions**. Il faut que vous **alliez**. Il faut qu'elles **aillent**.

**Прогон:** каждую гамму вслух — 3 раза подряд, без пауз. Потом 5 раз быстро.

## Языковая матрица

Принцип зеркала: ответ перестраивает лицо из вопроса. Триггер слева → ответ в subjonctif.

| Вопрос (indicatif) | Реакция (subjonctif) |
|--------------------|----------------------|
| Tu pars maintenant ? (ответь: да) | Oui, **il faut que je parte** maintenant. |
| Vous finissez le travail ? (ответь: да) | Oui, **il faut que nous finissions** le travail. |
| Il vient à la fête ? (ответь: да) | Oui, **je veux qu'il vienne** à la fête. |
| Ils savent la vérité ? (ответь: нет, не хочу) | Non, **je ne veux pas qu'ils sachent** la vérité. |
| Tu fais les courses ? (ответь: да) | Oui, **il faut que je fasse** les courses. |
| Elle est à l'heure ? (ответь: надеюсь) | J'espère, **il faut qu'elle soit** à l'heure. |
| Nous prenons le métro ? (ответь: да) | Oui, **il faut que nous prenions** le métro. |
| Vous pouvez m'aider ? (ответь: конечно) | Bien sûr, **je suis content que vous puissiez** m'aider. |
| Tu as le temps ? (ответь: да, но нужно больше) | Oui, mais **il faut que j'aie** plus de temps. |
| Il va chez le médecin ? (ответь: да) | Oui, **il faut qu'il aille** chez le médecin. |

Прогон: закрой правую колонку, отвечай вслух. Потом наоборот.

Перед тобой инфинитив. Скажи: у глагола **одна** основа в subjonctif или **две** (сапог)? Если две — назови обе.

4. recevoir
5. mettre

<summary>Ключ</summary>

## Памятка урока

```
ГЛАВНАЯ ФОРМУЛА:
  ils + présent → отбрось -ent → добавь окончания:
    je   -e        nous  -ions
    tu   -es       vous  -iez
    il   -e        ils   -ent

ПРИМЕР:
  parler  → ils parlent → parl-
    que je parle, que tu parles, qu'il parle,
    que nous parlions, que vous parliez, qu'ils parlent

САПОГ (две основы, как в présent indicatif):
  je/tu/il/ils → основа ILS  (prenn-, vienn-, boiv-)
  nous/vous    → основа NOUS (pren-, ven-, buv-)
  Это те же глаголы-сапоги из Урока 11: prendre, venir, boire,
  recevoir, devoir, croire, voir, vouloir, aller.

СЕМЬ НЕПРАВИЛЬНЫХ (наизусть):
  être    → sois, sois, soit, soyons, soyez, soient
  avoir   → aie, aies, ait, ayons, ayez, aient
  aller*  → aille, ailles, aille, allions, alliez, aillent
  faire   → fasse … fassions … fassent
  pouvoir → puisse … puissions … puissent
  savoir  → sache … sachions … sachent
  vouloir*→ veuille, veuilles, veuille, voulions, vouliez, veuillent
  (* = сапог)

БЕЗЛИЧНЫЕ:
  falloir → qu'il faille
  pleuvoir → qu'il pleuve

ТРИГГЕРЫ ДЛЯ ТРЕНИРОВКИ:
  il faut que …       (надо, чтобы)
  je veux que …       (я хочу, чтобы)
  je suis content que (я рад, что)
  bien que …          (хотя)
  pour que …          (чтобы — цель)
  avant que …         (прежде чем)

ЛОВУШКИ:
  • Глаголы на -ER: формы je/tu/il/ils в subjonctif = indicatif. Не паникуй.
  • que nous prenions = imparfait nous prenions (омофоны). Различает только que.
  • aie / aies / ait / aient — все «э». Различает только письмо.
  • reçoive — не забудь cédille перед o.
```
