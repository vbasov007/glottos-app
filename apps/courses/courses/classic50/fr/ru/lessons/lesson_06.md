# Урок 6: Прилагательные — согласование и место
**Словарный запас: цвета, размер, характер, внешность**

---

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — медленно, осознанно, прогоняя через все четыре формы
3. **Ускоряйся** — повторяй, пока согласование не будет вылетать на автомате

> Знать правило = 5%. Натренировать рот = 95%.
> Цель урока: чтобы при виде «маленький стол» у тебя уже горело окончание «-e» или его отсутствие.

---

## Часть 1: Главный принцип

Французское прилагательное **согласуется** с существительным в роде и числе. У него четыре формы:

| | Единственное | Множественное |
|---|---|---|
| **Мужской** | <say>grand</say> | <say>grands</say> |
| **Женский** | <say>grande</say> | <say>grandes</say> |

**Базовое правило:**

> Женский = + **e**. Множественное = + **s**. Женский множественный = + **es**.

Произношение — главная ловушка:
- *<say>grand</say>* → «гран» (d немое)
- *<say>grande</say>* → «гранд» (e в конце оживляет d!)
- *<say>grands</say>* → «гран» (s немое, звучит как мужской ед.)
- *<say>grandes</say>* → «гранд» (звучит как женский ед.)

> **На слух различаются только род (по согласной), не число.** *<say>Un grand chat</say>* и *<say>des grands chats</say>* звучат одинаково — отличает артикль.

---

## Часть 2: Когда «-e» ничего не меняет

Если прилагательное в мужском роде уже **заканчивается на немое -e**, женская форма выглядит и звучит так же.

| Мужской | Женский | Произношение |
|---|---|---|
| <say>jeune</say> | <say>jeune</say> | «жён» / «жён» |
| <say>rouge</say> | <say>rouge</say> | «руж» / «руж» |
| <say>jaune</say> | <say>jaune</say> | «жон» / «жон» |
| <say>triste</say> | <say>triste</say> | «трист» / «трист» |
| <say>facile</say> | <say>facile</say> | «фасиль» / «фасиль» |

Множественное добавляет `s` (немое): *<say>jeunes</say>*, *<say>rouges</say>*.

> **Без изменений.** Если кончается на `-e` без акцента — женский совпадает с мужским.

## Часть 3: Нерегулярные женские формы

Эти модели надо знать наизусть — они продуктивны.

| Окончание М | Окончание Ж | Пример М → Ж |
|---|---|---|
| **-eux** | **-euse** | <say>heureux</say> → <say>heureuse</say> (счастливый/-ая) |
| **-eur** | **-euse** | <say>menteur</say> → <say>menteuse</say> (лживый/-ая) |
| **-f** | **-ve** | <say>sportif</say> → <say>sportive</say>, <say>neuf</say> → <say>neuve</say> |
| **-er** | **-ère** | <say>cher</say> → <say>chère</say>, <say>premier</say> → <say>première</say> |
| **-on / -en / -el / -et** | удваивает согласную + **e** | <say>bon</say> → <say>bonne</say>, <say>italien</say> → <say>italienne</say>, <say>naturel</say> → <say>naturelle</say>, <say>muet</say> → <say>muette</say> |
| **-c** | **-che** или **-que** | <say>blanc</say> → <say>blanche</say>, <say>public</say> → <say>publique</say> |
| **-x** (другие) | **-sse** | <say>faux</say> → <say>fausse</say>, <say>roux</say> → <say>rousse</say> |

**Совсем нерегулярные — учи парами:**

| М | Ж | Перевод |
|---|---|---|
| <say>beau</say> | <say>belle</say> | красивый |
| <say>nouveau</say> | <say>nouvelle</say> | новый |
| <say>vieux</say> | <say>vieille</say> | старый |
| <say>blanc</say> | <say>blanche</say> | белый |
| <say>frais</say> | <say>fraîche</say> | свежий |
| <say>long</say> | <say>longue</say> | длинный |
| <say>gentil</say> | <say>gentille</say> | милый |
| <say>doux</say> | <say>douce</say> | мягкий, нежный |

## Часть 4: Множественное число — ловушки

База: + `s`. Но:

| Если оканчивается на… | Множественное | Пример |
|---|---|---|
| **-s, -x, -z** | без изменений | *<say>un gros chat</say>* → *<say>des gros chats</say>* |
| **-eau** | + **x** | *<say>beau</say>* → *<say>beaux</say>*, *<say>nouveau</say>* → *<say>nouveaux</say>* |
| **-al** | → **-aux** | *<say>national</say>* → *<say>nationaux</say>*, *<say>normal</say>* → *<say>normaux</say>* |

Женские формы множатся обычным `-s`: *<say>belles</say>*, *<say>nationales</say>*, *<say>grosses</say>*.

---

## Часть 5: Место прилагательного — после существительного

**Правило по умолчанию:** прилагательное идёт **после** существительного.

- *<say>une voiture rouge</say>* — красная машина
- *<say>un homme intelligent</say>* — умный мужчина
- *<say>des chats noirs</say>* — чёрные коты
- *<say>une femme française</say>* — француженка

Это противоположно русскому. Не «красная машина», а «машина красная».

Так стоят **все** цвета, национальности, формы, и большинство описательных прилагательных.

---

## Часть 6: BAGS-прилагательные — перед существительным

Небольшая, но очень частая группа стоит **перед** существительным. Запомни аббревиатуру **BAGS**:

| Буква | Значение | Прилагательные |
|---|---|---|
| **B** — Beauty | красота | <say>beau</say>, <say>joli</say> |
| **A** — Age | возраст | <say>jeune</say>, <say>vieux</say>, <say>nouveau</say> |
| **G** — Goodness | оценка | <say>bon</say>, <say>mauvais</say>, <say>gentil</say> |
| **S** — Size | размер | <say>grand</say>, <say>petit</say>, <say>gros</say>, <say>long</say> |

Примеры:
- *<say>une belle maison</say>* — красивый дом
- *<say>un jeune homme</say>* — молодой человек
- *<say>un bon film</say>* — хороший фильм
- *<say>une petite table</say>* — маленький стол
- *<say>un grand livre</say>* — большая книга

> **Ловушка!** *<say>grand homme</say>* (перед) = «великий человек»; *<say>homme grand</say>* (после) = «высокий мужчина». Место меняет смысл.

---

## Часть 7: beau / nouveau / vieux — пятая форма

Эти три прилагательные имеют **особую форму** перед существительным **мужского рода**, начинающимся с **гласной или немого h**. Это сделано, чтобы избежать зияния (двух гласных подряд).

| М перед согл. | М перед гласной | Ж | Мн. М | Мн. Ж |
|---|---|---|---|---|
| <say>beau</say> | <say>bel</say> | <say>belle</say> | <say>beaux</say> | <say>belles</say> |
| <say>nouveau</say> | <say>nouvel</say> | <say>nouvelle</say> | <say>nouveaux</say> | <say>nouvelles</say> |
| <say>vieux</say> | <say>vieil</say> | <say>vieille</say> | <say>vieux</say> | <say>vieilles</say> |

Примеры:
- *<say>un beau garçon</say>* — красивый мальчик
- *<say>un bel homme</say>* — красивый мужчина (не «beau homme»!)
- *<say>un nouvel ami</say>* — новый друг
- *<say>un vieil arbre</say>* — старое дерево

Женская форма уже сама заканчивается на согласную — там проблем нет: *<say>une belle amie</say>*.

---

## Часть 8: Артикль `des` → `de` перед прилагательным

Когда BAGS-прилагательное стоит **перед** существительным во **множественном** числе, артикль `des` превращается в `de`:

- *<say>des amis</say>* → *<say>de bons amis</say>* (хорошие друзья)
- *<say>des films</say>* → *<say>de grands films</say>* (большие фильмы)
- *<say>des maisons</say>* → *<say>de belles maisons</say>* (красивые дома)

Если прилагательное **после** существительного — артикль остаётся `des`:
- *<say>des films intéressants</say>* (интересные фильмы)

## Часть 10: Языковая гамма — прогони через четыре формы

Возьми прилагательное и прогоняй: М ед. → Ж ед. → М мн. → Ж мн.

**heureux (счастливый):**
- *<say>un homme heureux</say>* — счастливый мужчина
- *<say>une femme heureuse</say>* — счастливая женщина
- *<say>des hommes heureux</say>* — счастливые мужчины
- *<say>des femmes heureuses</say>* — счастливые женщины

**petit (маленький):**
- *<say>un petit chat</say>* — маленький кот
- *<say>une petite voiture</say>* — маленькая машина
- *<say>de petits chats</say>* — маленькие коты (des → de!)
- *<say>de petites voitures</say>* — маленькие машины

**beau (красивый):**
- *<say>un beau livre</say>* — красивая книга
- *<say>un bel arbre</say>* — красивое дерево (перед гласной!)
- *<say>une belle maison</say>* — красивый дом
- *<say>de beaux livres</say>* — красивые книги
- *<say>de belles maisons</say>* — красивые дома

**sportif (спортивный):**
- *<say>un garçon sportif</say>*
- *<say>une fille sportive</say>*
- *<say>des garçons sportifs</say>*
- *<say>des filles sportives</say>*

> Прогоняй **вслух**, чувствуй, как `-e` оживляет конечную согласную.

---

## Часть 11: Языковая матрица — описание через être и avoir

Соедини с прошлыми уроками: глагол *être* (Урок 4) — для характеристик, *avoir* (Урок 5) — для частей тела и черт.

| Вопрос | Ответ |
|---|---|
| <say>Comment est ton frère ?</say> | <say>Il est grand et sympathique.</say> |
| <say>Comment sont tes amis ?</say> | <say>Ils sont sportifs et gentils.</say> |
| <say>Comment est ta sœur ?</say> | <say>Elle est petite et heureuse.</say> |
| <say>Tu as les yeux de quelle couleur ?</say> | <say>J'ai les yeux verts.</say> |
| <say>Elle a les cheveux longs ?</say> (ответь: нет, короткие) | <say>Non, elle a les cheveux courts.</say> |
| <say>Il y a une nouvelle voiture ?</say> (ответь: да, красивая красная) | <say>Oui, il y a une belle voiture rouge.</say> |

Прогони своими словами: опиши себя, друга, родственника.

## Часть 12: Упражнения

Согласуй прилагательное в скобках:

2. un (beau) homme → …
3. des amis (sportif) → …
4. une fille (heureux) → …
5. des maisons (blanc) → …
6. un (vieux) ami → …
7. une (nouveau) idée → …
8. des chats (gris) → …
9. une histoire (long) → …
10. de (gentil) enfants → …

<details>
<summary>Ключ</summary>

1. *une voiture **rouge*** (без изменений)
2. *un **bel** homme* (перед гласной!)
3. *des amis **sportifs*** (-f → -fs)
4. *une fille **heureuse*** (-eux → -euse)
5. *des maisons **blanches*** (blanc → blanche → blanches)
6. *un **vieil** ami* (перед гласной!)
7. *une **nouvelle** idée*
8. *des chats **gris*** (уже на -s, не меняется)
9. *une **longue** histoire* (BAGS — перед; нерегулярная Ж)
10. *de **gentils** enfants* (des → de перед BAGS; gentil + s)

</details>

</details>

### Упражнение 7. Диалог — проговори вслух

> — Tu as un frère ?
> — Oui, j'ai un grand frère. Il s'appelle Marc.
> — Comment est-il ?
> — Il est grand, brun, sportif. Il a les yeux verts.
> — Et il est sympa ?
> — Oui, très ! Il est gentil et intelligent. Mais un peu paresseux.
> — Et tes parents ?
> — Mon père est vieux et fatigué. Ma mère est belle et heureuse.

Повтори 3 раза. Потом расскажи по памяти про свою семью.

## Памятка урока

```
ЧЕТЫРЕ ФОРМЫ ПРИЛАГАТЕЛЬНОГО:
              М ед.    Ж ед.     М мн.    Ж мн.
  grand   →   grand    grande    grands   grandes
                «гран»  «гранд»   «гран»   «гранд»

  Ж = + e (оживляет конечную согласную)
  Мн = + s (немое)
  Уже на -e → не меняется (jeune, rouge, triste)
  Уже на -s/-x → не меняется во мн. (gris, vieux)

НЕРЕГУЛЯРНЫЕ ЖЕНСКИЕ:
  -eux  → -euse   (heureux → heureuse)
  -f    → -ve     (sportif → sportive)
  -er   → -ère    (cher → chère)
  -on/-en/-el/-et → удвоение + e (bon → bonne, italien → italienne)
  -c    → -che    (blanc → blanche)

ПЯТАЯ ФОРМА (перед гласной/немым h, М ед.):
  beau    → bel    (un bel homme)
  nouveau → nouvel (un nouvel ami)
  vieux   → vieil  (un vieil arbre)

МНОЖЕСТВЕННОЕ — ЛОВУШКИ:
  -eau → -eaux  (beaux, nouveaux)
  -al  → -aux   (nationaux)

МЕСТО:
  ПОСЛЕ — большинство (цвет, форма, национальность, характер):
    une voiture rouge, un homme intelligent
  ПЕРЕД — BAGS:
    B — Beauty:   beau, joli
    A — Age:      jeune, vieux, nouveau
    G — Goodness: bon, mauvais, gentil
    S — Size:     grand, petit, gros, long

DES → DE:
  des amis → de bons amis (BAGS перед — артикль сокращается)
  des amis sympas (после — артикль остаётся)

СЛОВАРЬ-КОСТЯК:
  Цвета: rouge, jaune, noir, blanc, vert, bleu, gris, brun, rose
  Размер: grand, petit, gros, long, court
  Характер: bon, gentil, méchant, heureux, triste, intelligent,
            timide, sérieux, paresseux, sportif
  Внешность: beau, joli, laid, jeune, vieux, nouveau, mince, fort

МАТРИЦА — описывай через être + avoir:
  Il est grand et sympathique.
  Elle a les yeux verts et les cheveux longs.
```
