# Урок 11: Неправильные глаголы настоящего времени
**Словарный запас: pouvoir, vouloir, devoir, savoir, voir, dire, mettre, lire, écrire + модальные сцены**

---

## Как работать с этим уроком

1. **Прочитай** парадигмы — пойми, где сидит «излом» в спряжении (5 минут)
2. **Прогони** каждый глагол вслух через все 6 лиц — без пауз
3. **Соедини** модальный глагол + инфинитив: *je veux manger, tu peux venir, il doit partir* — это половина живой речи

> Эти девять глаголов — **скелет** французского. Без них ни одной фразы.
> **Особо:** запомни форму **ils/elles** каждого глагола — она станет основой сослагательного наклонения в Уроке 31.

---

## Часть 1: Тройка модальных — pouvoir, vouloir, devoir

Эти три глагола живут вместе. Спрягаются по схожей модели: «излом» между ед.ч. и мн.ч.

### pouvoir (мочь, иметь возможность)

| Лицо | Форма | Произношение |
|------|-------|--------------|
| je | **peux** | «пё» |
| tu | **peux** | «пё» |
| il / elle | **peut** | «пё» |
| nous | **pouvons** | «пу-вон» |
| vous | **pouvez** | «пу-ве» |
| ils / elles | **peuvent** | «пёв» |

### vouloir (хотеть)

| Лицо | Форма | Произношение |
|------|-------|--------------|
| je | **veux** | «вё» |
| tu | **veux** | «вё» |
| il / elle | **veut** | «вё» |
| nous | **voulons** | «ву-лон» |
| vous | **voulez** | «ву-ле» |
| ils / elles | **veulent** | «вёль» |

### devoir (должен; быть должным)

| Лицо | Форма | Произношение |
|------|-------|--------------|
| je | **dois** | «дуа» |
| tu | **dois** | «дуа» |
| il / elle | **doit** | «дуа» |
| nous | **devons** | «дёвон» |
| vous | **devez** | «дёве» |
| ils / elles | **doivent** | «дуав» |

> **Паттерн «обуви»:** в ед.ч. и в 3-м лице мн.ч. — сильный, «полный» гласный (peux, peut, peuvent). У nous/vous — «слабая» форма с -ou-/-e-. Это классическая французская модель **«сапог»** (boot pattern): 4 сильные формы (je, tu, il, ils) окружают 2 слабые (nous, vous).

---

## Часть 2: Модальная конструкция — модальный + инфинитив

Эти три глагола **сами по себе мало значат**. Их сила — в связке с инфинитивом.

| Конструкция | Значение | Пример |
|-------------|----------|--------|
| **pouvoir** + infinitif | мочь, иметь возможность / разрешение | *Je peux venir demain.* |
| **vouloir** + infinitif | хотеть сделать | *Tu veux manger ?* |
| **devoir** + infinitif | должен, обязан | *Nous devons partir.* |

**Никаких предлогов между ними!** Не *je peux à venir*, а *je peux venir*.

### Вежливые формы (запомни сразу)

| Французский | Русский |
|-------------|---------|
| Je voudrais… | Я хотел(а) бы… |
| Pourriez-vous… ? | Не могли бы Вы… ? |
| Vous devriez… | Вам следовало бы… |

> Это **conditionnel** — пока без объяснений, просто как штампы. Полная теория — Урок 28.

### Ловушки

- **vouloir** + предмет (без инфинитива) = «хотеть что-то»: *Je veux un café.*
- **devoir** + сумма = «быть должным (денег)»: *Tu me dois dix euros.*
- **pouvoir** в вопросе с инверсией: *Puis-je…?* (НЕ *peux-je*) — единственная аномалия.

## Часть 3: savoir vs connaître — два «знать»

Французский разделяет «знать» на два глагола. Русский — нет. Это ловушка.

### savoir (знать факт; уметь)

| Лицо | Форма |
|------|-------|
| je sais | nous savons |
| tu sais | vous savez |
| il/elle sait | ils/elles **savent** |

**savoir** = знать **информацию** или **уметь** (после идёт инфинитив):
- *Je sais où il habite.* — Я знаю, где он живёт.
- *Tu sais nager ?* — Умеешь плавать?
- *Elle sait que c'est vrai.* — Она знает, что это правда.

### connaître (быть знакомым)

| Лицо | Форма |
|------|-------|
| je connais | nous connaissons |
| tu connais | vous connaissez |
| il/elle connaît | ils/elles **connaissent** |

**connaître** = быть знакомым с **человеком, местом, произведением** (после идёт существительное):
- *Je connais Paris.* — Я знаю Париж (бывал там).
- *Tu connais Marie ?* — Знаешь Мари?
- *Nous connaissons cette chanson.* — Знаем эту песню.

> **Правило большого пальца:** *savoir* + **что/как/уметь**; *connaître* + **кого/что (как объект)**. Если можно поставить «уметь» или «факт, что…» — это savoir. Если «знаком с…» — connaître.

## Часть 4: voir, dire — короткие, но коварные

### voir (видеть)

| Лицо | Форма |
|------|-------|
| je vois | nous voyons |
| tu vois | vous voyez |
| il/elle voit | ils/elles **voient** |

- *Je vois la mer.* — Я вижу море.
- *Vous voyez la différence ?* — Видите разницу?

> **Похож на devoir по модели**: vois/vois/voit + voyons/voyez/**voient** (произношение: «вуа/вуайон/вуайе/вуа»). Буква *y* появляется только в nous/vous.

### dire (говорить, сказать)

| Лицо | Форма |
|------|-------|
| je dis | nous disons |
| tu dis | **vous dites** |
| il/elle dit | ils/elles **disent** |

> **Ловушка!** *vous dites* — а НЕ *vous disez*. Это одна из всего **трёх** форм во французском, где vous не заканчивается на -ez: *vous êtes, vous faites, vous dites*. Запомни их вместе.

- *Je dis la vérité.* — Я говорю правду.
- *Qu'est-ce que tu dis ?* — Что ты говоришь?

## Часть 5: mettre, lire, écrire — действия с предметом

### mettre (класть, ставить, надевать)

| Лицо | Форма |
|------|-------|
| je mets | nous mettons |
| tu mets | vous mettez |
| il/elle met | ils/elles **mettent** |

**Универсальный глагол:**
- *Je mets le livre sur la table.* — Кладу книгу на стол.
- *Elle met un manteau.* — Она надевает пальто.
- *Nous mettons la table.* — Накрываем на стол.

> По этой модели спрягаются **permettre** (разрешать), **promettre** (обещать), **remettre** (вернуть, отложить).

### lire (читать)

| Лицо | Форма |
|------|-------|
| je lis | nous lisons |
| tu lis | vous lisez |
| il/elle lit | ils/elles **lisent** |

### écrire (писать)

| Лицо | Форма |
|------|-------|
| j'écris | nous écrivons |
| tu écris | vous écrivez |
| il/elle écrit | ils/elles **écrivent** |

> **Паттерн:** в ед.ч. — короткая основа (lis-, écri-), во мн.ч. — длинная (lis**ons** / écri**vons**). Буква **v** в écrire появляется только во мн.ч.

## Часть 6: Форма ils/elles — фундамент будущего

Запомни эту таблицу **сейчас**. В Уроке 31 ты увидишь, что от неё растёт сослагательное наклонение.

| Глагол | ils/elles форма | Корень subjonctif (предвестник) |
|--------|-----------------|---------------------------------|
| pouvoir | ils **peuvent** | que je puisse (особый) |
| vouloir | ils **veulent** | que je veuille (особый) |
| devoir | ils **doivent** | que je doive |
| savoir | ils **savent** | que je sache (особый) |
| voir | ils **voient** | que je voie |
| dire | ils **disent** | que je dise |
| mettre | ils **mettent** | que je mette |
| lire | ils **lisent** | que je lise |
| écrire | ils **écrivent** | que j'écrive |

> **Правило-предвестник:** Для большинства глаголов сослагательное наклонение строится так: бери форму **ils**, отрезай -ent, добавляй личные окончания. *ils mettent → mett- → que je mette*. Подробно — в Уроке 31. Сейчас просто закрепи звучание формы ils.

## Языковая гамма

Прогони каждый глагол через все 6 лиц **вслух**, не глядя в таблицу. Скорость важнее точности на первом заходе.

**Гамма 1 — pouvoir + venir:**
> Je peux venir. Tu peux venir. Il peut venir. Nous pouvons venir. Vous pouvez venir. Ils peuvent venir.

**Гамма 2 — vouloir + un café:**
> Je veux un café. Tu veux un café. Elle veut un café. Nous voulons un café. Vous voulez un café. Elles veulent un café.

**Гамма 3 — devoir + partir:**
> Je dois partir. Tu dois partir. Il doit partir. Nous devons partir. Vous devez partir. Ils doivent partir.

**Гамма 4 — savoir + nager:**
> Je sais nager. Tu sais nager. Elle sait nager. Nous savons nager. Vous savez nager. Elles savent nager.

**Гамма 5 — dire la vérité:**
> Je dis la vérité. Tu dis la vérité. Il dit la vérité. Nous disons la vérité. **Vous dites** la vérité. Ils disent la vérité.

**Гамма 6 — lire un livre:**
> Je lis un livre. Tu lis un livre. Elle lit un livre. Nous lisons un livre. Vous lisez un livre. Elles lisent un livre.

**Гамма 7 — écrire une lettre:**
> J'écris une lettre. Tu écris une lettre. Il écrit une lettre. Nous écrivons une lettre. Vous écrivez une lettre. Ils écrivent une lettre.

  Je voudrais…   Pourriez-vous… ?   Vous devriez…

SAVOIR vs CONNAÎTRE:
  savoir    + факт / + que / + infinitif (уметь)
            Je sais nager. Je sais où il est.
  connaître + человек / место / произведение
            Je connais Paris. Je connais Marie.

ОСТАЛЬНЫЕ:
  voir   : vois / vois / voit // voyons / voyez // voient
  dire   : dis  / dis  / dit  // disons / DITES // disent ⚠
  mettre : mets / mets / met  // mettons / mettez // mettent
  lire   : lis  / lis  / lit  // lisons / lisez // lisent
  écrire : écris / écris / écrit // écrivons / écrivez // écrivent

ТРИ ИСКЛЮЧЕНИЯ ДЛЯ VOUS (НЕ -ez):
  vous ÊTES   vous FAITES   vous DITES

ФОРМА ils — ФУНДАМЕНТ БУДУЩЕГО:
  peuvent / veulent / doivent / savent / voient
  disent / mettent / lisent / écrivent
  → отсюда вырастет subjonctif (Урок 31)

ИДИОМЫ:
  Ça veut dire…    On dit que…    Vouloir, c'est pouvoir.
  Tu vois ?        Voyons !       Comme on dit.

БЕЗЛИЧНОЕ:
  Il faut + infinitif = надо, нужно (Il faut partir.)
