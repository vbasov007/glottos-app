# Текст к уроку 41: Новости дня — se-passive в действии
**Тема: базовые новостные ситуации с пассивом через se**

---

## Текст для аудирования и повторения

1. Aquí se vende pan fresco cada mañana.
2. En esta tienda se hablan tres idiomas.
3. Se alquilan habitaciones por semanas.
4. En la oficina se firman muchos contratos.
5. Se publica el periódico todos los días.
6. En este barrio se construyen casas nuevas.
7. No se permite fumar en el restaurante.
8. Se prohíbe entrar con perros.
9. Aquí se trabaja mucho.
10. En España se cena tarde.
11. En México se come comida picante.
12. ¿Cómo se dice «book» en español?
13. ¿Cómo se escribe tu nombre?
14. Se necesita personal con experiencia.
15. Se busca camarero para fines de semana.
16. En esta clase se aprende rápido.
17. Se inaugura el museo el sábado.
18. Se anuncian las noticias a las ocho.
19. Se aprueban las leyes en el parlamento.
20. Se eligen los representantes cada cuatro años.
21. Se celebra la fiesta el día doce.
22. Se organizan conciertos en el parque.
23. En este país se vive bien.
24. Se reciben pedidos en línea.
25. Aquí se preparan platos típicos.
26. Se cocina con aceite de oliva.
27. Se sirve el café después de la cena.
28. En la radio se escucha música clásica.
29. Se rechazan las propuestas malas.
30. Se confirma la noticia oficialmente.
