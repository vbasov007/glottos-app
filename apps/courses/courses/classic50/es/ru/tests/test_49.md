# Тест к уроку 49: Глаголы изменения — карта «стать»

**Как сказать по-испански?**
Переведи фразы на испанский язык. Главное — выбрать **правильный глагол** «стать»: *ponerse, volverse, hacerse, llegar a ser, convertirse en, quedarse*.
