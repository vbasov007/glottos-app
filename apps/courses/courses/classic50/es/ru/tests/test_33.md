# Тест к уроку 33: Subjuntivo после эмоций и оценок

**Как сказать по-испански?**
Переведи фразы на испанский. Тренируем триггеры: эмоции (*me alegro de que, siento que, me gusta que…*), оценки (*es bueno que, es una pena que…*), реакции (*¡qué pena que…!, ¡qué bien que…!*). Не забывай правило двух подлежащих: одно лицо → инфинитив, два лица → *que* + subjuntivo.
