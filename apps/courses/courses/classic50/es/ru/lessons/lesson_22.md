# Урок 22: Pretérito indefinido — НЕправильные глаголы
**Словарный запас: иррегулярные глаголы прошедшего времени (около 30 единиц)**

---

## Как работать с этим уроком

1. **Прочитай** — пойми три большие группы (U-, I-, J-основы) и три «диких» глагола (ser/ir, dar, ver).
2. **Прогони таблицы вслух** — каждую форму, не глазами, а ртом.
3. **Дрилл через шкалу** — для одного глагола в один заход: yo, tú, él, nosotros, vosotros, ellos. Без пауз.
4. **Матрицу — вопрос-ответ** — отвечай «вчера» на вопросы партнёра (или себя).

> Знать, что *tener* в прошедшем — *tuve*, — это 10%. Сказать *tuve, tuviste, tuvo, tuvimos, tuvisteis, tuvieron* за 5 секунд — это 90%.
> Новость хорошая: 90% всех иррегулярных preterite ведут себя по **одной схеме окончаний**. Выучи окончания — и ты получишь 14 глаголов сразу.

---

## Часть 1: Главная идея — единый набор окончаний

В уроке 21 у тебя были обычные окончания (-é, -aste, -ó, …). У иррегулярных глаголов **окончания другие** — и они **одинаковые** почти для всех.

| Лицо | Окончание |
|------|-----------|
| yo | **-e** |
| tú | **-iste** |
| él / ella / usted | **-o** |
| nosotros | **-imos** |
| vosotros | **-isteis** |
| ellos / ellas / ustedes | **-ieron** (или **-eron** после j) |

> **Ловушка №1, главная:** в 1-м и 3-м лице ед. ч. (yo, él) — **БЕЗ тильды**!
> *tuve*, *tuvo* — но не *tuvé*, не *tuvó*.
> *hice*, *hizo* — но не *hicé*, не *hizó*.
> Это отличает иррегулярные preterite от регулярных (*hablé, habló* — с тильдой!).

Если ты выучишь **основу** (stem) каждого иррегулярного глагола, ты получишь всю парадигму бесплатно — окончания всегда одни и те же.

---

## Часть 2: U-основы — самая большая группа

Семь глаголов: вместо своей гласной они вставляют в основу **u**.

| Инфинитив | Основа | Перевод |
|-----------|--------|---------|
| andar | **anduv-** | ходить |
| estar | **estuv-** | быть (находиться) |
| tener | **tuv-** | иметь |
| poder | **pud-** | мочь |
| poner | **pus-** | класть, ставить |
| saber | **sup-** | знать |
| caber | **cup-** | вмещаться |

### Парадигма (на примере *tener*)

| Лицо | Форма |
|------|-------|
| yo | **tuve** |
| tú | **tuviste** |
| él / ella / Ud. | **tuvo** |
| nosotros | **tuvimos** |
| vosotros | **tuvisteis** |
| ellos / ellas / Uds. | **tuvieron** |

Точно по той же модели: *anduve, anduviste, anduvo, anduvimos, anduvisteis, anduvieron*; *estuve … estuvieron*; *pude … pudieron*; *puse … pusieron*; *supe … supieron*; *cupe … cupieron*.

> **Семантика, важная для перевода:**
> - *supe* (от *saber*) — **«узнал»** (а не «знал»), то есть момент получения информации. *Lo supe ayer.* — «Я узнал об этом вчера».
> - *pude* (от *poder*) — **«смог»**, *no pude* — **«не смог»**.
> - *tuve que + inf.* — **«пришлось»**: *Tuve que ir al médico.* — «Мне пришлось пойти к врачу».
> - *conocí* (это уже из урока 21, регулярное) — **«познакомился»**, а не «знал».

## Часть 3: I-основы — три глагола

В основе появляется **i**.

| Инфинитив | Основа | Перевод |
|-----------|--------|---------|
| hacer | **hic-** | делать |
| querer | **quis-** | хотеть, любить |
| venir | **vin-** | приходить |

### Парадигма *hacer*

| Лицо | Форма |
|------|-------|
| yo | **hice** |
| tú | **hiciste** |
| él / ella / Ud. | **hizo** |
| nosotros | **hicimos** |
| vosotros | **hicisteis** |
| ellos / ellas / Uds. | **hicieron** |

> **Ловушка №2, орфография:** в 3-м лице ед. ч. — **hizo** (с **z**, а не c!). Это сохраняет звук «θ»/«с» перед *o*. Если бы написали *hico*, читалось бы «ико» = «к»+«о».

*Querer*: *quise, quisiste, quiso, quisimos, quisisteis, quisieron*.
*Venir*: *vine, viniste, vino, vinimos, vinisteis, vinieron*.

> **Семантика:**
> - *quise + inf.* — **«попытался»**: *Quise abrir la puerta.* — «Я попытался открыть дверь».
> - *no quise + inf.* — **«отказался»**: *No quise ir.* — «Я отказался идти».

---

## Часть 4: J-основы — на согласную *j*

Основа оканчивается на **j**. Главная фишка: в **3-м лице мн.ч.** окончание не *-ieron*, а **-eron** (j «съедает» i).

| Инфинитив | Основа | Перевод |
|-----------|--------|---------|
| decir | **dij-** | сказать |
| traer | **traj-** | приносить |
| conducir | **conduj-** | вести (машину) |
| traducir | **traduj-** | переводить |
| producir | **produj-** | производить |

### Парадигма *decir*

| Лицо | Форма |
|------|-------|
| yo | **dije** |
| tú | **dijiste** |
| él / ella / Ud. | **dijo** |
| nosotros | **dijimos** |
| vosotros | **dijisteis** |
| ellos / ellas / Uds. | **dijeron** ← без *i*! |

*Traer*: *traje, trajiste, trajo, trajimos, trajisteis, **trajeron***.
*Conducir*: *conduje, condujiste, condujo, condujimos, condujisteis, **condujeron***.

> **Ловушка №3:** **не** *dijieron*, **не** *trajieron*. Все глаголы на *-ducir* и оба «traer/decir» — теряют *i* в *ellos*.

---

## Часть 5: Полностью «дикие» — ser, ir, dar, ver

Это четыре глагола, которые надо просто выучить наизусть. Они короткие, частотные, и встретятся в каждом тексте.

### Ser и Ir — одна форма на двоих!

Контекст всегда подскажет, какой именно глагол. *Fui a Madrid* — «я поехал», *Fui profesor* — «я был учителем».

| Лицо | ser / ir |
|------|----------|
| yo | **fui** |
| tú | **fuiste** |
| él / ella / Ud. | **fue** |
| nosotros | **fuimos** |
| vosotros | **fuisteis** |
| ellos / ellas / Uds. | **fueron** |

> **Ловушка №4:** *fui, fue* — **без тильды**. Это короткие односложные слова, тильда не нужна.

### Dar — спрягается как обычный -ER/-IR глагол

(хотя инфинитив на -AR — да, это странно).

| Лицо | dar |
|------|-----|
| yo | **di** |
| tú | **diste** |
| él / ella / Ud. | **dio** |
| nosotros | **dimos** |
| vosotros | **disteis** |
| ellos / ellas / Uds. | **dieron** |

### Ver — почти регулярный, но без тильды

| Лицо | ver |
|------|-----|
| yo | **vi** |
| tú | **viste** |
| él / ella / Ud. | **vio** |
| nosotros | **vimos** |
| vosotros | **visteis** |
| ellos / ellas / Uds. | **vieron** |

> **Ловушка №5:** *vi, vio, di, dio, fui, fue* — **никаких тильд**. В уроке 21 у регулярных глаголов были *hablé, habló* — а тут нет. Запомни: иррегулярные preterite и короткие односложные = без тильды.

---

## Часть 6: Стем-чейндж в -IR глаголах (3-е лицо)

Это не «настоящие» иррегулярные — окончания у них **регулярные** (-ió, -ieron). Но в основе меняется гласная **только в 3-м лице** ед. и мн. числа.

| Инфинитив | yo / tú / nos / vos | él / ellos | Перевод |
|-----------|---------------------|------------|---------|
| pedir (e→i) | pedí, pediste, pedimos, pedisteis | **pidió / pidieron** | просить |
| servir (e→i) | serví, serviste, servimos, servisteis | **sirvió / sirvieron** | подавать, служить |
| repetir (e→i) | repetí, repetiste, repetimos, repetisteis | **repitió / repitieron** | повторять |
| seguir (e→i) | seguí, seguiste, seguimos, seguisteis | **siguió / siguieron** | следовать, продолжать |
| sentir (e→ie/i) | sentí, sentiste, sentimos, sentisteis | **sintió / sintieron** | чувствовать |
| preferir (e→ie/i) | preferí, preferiste, preferimos, preferisteis | **prefirió / prefirieron** | предпочитать |
| dormir (o→ue/u) | dormí, dormiste, dormimos, dormisteis | **durmió / durmieron** | спать |
| morir (o→ue/u) | morí, moriste, morimos, moristeis | **murió / murieron** | умирать |

> **Главное:** меняется **только** *él/ella/Ud.* и *ellos/ellas/Uds.* Все остальные формы — обычные регулярные.

## Часть 8: Упражнения

### Упражнение 1. Шкала — прогон одного глагола по всем лицам

Произнеси вслух, чётко, без пауз, в порядке: yo → tú → él → nosotros → vosotros → ellos. Каждый глагол — два прохода.

1. **tener** (иметь)
2. **estar** (быть)
3. **hacer** (делать)
4. **ir** (идти)
5. **decir** (сказать)
6. **venir** (приходить)
7. **poner** (класть)
8. **traer** (принести)

<details>
<summary>Ключ</summary>

1. *tuve, tuviste, tuvo, tuvimos, tuvisteis, tuvieron*
2. *estuve, estuviste, estuvo, estuvimos, estuvisteis, estuvieron*
3. *hice, hiciste, hizo, hicimos, hicisteis, hicieron*
4. *fui, fuiste, fue, fuimos, fuisteis, fueron*
5. *dije, dijiste, dijo, dijimos, dijisteis, dijeron* (без *i*!)
6. *vine, viniste, vino, vinimos, vinisteis, vinieron*
7. *puse, pusiste, puso, pusimos, pusisteis, pusieron*
8. *traje, trajiste, trajo, trajimos, trajisteis, trajeron* (без *i*!)

</details>

### Упражнение 2. Поставь глагол в нужную форму (pretérito indefinido)

1. Ayer yo (tener) ___ mucho trabajo.
2. ¿Dónde (estar, tú) ___ anoche?
3. Mis amigos (venir) ___ a las ocho.
4. Nosotros (hacer) ___ la tarea juntos.
5. Ella (decir) ___ la verdad.
6. Los niños (dormir) ___ ocho horas.
7. ¿Qué (pedir, tú) ___ en el restaurante?
8. Pedro no (poder) ___ venir.
9. El camarero nos (servir) ___ el café.
10. Yo (ir) ___ al cine el sábado.
11. Mi madre (poner) ___ la mesa.
12. Los profesores (traer) ___ los libros.
13. Ayer (ser) ___ un día difícil.
14. Yo (ver) ___ a Carmen en la calle.
15. ¿Quién (conducir) ___ el coche?

<summary>Ключ</summary>

1. *tuve*
2. *estuviste*
3. *vinieron*
4. *hicimos*
5. *dijo*
6. *durmieron* (o→u в 3pl)
7. *pediste*
8. *pudo*
9. *sirvió* (e→i в 3sg)
10. *fui*
11. *puso*
12. *trajeron* (без *i*!)
13. *fue*
14. *vi* (без тильды)
15. *condujo*

</details>

### Упражнение 3. Матрица — вопрос «вчера» и ответ

Партнёр спрашивает в *tú*, ты отвечаешь в *yo*. Если занимаешься один — задавай вопрос вслух и сам отвечай.

| № | Вопрос | Возможный ответ |
|---|--------|-----------------|
| 1 | ¿Qué hiciste ayer? | Ayer fui al trabajo y vi a un amigo. |
| 2 | ¿Adónde fuiste el fin de semana? | Fui a la playa con mi familia. |
| 3 | ¿A qué hora viniste a casa? | Vine a las once de la noche. |
| 4 | ¿Qué pediste para cenar? | Pedí una pizza y un café. |
| 5 | ¿Dormiste bien anoche? | Sí, dormí ocho horas. |
| 6 | ¿Pudiste terminar el trabajo? | No, no pude. Tuve que volver hoy. |
| 7 | ¿Qué te dijo el jefe? | Me dijo que el proyecto está listo. |
| 8 | ¿Quién condujo el coche? | Lo condujo mi hermano. |

Прогони таблицу 2 раза. Потом задай те же вопросы, поменяв «вчера» на «на прошлой неделе» (*la semana pasada*).

### Упражнение 4. Найди и исправь ошибку

1. Ayer yo *tené* mucho trabajo.
2. Mi amigo *hació* la tarea.
3. Ellos *dijieron* la verdad.
4. Yo *fuí* al cine.
5. El niño *dormió* mucho.
6. Nosotros *trajeron* los libros.
7. Ella *vió* la película.
8. Yo *podé* hacerlo.

<details>
<summary>Ключ</summary>

1. *tuve* (U-основа, без тильды)
2. *hizo* (I-основа, с **z** в 3sg, без тильды)
3. *dijeron* (J-основа теряет *i* в 3pl)
4. *fui* (короткое слово, без тильды)
5. *durmió* (o→u в 3sg)
6. *trajimos* (1pl — *imos*; в задании были «мы», а форма *trajeron* = «они»)
7. *vio* (без тильды)
8. *pude* (U-основа)

</details>

### Упражнение 6. Мини-рассказ — продолжи историю

Прочти первую фразу и придумай по-испански ещё **5 фраз** в pretérito indefinido, используя хотя бы три иррегулярных глагола из урока.

> Ayer fue un día muy interesante. Por la mañana…

<details>
<summary>Пример</summary>

> *Ayer fue un día muy interesante. Por la mañana fui al mercado y compré frutas. Después vine a casa y puse todo en la mesa. Mi hermana vino a comer y me trajo un libro. Yo le di las gracias y le dije que estuvo perfecto. Por la noche dormimos muy bien.*

</details>

### Упражнение 7. Диалог — проговори оба голоса

> — ¿Qué tal el fin de semana?
> — Muy bien, gracias. El sábado fui a la montaña con Pedro.
> — ¿Y qué hicisteis allí?
> — Anduvimos mucho, vimos un lago precioso y comimos en un pueblo pequeño.
> — ¿A qué hora volvisteis?
> — Volvimos tarde. Llegué a casa a las once y me dormí enseguida.
> — ¿Pudiste descansar el domingo?
> — Sí, dormí hasta las diez. Después hice la compra y leí un libro.
> — ¡Qué fin de semana más bueno!
> — Sí, fue genial.

Повтори 3 раза. Затем закрой текст и пересоставь по памяти.

## Памятка урока

```
PRETÉRITO INDEFINIDO — НЕправильные глаголы.

ОБЩИЕ ОКОНЧАНИЯ (для ВСЕХ, кроме ser/ir/dar/ver):
  yo        -e
  tú        -iste
  él        -o
  nosotros  -imos
  vosotros  -isteis
  ellos     -ieron   (после J → -eron)

  ▶ Никаких тильд на yo и él!  tuve, tuvo (НЕ tuvé, tuvó)

U-ОСНОВЫ (вставляют u):
  andar  → anduv-       estar  → estuv-
  tener  → tuv-         poder  → pud-
  poner  → pus-         saber  → sup-
  caber  → cup-

I-ОСНОВЫ:
  hacer  → hic-   (но 3sg: HIZO — c → z перед o!)
  querer → quis-
  venir  → vin-

J-ОСНОВЫ (3pl без i):
  decir     → dij-   → dijeron
  traer     → traj-  → trajeron
  conducir  → conduj-→ condujeron
  traducir  → traduj-→ tradujeron

«ДИКИЕ» — учить наизусть:
  ser / ir : fui, fuiste, fue, fuimos, fuisteis, fueron
  dar      : di, diste, dio, dimos, disteis, dieron
  ver      : vi, viste, vio, vimos, visteis, vieron

  ▶ Все короткие: vi, vio, di, dio, fui, fue — БЕЗ тильды.

-IR ГЛАГОЛЫ СО СМЕНОЙ ОСНОВЫ (только 3sg / 3pl):
  pedir   → pidió, pidieron     servir  → sirvió, sirvieron
  repetir → repitió, repitieron seguir  → siguió, siguieron
  sentir  → sintió, sintieron   preferir→ prefirió, prefirieron
  dormir  → durmió, durmieron   morir   → murió, murieron

СЕМАНТИКА:
  supe = «узнал»     pude = «смог»
  no pude = «не смог»  no quise = «отказался»
  quise = «попытался»  tuve que = «пришлось»

ВРЕМЕННЫЕ МАРКЕРЫ:
  ayer, anoche, anteayer
  la semana pasada, el año pasado
```
