# Урок 10: Hay vs está/están. Безличный haber. Квантификаторы
**Словарный запас: мебель и обстановка, mucho/poco/bastante/demasiado/algún, порядковые числительные 1–10**

---

## Как работать с этим уроком

1. **Прочитай** — пойми разницу между *hay* (есть в наличии) и *está/están* (находится где-то) (5 минут)
2. **Проговори** — каждую пару контрастно: «есть стол» vs «стол находится в кухне»
3. **Ускоряйся** — пока выбор *hay* / *está* не станет автоматическим

> Это последний урок блока A1. После него — внутренний хребет языка собран: <say>ser</say>, <say>estar</say>, <say>hay</say>, настоящее время, согласование, отрицание, вопросы.

---

## Часть 1: Главное правило — hay против está/están

> **<say>Hay</say>** = «есть / имеется» (что-то существует, без указания, какое именно).
> **<say>Está</say> / <say>están</say>** = «находится / находятся» (известное, конкретное — где оно).

| Hay | Está / Están |
|-----|--------------|
| **существование**, **наличие** | **местоположение** известного |
| с **<say>un</say> / <say>una</say> / <say>unos</say> / <say>unas</say>**, числами, квантификаторами | с **<say>el</say> / <say>la</say> / <say>los</say> / <say>las</say>**, именами собственными, притяжательными |
| вводит новое в разговор | говорит о уже известном |
| **всегда одна форма** — *<say>hay</say>* (ед. и мн.!) | согласуется: *<say>está</say>* (ед.) / *<say>están</say>* (мн.) |

| Hay (новое, безличное) | Está / Están (конкретное) |
|------------------------|----------------------------|
| *<say>Hay un libro en la mesa.</say>* — На столе (есть) книга. | *<say>El libro está en la mesa.</say>* — Книга (эта самая) — на столе. |
| *<say>Hay tres sillas.</say>* — Есть три стула. | *<say>Las sillas están en la cocina.</say>* — Стулья — на кухне. |
| *<say>¿Hay una farmacia cerca?</say>* — Тут есть аптека? | *<say>¿Dónde está la farmacia?</say>* — Где аптека? |
| *<say>Hay mucha gente.</say>* — Много народу. | *<say>María está aquí.</say>* — Мария здесь. |

> **Тест на выбор:** можешь подставить «<say>el</say> / <say>la</say> / <say>mi</say> / <say>Juan</say>»? → *<say>está</say>/<say>están</say>*. Подставляешь «<say>un</say> / <say>una</say> / <say>dos</say> / <say>mucho</say>»? → *<say>hay</say>*.

## Часть 2: Hay — безличный, всегда одинаковый

> **<say>Hay</say>** — это форма глагола *<say>haber</say>* третьего лица единственного числа в безличном употреблении. **Не меняется** ни по числу, ни по роду.

| Русский | Испанский |
|---------|-----------|
| Есть стол. | *<say>Hay una mesa.</say>* |
| Есть **столы**. | *<say>Hay mesas.</say>* (НЕ ~~han~~ mesas) |
| Есть **много** столов. | *<say>Hay muchas mesas.</say>* |
| **Нет** столов. | *<say>No hay mesas.</say>* |
| Есть **что-нибудь**? | *<say>¿Hay algo?</say>* |
| **Ничего** нет. | *<say>No hay nada.</say>* |

> **Ловушка №1:** *<say>Hay</say>* — это **одно слово**, не *<say>ha</say> + <say>y</say>*. Произносится «**ай**», как русское «ай».
> **Ловушка №2:** Никогда не говори ~~*hayn*~~ или ~~*hayan*~~ для множественного. Это безличная форма — всегда *<say>hay</say>*.
> **Ловушка №3:** ~~*Hay el libro*~~ — **запрещено**. После *<say>hay</say>* не бывает определённого артикля. Если хочешь сказать «книга на столе», нужен *<say>está</say>*: *<say>el libro está en la mesa</say>*.

## Часть 3: Квантификаторы — сколько именно?

Квантификаторы согласуются с существительным по роду и числу (кроме нескольких неизменяемых).

### Согласующиеся: mucho, poco, demasiado, todo, otro

| Форма | Пример |
|-------|--------|
| *<say>mucho</say> / <say>mucha</say> / <say>muchos</say> / <say>muchas</say>* | *<say>Hay mucha agua.</say>* — Много воды. / *<say>muchos libros</say>* — много книг. |
| *<say>poco</say> / <say>poca</say> / <say>pocos</say> / <say>pocas</say>* | *<say>Hay poca gente.</say>* — Мало народу. / *<say>pocos coches</say>* — мало машин. |
| *<say>demasiado</say> / <say>demasiada</say> / <say>demasiados</say> / <say>demasiadas</say>* | *<say>Hay demasiado ruido.</say>* — Слишком много шума. |
| *<say>todo</say> / <say>toda</say> / <say>todos</say> / <say>todas</say>* | *<say>todos los días</say>* — каждый день / все дни. |
| *<say>otro</say> / <say>otra</say> / <say>otros</say> / <say>otras</say>* | *<say>otra silla</say>* — другой стул. **Без артикля!** Не ~~*una otra silla*~~. |

### Неизменяемые: bastante, suficiente, más, menos

| Форма | Пример |
|-------|--------|
| *<say>bastante</say> / <say>bastantes</say>* (только по числу) | *<say>Hay bastante comida.</say>* — Достаточно еды. / *<say>bastantes problemas</say>* — достаточно проблем. |
| *<say>suficiente</say> / <say>suficientes</say>* (только по числу) | *<say>Hay suficiente tiempo.</say>* — Достаточно времени. |
| *<say>más</say>* | *<say>Hay más sillas aquí.</say>* — Здесь больше стульев. |
| *<say>menos</say>* | *<say>Hay menos gente hoy.</say>* — Сегодня меньше народу. |

### Algún / ningún — «какой-то» / «никакой»

Перед существительным мужского рода единственного числа теряют *-o* (как *uno → un*).

| Полная форма | Усечённая (перед m. sg.) | Пример |
|--------------|---------------------------|--------|
| *<say>alguno</say>* | **<say>alguno → algún</say>** | *<say>algún día</say>* — какой-нибудь день |
| *<say>ninguno</say>* | **<say>ninguno → ningún</say>** | *<say>ningún problema</say>* — никаких проблем |
| *<say>alguna</say> / <say>algunos</say> / <say>algunas</say>* | без изменений | *<say>alguna idea</say>* — какая-нибудь идея |
| *<say>ninguna</say>* | без изменений | *<say>ninguna duda</say>* — никаких сомнений |

> *<say>Ningún</say> / <say>ninguna</say>* почти всегда **единственного числа**: «никаких» по-испански = «никакого ни одного». *<say>No hay ningún libro</say>* = нет ни одной книги.

## Часть 4: Hay + квантификатор — рабочая комбинация

Это базовая структура для описания пространства.

| Испанский | Русский |
|-----------|---------|
| *<say>En el salón hay un sofá y dos sillones.</say>* | В гостиной (есть) диван и два кресла. |
| *<say>Hay muchos libros en la estantería.</say>* | На полке много книг. |
| *<say>No hay nadie en la oficina.</say>* | В офисе никого нет. |
| *<say>¿Hay algún restaurante por aquí?</say>* | Тут есть какой-нибудь ресторан? |
| *<say>Hay bastante espacio.</say>* | Достаточно места. |
| *<say>Hay demasiada gente.</say>* | Слишком много народу. |
| *<say>No hay ningún problema.</say>* | Никаких проблем. |

> **Заметь:** *<say>no hay nadie</say>*, *<say>no hay nada</say>*, *<say>no hay ningún</say>*. В испанском **двойное отрицание обязательно** (повторение R2 → L8): сначала *<say>no</say>* перед глаголом, потом отрицательное слово после.

## Часть 5: Hay vs estar — типичные ситуации

| Ситуация | Hay (наличие) | Está/están (где конкретно) |
|----------|---------------|------------------------------|
| Спрашиваем, существует ли | *<say>¿Hay una farmacia cerca?</say>* | — |
| Спрашиваем, где конкретное | — | *<say>¿Dónde está la farmacia?</say>* |
| Перечисляем мебель в комнате | *<say>Hay una mesa, dos sillas y un sofá.</say>* | — |
| Говорим, где стоит наш диван | — | *<say>El sofá está al lado de la ventana.</say>* |
| Сколько людей в зале | *<say>Hay mucha gente.</say>* | — |
| Где находится Мария | — | *<say>María está en la cocina.</say>* |
| Погода (R1 → L9) | *<say>Hay sol. Hay viento. Hay niebla.</say>* | — |
| Время / самочувствие | — | *<say>Está nublado. Estoy bien.</say>* |

> **Важно про погоду:** часть выражений с погодой использует *<say>hay</say>* (*<say>hay sol</say>, <say>hay viento</say>, <say>hay nubes</say>, <say>hay niebla</say>, <say>hay tormenta</say>*), часть — *<say>está</say>* (*<say>está nublado</say>, <say>está despejado</say>, <say>está soleado</say>*) — потому что *<say>nublado</say> / <say>soleado</say>* — это прилагательные/причастия, а *<say>sol</say> / <say>viento</say>* — существительные.

## Часть 6: Порядковые числительные 1–10

Употребляются как прилагательные: согласуются по роду и числу. Стоят обычно **перед** существительным.

| № | Мужской род | Женский род |
|---|-------------|-------------|
| 1-й | <say>primero</say> (→ **<say>primer</say>** перед m.sg.) | <say>primera</say> |
| 2-й | <say>segundo</say> | <say>segunda</say> |
| 3-й | <say>tercero</say> (→ **<say>tercer</say>** перед m.sg.) | <say>tercera</say> |
| 4-й | <say>cuarto</say> | <say>cuarta</say> |
| 5-й | <say>quinto</say> | <say>quinta</say> |
| 6-й | <say>sexto</say> | <say>sexta</say> |
| 7-й | <say>séptimo</say> | <say>séptima</say> |
| 8-й | <say>octavo</say> | <say>octava</say> |
| 9-й | <say>noveno</say> | <say>novena</say> |
| 10-й | <say>décimo</say> | <say>décima</say> |

> **Ловушка с усечением:** *<say>primero</say>* и *<say>tercero</say>* теряют *-o* перед существительным мужского рода единственного числа (как *<say>uno</say>*, *<say>alguno</say>*, *<say>bueno</say>*, *<say>malo</say>* — повторение R3 → L6):
> *<say>el primer día</say>* (первый день), но *<say>la primera vez</say>* (первый раз);
> *<say>el tercer piso</say>* (третий этаж), но *<say>la tercera puerta</say>* (третья дверь).

| Пример | Перевод |
|--------|---------|
| *<say>Vivo en el primer piso.</say>* | Я живу на первом этаже. |
| *<say>Es mi tercera clase de español.</say>* | Это мой третий урок испанского. |
| *<say>La quinta puerta a la derecha.</say>* | Пятая дверь направо. |
| *<say>Es la segunda vez.</say>* | Это второй раз. |

> С 11 и далее в речи чаще используют **количественные** числительные: *<say>el siglo XXI</say>* читается *<say>el siglo veintiuno</say>* (двадцать первый век), не *<say>vigésimo primero</say>*. Порядковые от 11+ — книжные.

## Часть 8: Языковая матрица — hay vs está

Прогоняй каждую пару вслух, пока выбор не станет автоматическим.

| Вопрос | Ответ |
|--------|-------|
| <say>¿Hay un sofá en el salón?</say> (ответь: да) | <say>Sí, hay un sofá.</say> |
| <say>¿Dónde está el sofá?</say> (ответь: у окна) | <say>El sofá está al lado de la ventana.</say> |
| <say>¿Hay libros en la estantería?</say> (ответь: да) | <say>Sí, hay muchos libros.</say> |
| <say>¿Dónde están mis libros?</say> (ответь: на полке) | <say>Tus libros están en la estantería.</say> |
| <say>¿Hay alguien en casa?</say> (ответь: нет) | <say>No, no hay nadie.</say> |
| <say>¿Dónde está María?</say> (ответь: на кухне) | <say>María está en la cocina.</say> |
| <say>¿Hay algún problema?</say> (ответь: нет) | <say>No, no hay ningún problema.</say> |
| <say>¿Cuántas sillas hay?</say> (ответь: четыре) | <say>Hay cuatro sillas.</say> |
| <say>¿Hay bastante comida?</say> (ответь: да) | <say>Sí, hay bastante.</say> |
| <say>¿Está abierta la tienda?</say> (ответь: нет) | <say>No, está cerrada.</say> |

### Упражнение 7. Диалог — проговори вслух

> — ¿Hay un hotel cerca de aquí?

> — ¿Cuál es mejor?
> — El segundo es más bonito, pero el primero es más barato.

## Памятка урока

```
HAY vs ESTÁ/ESTÁN — главный контраст:

  HAY            = существует / есть в наличии (НОВОЕ)
                   + un / una / unos / unas / число / mucho / algún / nada
                   ВСЕГДА одна форма: hay (ед. и мн.!)
                   Hay un libro. Hay tres libros. Hay muchos libros.
                   ¿Hay algo? — No, no hay nada.

  ESTÁ / ESTÁN   = находится где-то (КОНКРЕТНОЕ, ИЗВЕСТНОЕ)
                   + el / la / los / las / mi / tu / имя
                   согласуется: está (ед.) / están (мн.)
                   El libro está en la mesa.
                   Los libros están en la mesa.

  Тест: можешь сказать "el / mi / Juan"? → está/están.
        Подставляешь "un / dos / mucho"? → hay.

ЗАПРЕЩЕНО:
  ✗ Hay el libro  → ✓ El libro está aquí.  ИЛИ  ✓ Hay un libro.
  ✗ Han mesas      → ✓ Hay mesas.
  ✗ Hayn mucha gente → ✓ Hay mucha gente.

КВАНТИФИКАТОРЫ:
  Согласуются по роду+числу:
    mucho / mucha / muchos / muchas
    poco / poca / pocos / pocas
    demasiado / demasiada / demasiados / demasiadas
    todo / toda / todos / todas
    otro / otra / otros / otras  (БЕЗ артикля перед otro!)

  Только по числу:
    bastante / bastantes
    suficiente / suficientes

  Усечение перед m.sg. (как uno → un):
    alguno → algún día        alguna idea
    ninguno → ningún problema ninguna duda
    (ningún почти всегда ед.ч.: no hay ningún libro)

ДВОЙНОЕ ОТРИЦАНИЕ (повтор L8):
  No hay nada.
  No hay nadie.
  No hay ningún problema.

ПОРЯДКОВЫЕ 1–10:
  1 primero (→ primer m.sg.)   6 sexto
  2 segundo                    7 séptimo
  3 tercero (→ tercer m.sg.)   8 octavo
  4 cuarto                     9 noveno
  5 quinto                     10 décimo

  el primer día / la primera vez
  el tercer piso / la tercera puerta
  С 11+ в речи — количественные: el siglo veintiuno.

МЕБЕЛЬ (запомни род):
  la mesa, la silla, la cama, la lámpara, la alfombra,
  la cortina, la ventana, la puerta, la pared,
  el sillón, el sofá, el armario, el espejo, el cuadro,
  el suelo, el techo, el piso, el estante, la estantería,
  la habitación.

ПОГОДА (повтор R1 → L9):
  Hay sol / viento / niebla / nubes / tormenta.   ← hay + сущ.
  Está nublado / soleado / despejado.             ← está + прил.
  Hace frío / calor / sol / viento.               ← hace + сущ. (L9)
```

> **Поздравляю — это конец блока A1.** Хребет собран: ser, estar, hay, настоящее правильных глаголов, отрицание, вопросы, артикли, согласование, числа 0–20, дни, месяцы, семья, еда, дом, мебель. Дальше — **Тест «Aprendiz»**, проверь, что всё на месте.
>
> **Урок 11 (старт A2):** глаголы с изменением корневой гласной — *querer, poder, dormir, pedir*. Узнаешь, почему *quiero* пишется с *ie*, а *queremos* — без, и как одной картой накрыть треть всех неправильных глаголов испанского.
