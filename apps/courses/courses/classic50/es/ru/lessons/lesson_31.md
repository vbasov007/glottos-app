## Как работать с этим уроком

1. **Прочитай** правило вывода форм — оно одно и оно простое (3 минуты).
2. **Прогоняй таблицы вслух** — лицо за лицом, глагол за глаголом. Без беглости на формах остальные пять уроков (L32–L36) встанут.
3. **Делай матричные упражнения** — «дай мне subjuntivo от *tener* для *nosotros*» — пока ответ не вылетает за полсекунды.

> Этот урок — **только формы**. Никаких «когда употреблять». Употребления раскладываются по полочкам в L32–L36.
> Хорошая новость: правило вывода одно и работает почти всегда. Плохая новость: исключений 6, и их надо знать наизусть.
> Если ты честно прошёл L12 (irregular yo), 80% работы уже сделано: subjuntivo строится **от yo-формы настоящего времени**.

## Часть 1: Главное правило — «от yo-формы, минус -o, плюс ПРОТИВОПОЛОЖНАЯ гласная»

Возьми глагол в **настоящем времени**, форму **yo**, отбрось **-o** — получишь основу subjuntivo. К ней добавь окончания:

- глаголы на **-AR** → окончания на **-e** (противоположная гласная)
- глаголы на **-ER / -IR** → окончания на **-a** (противоположная гласная)

Вот и весь фокус. Дальше — варианты применения.

| Тип | yo-форма наст. | основа | + окончания |
|-----|----------------|--------|-------------|
| -AR | habl**o** | habl- | -e, -es, -e, -emos, -éis, -en |
| -ER | com**o** | com- | -a, -as, -a, -amos, -áis, -an |
| -IR | viv**o** | viv- | -a, -as, -a, -amos, -áis, -an |

### Полные таблицы — правильные глаголы

**hablar** (говорить):

| Лицо | Форма |
|------|-------|
| yo | hable |
| tú | hables |
| él / ella / usted | hable |
| nosotros | hablemos |
| vosotros | habléis |
| ellos / ustedes | hablen |

**comer** (есть):

| Лицо | Форма |
|------|-------|
| yo | coma |
| tú | comas |
| él / ella / usted | coma |
| nosotros | comamos |
| vosotros | comáis |
| ellos / ustedes | coman |

**vivir** (жить):

| Лицо | Форма |
|------|-------|
| yo | viva |
| tú | vivas |
| él / ella / usted | viva |
| nosotros | vivamos |
| vosotros | viváis |
| ellos / ustedes | vivan |

> **Обрати внимание:** 1-е и 3-е лицо ед. числа **совпадают** — *hable* и *hable*, *coma* и *coma*. Это нормально, как и в imperfecto. Контекст и местоимение снимут двусмысленность.
> **Тильды:** только в форме *vosotros* — *habléis, comáis, viváis*. Не забывай.

## Часть 2: Правило в действии — нерегулярная yo-форма тянется во ВСЕ лица

Если в настоящем времени у тебя «странная» yo-форма (L12: *tengo, hago, conozco, pongo, salgo, digo, oigo, traigo, vengo, veo*), эта странность **попадает в основу subjuntivo и сохраняется везде**.

| Инфинитив | yo (presente) | основа | yo (subj.) | nosotros (subj.) | ellos (subj.) |
|-----------|---------------|--------|------------|------------------|---------------|
| tener | teng**o** | teng- | tenga | tengamos | tengan |
| hacer | hag**o** | hag- | haga | hagamos | hagan |
| poner | pong**o** | pong- | ponga | pongamos | pongan |
| salir | salg**o** | salg- | salga | salgamos | salgan |
| decir | dig**o** | dig- | diga | digamos | digan |
| venir | veng**o** | veng- | venga | vengamos | vengan |
| oír | oig**o** | oig- | oiga | oigamos | oigan |
| traer | traig**o** | traig- | traiga | traigamos | traigan |
| conocer | conozc**o** | conozc- | conozca | conozcamos | conozcan |

> **Главное:** в отличие от настоящего времени, где «нерегулярная» yo стоит особняком, в subjuntivo **все шесть лиц** получают эту основу. *tengo / tienes / tiene*, но *tenga / tengas / tenga / tengamos / tengáis / tengan* — все с *teng-*.

## Часть 3: Глаголы с чередованием в основе (stem-changing)

Здесь правило **усложняется на одну строчку**:

- **-AR / -ER** с чередованием (e→ie, o→ue): чередуют там же, где в настоящем времени — **1/2/3 ед. и 3 мн.**, но **НЕ в nosotros / vosotros**.
- **-IR** с чередованием: чередуют **везде**, включая *nosotros / vosotros*, но в этих двух формах — **по-особому** (e→i вместо e→ie; o→u вместо o→ue).

### -AR / -ER: pensar (e→ie), poder (o→ue)

**pensar** (думать):

| Лицо | Форма |
|------|-------|
| yo | piense |
| tú | pienses |
| él / ella / usted | piense |
| nosotros | **pensemos** |
| vosotros | **penséis** |
| ellos / ustedes | piensen |

**poder** (мочь):

| Лицо | Форма |
|------|-------|
| yo | pueda |
| tú | puedas |
| él / ella / usted | pueda |
| nosotros | **podamos** |
| vosotros | **podáis** |
| ellos / ustedes | puedan |

> **Замечай:** в *nosotros / vosotros* основа возвращается к «спокойному» виду — *pens-, pod-*. Как в настоящем времени.

### -IR: чередование идёт ВЕЗДЕ (включая nosotros / vosotros)

**sentir** (чувствовать, сожалеть, e→ie / e→i):

| Лицо | Форма |
|------|-------|
| yo | sienta |
| tú | sientas |
| él / ella / usted | sienta |
| nosotros | **sintamos** |
| vosotros | **sintáis** |
| ellos / ustedes | sientan |

**pedir** (просить, e→i):

| Лицо | Форма |
|------|-------|
| yo | pida |
| tú | pidas |
| él / ella / usted | pida |
| nosotros | **pidamos** |
| vosotros | **pidáis** |
| ellos / ustedes | pidan |

**dormir** (спать, o→ue / o→u):

| Лицо | Форма |
|------|-------|
| yo | duerma |
| tú | duermas |
| él / ella / usted | duerma |
| nosotros | **durmamos** |
| vosotros | **durmáis** |
| ellos / ustedes | duerman |

> **Запомни картинку:** у -IR-чередующихся в *nosotros / vosotros* гласная **слабая** — *sintamos, pidamos, durmamos*. В остальных лицах — как в настоящем (*siento → sienta*, *pido → pida*, *duermo → duerma*).

## Часть 5: Связь с уроками 29, 30 и 12 — теперь всё встаёт на места

**Урок 12 (нерегулярные yo).** Ты учил *tengo, hago, pongo, salgo, digo, conozco, oigo, traigo* в настоящем времени. Сейчас выясняется: ровно эти основы и строят весь subjuntivo. Если ты их забыл — открой L12 и прогони ещё раз. Без них subjuntivo не построишь.

**Урок 30 (отрицательный императив).** Помнишь *no hables, no comas, no salgas, no tengas, no vayas*? Это были не «специальные формы императива», а **именно subjuntivo**. Так что половину форм этого урока ты уже знаешь.

**Урок 29 (утвердительный императив).** *Tú* в утвердительном — отдельная форма (*habla, come, vive*), а вот *usted, nosotros, ustedes* — снова subjuntivo: *hable usted, comamos, vivan ustedes*. То есть весь императив, кроме *tú affirmative* и *vosotros affirmative*, **берётся прямо из subjuntivo**. Один набор форм работает в трёх местах.

| Конструкция | Из какого набора форм |
|-------------|------------------------|
| Императив *tú* (утв.) | специальная форма (L29) |
| Императив *vosotros* (утв.) | специальная форма (L29) |
| Императив *usted / nosotros / ustedes* (утв.) | **subjuntivo** (L29) |
| Любой императив отрицательный (*no + verbo*) | **subjuntivo** (L30) |
| Сослагательное в придаточных | **subjuntivo** (L32–L36) |

> **Вывод:** ты учишь не «ещё одно наклонение», а **ядро всей продвинутой грамматики**. Один комплект форм — куча применений.

### Упражнение 9. Диалог — проговори вслух

> — Quiero que vengas mañana.
> — Ojalá pueda. Tengo mucho trabajo.
> — Espero que termines pronto.
> — Yo también. Ojalá no sea tarde.
> — Pues que tengas suerte.
> — Gracias. Hasta mañana.

Все глаголы после *que* и *ojalá* — в subjuntivo. Найди их: *vengas, pueda, termines, sea, tengas*.
