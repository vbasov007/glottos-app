# Урок 29: Утвердительный императив. Команды и инструкции
**Словарный запас: глаголы-инструкции, рецепты, дорога, классные команды**

---

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — каждую форму, через все пять «адресатов»: *tú, vosotros, usted, ustedes, nosotros*
3. **Ускоряйся** — гоняй scale и matrix, пока команды не будут вылетать рефлексом

> Императив — это форма, которой ты *действуешь языком*. Не описываешь, не спрашиваешь, не предполагаешь — **командуешь**. И испанский даёт тебе пять разных «командных голосов»: для друга, для группы друзей, для незнакомца, для группы незнакомцев и для «нас вместе». В этом уроке — только **утвердительные** команды («сделай»). Отрицательные («не делай») — в Уроке 30.

---

## Часть 1: Зачем нужен императив

В русском всё просто: «иди» — это и для «ты», и для «вы» через одну форму с -те («идите»). Испанский здесь честнее и подробнее — у него пять отдельных форм:

| Кому говоришь | Форма | Пример (hablar) |
|---------------|-------|-----------------|
| другу (ты) | tú | *¡habla!* — говори! |
| друзьям (вы) | vosotros | *¡hablad!* — говорите! (Испания) |
| незнакомцу (Вы) | usted | *¡hable!* — говорите! (формально) |
| группе незнакомцев / в Лат. Америке всегда | ustedes | *¡hablen!* |
| «давайте мы вместе» | nosotros | *¡hablemos!* — давайте говорить! |

> **Главное:** выбор формы — это **выбор регистра и адресата**. Перепутаешь *tú* и *usted* — это как в русском перейти с «ты» на «Вы» посреди разговора. Слышно сразу.

## Часть 2: Форма tú — самая частая

**Правило простое.** Возьми 3-е лицо единственного числа настоящего времени — и это уже команда для *tú*. Никаких изменений.

| Инфинитив | 3 л. ед. (él habla) | Команда tú |
|-----------|---------------------|------------|
| hablar | habla | *¡habla!* — говори! |
| comer | come | *¡come!* — ешь! |
| vivir | vive | *¡vive!* — живи! |
| escuchar | escucha | *¡escucha!* — слушай! |
| leer | lee | *¡lee!* — читай! |
| abrir | abre | *¡abre!* — открой! |
| escribir | escribe | *¡escribe!* — пиши! |
| cerrar | cierra | *¡cierra!* — закрой! (e→ie сохраняется) |
| volver | vuelve | *¡vuelve!* — вернись! (o→ue сохраняется) |
| pedir | pide | *¡pide!* — попроси! (e→i сохраняется) |

> Изменения корня из настоящего времени **переходят** в императив. Если *vuelve* в настоящем — то и в команде *¡vuelve!*.

## Часть 3: Восемь нерегулярных команд tú — выучи как иконы

Это **самые частотные** глаголы языка. Их короткие командные формы — почти отдельные слова. Выучи их как набор.

| Инфинитив | Команда tú | Перевод |
|-----------|------------|---------|
| venir | **ven** | приди / иди сюда |
| decir | **di** | скажи |
| salir | **sal** | выйди / уходи |
| hacer | **haz** | сделай |
| tener | **ten** | держи / имей |
| ir | **ve** | иди |
| poner | **pon** | положи / поставь |
| ser | **sé** | будь |

> **Мнемоника:** *Ven, di, sal, haz, ten, ve, pon, sé* — проговори вслух как считалку 10 раз. Эти 8 форм покрывают 80% бытовых команд.

**Примеры в живой речи:**

| Команда | Смысл |
|---------|-------|
| *¡Ven aquí!* | Иди сюда! |
| *Dime la verdad.* | Скажи мне правду. |
| *Sal de la cocina.* | Выйди из кухни. |
| *Haz tu tarea.* | Сделай (своё) задание. |
| *Ten cuidado.* | Будь осторожен. (букв. «имей осторожность») |
| *Ve a casa.* | Иди домой. |
| *Pon la mesa.* | Накрой на стол. (букв. «поставь стол») |
| *Sé bueno.* | Будь хорошим. |

> **Ловушка:** *ve* — это команда **двух** глаголов: *ir* («иди») и *ver* («смотри», от 3 л. ед. *ve*). Различай по контексту: *Ve a casa* — «иди домой»; *Ve la película* — «посмотри фильм».

## Часть 4: Форма vosotros — отбрось -r, поставь -d

**Правило одно, без исключений.** Бери инфинитив, убирай -r, ставь -d.

| Инфинитив | Команда vosotros |
|-----------|------------------|
| hablar | *¡hablad!* |
| comer | *¡comed!* |
| vivir | *¡vivid!* |
| venir | *¡venid!* |
| salir | *¡salid!* |
| hacer | *¡haced!* |
| decir | *¡decid!* |
| ir | *¡id!* |

> **Заметь:** все 8 нерегулярных tú в форме *vosotros* становятся **регулярными**! *Haz* для друга — *haced* для друзей. Это правило экономит мозг.

> **Где это используется?** В Испании — каждый день. В Латинской Америке — почти никогда (там вместо *vosotros* используют *ustedes* для любой группы, даже близких друзей). Если едешь в Мадрид — учи. Если в Мехико — узнавай в лицо, но используй *ustedes*.

## Часть 5: Формы usted и ustedes — заимствованы из subjuntivo

Здесь форма — **не** из настоящего времени, а из настоящего сослагательного (presente de subjuntivo). Полностью этот режим — в Уроках 30–31. Сейчас запомни механику: **бери 1 л. ед. настоящего, убирай -o, добавляй обратное окончание**.

- глаголы на -ar → окончание -e (usted), -en (ustedes)
- глаголы на -er / -ir → окончание -a (usted), -an (ustedes)

| Инфинитив | yo (наст.) | usted | ustedes |
|-----------|-----------|-------|---------|
| hablar | hablo | *¡hable!* | *¡hablen!* |
| comer | como | *¡coma!* | *¡coman!* |
| vivir | vivo | *¡viva!* | *¡vivan!* |
| escribir | escribo | *¡escriba!* | *¡escriban!* |
| venir | vengo | *¡venga!* | *¡vengan!* |
| decir | digo | *¡diga!* | *¡digan!* |
| hacer | hago | *¡haga!* | *¡hagan!* |
| tener | tengo | *¡tenga!* | *¡tengan!* |
| poner | pongo | *¡ponga!* | *¡pongan!* |
| salir | salgo | *¡salga!* | *¡salgan!* |

**Полностью особые** (yo-форма заканчивается не на -o):

| Инфинитив | usted | ustedes |
|-----------|-------|---------|
| ir | *¡vaya!* | *¡vayan!* |
| ser | *¡sea!* | *¡sean!* |
| dar | *¡dé!* | *¡den!* |
| estar | *¡esté!* | *¡estén!* |
| saber | *¡sepa!* | *¡sepan!* |

> Эти формы ты увидишь в любой официальной табличке: *No fume* — «Не курите»; *Empuje* — «Толкните»; *Tire* — «Потяните на себя».

## Часть 6: Форма nosotros — «давайте» сделаем это

То же правило: бери 1 л. ед. настоящего, убирай -o, окончание:

- глаголы на -ar → -emos
- глаголы на -er / -ir → -amos

| Инфинитив | Команда nosotros | Перевод |
|-----------|------------------|---------|
| hablar | *¡hablemos!* | давайте говорить! |
| comer | *¡comamos!* | давайте есть! |
| vivir | *¡vivamos!* | давайте жить! |
| escuchar | *¡escuchemos!* | давайте слушать! |
| empezar | *¡empecemos!* | давайте начнём! |
| ir | *¡vamos!* | пойдём! (нерег.) |

> **Особый случай *ir*:** утвердительное «пойдём» — это *¡vamos!*, а не *¡vayamos!*. Это самая частая фраза-команда вообще: *¡Vamos!* — «Идём! / Поехали! / Давай!»

> **Альтернатива:** в живой речи вместо *comamos* часто говорят *vamos a comer* — буквально «пойдём поесть». Это **самый частый разговорный способ** сказать «давайте». Но классическая форма — *comamos* — тоже звучит нормально.

## Часть 7: Местоимения липнут к команде

В **утвердительном** императиве местоимения (прямое, косвенное, возвратное) **присоединяются** к глаголу, в один слог. Это правило знакомо тебе с уроков 15–18 (с инфинитивом и герундием — там тоже можно липнуть). Здесь же — **обязательно**.

| Без местоимения | С местоимением | Перевод |
|-----------------|----------------|---------|
| *¡Da!* (от dar) | *¡Dame!* | дай мне! |
| *¡Llama!* | *¡Llámame!* | позвони мне! |
| *¡Compra!* | *¡Cómpramelo!* | купи мне это! |
| *¡Levanta!* | *¡Levántate!* | встань! (возвратный) |
| *¡Di!* | *¡Dímelo!* | скажи мне это! |
| *¡Pon!* | *¡Ponlo aquí!* | положи это сюда! |
| *¡Habla!* | *¡Háblame!* | поговори со мной! |
| *¡Come!* | *¡Cómelo!* | съешь это! |

### Тильда — обязательное приложение

Когда местоимение прилипает, **ударение остаётся на старом месте** (на корне глагола). Но по правилу окончания слово может попасть в *esdrújula* / *sobreesdrújula* — а это всегда с тильдой.

| Команда | Объяснение |
|---------|------------|
| *llama* (без тильды) → *llámame* (с тильдой) | стало 3 слога, ударение на третьем с конца → *esdrújula* → тильда обязательна |
| *compra* → *cómpramelo* | стало 4 слога, ударение на четвёртом с конца → *sobreesdrújula* → тильда обязательна |
| *di* → *dímelo* | стало 3 слога, ударение третьим с конца → тильда |
| *levanta* → *levántate* | то же самое: 4 слога, ударение третьим с конца → тильда |

> **Правило проще, чем кажется:** **если присоединил местоимение и в результате слов получилось 3 и более слогов — почти всегда нужна тильда** там, где было ударение исходной команды.

### Возвратные глаголы — отдельный случай

С возвратными глаголами **vosotros** теряет финальное -d:

| Инфинитив | Без возвратности | Возвратный (vosotros) |
|-----------|------------------|------------------------|
| levantarse | *¡levantad!* | *¡levantaos!* (не *levantados*!) |
| sentarse | *¡sentad!* | *¡sentaos!* |

С **nosotros** — теряется финальное -s:

| Инфинитив | Без возвратности | Возвратный (nosotros) |
|-----------|------------------|------------------------|
| levantarse | *¡levantemos!* | *¡levantémonos!* |
| irse | *¡vamos!* | *¡vámonos!* — «пошли отсюда!» |

> *¡Vámonos!* — выучи как готовую фразу. Это самое разговорное «пошли!» во всём испанском.

## Часть 8: Scale — прогони императив через все пять форм

Возьми один глагол и проговори его команду **в пяти регистрах подряд**. Это и есть scale-упражнение урока. Гоняй ежедневно.

| ustedes | *¡vengan!* |

## Часть 9: Matrix — «скажи мне, что делать»

Это упражнение «вопрос → команда»: ты задаёшь себе вопрос, и сам отвечаешь командой в нужном регистре. Гоняется парами.

| Вопрос | Команда (tú) | Команда (usted) |
|--------|--------------|------------------|
| ¿Qué hago? | *¡Habla!* | *¡Hable!* |
| ¿Adónde voy? | *¡Ve a casa!* | *¡Vaya a casa!* |
| ¿Qué digo? | *¡Di la verdad!* | *¡Diga la verdad!* |
| ¿Cuándo vengo? | *¡Ven mañana!* | *¡Venga mañana!* |
| ¿Dónde lo pongo? | *¡Ponlo aquí!* | *¡Póngalo aquí!* |
| ¿Qué hago con esto? | *¡Cómelo!* | *¡Cómalo!* |
| ¿Cuándo salgo? | *¡Sal ahora!* | *¡Salga ahora!* |
| ¿Cómo soy? | *¡Sé paciente!* | *¡Sea paciente!* |

> **Зачем matrix:** в реальной речи команды почти всегда приходят в ответ на чей-то вопрос («Куда мне это положить?», «Что мне делать?»). Тренируй именно эту пару — спросил → скомандовал.

### Упражнение 10. Диалог — проговори вслух

> — Mamá, ¿qué hago ahora?
> — Pon la mesa, por favor.
> — Vale. ¿Y después?
> — Después, ven a la cocina y ayúdame.
> — ¿Llamo a mi hermano?
> — Sí, llámalo. Y dile que la cena está casi lista.

Повтори диалог 3 раза. Замени *tú*-команды на *usted*-команды и проговори ещё раз — это будет тот же диалог, но «к незнакомцу».

## Памятка урока

```
УТВЕРДИТЕЛЬНЫЙ ИМПЕРАТИВ — 5 ФОРМ ДЛЯ 5 АДРЕСАТОВ

   tú       — другу
   vosotros — друзьям (Испания)
   usted    — на «Вы» одному
   ustedes  — на «Вы» группе (и: любая группа в Лат. Ам.)
   nosotros — «давайте мы»

ОБРАЗОВАНИЕ:

   tú        = 3 л. ед. настоящего времени
               habla, come, vive

   vosotros  = инфинитив без -r + d
               hablad, comed, vivid

   usted     = 1 л. ед. наст. без -o + противоположное окончание
               (-ar → -e; -er/-ir → -a)
               hable, coma, viva

   ustedes   = usted + n
               hablen, coman, vivan

   nosotros  = -ar → -emos;  -er/-ir → -amos
               hablemos, comamos, vivamos
               (исключение: ir → ¡vamos!)

ВОСЕМЬ НЕРЕГУЛЯРНЫХ tú:

   ven (venir)   di (decir)    sal (salir)   haz (hacer)
   ten (tener)   ve (ir)       pon (poner)   sé (ser)

ПОЛНОСТЬЮ ОСОБЫЕ usted/ustedes:

   ir    → vaya / vayan
   ser   → sea / sean
   dar   → dé / den
   estar → esté / estén
   saber → sepa / sepan

МЕСТОИМЕНИЯ:

   В УТВЕРДИТЕЛЬНОМ императиве — ПРИЛИПАЮТ к концу глагола.
   Часто требуется тильда там, где было ударение исходной команды:

      llama  → llámame
      compra → cómpramelo
      di     → dímelo
      levanta → levántate
      pon    → ponlo (2 слога — без тильды)

   Возвратное vosotros: теряет -d
      levantad → levantaos

   Возвратное nosotros: теряет -s
      vámonos, levantémonos

ОЧЕНЬ ЧАСТЫЕ ФРАЗЫ:

   ¡Ven aquí!         Иди сюда!
   ¡Dime!             Скажи мне!
   ¡Mira!             Смотри!
   ¡Escucha!          Слушай!
   ¡Espera!           Подожди!
   ¡Vamos!            Идём!
   ¡Vámonos!          Пошли (отсюда)!
   ¡Ten cuidado!      Будь осторожен!
   ¡Hazlo!            Сделай это!
   ¡Pon la mesa!      Накрой на стол!
```

> **Следующий шаг:** Урок 30 — **отрицательный** императив («не делай»). Это совсем другая форма (целиком на subjuntivo): *¡no hables!*, *¡no comas!*, *¡no vayas!*. И местоимения там встают **перед** глаголом: *no me lo digas*. Узнаешь самую красивую асимметрию испанской грамматики.
