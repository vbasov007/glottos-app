# Урок 11: Глаголы с изменением корневой гласной в настоящем времени
**Словарный запас: стем-меняющиеся глаголы по типам (querer, pensar, poder, dormir, pedir, servir, empezar, volver)**

---

## Как работать с этим уроком

1. **Прочитай** правило (5 минут) — пойми, ЧТО меняется и КОГДА.
2. **Проговори** все шесть форм вслух — медленно, с ударением на сапог.
3. **Прогоняй гамму** — пока «quiero / queremos» не вылетает само.

> В уроке 7 ты выучил правильный настоящий: *hablo, hablas, habla, hablamos, habláis, hablan*. Теперь — первая лёгкая аномалия. Окончания остаются те же. Меняется только **корневая гласная**, и только под **ударением**.

---

## Часть 1: Главная идея — «сапог» (la bota)

Корневая гласная меняется в **4 формах из 6**: *yo, tú, él, ellos*. В *nosotros* и *vosotros* — **остаётся как в инфинитиве**. Эти четыре формы, если выписать парадигму столбиком, образуют «сапог»:

```
   yo       │ MЕНЯЕТСЯ │  ┐
   tú       │ MЕНЯЕТСЯ │  │  голенище сапога
   él/ella  │ MЕНЯЕТСЯ │  ┘
   nosotros │  как есть │  ← подошва
   vosotros │  как есть │  ← подошва
   ellos    │ MЕНЯЕТСЯ │  ← пятка
```

**Почему так?** В *nosotros / vosotros* ударение падает на **окончание** (*ha-bla-mos*, *ha-bláis*). Корневая гласная безударная — спокойно. В остальных формах ударение **на корне** — гласная «ломается».

> **Правило одной строкой:** ударная корневая гласная → меняется. Безударная → нет.

---

## Часть 2: Тип e → ie

Самая частая группа. Корневая **e** под ударением становится **ie**.

### Querer (хотеть) — образец e→ie

| Лицо | Форма | Корень |
|------|-------|--------|
| yo | **quiero** | qu**ie**r- |
| tú | **quieres** | qu**ie**r- |
| él / ella / usted | **quiere** | qu**ie**r- |
| nosotros | **queremos** | qu**e**r- ← без изменения |
| vosotros | **queréis** | qu**e**r- ← без изменения |
| ellos / ellas / ustedes | **quieren** | qu**ie**r- |

### Другие глаголы e→ie

| Инфинитив | Перевод | yo-форма | nosotros |
|-----------|---------|----------|----------|
| pensar | думать | pienso | pensamos |
| empezar | начинать | empiezo | empezamos |
| cerrar | закрывать | cierro | cerramos |
| entender | понимать | entiendo | entendemos |
| perder | терять | pierdo | perdemos |
| preferir | предпочитать | prefiero | preferimos |
| sentir | чувствовать | siento | sentimos |
| venir | приходить ⚠ | **vengo** / vienes / viene… | venimos |

> **Ловушка:** *venir* и *tener* — комбо: **e→ie + неправильная yo** (*vengo, tengo*). Это будет в уроке 12. Пока запомни: *yo vengo, tú vienes, él viene, nosotros venimos, vosotros venís, ellos vienen.*

---

## Часть 3: Тип o → ue

Корневая **o** под ударением становится **ue**.

### Poder (мочь) — образец o→ue

| Лицо | Форма |
|------|-------|
| yo | **puedo** |
| tú | **puedes** |
| él / ella / usted | **puede** |
| nosotros | **podemos** |
| vosotros | **podéis** |
| ellos / ellas / ustedes | **pueden** |

### Другие глаголы o→ue

| Инфинитив | Перевод | yo-форма | nosotros |
|-----------|---------|----------|----------|
| dormir | спать | duermo | dormimos |
| volver | возвращаться | vuelvo | volvemos |
| encontrar | находить / встречать | encuentro | encontramos |
| contar | считать / рассказывать | cuento | contamos |
| recordar | помнить / напоминать | recuerdo | recordamos |
| costar | стоить | cuesta (обычно 3 л.) | cuestan |
| morir | умирать | muero | morimos |
| soñar (con) | мечтать / видеть во сне | sueño | soñamos |
| almorzar | обедать | almuerzo | almorzamos |

### Исключение: jugar (u → ue)

Единственный глагол, где меняется **u → ue**, а не o.

| Лицо | Форма |
|------|-------|
| yo | **juego** |
| tú | **juegas** |
| él | **juega** |
| nosotros | **jugamos** |
| vosotros | **jugáis** |
| ellos | **juegan** |

> **Запомни:** *jugar al fútbol*, *jugar al tenis* — играть **в** (с предлогом *a* + el → al).

---

## Часть 4: Тип e → i (только -IR глаголы!)

Корневая **e** под ударением становится **i**. Эта группа существует **только среди -IR глаголов**. Если глагол на -AR или -ER — он НИКОГДА не e→i.

### Pedir (просить / заказывать) — образец e→i

| Лицо | Форма |
|------|-------|
| yo | **pido** |
| tú | **pides** |
| él / ella / usted | **pide** |
| nosotros | **pedimos** |
| vosotros | **pedís** |
| ellos / ellas / ustedes | **piden** |

### Другие глаголы e→i

| Инфинитив | Перевод | yo-форма | nosotros |
|-----------|---------|----------|----------|
| servir | подавать / служить | sirvo | servimos |
| repetir | повторять | repito | repetimos |
| seguir | следовать / продолжать ⚠ | **sigo** / sigues / sigue… | seguimos |
| conseguir | добиваться / доставать | **consigo** / consigues… | conseguimos |
| reír | смеяться | río | reímos |
| vestir(se) | одевать(ся) | visto / me visto | vestimos |

> **Ловушка с seguir:** в yo-форме исчезает «u»: *sigo* (а не *siguo*). Это потому что *gu+o* = «гуо», а нужно «го» — пишется без u. Тот же фокус с *conseguir → consigo*.

---

## Часть 5: Pedir vs preguntar — две «спрашивать»

Самая частая ошибка русскоговорящих. По-русски одно слово, по-испански — два разных:

| Глагол | Что значит | Пример |
|--------|------------|--------|
| **preguntar** | задавать вопрос | *Te pregunto: ¿dónde vives?* — Я тебя спрашиваю: где ты живёшь? |
| **pedir** | просить что-то (объект / услугу) | *Te pido un favor.* — Прошу тебя об одолжении. *Pido un café.* — Заказываю кофе. |

> **Запомни:** *preguntar* — про **информацию**, *pedir* — про **вещь** или **действие**.
> *preguntar* — правильный (-AR), не меняется! *Pregunto, preguntas, pregunta…*
> *pedir* — e→i: *pido, pides, pide, pedimos, pedís, piden.*

---

## Часть 6: Как угадать, что глагол стем-меняющийся?

Никак. Это запоминается. Но есть две подсказки:

1. **В словаре** часто помечают: *querer (ie)*, *poder (ue)*, *pedir (i)*.
2. **Семьи слов:** если знаешь *pensar (ie)*, то *empezar* — тоже *(ie)*. Если знаешь *poder*, то *volver, dormir* — тоже *(ue)*. Похожие гласные в корне часто ведут себя одинаково.

> **Главное:** запомни **первое лицо ед. ч.** (yo) — и все остальные «сапожные» формы строятся по тому же корню.

## Языковая гамма

Прогоняй каждый глагол через все 6 лиц вслух. Слушай, где «сапог», а где — нет.

### Гамма 1: querer (e→ie)

> yo **quiero** — tú **quieres** — él **quiere** — nosotros **queremos** — vosotros **queréis** — ellos **quieren**

### Гамма 2: poder (o→ue)

> yo **puedo** — tú **puedes** — él **puede** — nosotros **podemos** — vosotros **podéis** — ellos **pueden**

### Гамма 3: pedir (e→i)

> yo **pido** — tú **pides** — él **pide** — nosotros **pedimos** — vosotros **pedís** — ellos **piden**

### Гамма 4: dormir (o→ue, -IR)

> yo **duermo** — tú **duermes** — él **duerme** — nosotros **dormimos** — vosotros **dormís** — ellos **duermen**

### Гамма 5: empezar (e→ie, -AR)

> yo **empiezo** — tú **empiezas** — él **empieza** — nosotros **empezamos** — vosotros **empezáis** — ellos **empiezan**

### Гамма 6: jugar (u→ue, уникальный)

> yo **juego** — tú **juegas** — él **juega** — nosotros **jugamos** — vosotros **jugáis** — ellos **juegan**

## Языковая матрица

Принцип: ответ заложен в вопросе. Сначала тренируй однотипные обмены, потом меняй лица.

### Матрица 1: querer — что хочешь / хотите

| Вопрос | Ответ |
|--------|-------|
| ¿Qué quieres tomar? | **Quiero** un café. |
| ¿Qué quiere usted? | **Quiero** agua, por favor. |
| ¿Qué queréis comer? | **Queremos** pizza. |
| ¿Qué quieren ellos? (ответь: отдохнуть) | **Quieren** descansar. |

### Матрица 2: poder — можешь / можете

| Вопрос | Ответ |
|--------|-------|
| ¿Puedes venir mañana? | Sí, **puedo**. / No, no **puedo**. |
| ¿Podéis ayudarme? | Sí, **podemos** ayudarte. |
| ¿Puede usted hablar más despacio? | Sí, claro, **puedo**. |
| ¿Pueden ellos esperar? (ответь: да, могут) | Sí, **pueden** esperar diez minutos. |

### Матрица 3: pedir vs preguntar

| Вопрос | Ответ |
|--------|-------|
| ¿Qué pides en el bar? | **Pido** una cerveza. |
| ¿Qué le preguntas al profesor? | Le **pregunto** dónde está la biblioteca. |
| ¿Qué piden ellos en el restaurante? (ответь: счёт) | **Piden** la cuenta. |
| ¿Qué preguntan los turistas? (ответь: как дойти до музея) | **Preguntan** cómo llegar al museo. |

### Матрица 4: dormir / volver — рутина

| Вопрос | Ответ |
|--------|-------|
| ¿A qué hora vuelves a casa? | **Vuelvo** a las siete. |
| ¿Cuántas horas duermes? | **Duermo** ocho horas. |
| ¿A qué hora volvéis del trabajo? | **Volvemos** a las seis. |
| ¿Dónde duermen los niños? (ответь: в спальне) | **Duermen** en su habitación. |

### Упражнение 5. Контраст с правильными глаголами (вспоминаем урок 7)

Один глагол в паре — правильный (-AR/-ER/-IR без изменений), второй — стем-меняющийся. Не перепутай!

1. Yo \_\_\_\_\_ (hablar) español y mi amigo \_\_\_\_\_ (entender) francés.
2. Nosotros \_\_\_\_\_ (estudiar) por la mañana y \_\_\_\_\_ (empezar) las clases a las ocho.
3. Tú \_\_\_\_\_ (comer) en casa, pero yo \_\_\_\_\_ (almorzar) en la oficina.
4. Ella \_\_\_\_\_ (escribir) un libro y \_\_\_\_\_ (pensar) en el final.
5. Los estudiantes \_\_\_\_\_ (leer) el texto y \_\_\_\_\_ (preferir) los cuentos cortos.

<details>
<summary>Ключ</summary>

1. *hablo … entiende* (hablar — правильный; entender — e→ie)
2. *estudiamos … empezamos* (оба — nosotros, но empezar здесь без изменения, потому что подошва!)
3. *comes … almuerzo* (comer — правильный; almorzar — o→ue)
4. *escribe … piensa* (escribir — правильный; pensar — e→ie)
5. *leen … prefieren* (leer — правильный; preferir — e→ie)

</details>

### Упражнение 6. Вспоминаем ser vs estar (урок 9) + новые глаголы

Поставь *ser*, *estar* или нужный стем-меняющийся глагол.

1. Hoy \_\_\_\_\_ (yo, estar) muy cansado, \_\_\_\_\_ (querer) dormir.
2. El café \_\_\_\_\_ (estar) frío, no lo \_\_\_\_\_ (yo, poder) beber.
3. María \_\_\_\_\_ (ser) muy inteligente, siempre \_\_\_\_\_ (entender) todo.
4. ¿Dónde \_\_\_\_\_ (estar) mis llaves? Nunca las \_\_\_\_\_ (yo, encontrar).
5. Tú \_\_\_\_\_ (ser) un buen amigo, siempre me \_\_\_\_\_ (ayudar — правильный!).

<details>
<summary>Ключ</summary>

1. *estoy … quiero* (состояние → estar; хотеть → querer e→ie)
2. *está … puedo* (временное состояние → estar)
3. *es … entiende* (характеристика → ser; entender e→ie)
4. *están … encuentro* (местоположение → estar; encontrar o→ue)
5. *eres … ayudas* (характеристика → ser; ayudar — правильный -AR)

</details>

### Упражнение 7. Мини-диалог — проговори вслух

> — ¡Hola! ¿Qué quieres tomar?
> — **Quiero** un café, por favor. ¿Y tú?
> — Yo **prefiero** un té. ¿**Pedimos** algo de comer?
> — Sí, pero no **puedo** quedarme mucho. **Empiezo** a trabajar a las dos.
> — Vale. **Pido** dos tostadas, ¿te parece?
> — Perfecto. Oye, ¿**recuerdas** dónde está el baño?
> — Creo que **está** al fondo, a la derecha.
> — Gracias. Ahora **vuelvo**.

Повтори трижды. Потом замени роли: ты → usted, или ты → vosotros (двое в кафе).

## Памятка урока

```
СТЕМ-МЕНЯЮЩИЕСЯ ГЛАГОЛЫ (PRESENTE) — ПРАВИЛО САПОГА

         yo       │ MЕНЯЕТСЯ │ ┐
         tú       │ MЕНЯЕТСЯ │ │ ГОЛЕНИЩЕ
         él       │ MЕНЯЕТСЯ │ ┘
         nosotros │  как есть │ ← ПОДОШВА (ударение на окончание)
         vosotros │  как есть │ ← ПОДОШВА
         ellos    │ MЕНЯЕТСЯ │ ← ПЯТКА

ТРИ ТИПА:

e → ie     querer  → quiero, quieres, quiere, queremos, queréis, quieren
           pensar, empezar, cerrar, entender, perder, preferir, sentir

o → ue     poder   → puedo, puedes, puede, podemos, podéis, pueden
           volver, dormir, encontrar, contar, recordar, costar, soñar, almorzar

u → ue     jugar   → juego, juegas, juega, jugamos, jugáis, juegan
           (только этот глагол!)

e → i      pedir   → pido, pides, pide, pedimos, pedís, piden
           servir, repetir, seguir (yo: sigo!), conseguir
           ⚠ ТОЛЬКО -IR ГЛАГОЛЫ!

ОКОНЧАНИЯ — ТЕ ЖЕ, что у правильных глаголов из урока 7.
Меняется ТОЛЬКО корневая гласная.

ЛОВУШКИ:
• preguntar (правильный!) = задавать вопрос
• pedir (e→i)             = просить, заказывать
• seguir → yo sigo (без u, не siguo)
• venir, tener — двойная неправильность (yo vengo / tengo) — урок 12

КЛЮЧЕВЫЕ ФРАЗЫ:
  Quiero un café.            — Хочу кофе.
  No puedo, lo siento.       — Не могу, извини.
  ¿Cuánto cuesta?            — Сколько стоит?
  Pido la cuenta.            — Прошу счёт.
  Vuelvo a casa.             — Возвращаюсь домой.
  No entiendo nada.          — Ничего не понимаю.
  Prefiero el té.            — Предпочитаю чай.
  Juego al fútbol.           — Играю в футбол.
```

> **Следующий шаг:** Урок 12 — Неправильные yo-формы: *tengo, hago, pongo, salgo, conozco, doy, veo, sé*. Один глагол — одна ломаная форма (yo), а остальные пять — нормальные. Плюс комбо: *tener* и *venir* — стем-меняющиеся **И** с неправильной yo.
