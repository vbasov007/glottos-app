# Урок 30: Отрицательный императив. «Не делай!», знаки и запреты
**Словарный запас: запреты, предупреждения, правила безопасности, надписи на табличках**

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — медленно, осознанно, отрабатывая «переключение» окончаний
3. **Ускоряйся** — повторяй пары «делай / не делай», пока переход не будет автоматическим

> В уроке 29 ты научился командовать: *¡habla!*, *¡come!*, *¡ven!*. Это легко. Но как только нужно сказать «**не** говори», «**не** ешь», «**не** приходи» — испанский подменяет форму. И подменяет на ту самую форму, которая называется **сослагательное наклонение** (subjuntivo). Это «контрабанда»: subjuntivo проникает в твою речь уже сейчас, через отрицательные команды, чтобы в уроке 31 он не показался чем-то незнакомым.

## Часть 1: Главное правило отрицательного императива

> **Все отрицательные команды (для всех лиц — *tú*, *usted*, *nosotros*, *vosotros*, *ustedes*) берутся из настоящего сослагательного наклонения.** Утвердительная команда — одна форма. Отрицательная — совсем другая.

Сравни:

| Утвердительно | Отрицательно |
|---------------|--------------|
| ¡Habla! — Говори! | No **hables**. — Не говори. |
| ¡Come! — Ешь! | No **comas**. — Не ешь. |
| ¡Vive aquí! — Живи здесь! | No **vivas** aquí. — Не живи здесь. |
| ¡Ven! — Иди сюда! | No **vengas**. — Не приходи. |
| ¡Sal! — Выйди! | No **salgas**. — Не выходи. |
| ¡Haz! — Сделай! | No **hagas**. — Не делай. |
| ¡Di! — Скажи! | No **digas**. — Не говори. |
| ¡Da! — Дай! | No **des**. — Не давай. |
| ¡Ve! — Иди! | No **vayas**. — Не ходи. |
| ¡Sé bueno! — Будь хорошим! | No **seas** malo. — Не будь плохим. |

> **Главное:** в утвердительной форме *tú*-команда выглядит как 3-е лицо настоящего времени (*habla*, *come*, *vive*). В отрицательной — окончания **меняются местами**: глаголы на -ar получают окончания с **e**, глаголы на -er/-ir получают окончания с **a**. Это и есть subjuntivo.

## Часть 2: Как вывести форму — рецепт «yo-форма → противоположная гласная»

Алгоритм работает для **любого** глагола, кроме горстки исключений.

> **Шаг 1.** Возьми глагол в 1-м лице настоящего времени (yo-форму).
> **Шаг 2.** Отбрось окончание **-o**.
> **Шаг 3.** Прибавь окончание с **противоположной** гласной:
>   – глаголы на **-ar** → окончания на **-e**
>   – глаголы на **-er / -ir** → окончания на **-a**

### Глаголы на -AR (окончания на -e)

| Лицо | hablar (yo hablo → habl-) |
|------|---------------------------|
| tú | no habl**es** |
| usted | no habl**e** |
| nosotros | no habl**emos** |
| vosotros | no habl**éis** |
| ustedes | no habl**en** |

### Глаголы на -ER / -IR (окончания на -a)

| Лицо | comer (yo como → com-) | vivir (yo vivo → viv-) |
|------|------------------------|------------------------|
| tú | no com**as** | no viv**as** |
| usted | no com**a** | no viv**a** |
| nosotros | no com**amos** | no viv**amos** |
| vosotros | no com**áis** | no viv**áis** |
| ustedes | no com**an** | no viv**an** |

> **Рецепт «yo-форма» работает и для неправильных глаголов** — потому что неправильность yo-формы переходит в команду.

### Неправильные глаголы — yo-форма решает всё

| Глагол | yo-форма | no … (tú) | no … (ustedes) |
|--------|----------|-----------|-----------------|
| tener (иметь) | tengo | no **tengas** | no **tengan** |
| venir (приходить) | vengo | no **vengas** | no **vengan** |
| salir (выходить) | salgo | no **salgas** | no **salgan** |
| hacer (делать) | hago | no **hagas** | no **hagan** |
| decir (говорить) | digo | no **digas** | no **digan** |
| poner (класть) | pongo | no **pongas** | no **pongan** |
| traer (приносить) | traigo | no **traigas** | no **traigan** |
| oír (слышать) | oigo | no **oigas** | no **oigan** |
| conocer (быть знакомым) | conozco | no **conozcas** | no **conozcan** |

### Пять «настоящих» исключений — выучи списком

Эти глаголы не идут от yo-формы. Их subjuntivo надо запомнить:

| Глагол | no … (tú) | no … (usted) | no … (nosotros) | no … (vosotros) | no … (ustedes) |
|--------|-----------|--------------|-----------------|------------------|------------------|
| dar (давать) | no **des** | no **dé** | no **demos** | no **deis** | no **den** |
| estar (быть/находиться) | no **estés** | no **esté** | no **estemos** | no **estéis** | no **estén** |
| ir (идти) | no **vayas** | no **vaya** | no **vayamos** | no **vayáis** | no **vayan** |
| saber (знать) | no **sepas** | no **sepa** | no **sepamos** | no **sepáis** | no **sepan** |
| ser (быть) | no **seas** | no **sea** | no **seamos** | no **seáis** | no **sean** |

> **Ловушка:** *ir* в утвердительной форме — *¡vé!*, а в отрицательной — *no vayas*. Полная смена основы. То же с *ser*: *¡sé bueno!* — *no seas malo*.

## Часть 3: Местоимения «переезжают» — главное отличие от утвердительной команды

В уроке 29 ты усвоил: в утвердительной команде местоимения **прилипают сзади** к глаголу: *¡dímelo!*, *¡levántate!*, *¡dénselo!*.

В отрицательной команде всё **переворачивается**: местоимения уходят **вперёд**, ставятся **отдельно** перед глаголом.

| Утвердительно (сзади, слитно) | Отрицательно (спереди, раздельно) |
|-------------------------------|------------------------------------|
| ¡Dímelo! — Скажи мне это! | No **me lo** digas. — Не говори мне это. |
| ¡Levántate! — Вставай! | No **te** levantes. — Не вставай. |
| ¡Cómpralo! — Купи его! | No **lo** compres. — Не покупай его. |
| ¡Dáselo! — Дай ему это! | No **se lo** des. — Не давай ему это. |
| ¡Vete! — Уходи! | No **te** vayas. — Не уходи. |
| ¡Siéntense! — Садитесь! | No **se** sienten. — Не садитесь. |
| ¡Háganlo! — Сделайте это! | No **lo** hagan. — Не делайте это. |

> **Запомни схему:**
> Утвердительно: ГЛАГОЛ + местоимение (одно слово, часто с тильдой) → *cómpralo*
> Отрицательно: **no** + местоимение + ГЛАГОЛ (раздельно, без тильды) → *no lo compres*

### Порядок местоимений — тот же

Если местоимений два, порядок прежний: **se → te/me → lo/la/los/las**.

- No **me lo** digas. — Не говори мне это.
- No **te lo** comas. — Не ешь это (его себе).
- No **se lo** des. — Не давай ему это.
- No **nos lo** pidan. — Не просите у нас это.

### Возвратные глаголы

- *levantarse* → no **te** levantes / no **se** levante / no **nos** levantemos / no **os** levantéis / no **se** levanten
- *preocuparse* → no **te** preocupes / no **se** preocupe — Не волнуйся / Не волнуйтесь.
- *irse* → no **te** vayas / no **se** vaya — Не уходи / Не уходите.
- *quedarse* → no **te** quedes — Не оставайся.
- *acostarse* → no **te** acuestes — Не ложись.

## Часть 4: «Шкала строгости» — от мягкого совета до жёсткого запрета

Одно и то же содержание звучит совсем по-разному в зависимости от формы.

| Уровень | Пример | Когда |
|---------|--------|-------|
| Мягкий совет | *Mejor no fumes aquí.* — Лучше не кури здесь. | дружеский тон |
| Прямая просьба | *No fumes, por favor.* — Не кури, пожалуйста. | вежливая просьба |
| Команда (tú) | *No fumes.* — Не кури. | взрослый ребёнку, друг другу |
| Команда (usted) | *No fume, por favor.* — Не курите, пожалуйста. | к незнакомому |
| Категорично | *¡No fumes aquí!* — Не кури здесь! | повышенный тон |
| Знак / правило | **No fumar.** — Не курить. (инфинитив!) | надписи, инструкции |
| Жёсткий запрет | **Prohibido fumar.** — Курить запрещено. | официальный знак |

> **Заметь:** на знаках и в инструкциях испанский часто использует **инфинитив**: *No fumar*, *No entrar*, *No tocar*. Это «безличная» форма — не обращена ни к кому конкретно. Команды (*no fumes / no fume*) — обращены к человеку.

## Часть 5: «Матрица команд» — родитель и ребёнок

Прокрути в голове сценку: мама с маленьким ребёнком на улице. Каждая фраза — отрицательная команда в *tú*-форме.

| Ситуация | Команда |
|----------|---------|
| Ребёнок тянется к розетке | ¡No **toques** eso! |
| Ребёнок бежит к дороге | ¡No **corras**! |
| Ребёнок кричит в магазине | No **grites**, por favor. |
| Ребёнок хочет конфету перед обедом | No **comas** dulces ahora. |
| Ребёнок открывает окно | No **abras** la ventana. |
| Ребёнок дразнит собаку | No **molestes** al perro. |
| Ребёнок капризничает | No **llores**, no pasa nada. |
| Ребёнок отстаёт | No **te quedes** atrás. |
| Ребёнок убегает | No **te vayas** lejos. |
| Ребёнок боится | No **tengas** miedo. |
| Ребёнок просит купить игрушку | No **me pidas** más juguetes. |
| Ребёнок что-то ломает | No **rompas** eso. |
| Ребёнок мешает чужому разговору | No **interrumpas**. |
| Ребёнок лезет на стол | No **subas** ahí. |
| Ребёнок ругается | No **digas** esas palabras. |

То же — формально, в формате *usted* (от учителя, врача, начальника):

- No **se preocupe**. — Не волнуйтесь.
- No **se mueva**. — Не двигайтесь.
- No **hable** así. — Не говорите так.
- No **se vaya** todavía. — Не уходите пока.
- No **tenga** miedo. — Не бойтесь.
- No **olvide** los documentos. — Не забудьте документы.

```

  1. Возьми yo-форму настоящего времени (yo hablo, yo como, yo tengo)
  2. Отбрось -o → habl-, com-, teng-

       -AR → -e (no hables / no hable / no hablemos / no habléis / no hablen)
```

> **Следующий шаг:** Урок 31 — сослагательное наклонение (subjuntivo) во всей красе. Формы ты уже знаешь — мы их «контрабандой» провели через отрицательные команды.
