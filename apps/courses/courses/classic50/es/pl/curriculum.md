# Program kursu języka hiszpańskiego (50 lekcji)
**Glottos Matrix — strukturalny kurs językowy**

> Zasada: 5% — zrozumieć regułę, 95% — wytrenować aparat mowy. Każdy temat przerabiamy przez matrycę: yo → tú → él/ella → nosotros → vosotros → ellos.

## Jak pracować z każdą lekcją

1. **Teoria** — obejrzyj wideo / przeczytaj regułę (5 minut)
2. **Tłumaczenie** — przetłumacz frazy z polskiego na hiszpański, sprawdź klucz
3. **Gamy językowe** — przegnaj nową regułę przez wszystkie osoby (yo → tú → él → nosotros → vosotros → ellos)
4. **Matryca** — przepracuj pytania-odpowiedzi w trzech scenariuszach: ty (l.poj.) → wy (l.mn.) → osoby trzecie
5. **Zasada lustra** — odpowiedź już siedzi w pytaniu, nie buduj od zera

## Triki na zapamiętywanie

- **Rodzaj**: 80% wynika z końcówki (-o → m, -a → f, -ción → f, -dad → f). Wyjątki trzymaj na mini-liście: *el problema*, *el día*, *el mapa*, *la mano*, *la foto*, *la moto*
- **Akcent + tilde**: agudas (ostatnia sylaba — tilde, jeśli kończy się na samogłoskę/n/s), llanas (przedostatnia — tilde, jeśli NIE na samogłoskę/n/s), esdrújulas (trzecia od końca — ZAWSZE tilde)
- **Ser vs estar**: ser = stałe (tożsamość, pochodzenie, zawód), estar = przejściowe / położenie / rezultat
- **Czasy przeszłe**: indefinido = „i tyle" (jedna akcja, początek lub koniec), imperfecto = „…" (trwało, było tłem, było nawykiem)
- **Subjuntivo**: zapamiętaj **WEIRDO** — Will (wola), Emotion (emocja), Impersonal (bezosobowe), Recommendation (rada), Doubt (wątpliwość), Ojalá (oby)
- **Rolowane *rr* i *ñ***: polskie *r* i *ń* — Heimvorteil. Ale *v* czyta się jak *b* (*vino* = „bino")
