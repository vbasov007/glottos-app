# Text for Lesson 42: *Por* and *para* in everyday life
**Theme: basic por/para situations — gifts, price, time, direction**

---

## Listening text

1. Este regalo es para mi madre.
2. La carta es para ti.
3. Compré flores para mi novia.
4. Pagué diez euros por las flores.
5. Gracias por todo.
6. Por favor, una mesa para dos.
7. Salgo para Madrid mañana.
8. El tren para Barcelona sale a las ocho.
9. Vamos para casa.
10. Pasamos por el parque cada día.
11. Camino por la calle.
12. Estudio español para viajar.
13. Trabajo para vivir.
14. El informe es para el viernes.
15. Necesito la traducción para mañana.
16. Para mí, es demasiado caro.
17. Para mi gusto, es perfecto.
18. Para extranjero, hablas muy bien.
19. Lo hice por amor.
20. Llego tarde por el tráfico.
21. Estudié por tres horas anoche.
22. Por la mañana tomo café.
23. Por la tarde leo un libro.
24. Por la noche veo la tele.
25. Por suerte, hace buen tiempo.
26. Por desgracia, no puedo ir.
27. Por supuesto que sí.
28. Por fin es viernes.
29. Por ahora, estoy bien.
30. Para siempre amigos.
