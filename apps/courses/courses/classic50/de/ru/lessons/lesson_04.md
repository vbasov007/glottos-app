# Урок 4: Простые глаголы (machen, lernen, wohnen, arbeiten, spielen)
**Словарный запас: Профессии**

---

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — медленно, осознанно, каждое окончание
3. **Ускоряйся** — повторяй, пока фразы не будут вылетать сами

> Ты уже знаешь sein и haben. Теперь — обычные глаголы. Хорошая новость: 80% немецких глаголов спрягаются одинаково. Выучи один шаблон — и у тебя 80% всех глаголов в кармане.

---

## Часть 1: Шаблон спряжения

Любой правильный глагол в инфинитиве заканчивается на **-en**: <say>mach**en**</say>, <say>lern**en**</say>, <say>wohn**en**</say>.

Чтобы проспрягать глагол:

1. Убери **-en** → получи основу: <say>**mach-**</say>, <say>**lern-**</say>, <say>**wohn-**</say>
2. Добавь окончание по лицу.

| Местоимение | Окончание | machen (делать) |
|---|---|---|
| <say>**ich**</say> | **-e** | <say>mach**e**</say> |
| <say>**du**</say> | **-st** | <say>mach**st**</say> |
| <say>**er / sie / es**</say> | **-t** | <say>mach**t**</say> |
| <say>**wir**</say> | **-en** | <say>mach**en**</say> |
| <say>**ihr**</say> | **-t** | <say>mach**t**</say> |
| <say>**Sie / sie**</say> (вежл. / они) | **-en** | <say>mach**en**</say> |

Это единственная таблица, которую тебе нужно запомнить. Всё.

---

## Часть 2: Главный хак — три окончания

> **ich = -e, du = -st, er = -t. Остальное = инфинитив.**

Разверни:

- **ich** → **-e** (единственное уникальное)
- **du** → **-st** (всегда слышишь «ст»)
- **er / sie / es** → **-t** (одна буква, третье лицо)
- **wir** → **-en** (= инфинитив)
- **ihr** → **-t** (совпадает с er)
- **Sie / sie** → **-en** (= инфинитив)

Фактически тебе нужно помнить только три окончания: **-e**, **-st**, **-t**. Остальное — инфинитив.

---

## Часть 3: Ловушка — основы на -t, -d, -n

Некоторые глаголы имеют основу, которая заканчивается на **-t**, **-d** или **-n** (после согласной): <say>**arbeit-en**</say>, <say>**find-en**</say>, <say>**öffn-en**</say>.

Попробуй сказать «<say>du arbeitst</say>» — язык сломаешь. Поэтому немцы вставляют **-e-** перед окончаниями **-st** и **-t**.

| Лицо | machen (обычный) | arbeiten (-t в основе) |
|------|-------------------|------------------------|
| <say>ich</say> | <say>mach**e**</say> | <say>arbeit**e**</say> |
| <say>du</say> | <say>mach**st**</say> | <say>arbeit**e**st</say> |
| <say>er/sie/es</say> | <say>mach**t**</say> | <say>arbeit**e**t</say> |
| <say>wir</say> | <say>mach**en**</say> | <say>arbeit**en**</say> |
| <say>ihr</say> | <say>mach**t**</say> | <say>arbeit**e**t</say> |
| <say>sie/Sie</say> | <say>mach**en**</say> | <say>arbeit**en**</say> |

> **Ловушка!** <say>du mach**st**</say> — но <say>du arbeit**e**st</say>. <say>Er mach**t**</say> — но <say>er arbeit**e**t</say>.
> Буква **-e-** спасает от скопления согласных.

То же самое с <say>**finden**</say>: <say>du find**e**st</say>, <say>er find**e**t</say>. И с <say>**öffnen**</say>: <say>du öffn**e**st</say>, <say>er öffn**e**t</say>.

| <say>Musiker</say> / <say>Musikerin</say> | музыкант |
| <say>Friseur</say> / <say>Friseurin</say> | парикмахер |

## Часть 6: Языковые гаммы

### Гамма 1 — <say>wohnen</say>

```
Ich wohne in Berlin
Du wohnst in Berlin
Er wohnt in Berlin
Sie wohnt in Berlin
Wir wohnen in Berlin
Ihr wohnt in Berlin
Sie wohnen in Berlin
```

Повтори 3 раза. На четвёртый — замени «<say>Berlin</say>» на «<say>München</say>», «<say>Hamburg</say>», «<say>Köln</say>».

### Гамма 2 — <say>arbeiten</say> (обрати внимание на -e-)

```
Ich arbeite als Lehrer
Du arbeitest als Lehrer
Er arbeitet als Lehrer
Sie arbeitet als Ärztin
Wir arbeiten als Ingenieure
Ihr arbeitet als Kellner
Sie arbeiten als Polizisten
```

> **Ловушка повторяется:** <say>du arbeit**e**st</say>, <say>er arbeit**e**t</say>, <say>ihr arbeit**e**t</say>. Не пропускай эту **-e-**.

### Гамма 3 — <say>spielen</say> + дополнение

```
Ich spiele jeden Tag Fußball
Du spielst jeden Tag Fußball
Er spielt jeden Tag Fußball
Sie spielt jeden Tag Gitarre
Wir spielen jeden Tag Tennis
Ihr spielt jeden Tag Klavier
Sie spielen jeden Tag Volleyball
```

## Часть 7: Языковая матрица

### Матрица 1 — «Ты ↔ Я»

```
Wo wohnst du? – Ich wohne in München
Was machst du? – Ich lerne Deutsch
Was trinkst du? – Ich trinke Tee
Was spielst du? – Ich spiele Gitarre
Was kochst du heute? – Ich koche Suppe
Wo arbeitest du? – Ich arbeite in einem Büro
```

### Матрица 2 — Третьи лица

```
Was macht er beruflich? – Er arbeitet als Ingenieur
Wo wohnt sie? – Sie wohnt in Frankfurt
Was lernt er? – Er lernt Spanisch
Was kauft sie? – Sie kauft Milch und Brot
Was hört er? – Er hört Musik
Wo arbeiten sie? – Sie arbeiten in einem Krankenhaus
```

### Матрица 3 — Ситуативная (профессия)

```
Was sind Sie von Beruf? – Ich bin Programmierer. Ich arbeite bei Siemens
Was machst du beruflich? – Ich bin Student. Ich lerne Medizin
Wo arbeitet dein Vater? – Er arbeitet als Mechaniker
Was macht deine Mutter? – Sie arbeitet als Krankenschwester
Wo wohnt deine Familie? – Meine Familie wohnt in Berlin
Was kochst du am Wochenende? – Ich koche Pasta und Pizza
```

### Упражнение 3. Контраст — обычная основа vs основа на -t/-d

Выбери правильную форму.

1. du (machst / machest)
2. du (arbeitst / arbeitest)
3. er (lernt / lernet)
4. er (findt / findet)
5. ihr (spielt / spielet)
6. ihr (arbeitt / arbeitet)
7. du (kochst / kochest)
8. er (öffnt / öffnet)

<details>
<summary>Ключ</summary>

1. *machst* (обычная основа — без -e-)
2. *arbeitest* (основа на -t — нужна -e-)
3. *lernt* (обычная основа)
4. *findet* (основа на -d — нужна -e-)
5. *spielt* (обычная основа)
6. *arbeitet* (основа на -t — нужна -e-)
7. *kochst* (обычная основа)
8. *öffnet* (основа на -n после согласной — нужна -e-)

</details>

## Памятка урока

```
ПРАВИЛЬНЫЕ ГЛАГОЛЫ = ОСНОВА + ОКОНЧАНИЕ

Инфинитив = основа + -en (mach + en)

Окончания:
  ich    → -e
  du     → -st
  er/sie → -t
  wir    → -en
  ihr    → -t
  sie/Sie→ -en

Основа на -t / -d / -n (после согл.) → вставляй -e-:
  du arbeit**e**st, er arbeit**e**t, ihr arbeit**e**t
  du find**e**st, er find**e**t
  du öffn**e**st, er öffn**e**t

Глаголы урока:
  machen, lernen, wohnen, arbeiten, spielen,
  trinken, kaufen, hören, kochen, fragen

Профессии (-er → -erin для женского рода):
  Lehrer, Arzt, Ingenieur, Fahrer, Kellner,
  Verkäufer, Koch, Programmierer, Polizist,
  Mechaniker, Student, Musiker, Friseur,
  Krankenschwester

Ключевые фразы:
  Was sind Sie von Beruf?
  Was machst du beruflich?
  Ich bin … von Beruf.
```
