## Часть 1: Кто-то должен показать род

В русском прилагательное всегда показывает род: красн**ый** дом, красн**ая** машина, красн**ое** окно. В немецком — то же самое, но с хитростью: перед существительным часто стоит артикль, и он уже показывает род. Если артикль справился — прилагательное расслабляется. Если нет — берёт работу на себя.

> Род — это табличка, которую кто-то должен нести. Артикль несёт — прилагательное идёт налегке. Артикль слабый или его нет — прилагательное тащит.

---

## Часть 2: Главный хак — «Кто показывает род?»

> **Артикль показал род чётко → прилагательное получает слабое -e / -en.**
> **Артикль НЕ показал род → прилагательное получает сильное окончание (как у артикля).**

Три типа склонения — это просто три ситуации:

1. **После der/die/das** — артикль показывает род. Прилагательное: **-e** или **-en**.
2. **После ein/mein/kein** — у «ein» в Nom муж./ср. нет окончания → прилагательное подхватывает: ein groß**er** Mann, ein klein**es** Kind.
3. **Без артикля** — прилагательное тащит всё. Окончания = окончания артикля.

---

## Часть 3: Тип 1 — после der-слов (слабое склонение)

После der, die, das, dieser, jeder, welcher, alle:

|  | Мужской | Средний | Женский | Множ. число |
|---|---|---|---|---|
| **Nom** | der groß**e** Mann | das klein**e** Kind | die jung**e** Frau | die alt**en** Leute |
| **Akk** | den groß**en** Mann | das klein**e** Kind | die jung**e** Frau | die alt**en** Leute |
| **Dat** | dem groß**en** Mann | dem klein**en** Kind | der jung**en** Frau | den alt**en** Leuten |
| **Gen** | des groß**en** Mannes | des klein**en** Kindes | der jung**en** Frau | der alt**en** Leute |

1. **Nom ед. число** — везде **-e**. Множественное — **-en**.
2. **Всё остальное** — **-en**. Точка.
3. Akk женского и среднего = Nom (тоже **-e**). Только мужской в Akk меняется на **-en**.

> Короче: после der-слов прилагательное знает только два окончания — **-e** и **-en**. Просто и красиво.

---

## Часть 4: Тип 2 — после ein-слов (смешанное склонение)

После ein, eine, mein, dein, sein, ihr, unser, kein:

|  | Мужской | Средний | Женский | Множ. число |
|---|---|---|---|---|
| **Nom** | ein groß**er** Mann | ein klein**es** Kind | eine jung**e** Frau | meine alt**en** Freunde |
| **Akk** | einen groß**en** Mann | ein klein**es** Kind | eine jung**e** Frau | meine alt**en** Freunde |
| **Dat** | einem groß**en** Mann | einem klein**en** Kind | einer jung**en** Frau | meinen alt**en** Freunden |
| **Gen** | eines groß**en** Mannes | eines klein**en** Kindes | einer jung**en** Frau | meiner alt**en** Freunde |

Отличие от Типа 1 — ровно **две клетки**:

1. **Nom муж.** — ein groß**er** Mann. «Ein» без окончания → прилагательное подхватывает: **-er**.
2. **Nom/Akk ср.** — ein klein**es** Kind. «Ein» без окончания → прилагательное: **-es**.

> **Ловушка!** «Ein großer Mann» (Nom) vs. «einen großen Mann» (Akk). В Akk у «einen» уже есть -en → артикль показал род → прилагательное возвращается к слабому -en.

---

## Часть 5: Тип 3 — без артикля (сильное склонение)

Без артикля прилагательное берёт на себя всё. Окончания совпадают с окончаниями определённого артикля:

|  | Мужской | Средний | Женский | Множ. число |
|---|---|---|---|---|
| **Nom** | groß**er** Mann | kalt**es** Bier | frisch**e** Milch | alt**e** Freunde |
| **Akk** | groß**en** Mann | kalt**es** Bier | frisch**e** Milch | alt**e** Freunde |
| **Dat** | groß**em** Mann | kalt**em** Bier | frisch**er** Milch | alt**en** Freunden |
| **Gen** | groß**en** Mannes | kalt**en** Kindes | frisch**er** Milch | alt**er** Freunde |

Без артикля чаще всего: вещества (**kaltes Bier**, **frische Milch**), множественное число (**gute Freunde**), абстрактные понятия (**großer Mut**).

> **Ловушка!** Gen мужского и среднего без артикля: не -es, а **-en** (trotz großen Mutes). Но это редкость — не зацикливайся.

## Часть 7: Языковые гаммы

### Гамма 1 — описание с определённым артиклем (Nom/Akk)

```
Ich sehe den schlanken Mann dort
Du siehst den schlanken Mann dort
Er sieht den schlanken Mann dort
Sie sieht den schlanken Mann dort
Wir sehen den schlanken Mann dort
Ihr seht den schlanken Mann dort
Sie sehen den schlanken Mann dort
```

> Sehen — с изменением гласной: du s**ie**hst, er s**ie**ht. Akkusativ мужского → -en.

### Гамма 2 — описание с неопределённым артиклем (Nom)

```
Ich bin ein geduldiger Mensch
Du bist ein geduldiger Mensch
Er ist ein geduldiger Mensch
Sie ist eine geduldige Frau
Wir sind geduldige Menschen
Ihr seid geduldige Menschen
Sie sind geduldige Menschen
```

> «Ein geduldiger Mensch» (ein без окончания → -er). «Eine geduldige Frau» (eine с -e → прилагательное слабое -e). Мн. число без артикля → сильное: geduldig**e** Menschen.

### Гамма 3 — описание с Dativ

```
Ich helfe meinem ehrgeizigen Kollegen
Du hilfst deinem ehrgeizigen Kollegen
Er hilft seinem ehrgeizigen Kollegen
Sie hilft ihrem ehrgeizigen Kollegen
Wir helfen unserem ehrgeizigen Kollegen
Ihr helft eurem ehrgeizigen Kollegen
Sie helfen ihrem ehrgeizigen Kollegen
```

> Dat муж. + ein-слово: «meinem» показал род (-em) → прилагательное: -en.

### Упражнение 1. Подстановка — вставь правильное окончание прилагательного

Вставь окончание. В скобках дан тип артикля и падеж для подсказки.

1. Der jung___ Mann ist Arzt. (der, Nom)
2. Ich suche eine zuverlässig___ Mitarbeiterin. (eine, Akk)
3. Er trinkt kalt___ Wasser. (без артикля, Akk)
4. Sie hilft dem alt___ Nachbarn. (dem, Dat)
5. Ein ehrgeizig___ Student lernt jeden Tag. (ein, Nom, m)
6. Wir brauchen frisch___ Luft. (без артикля, Akk)
7. Das lockig___ Haar steht ihr gut. (das, Nom)
8. Er arbeitet mit einer geduldig___ Lehrerin. (einer, Dat)

<details>
<summary>Ключ</summary>

1. *Der jung**e** Mann ist Arzt.* (der + Nom → -e)
2. *Ich suche eine zuverlässig**e** Mitarbeiterin.* (eine + Akk жен. = Nom → -e)
3. *Er trinkt kalt**es** Wasser.* (без артикля, Akk ср. род → -es)
4. *Sie hilft dem alt**en** Nachbarn.* (dem + Dat → -en)
5. *Ein ehrgeizig**er** Student lernt jeden Tag.* (ein + Nom муж. → -er, «ein» без окончания)
6. *Wir brauchen frisch**e** Luft.* (без артикля, Akk жен. → -e)
7. *Das lockig**e** Haar steht ihr gut.* (das + Nom → -e)
8. *Er arbeitet mit einer geduldig**en** Lehrerin.* (einer + Dat → -en)

</details>

### Упражнение 3. Контраст — смени артикль, исправь окончание

1. **der** große Mann → **ein** groß___ Mann
2. **eine** junge Frau → **die** jung___ Frau
3. **das** kleine Kind → **ein** klein___ Kind
4. **kalter** Kaffee → **der** kalt___ Kaffee
5. **mein** alter Freund → **dieser** alt___ Freund
6. **die** neugierigen Kinder → neugierig___ Kinder (без артикля)

<details>
<summary>Ключ</summary>

1. *ein groß**er** Mann* (ein без окончания → -er)
2. *die jung**e** Frau* (die → -e)
3. *ein klein**es** Kind* (ein без окончания → -es)
4. *der kalt**e** Kaffee* (der → -e)
5. *dieser alt**e** Freund* (dieser → -e)
6. *neugierig**e** Kinder* (без артикля, Nom мн. → -e)

</details>

## Памятка урока

```
КТО-ТО ДОЛЖЕН ПОКАЗАТЬ РОД

Три типа склонения прилагательных:

Тип 1 (после der/die/das/dieser/jeder):
  Nom ед.ч. → -e    |  Всё остальное → -en

Тип 2 (после ein/mein/kein):
  = Тип 1, НО: Nom муж. → -er, Nom/Akk ср. → -es
  (потому что «ein» без окончания не показывает род)

Тип 3 (без артикля):
  Окончание = окончание определённого артикля
  (der → -er, das → -es, die → -e, dem → -em, den → -en)

ХАК: Артикль показал род → прилагательное -e/-en
     Артикль НЕ показал → прилагательное берёт сильное окончание

Внешность: schlank, mollig, lockig, glatt, blond, dunkelhaarig, blass, muskulös
Характер:  zuverlässig, ehrgeizig, geduldig, stur, großzügig, bescheiden,
           neugierig, selbstbewusst, einfühlsam, launisch
```
