# Урок 42: Passiv. Passiv Perfekt
**Словарный запас: Наука и медицина**

---

## Как работать с этим уроком

1. **Прочитай** правило — пойми логику (5 минут)
2. **Переведи** упражнения письменно — проверь по ключу
3. **Проговори вслух** — медленно, осознанно, анализируя каждую форму
4. **Ускоряйся** — гоняй матрицу, пока не будет вылетать на автомате

> К этому моменту ты уже знаешь Konjunktiv II — умеешь говорить о нереальных вещах. Теперь учимся говорить о действиях, когда неважно **кто** делает, а важно **что** делается. Это пассив — рабочая лошадка научных текстов и новостей.

---

## Часть 1: Что такое Passiv и зачем он нужен

В активе фокус на том, **кто** делает: Der Arzt operiert den Patienten.

В пассиве фокус на том, **что** делается: Der Patient **wird** operiert.

По-русски то же самое: «Врач оперирует пациента» vs. «Пациент **оперируется**». Лекарство **было разработано**. Эксперимент **проводится**. Вот это и есть пассив.

---

## Часть 2: Главный хак — три кита пассива

> **werden = помощник пассива. Partizip II = в конце. Кто делает = von + Dativ.**

Запомни эту формулу — и ты построишь любое пассивное предложение. Всё остальное — детали.

Ты уже знаешь werden как «становиться» и как помощник будущего времени. Не путай: werden + Infinitiv = будущее. werden + Partizip II = пассив.

---

## Часть 3: Passiv Präsens — формула и деятель

**Формула:** Subjekt + **werden** (спрягается) + ... + **Partizip II** (в конце)

| Лицо | werden |
|------|--------|
| ich | **werde** |
| du | **wirst** |
| er/sie/es | **wird** |
| wir | **werden** |
| ihr | **werdet** |
| sie/Sie | **werden** |

| Aktiv | Passiv |
|-------|--------|
| Der Forscher untersucht das Problem | Das Problem **wird** untersucht |
| Die Ärztin verschreibt die Therapie | Die Therapie **wird** verschrieben |
| Man testet das Medikament | Das Medikament **wird** getestet |

1. **Akkusativ-объект актива → Nominativ-подлежащее пассива.**
2. **Подлежащее актива исчезает** — или уходит в **von + Dativ**: Das Experiment **wird** von **dem Wissenschaftler** durchgeführt.
3. **Partizip II всегда в конце.** Без исключений.

> **Ловушка!** von + Dativ = **кто** делает. durch + Akkusativ = **чем/посредством чего**.
> Die Krankheit wird von **dem Arzt** behandelt. (Кем? Врачом.)
> Die Krankheit wird durch **eine neue Therapie** geheilt. (Чем? Новой терапией.)

---

## Часть 4: Passiv Perfekt — формула

**Формула:** Subjekt + **sein** (спрягается) + ... + **Partizip II** + **worden**

Не «geworden», а именно **worden** — без «ge-». Это специальная форма только для Passiv Perfekt.

| Aktiv Perfekt | Passiv Perfekt |
|---------------|----------------|
| Der Forscher **hat** das Gen **entdeckt** | Das Gen **ist** entdeckt **worden** |
| Die Firma **hat** das Medikament **entwickelt** | Das Medikament **ist** entwickelt **worden** |
| Man **hat** den Patienten **operiert** | Der Patient **ist** operiert **worden** |

> **Якорь:** sein + Partizip II + **worden** (без ge-!). sein спрягается, остальное — в конец.

| Лицо | Passiv Perfekt |
|------|----------------|
| ich / du / er | bin / bist / ist ... operiert **worden** |
| wir / ihr / sie | sind / seid / sind ... operiert **worden** |

---

## Часть 5: Aktiv → Passiv за 5 шагов

1. Найди **Akkusativ-объект** в активном предложении.
2. Сделай его **Nominativ** (подлежащее пассива).
3. Поставь **werden** (Präsens) или **sein ... worden** (Perfekt).
4. Глагол → **Partizip II** → в конец.
5. Деятель → **von + Dativ** (если нужен).

**Aktiv:** Die Wissenschaftlerin entwickelt **einen Impfstoff**.
**Passiv:** **Ein Impfstoff** wird von der Wissenschaftlerin entwickelt.

> **Ловушка!** Только переходные глаголы (с Akkusativ-объектом) образуют личный пассив. *~~Der Patient wird geholfen~~* — нельзя! helfen требует Dativ → **Dem Patienten wird geholfen** (безличный пассив).

## Часть 7: Языковые гаммы

### Гамма 1 — Passiv Präsens (werden + Partizip II)

Прогони вслух. Следи за формой werden:

```
Ich werde von dem Arzt untersucht
Du wirst von dem Arzt untersucht
Er wird von dem Arzt untersucht
Sie wird von dem Arzt untersucht
Wir werden von dem Arzt untersucht
Ihr werdet von dem Arzt untersucht
Sie werden von dem Arzt untersucht
```

> Все лица — разные формы werden. Partizip II «untersucht» не меняется. Никогда.

### Гамма 2 — Passiv Perfekt (sein + Partizip II + worden)

```
Ich bin gestern operiert worden
Du bist gestern operiert worden
Er ist gestern operiert worden
Sie ist gestern operiert worden
Wir sind gestern operiert worden
Ihr seid gestern operiert worden
Sie sind gestern operiert worden
```

> Обрати внимание: sein спрягается как обычно. «operiert worden» — неизменяемый блок в конце.

### Гамма 3 — Трансформация актив → пассив (наука)

Читай пары вслух — сначала актив, потом пассив:

```
Der Forscher entdeckt ein neues Gen – Ein neues Gen wird entdeckt
Die Firma entwickelt einen Impfstoff – Ein Impfstoff wird entwickelt
Die Ärztin behandelt den Patienten – Der Patient wird behandelt
Das Team führt das Experiment durch – Das Experiment wird durchgeführt
Man testet das Medikament im Labor – Das Medikament wird im Labor getestet
```

> durchführen — приставка отделяемая. В пассиве возвращается к Partizip II: durch**geführt**.

## Часть 8: Языковая матрица

### Матрица 1 — «Ты ↔ Я» (Passiv Präsens)

```
Wirst du oft von deinem Arzt untersucht? – Ja, ich werde regelmäßig untersucht
Wirst du von der Professorin betreut? – Ja, ich werde von ihr betreut
Wirst du gut behandelt? – Ja, ich werde sehr gut behandelt
Wirst du oft zu Experimenten eingeladen? – Nein, ich werde selten eingeladen
Wirst du über die Ergebnisse informiert? – Ja, ich werde immer sofort informiert
```

### Матрица 2 — «Третьи лица» (Passiv Perfekt)

```
Ist der Patient schon operiert worden? – Ja, er ist gestern operiert worden
Ist das Medikament getestet worden? – Ja, es ist erfolgreich getestet worden
Sind die Ergebnisse veröffentlicht worden? – Nein, sie sind noch nicht veröffentlicht worden
Ist das neue Gen entdeckt worden? – Ja, es ist von einem deutschen Forscher entdeckt worden
Ist der Impfstoff schon entwickelt worden? – Ja, er ist letztes Jahr entwickelt worden
```

### Матрица 3 — Наука и медицина (смешанный пассив)

```
Wie wird diese Krankheit behandelt? – Sie wird mit einer neuen Therapie behandelt
Wo wird das Experiment durchgeführt? – Es wird im Labor der Universität durchgeführt
Wann ist der Impfstoff entwickelt worden? – Er ist vor zwei Jahren entwickelt worden
Von wem ist das Gen entdeckt worden? – Es ist von einem internationalen Team entdeckt worden
Werden die Patienten regelmäßig untersucht? – Ja, sie werden alle drei Monate untersucht
```

## Часть 9: Упражнения

### Упражнение 1. Определи — Aktiv или Passiv?

Напиши A (актив) или P (пассив) рядом с каждым предложением:

1. Der Forscher untersucht die Probe.
2. Die Probe wird untersucht.
3. Das Medikament ist entwickelt worden.
4. Die Ärztin hat den Patienten behandelt.
5. Der Patient wird von der Ärztin behandelt.
6. Man führt ein Experiment durch.

<details>
<summary>Ключ</summary>

1. *A* (подлежащее — der Forscher — сам действует)
2. *P* (wird + Partizip II)
3. *P* (ist + Partizip II + worden)
4. *A* (hat + Partizip II = Perfekt Aktiv)
5. *P* (wird + von + Partizip II)
6. *A* (man + глагол = актив)

</details>

### Упражнение 2. Трансформация: Aktiv → Passiv Präsens

Перестрой предложения из актива в пассив. Деятеля убери.

1. Der Wissenschaftler entdeckt ein neues Virus.
2. Die Ärztin verschreibt eine Therapie.
3. Man testet das Medikament an Freiwilligen.
4. Der Forscher veröffentlicht die Studie.
5. Die Firma entwickelt einen neuen Impfstoff.
6. Das Team führt drei Experimente durch.

<details>
<summary>Ключ</summary>

1. *Ein neues Virus wird entdeckt.*
2. *Eine Therapie wird verschrieben.*
3. *Das Medikament wird an Freiwilligen getestet.*
4. *Die Studie wird veröffentlicht.*
5. *Ein neuer Impfstoff wird entwickelt.*
6. *Drei Experimente werden durchgeführt.*

</details>

### Упражнение 3. Трансформация: Aktiv → Passiv Perfekt

Перестрой в Passiv Perfekt. Деятеля укажи через von + Dativ.

1. Der Wissenschaftler hat ein Gen entdeckt.
2. Die Forscherin hat das Experiment durchgeführt.
3. Die Firma hat das Medikament entwickelt.
4. Der Arzt hat den Patienten operiert.
5. Das Team hat die Ergebnisse veröffentlicht.
6. Die Ärztin hat die Wirkung bewiesen.

<details>
<summary>Ключ</summary>

1. *Ein Gen ist von dem Wissenschaftler entdeckt worden.*
2. *Das Experiment ist von der Forscherin durchgeführt worden.*
3. *Das Medikament ist von der Firma entwickelt worden.*
4. *Der Patient ist von dem Arzt operiert worden.*
5. *Die Ergebnisse sind von dem Team veröffentlicht worden.*
6. *Die Wirkung ist von der Ärztin bewiesen worden.*

</details>

## Памятка урока

```
PASSIV PRÄSENS = werden + Partizip II
PASSIV PERFEKT = sein + Partizip II + worden (без ge-!)
werden: werde / wirst / wird / werden / werdet / werden
Кто делает? → von + Dativ | Чем? → durch + Akkusativ
Aktiv → Passiv: Akk-объект → Nominativ, глагол → Partizip II в конец
Только переходные глаголы (с Akk) → личный Passiv!

Словарь: Forschung (f), Therapie (f), Experiment (n), Medikament (n),
Labor (n), Impfstoff (m), entdecken, entwickeln, untersuchen, behandeln
```
