## Часть 1: Partizip II неправильных глаголов — другая формула

У правильных: **ge- + основа + -t** (gemacht, gekauft). У неправильных:

> **ge- + изменённая основа + -en**

schreib**en** → ge**schrieb**en, sprech**en** → ge**sproch**en, trink**en** → ge**trunk**en.

Корневая гласная **прыгает**. Окончание не -t, а **-en**. Два признака — и ты сразу узнаешь неправильный Partizip II.

Гласные меняются по группам. Запомни группу — знаешь десяток глаголов:

| Группа | Infinitiv → Partizip II | Ещё примеры |
|--------|------------------------|-------------|
| **ei → ie → ie** | schreiben → geschrieben | bleiben → geblieben, treiben → getrieben |
| **i → a → u** | trinken → getrunken | finden → gefunden, singen → gesungen |
| **e → a → o** | sprechen → gesprochen | nehmen → genommen, helfen → geholfen |
| **a → u → a** | fahren → gefahren | tragen → getragen, schlagen → geschlagen |

> **Якорь:** Четыре цепочки — 80% неправильных глаголов. Остальные учи отдельно: gehen → gegangen, stehen → gestanden, tun → getan.

---

## Часть 2: Когда ge- исчезает

### Правило 1: Неотделяемые приставки

Приставки **ver-, be-, er-, ent-, emp-, zer-, ge-, miss-** намертво приклеены к глаголу. Они не пускают ge- внутрь.

| Infinitiv | Partizip II | Перевод |
|-----------|-------------|---------|
| verstehen | verstanden | понимать |
| bekommen | bekommen | получать |
| erzählen | erzählt | рассказывать |
| empfehlen | empfohlen | рекомендовать |
| vergessen | vergessen | забывать |

> **Якорь:** Если глагол начинается с ver-, be-, er-, ent-, emp-, zer-, ge-, miss- — забудь про ge-. Его просто нет.

### Правило 2: Глаголы на -ieren

Глаголы на **-ieren** — заимствования. Без ge-, окончание **-t**:

| Infinitiv | Partizip II | Перевод |
|-----------|-------------|---------|
| studieren | studiert | учиться (в вузе) |
| telefonieren | telefoniert | звонить |
| trainieren | trainiert | тренироваться |

> **Ловушка!** trainieren → trainier**t** (не ge-trainier-t!). Приставки ge- нет, окончание -t. Два отличия от обычных глаголов.

### Правило 3: Отделяемые приставки — ge- внутри

Отделяемые приставки (auf-, an-, ein-, mit-...) — ge- **встаёт между** приставкой и основой:

| Infinitiv | Partizip II | Перевод |
|-----------|-------------|---------|
| aufstehen | auf**ge**standen | вставать |
| einkaufen | ein**ge**kauft | делать покупки |
| anfangen | an**ge**fangen | начинать |
| mitnehmen | mit**ge**nommen | брать с собой |

> **Якорь:** auf + ge + standen. Приставка ge- — как начинка в бутерброде.

## Часть 3: haben или sein?

Большинство глаголов строят Perfekt с **haben**. Группа глаголов требует **sein**:

1. **Перемещение А → Б:** gehen, fahren, fliegen, laufen, schwimmen, kommen, reisen
2. **Изменение состояния:** aufstehen, einschlafen, aufwachen, sterben, werden
3. **Исключения:** **sein** (Ich bin gewesen) и **bleiben** (Ich bin geblieben)

Всё остальное — **haben**.

> **Ловушка!** schwimmen с направлением — всегда sein. Без направления — в разговорном немцы тоже берут sein. Не парься — бери sein для глаголов движения.

## Часть 5: Языковые гаммы

### Гамма 1 — Perfekt с haben (тренировка неправильных глаголов)

```
Ich habe viel Wasser getrunken
Du hast viel Wasser getrunken
Er hat viel Wasser getrunken
Sie hat viel Wasser getrunken
Wir haben viel Wasser getrunken
Ihr habt viel Wasser getrunken
Sie haben viel Wasser getrunken
```

> trinken → getrunken (группа i-a-u). haben — нет перемещения.

### Гамма 2 — Perfekt с sein (перемещение)

```
Ich bin gestern fünf Kilometer gelaufen
Du bist gestern fünf Kilometer gelaufen
Er ist gestern fünf Kilometer gelaufen
Sie ist gestern fünf Kilometer gelaufen
Wir sind gestern fünf Kilometer gelaufen
Ihr seid gestern fünf Kilometer gelaufen
Sie sind gestern fünf Kilometer gelaufen
```

> laufen → gelaufen (перемещение → sein). Запомни: ihr seid!

### Гамма 3 — Perfekt с отделяемой приставкой (aufstehen)

```
Ich bin um sechs aufgestanden
Du bist um sechs aufgestanden
Er ist um sechs aufgestanden
Sie ist um sechs aufgestanden
Wir sind um sechs aufgestanden
Ihr seid um sechs aufgestanden
Sie sind um sechs aufgestanden
```

> auf-ge-standen: ge- между приставкой и основой. sein — изменение состояния.

## Памятка урока

```
НЕПРАВИЛЬНЫЙ PARTIZIP II = ge- + изменённая основа + -en

Группы: ei-ie-ie | i-a-u | e-a-o | a-u-a

БЕЗ ge-:  ver/be/er/ent/emp/zer/ge/miss → verstanden
          -ieren → trainiert
ge- ВНУТРИ: отделяемые приставки → auf-ge-standen

SEIN: перемещение (gehen, fahren, laufen, schwimmen)
      + изменение состояния (aufstehen, einschlafen)
      + sein/bleiben
Всё остальное → HABEN
```
