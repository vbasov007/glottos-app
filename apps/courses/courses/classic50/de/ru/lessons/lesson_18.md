## Часть 1: Предлоги, которые ВСЕГДА требуют Akkusativ

Пять предлогов. После них — всегда Akkusativ. Без исключений.

> **DuFüGeOhUm** — durch, für, gegen, ohne, um. Произнеси три раза. Это твоя мантра.

| Предлог | Перевод | Пример |
|---------|---------|--------|
| **durch** | через, сквозь | Wir gehen **durch den Park** |
| **für** | для, за | Das Geschenk ist **für meinen Bruder** |
| **gegen** | против | Sie spielt **gegen ihren Kollegen** |
| **ohne** | без | Ich trinke Kaffee **ohne den Zucker** |
| **um** | вокруг, в (время) | Wir laufen **um den See** |

> **Ловушка!** «Ohne» — без артикля в большинстве случаев: *ohne Zucker*, *ohne Problem*. Но если есть притяжательное — артикль нужен: *ohne meinen Schlüssel*.

---

## Часть 2: Предлоги, которые ВСЕГДА требуют Dativ

Семь предлогов. После них — только Dativ.

> **AusBeiMitNachSeitVonZu** — повтори как считалку. Это твой второй якорь.

| Предлог | Перевод | Пример |
|---------|---------|--------|
| **aus** | из | Er kommt **aus der Türkei** |
| **bei** | у, при, во время | Ich wohne **bei meinem Onkel** |
| **mit** | с | Sie fährt **mit dem Zug** |
| **nach** | после, в (страну/город) | **Nach dem Frühstück** lese ich die Zeitung |
| **seit** | с (момента времени) | Ich lerne Deutsch **seit einem Jahr** |
| **von** | от, из | Das ist ein Brief **von meiner Mutter** |
| **zu** | к | Ich gehe **zum Arzt** (zu + dem = zum) |

Запомни:

1. **«Mit» = всегда Dativ.** Mit dem Bus, mit der Bahn, mit einem Freund.
2. **«Zu» сливается**: zu + dem = **zum**, zu + der = **zur**.
3. **«Von» сливается**: von + dem = **vom**. Das Café ist nicht weit **vom** Hotel.

## Часть 3: Сводная таблица всех артиклей — Nom / Akk / Dat

Одна таблица, которая решает всё. Выучи наизусть.

### Определённые артикли

|  | Мужской (der) | Средний (das) | Женский (die) | Множ. число (die) |
|---|---|---|---|---|
| **Nominativ** | der | das | die | die |
| **Akkusativ** | de**n** | das | die | die |
| **Dativ** | de**m** | de**m** | der | den + **(-n)** |

### Неопределённые артикли

|  | Мужской | Средний | Женский |
|---|---|---|---|
| **Nominativ** | ein | ein | eine |
| **Akkusativ** | eine**n** | ein | eine |
| **Dativ** | eine**m** | eine**m** | einer |

1. **Akkusativ меняет только мужской:** der → de**n**, ein → eine**n**. Средний и женский — без изменений.
2. **Dativ: мужской и средний одинаковые.** Оба на **-m**: dem, einem.
3. **Женский в Dativ** — der / einer. Как мужской Nominativ — не путай, контекст подскажет.

## Часть 4: Дерево решений — как выбрать артикль за 3 секунды

Каждый раз задай себе три вопроса:

```
1. Какой род? → der / das / die
2. Какой падеж?
   Кто? → NOM | Кого/Что? → AKK | Кому? → DAT
   После durch/für/gegen/ohne/um → AKK
   После aus/bei/mit/nach/seit/von/zu → DAT
3. Определённый или неопределённый? → смотри таблицу
```

> **Ловушка!** Wechselpräpositionen (in, an, auf, über, unter, vor, hinter, neben, zwischen) — это особая группа. Они требуют Akkusativ при движении (wohin?) и Dativ при местонахождении (wo?). Не путай их с «всегда-Akkusativ» и «всегда-Dativ» предлогами.

## Часть 6: Языковые гаммы

### Гамма 1 — für + Akkusativ (забота о ком-то)

```
Ich kaufe ein Geschenk für meinen Vater
Du kaufst ein Geschenk für deinen Vater
Er kauft ein Geschenk für seinen Vater
Sie kauft ein Geschenk für ihren Vater
Wir kaufen ein Geschenk für unseren Vater
Ihr kauft ein Geschenk für euren Vater
Sie kaufen ein Geschenk für ihren Vater
```

> Обрати внимание: «für» всегда тянет Akkusativ. Мужской род: mein → mein**en**, dein → dein**en**.

### Гамма 2 — mit + Dativ (с кем / с чем)

```
Ich fahre mit dem Zug zur Arbeit
Du fährst mit dem Zug zur Arbeit
Er fährt mit dem Zug zur Arbeit
Sie fährt mit dem Zug zur Arbeit
Wir fahren mit dem Zug zur Arbeit
Ihr fahrt mit dem Zug zur Arbeit
Sie fahren mit dem Zug zur Arbeit
```

> Глагол fahren — с изменением гласной: du f**ä**hrst, er f**ä**hrt. И «mit» + «zu» — оба Dativ в одном предложении.

### Гамма 3 — geben + Dativ + Akkusativ (два объекта)

```
Ich gebe dem Nachbarn das Paket
Du gibst dem Nachbarn das Paket
Er gibt dem Nachbarn das Paket
Sie gibt dem Nachbarn das Paket
Wir geben dem Nachbarn das Paket
Ihr gebt dem Nachbarn das Paket
Sie geben dem Nachbarn das Paket
```

> Порядок: КОМУ (Dativ) + ЧТО (Akkusativ). Глагол geben: du g**i**bst, er g**i**bt.

## Часть 8: Упражнения

### Упражнение 1. Определи предлог и падеж — впиши артикль

Вставь правильный артикль. В скобках укажи падеж (N), (A) или (D).

1. Er geht durch ___ Stadt ( ).
2. Sie kommt aus ___ Schule ( ).
3. Wir fahren mit ___ Bus ( ).
4. Das ist ein Geschenk für ___ Tochter ( ).
5. Ich wohne seit ___ Monat hier ( ).
6. Er läuft ohne ___ Jacke ( ) durch ___ Regen ( ).
7. Sie geht zu ___ Bäckerei ( ).
8. Der Brief ist von ___ Arzt ( ).

<details>
<summary>Ключ</summary>

1. *Er geht durch **die** Stadt (A).* (durch → всегда Akk, die Stadt → die в Akk не меняется)
2. *Sie kommt aus **der** Schule (D).* (aus → всегда Dat, die → der)
3. *Wir fahren mit **dem** Bus (D).* (mit → всегда Dat, der Bus → dem)
4. *Das ist ein Geschenk für **die/meine** Tochter (A).* (für → всегда Akk, die → die)
5. *Ich wohne seit **einem** Monat hier (D).* (seit → всегда Dat, der Monat → einem)
6. *Er läuft ohne **die/eine** Jacke (A) durch **den** Regen (A).* (ohne и durch → оба Akk)
7. *Sie geht **zur** Bäckerei (D).* (zu → всегда Dat, zu + der = zur)
8. *Der Brief ist **vom** Arzt (D).* (von → всегда Dat, von + dem = vom)

</details>

### Упражнение 3. Контраст Akkusativ / Dativ — выбери правильный вариант

В каждой паре одно предложение с Akkusativ, другое — с Dativ. Переведи оба.

1. Я покупаю цветы **для** соседки. — Я дарю цветы **соседке**.
2. Он бежит **через** мост. — Он стоит **на** мосту.
3. Она едет **без** мужа. — Она едет **с** мужем.
4. Посылка **от** моего брата. — Посылка **для** моего брата.

<details>
<summary>Ключ</summary>

1. *Ich kaufe Blumen **für die** Nachbarin (A). — Ich schenke **der** Nachbarin Blumen (D).*
2. *Er läuft **durch die** Brücke... → Er läuft **über die** Brücke (A, движение). — Er steht **auf der** Brücke (D, место).* (Wechselpräposition!)
3. *Sie fährt **ohne** ihren Mann (A). — Sie fährt **mit** ihrem Mann (D).*
4. *Das Paket ist **von** meinem Bruder (D). — Das Paket ist **für** meinen Bruder (A).*

</details>

## Памятка урока

```
AKKUSATIV-ПРЕДЛОГИ: durch, für, gegen, ohne, um → DuFüGeOhUm
DATIV-ПРЕДЛОГИ: aus, bei, mit, nach, seit, von, zu → AusBeiMitNachSeitVonZu

СВОДНАЯ ТАБЛИЦА:
             Mask (der)    Neutr (das)   Fem (die)    Plural (die)
NOM          der / ein     das / ein     die / eine   die
AKK          den / einen   das / ein     die / eine   die
DAT          dem / einem   dem / einem   der / einer  den + (-n)

АЛГОРИТМ:  Вопрос → Падеж → Род → Артикль

Слияния: zu + dem = zum | zu + der = zur | von + dem = vom | bei + dem = beim

Порядок слов: КТО + глагол + КОМУ (D) + ЧТО (A)
```
