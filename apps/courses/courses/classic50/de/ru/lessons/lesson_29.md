## Часть 1: Зачем немцам два прошедших времени

В немецком два прошедших. У каждого своя территория:

- **Perfekt** — разговорная речь. Болтаешь с другом — Perfekt.
- **Präteritum** — письменная речь. Газеты, романы, сказки, доклады.

> Открыл газету — там Präteritum. Открыл рот — там Perfekt.

Исключение: **sein** (war), **haben** (hatte) и **модальные** (konnte, musste, wollte) используют Präteritum даже в речи. «Ich habe gehabt» звучит нелепо даже для немцев.

---

## Часть 2: Главный хак — два типа глаголов, два паттерна

Запомни формулу:

> **Правильные = основа + TE. Неправильные = новая основа, без TE.**

machen → mach**te** (суффикс -te = правильный). gehen → **ging** (основа изменилась, -te нет = неправильный). Видишь -te — правильный. Нет -te — неправильный. Два лагеря, без исключений.

---

## Часть 3: Спряжение правильных глаголов

Берёшь основу, приклеиваешь **-te**, добавляешь личное окончание:

| Лицо | machen | arbeiten |
|------|--------|----------|
| ich | mach**te** | arbeit**ete** |
| du | mach**test** | arbeit**etest** |
| er/sie/es | mach**te** | arbeit**ete** |
| wir | mach**ten** | arbeit**eten** |
| ihr | mach**tet** | arbeit**etet** |
| sie/Sie | mach**ten** | arbeit**eten** |

Три вещи:

1. **Ich = er/sie/es** — оба без окончания после -te. Два по цене одного.
2. **Wir = sie/Sie** — оба -ten.
3. Основа на **-t, -d** (arbeiten, reden) — буфер **-e-**: arbeit**ete**, red**ete**.

> **Ловушка!** Не путай Präteritum с Partizip II. «Er machte» (Präteritum) — НЕ «Er hat gemacht» (Perfekt). Это два разных времени с одним значением, но разной территорией.

---

## Часть 4: Спряжение неправильных глаголов

Неправильные меняют корневую гласную. Суффикса -te нет:

| Лицо | gehen → ging | sprechen → sprach |
|------|-------------|-------------------|
| ich | ging | sprach |
| du | ging**st** | sprach**st** |
| er/sie/es | ging | sprach |
| wir | ging**en** | sprach**en** |
| ihr | ging**t** | sprach**t** |
| sie/Sie | ging**en** | sprach**en** |

Снова: **ich = er/sie/es** (без окончания), **wir = sie/Sie** (-en).

Ключевые неправильные глаголы — группируй по паттерну гласных:

| Паттерн | Infinitiv → Präteritum |
|---------|----------------------|
| **ei → ie** | schreiben → schrieb, bleiben → blieb |
| **e → a** | sprechen → sprach, geben → gab, lesen → las |
| **a → u** | fahren → fuhr |
| **e → a (другие)** | sehen → sah, kommen → kam, nehmen → nahm |
| **особые** | gehen → ging, finden → fand, stehen → stand |

> **Якорь:** Три глагола из группы запомнил — десять уже знаешь. ei→ie — самая большая группа.

---

## Часть 5: Когда Perfekt, когда Präteritum — карта

| Ситуация | Время | Пример |
|----------|-------|--------|
| Разговор | **Perfekt** | Ich habe einen Film gesehen |
| Газета / книга | **Präteritum** | Der Präsident sprach vor dem Parlament |
| sein, haben, модальные — везде | **Präteritum** | Ich war müde. Ich konnte nicht kommen |

> **Ловушка!** Немец никогда не скажет «Ich bin müde gewesen». Он скажет «Ich **war** müde». Для sein и haben — Präteritum даже в речи.

## Часть 9: Упражнения

### Упражнение 1. Поставь глагол в Präteritum (er/sie/es)

Определи тип глагола и напиши форму для третьего лица.

1. machen → er ___
2. gehen → sie ___
3. lernen → es ___
4. sprechen → er ___
5. arbeiten → sie ___
6. kommen → er ___
7. sehen → er ___
8. passieren → es ___

<details>
<summary>Ключ</summary>

1. *er **machte*** (правильный: основа + te)
2. *sie **ging*** (неправильный: e→i)
3. *es **lernte*** (правильный: основа + te)
4. *er **sprach*** (неправильный: e→a)
5. *sie **arbeitete*** (правильный: основа на -t, буфер -ete)
6. *er **kam*** (неправильный: o→a)
7. *er **sah*** (неправильный: e→a)
8. *es **passierte*** (правильный: основа + te)

</details>

### Упражнение 2. Трансформация: перепиши из Perfekt в Präteritum

1. Ich habe die Zeitung gelesen.
2. Sie hat einen Artikel geschrieben.
3. Er ist gestern nach Berlin gefahren.
4. Die Journalistin hat über das Ereignis berichtet.
5. Das Konzert hat gestern stattgefunden.
6. Ich habe mit dem Kollegen gesprochen.
7. Sie sind um 8 Uhr gekommen.

<details>
<summary>Ключ</summary>

1. *Ich **las** die Zeitung.*
2. *Sie **schrieb** einen Artikel.*
3. *Er **fuhr** gestern nach Berlin.*
4. *Die Journalistin **berichtete** über das Ereignis.*
5. *Das Konzert **fand** gestern **statt**.* (отделяемая приставка!)
6. *Ich **sprach** mit dem Kollegen.*
7. *Sie **kamen** um 8 Uhr.*

</details>

## Памятка урока

```
PRÄTERITUM = ТЕКСТЫ. PERFEKT = РЕЧЬ.
Исключение: war, hatte, модальные — Präteritum даже в речи.

Правильные: основа + te    |  Неправильные: новая основа
  ich machte / er machte    |    ich ging / er ging
  du machtest               |    du gingst
  wir machten / sie machten |    wir gingen / sie gingen

Паттерны: ei→ie (schrieb), e→a (sprach, gab, las), a→u (fuhr)

Новости: Zeitung (f), Nachricht (f), Bericht (m), Artikel (m),
Sendung (f), Schlagzeile (f), Ereignis (n), Zuschauer (m)
```
