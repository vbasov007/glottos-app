# Урок 10: Введение в Dativ (дательный падеж)
**Словарный запас: Ориентация в городе**

---

## Как работать с этим уроком

1. **Прочитай** правило — пойми логику (5 минут)
2. **Переведи** упражнения письменно — проверь по ключу
3. **Проговори вслух** — медленно, осознанно, анализируя каждый артикль
4. **Ускоряйся** — гоняй матрицу, пока не будет вылетать на автомате

> К этому моменту ты уже знаешь Akkusativ. Теперь добавляем второй падеж — Dativ. Не бойся: принцип тот же. Задай вопрос → определи падеж → поставь правильный артикль.

---

## Часть 1: Что такое Dativ и зачем он нужен

Akkusativ отвечал на вопрос <say>**Wen? Was?**</say> (Кого? Что?) — это объект действия.

Dativ отвечает на вопрос <say>**Wem?**</say> (Кому? Чему?) — это тот, кому что-то достаётся, кому помогают, кому отвечают.

> Я дарю **брату** подарок. Кому? → Брату → **Dativ**.
> Я вижу **брата**. Кого? → Брата → **Akkusativ**.

Чувствуешь разницу? Akkusativ — на кого направлено действие. Dativ — кому от этого действия польза (или вред).

---

## Часть 2: Главный хак — буква «M»

Запомни одну букву и ты никогда не перепутаешь Dativ:

> <say>**Wem?**</say> → M → <say>**deM**</say>, <say>**eineM**</say>, <say>**meineM**</say>

Русское «кому» — в нём есть **М**. Немецкое «<say>wem</say>» — заканчивается на **М**. И все артикли мужского и среднего рода в Dativ заканчиваются на **-M**.

Это не совпадение. Это твой якорь.

---

## Часть 3: Таблица артиклей в Dativ

|  | Мужской (der) | Средний (das) | Женский (die) | Множ. число (die) |
|---|---|---|---|---|
| **Определённый** | <say>de**m**</say> | <say>de**m**</say> | <say>der</say> | <say>den</say> + **(-n)** |
| **Неопределённый** | <say>eine**m**</say> | <say>eine**m**</say> | <say>einer</say> | – |
| **Притяжательный** | <say>meine**m**</say> | <say>meine**m**</say> | <say>meiner</say> | <say>meinen</say> + **(-n)** |

Три вещи, которые надо вбить в голову:

1. **Мужской и средний** в Dativ — одинаковые. Оба на **-m**. Два по цене одного.
2. **Женский** — <say>der</say> / <say>einer</say>. Да, <say>der</say> — как у мужского в Nominativ. Не путай по контексту.
3. **Множественное число** — <say>den</say>, и к самому существительному прибавляется **-n**: <say>den Kinder**n**</say>, <say>den Freunde**n**</say>, <say>den Kolleg**en**</say>. Забудешь это -n — мгновенно узнают иностранца.

---

## Часть 4: Глаголы, которые требуют Dativ

Эти глаголы ВСЕГДА тянут за собой Dativ. Не думай — просто помни:

| Глагол | Перевод | Пример |
|--------|---------|--------|
| <say>**helfen**</say> | помогать | <say>Ich helfe **dem Nachbarn**</say> |
| <say>**danken**</say> | благодарить | <say>Ich danke **meiner Lehrerin**</say> |
| <say>**antworten**</say> | отвечать | <say>Er antwortet **dem Chef**</say> |
| <say>**gefallen**</say> | нравиться | <say>Die Stadt gefällt **meinem Onkel**</say> |
| <say>**gehören**</say> | принадлежать | <say>Das Fahrrad gehört **dem Mädchen**</say> |
| <say>**gratulieren**</say> | поздравлять | <say>Wir gratulieren **unserem Opa**</say> |
| <say>**glauben**</say> | верить | <say>Ich glaube **dir**</say> |
| <say>**zeigen**</say> | показывать | <say>Sie zeigt **dem Touristen** den Weg</say> |
| <say>**schenken**</say> | дарить | <say>Er schenkt **seiner Frau** Blumen</say> |
| <say>**geben**</say> | давать | <say>Ich gebe **dem Kellner** Trinkgeld</say> |

> **Ловушка!** «Спрашивать» (<say>fragen</say>) — это Akkusativ. «Отвечать» (<say>antworten</say>) — это Dativ.
> <say>Ich frage de**n** Nachbar**n**</say> (A). <say>Ich antworte de**m** Nachbar**n**</say> (D).
> Одно слово — разный падеж. Вот почему вопрос «<say>Wen?</say>» или «<say>Wem?</say>» решает всё.

---

## Часть 5: Порядок слов — Dativ перед Akkusativ

Когда в предложении два объекта — **Dativ всегда первый**:

> <say>Ich zeige **dem Gast** (D) **das Zimmer** (A).</say>
> <say>Sie gibt **ihrer Tochter** (D) **einen Schlüssel** (A).</say>
> <say>Er schenkt **seinem Kollegen** (D) **eine Uhr** (A).</say>

Шаблон: КТО + глагол + **КОМУ (D)** + **ЧТО (A)**

## Часть 7: Языковые гаммы

### Гамма 1 — <say>helfen</say> + Dativ (таблица в глубину)

Прогони вслух, меняя только местоимение и окончание глагола:

```
Ich helfe meinem Nachbarn
Du hilfst deinem Nachbarn
Er hilft seinem Nachbarn
Sie hilft ihrem Nachbarn
Wir helfen unserem Nachbarn
Ihr helft eurem Nachbarn
Sie helfen ihrem Nachbarn
```

> Обрати внимание: <say>helfen</say> — глагол с изменением корневой гласной! <say>Du h**i**lfst</say>, <say>er h**i**lft</say>.

### Гамма 2 — <say>gefallen</say> + Dativ

```
Mir gefällt diese Stadt
Dir gefällt diese Stadt
Ihm gefällt diese Stadt
Ihr gefällt diese Stadt
Uns gefällt diese Stadt
Euch gefällt diese Stadt
Ihnen gefällt diese Stadt
```

### Гамма 3 — <say>zeigen</say> + Dativ + Akkusativ (два объекта)

```

Sie zeigt ihrer Kollegin den Weg
Wir zeigen unserer Kollegin den Weg

Sie zeigen ihrer Kollegin den Weg
```

## Часть 8: Языковая матрица (вопрос-ответ)

### Матрица 1 — «Ты → Я» и обратно

```

```


## Часть 9: Упражнения

### Упражнение 1. Определи падеж — поставь (A) или (D)

Перепиши, вставив правильный артикль. В скобках отметь падеж.

1. Ich zeige ___ Nachbarin ( ) ___ Garten ( ).
2. Sie fragt ___ Polizisten ( ).
3. Er dankt ___ Ärztin ( ).
4. Wir schenken ___ Opa ( ) ___ Buch ( ).
5. Du antwortest ___ Lehrerin ( ).
6. Das Haus gehört ___ Familie ( ).
7. Ich gebe ___ Hund ( ) ___ Knochen ( ).
8. Sie gratuliert ___ Freundin ( ).

<details>
<summary>Ключ</summary>

1. Ich zeige **der** Nachbarin (D) **den** Garten (A).
2. Sie fragt **den** Polizisten (A). *(fragen = Akkusativ!)*
3. Er dankt **der** Ärztin (D).
4. Wir schenken **dem** Opa (D) **das** Buch (A).
5. Du antwortest **der** Lehrerin (D).
6. Das Haus gehört **der** Familie (D).
7. Ich gebe **dem** Hund (D) **den** Knochen (A).
8. Sie gratuliert **der** Freundin (D).

</details>

6. Мы показываем нашим гостям ратушу.
7. Она благодарит своего коллегу.

<details>
<summary>Ключ</summary>

### Упражнение 3. Контраст Akkusativ / Dativ — переведи пары

1. Я спрашиваю врача. — Я отвечаю врачу.
2. Она видит соседку. — Она помогает соседке.
3. Мы знаем учителя. — Мы благодарим учителя.
4. Ты ищешь друга. — Ты веришь другу.

<details>
<summary>Ключ</summary>

1. *Ich frage **den** Arzt (A). – Ich antworte **dem** Arzt (D).*
2. *Sie sieht **die** Nachbarin (A). – Sie hilft **der** Nachbarin (D).*
3. *Wir kennen **den** Lehrer (A). – Wir danken **dem** Lehrer (D).*
4. *Du suchst **den** Freund (A). – Du glaubst **dem** Freund (D).*

</details>

## Памятка урока

```
DATIV = WEM? = «M»

der → dem     |  ein → einem     |  mein → meinem
das → dem     |  ein → einem     |  mein → meinem
die → der     |  eine → einer    |  meine → meiner
die (мн.) → den + (-n) к существительному

Порядок слов: КТО + глагол + КОМУ (D) + ЧТО (A)

Глаголы-магниты Dativ:
helfen, danken, antworten, gefallen, gehören,
gratulieren, glauben, zeigen, schenken, geben
```
