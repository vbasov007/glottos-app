# Урок 48: Союзы indem и so dass
**Словарный запас: Публичные выступления**

---

## Как работать с этим уроком

1. **Прочитай** правило — пойми логику (5 минут)
2. **Переведи** упражнения письменно — проверь по ключу
3. **Проговори вслух** — медленно, осознанно, анализируя порядок слов в каждом придаточном
4. **Ускоряйся** — гоняй матрицу, пока не будет вылетать на автомате

> К этому моменту ты владеешь Plusquamperfekt и умеешь выстраивать цепочки событий в прошлом. Теперь добавляем два союза, без которых невозможна связная речь: **indem** (каким способом?) и **so dass** (с каким результатом?). Это инструменты для аргументации и выступлений.

---

## Часть 1: indem — «тем, что» / «путём того, что»

По-русски ты говоришь: «Я улучшил немецкий **тем, что** каждый день читал». В немецком для этого есть **indem**.

> Ich habe mein Deutsch verbessert, **indem** ich jeden Tag **gelesen habe**.

indem отвечает на вопрос **Wie? Auf welche Weise?** — каким способом? каким образом?

Главное правило: в придаточном с indem глагол стоит **в конце**.

> Man lernt eine Sprache, **indem** man jeden Tag **übt**.
> Ты учишь язык тем, что каждый день тренируешься.

---

## Часть 2: Хак — indem = «by doing»

Если знаешь английский — используй якорь:

> **indem = by (doing something)**
> Man überzeugt andere, indem man gute Argumente bringt.
> You convince others **by** bringing good arguments.

Не знаешь английский? Тогда якорь попроще:

> **indem = «путём того, что...»** → глагол в КОНЕЦ.

indem — это всегда **способ / метод**. Не цель, не результат, а именно **как ты это делаешь**.

---

## Часть 3: indem vs. durch + Akkusativ

Одну и ту же мысль можно выразить двумя способами:

| Конструкция | Пример | Перевод |
|-------------|--------|---------|
| **indem** + придаточное | Man lernt, **indem** man viel **liest** | Учишься тем, что много читаешь |
| **durch** + Akk (сущ.) | Man lernt **durch** viel**es** Lesen | Учишься путём чтения |

Три вывода:

1. **indem** требует полное придаточное с подлежащим и глаголом в конце.
2. **durch + Akkusativ** требует существительное (часто отглагольное: Lesen, Üben, Arbeiten).
3. В разговорной речи **indem** звучит естественнее. **Durch** — более книжный вариант.

> **Ловушка!** Не путай indem с damit. **Indem** = способ (как?). **Damit** = цель (зачем?).
> Er spricht langsam, **indem** er jedes Wort betont. (Способ: как именно он говорит медленно?)
> Er spricht langsam, **damit** alle ihn verstehen. (Цель: зачем он говорит медленно?)

---

## Часть 4: so dass / sodass — «так что» (результат)

Теперь вторая конструкция. По-русски: «Он говорил громко, **так что** все его услышали».

> Er hat laut gesprochen, **so dass** alle ihn gehört haben.

so dass (пишется и слитно: **sodass**) вводит **результат** предыдущего действия.

> Die Präsentation war gut strukturiert, **sodass** das Publikum alles **verstanden hat**.
> Презентация была хорошо структурирована, так что публика всё поняла.

---

## Часть 5: Тройной контраст — damit / so dass / indem

Это ключевое различие. Вбей его в голову:

| Союз | Значение | Вопрос | Пример |
|------|----------|--------|--------|
| **damit** | цель, намерение (чтобы) | Wozu? | Er übt, **damit** er besser **wird** |
| **so dass** | фактический результат (так что) | Mit welchem Ergebnis? | Er hat viel geübt, **so dass** er besser **geworden ist** |
| **indem** | способ, метод (тем что) | Wie? | Er wird besser, **indem** er jeden Tag **übt** |

> **Якорь:** damit = **хочу** (цель). so dass = **получилось** (результат). indem = **делаю так** (способ).

Всё на одной ситуации:

> Ich bereite mich vor, **damit** mein Vortrag gelingt. (Цель: чтобы удался.)
> Ich habe mich vorbereitet, **so dass** mein Vortrag gelungen ist. (Результат: так что удался.)
> Ich bereite mich vor, **indem** ich jeden Punkt durchgehe. (Способ: прорабатываю каждый пункт.)

## Часть 7: Языковые гаммы

### Гамма 1 — indem + überzeugen (способ убеждения)

```
Ich überzeuge das Publikum, indem ich klare Argumente bringe
Du überzeugst das Publikum, indem du klare Argumente bringst
Er überzeugt das Publikum, indem er klare Argumente bringt
Sie überzeugt das Publikum, indem sie klare Argumente bringt
Wir überzeugen das Publikum, indem wir klare Argumente bringen
Ihr überzeugt das Publikum, indem ihr klare Argumente bringt
Sie überzeugen das Publikum, indem sie klare Argumente bringen
```

> Обрати внимание: подлежащее в придаточном с indem **совпадает** с подлежащим в главном. Это типичная ситуация.

### Гамма 2 — so dass + Ergebnis (результат выступления)

```
Ich habe laut gesprochen, so dass alle mich gehört haben
Du hast laut gesprochen, so dass alle dich gehört haben
Er hat laut gesprochen, so dass alle ihn gehört haben
Sie hat laut gesprochen, so dass alle sie gehört haben
Wir haben laut gesprochen, so dass alle uns gehört haben
Ihr habt laut gesprochen, so dass alle euch gehört haben
Sie haben laut gesprochen, so dass alle sie gehört haben
```

### Гамма 3 — durch + Akkusativ (книжный вариант способа)

```
Ich verbessere meinen Vortrag durch ständiges Üben
Du verbesserst deinen Vortrag durch ständiges Üben
Er verbessert seinen Vortrag durch ständiges Üben
Sie verbessert ihren Vortrag durch ständiges Üben
Wir verbessern unseren Vortrag durch ständiges Üben
Ihr verbessert euren Vortrag durch ständiges Üben
Sie verbessern ihren Vortrag durch ständiges Üben
```

> Здесь durch + Akkusativ: ständig**es** Üben — субстантивированный инфинитив среднего рода (das Üben).

## Часть 8: Языковая матрица

### Матрица 1 — «Ты ↔ Я» (indem)

```
Wie bereitest du dich auf den Vortrag vor? – Ich bereite mich vor, indem ich die Gliederung schreibe
Wie fasst du die Ergebnisse zusammen? – Ich fasse die Ergebnisse zusammen, indem ich die Hauptpunkte wiederhole
Wie überzeugst du die Zuhörer? – Ich überzeuge sie, indem ich Beweise liefere
Wie verbesserst du deine Aussprache? – Ich verbessere sie, indem ich jeden Tag laut lese
Wie lernst du neue Vokabeln? – Ich lerne sie, indem ich Karteikarten benutze
Wie bleibst du ruhig bei der Präsentation? – Ich bleibe ruhig, indem ich tief durchatme
```

### Матрица 2 — «Третьи лица» (so dass + результат)

```
Hat der Redner gut gesprochen? – Ja, er hat so klar gesprochen, dass alle ihn verstanden haben
Hat sie ihre These gut begründet? – Ja, sie hat sie so gut begründet, dass niemand widersprochen hat
Haben die Studenten viel geübt? – Ja, sie haben so viel geübt, dass der Vortrag perfekt war
Hat der Professor laut genug gesprochen? – Ja, er hat laut gesprochen, so dass auch die letzte Reihe alles gehört hat
Hat die Rednerin das Publikum überzeugt? – Ja, sie hat es überzeugt, so dass alle applaudiert haben
Haben die Teilnehmer gute Argumente gebracht? – Ja, so dass die Diskussion sehr produktiv war
```

### Матрица 3 — Дебаты: аргумент и контраргумент

```
Meiner Meinung nach brauchen wir mehr Übung. – Da widerspreche ich. Wir brauchen bessere Methoden
Ich bin der Meinung, dass Theorie wichtiger ist. – Im Gegenteil. Man lernt besser, indem man praktiziert
Können Sie Ihren Standpunkt begründen? – Ja. Studien zeigen, dass Praxis effektiver ist
Wie kommen Sie zu diesem Schluss? – Ich schlussfolgere das aus den Ergebnissen der Forschung
Lassen Sie mich zusammenfassen. – Bitte, fassen Sie zusammen
Haben Sie noch ein Argument? – Ja. Ich möchte noch einen Beweis anführen
```

## Часть 9: Упражнения

### Упражнение 1. Вставь правильный союз: indem, so dass или damit

Выбери союз, который подходит по смыслу.

1. Er hat den Vortrag gut strukturiert, ___ das Publikum alles verstanden hat.
2. Man wird ein guter Redner, ___ man regelmäßig vor Leuten spricht.
3. Sie spricht langsam, ___ alle folgen können.
4. Ich bereite Folien vor, ___ meine Präsentation anschaulich wird.
5. Der Student hat seine These überzeugend dargestellt, ___ der Professor beeindruckt war.
6. Du kannst dein Argument stärken, ___ du konkrete Beispiele nennst.
7. Sie fasste alles zusammen, ___ sie die Hauptpunkte noch einmal nannte.

<details>
<summary>Ключ</summary>

1. *so dass* (результат: так что публика всё поняла)
2. *indem* (способ: тем, что регулярно выступаешь)
3. *damit* (цель: чтобы все могли следить)
4. *damit* (цель: чтобы презентация была наглядной)
5. *so dass* (результат: так что профессор был впечатлён)
6. *indem* (способ: тем, что называешь конкретные примеры)
7. *indem* (способ: тем, что назвала главные пункты)

</details>

### Упражнение 2. Переведи с русского на немецкий

1. Ты учишь немецкий тем, что каждый день слушаешь подкасты.
2. Она говорила так убедительно, что все с ней согласились.
3. Он готовится к докладу, читая научные статьи. (используй indem)
4. Оратор подвёл итог, так что дискуссия могла начаться.
5. Я улучшаю свою речь путём постоянной практики. (используй durch)
6. Мы можем сделать вывод, что этот аргумент сильнее.
7. По моему мнению, нужно привести больше доказательств.
8. Она возразила докладчику, приведя контраргумент. (используй indem)

<details>
<summary>Ключ</summary>

1. *Du lernst Deutsch, indem du jeden Tag Podcasts hörst.*
2. *Sie hat so überzeugend gesprochen, dass alle ihr zugestimmt haben.* (или: *..., so dass alle ihr zugestimmt haben.*)
3. *Er bereitet sich auf den Vortrag vor, indem er wissenschaftliche Artikel liest.*
4. *Der Redner hat zusammengefasst, so dass die Diskussion beginnen konnte.*
5. *Ich verbessere meine Rede durch ständige Praxis.*
6. *Wir können schlussfolgern, dass dieses Argument stärker ist.*
7. *Meiner Meinung nach muss man mehr Beweise anführen.*
8. *Sie hat dem Redner widersprochen, indem sie ein Gegenargument gebracht hat.*

</details>

### Упражнение 3. Трансформация: indem → durch + Akkusativ (и наоборот)

Перестрой предложения, сохраняя смысл.

1. Man überzeugt andere, **indem** man Fakten präsentiert. → Man überzeugt andere **durch**...
2. Sie verbessert ihre Aussprache **durch** tägliches Vorlesen. → Sie verbessert ihre Aussprache, **indem**...
3. Er stärkt sein Argument, **indem** er Statistiken zitiert. → Er stärkt sein Argument **durch**...
4. Wir lernen **durch** regelmäßige Wiederholung. → Wir lernen, **indem**...

<details>
<summary>Ключ</summary>

1. *Man überzeugt andere durch die Präsentation von Fakten.* (или: *durch das Präsentieren von Fakten*)
2. *Sie verbessert ihre Aussprache, indem sie jeden Tag laut vorliest.*
3. *Er stärkt sein Argument durch das Zitieren von Statistiken.*
4. *Wir lernen, indem wir regelmäßig wiederholen.*

</details>

### Упражнение 4. Контраст damit / so dass — переведи пары

1. Он говорит медленно, чтобы все поняли. — Он говорил медленно, так что все поняли.
2. Она готовит раздаточный материал, чтобы было нагляднее. — Она подготовила раздаточный материал, так что стало нагляднее.
<details>
<summary>Ключ</summary>

1. *Er spricht langsam, **damit** alle verstehen. — Er hat langsam gesprochen, **so dass** alle verstanden haben.*
2. *Sie bereitet ein Handout vor, **damit** es anschaulicher wird. — Sie hat ein Handout vorbereitet, **so dass** es anschaulicher geworden ist.*

</details>

### Упражнение 5. Матрица — отвечай по образцу

Прочитай вопрос и впиши ответ. Если ответ не приходит мгновенно — открой ключ и проговори.

1. Wie überzeugst du die Zuhörer? (ответь: ясными аргументами)
2. Wie fasst du die Ergebnisse zusammen? (ответь: повторяя важнейшие пункты)
3. Hat der Redner gut gesprochen? (ответь: да, так что все его поняли)
4. Können Sie Ihren Standpunkt begründen? (ответь: да, с помощью данных)
5. Wie lernst du neue Vokabeln? (ответь: ежедневно их повторяя)
6. Wie kommen Sie zu diesem Schluss? (ответь: анализируя факты)
7. Wie bereitet er sich auf den Vortrag vor? (ответь с indem)
8. Hat sie das Publikum überzeugt? (ответь с so... dass)
9. Wie stärkt der Redner sein Argument? (ответь с indem)

<details>
<summary>Ключи</summary>

1. *Ich überzeuge die Zuhörer, indem ich klare Argumente bringe.*
2. *Ich fasse die Ergebnisse zusammen, indem ich die wichtigsten Punkte wiederhole.*
3. *Ja, der Redner hat so gut gesprochen, dass alle ihn verstanden haben.*
4. *Ja, ich kann meinen Standpunkt mit Daten begründen.*
5. *Ich lerne neue Vokabeln, indem ich sie täglich wiederhole.*
6. *Ich komme zu diesem Schluss, indem ich die Fakten analysiere.*
7. *Er bereitet sich vor, indem er die Folien gründlich überarbeitet.*
8. *Ja, sie hat so gut gesprochen, dass das Publikum sie applaudierte.*
9. *Er stärkt es, indem er konkrete Beispiele nennt.*

</details>

## Памятка урока

```
INDEM = СПОСОБ (wie?) → глагол В КОНЕЦ
  Man lernt, indem man übt

SO DASS / SODASS = РЕЗУЛЬТАТ (mit welchem Ergebnis?) → глагол В КОНЕЦ
  Er sprach laut, so dass alle ihn hörten

DAMIT = ЦЕЛЬ (wozu?) → глагол В КОНЕЦ
  Er spricht laut, damit alle ihn hören

INDEM vs DURCH:
  indem + придаточное: ..., indem man viel liest
  durch + Akk (сущ.):  ...durch vieles Lesen

Словарь:
der Vortrag (m) — доклад       das Argument (n) — аргумент
die Zusammenfassung (f) — итог  schlussfolgern — делать вывод
überzeugen — убеждать           widersprechen (+D) — возражать
begründen — обосновывать        darstellen — излагать
```
