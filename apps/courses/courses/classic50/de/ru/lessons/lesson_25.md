## Часть 1: man — безликое «люди/ты/все»

В русском мы говорим: «Здесь **говорят** по-немецки». Кто говорит? Неважно. Все. Люди вообще.

В немецком для этого есть слово **man**. Это не «мужчина» (der Mann), а безличное местоимение. Значит «люди», «все», «ты вообще».

> **man = все/люди/ты (в общем смысле)**
> Man spricht hier Deutsch. — Здесь говорят по-немецки.
> Man muss jeden Tag üben. — Нужно заниматься каждый день.

> **man ВСЕГДА требует глагол в er-форме.** Man sprech**t**. Man muss. Man könn**te**. Без вариантов.

Не путай: **man** (безличное, одна n) и **Mann** (мужчина, две n).

## Часть 2: es — формальное подлежащее

В немецком предложение не может без подлежащего. Даже если по смыслу его нет — ставят **es** как заглушку. Три случая:

**1. Погода и природные явления:**

| Немецкий | Русский |
|----------|---------|
| Es regnet | Идёт дождь |
| Es schneit | Идёт снег |
| Es donnert | Гремит гром |
| Es ist windig | Ветрено |
| Es ist kalt | Холодно |
| Es ist heiß | Жарко |

**2. Ощущения и состояния:** Es ist spät (поздно), Es tut mir leid (мне жаль), Wie geht es dir? (как дела?)

**3. Оценки:** Es ist wichtig (важно), Es ist schön (красиво)

> Во всех этих случаях **es** нельзя перевести. Оно не значит «оно». Оно просто занимает место подлежащего — немецкий не терпит пустоты.

## Часть 3: es gibt — «есть/имеется»

Это конструкция, которую ты будешь использовать каждый день. «Es gibt» = «есть», «имеется», «существует».

> Es gibt einen Park. — Есть парк.
> Es gibt viele Restaurants. — Есть много ресторанов.
> Gibt es hier ein Café? — Здесь есть кафе?

Два факта: **es gibt** не меняется (только эта форма, никаких «es geben»). Вопрос — переставь: **Gibt es...?** Отрицание: **Es gibt kein/keine...**

> **Ловушка!** После «es gibt» ВСЕГДА стоит **Akkusativ**. Всегда. Без исключений.
> Es gibt ein**en** Fehler. (не ~~ein Fehler~~)
> Es gibt ein**en** Park. (не ~~ein Park~~)
> Es gibt ein**e** Bibliothek. (не меняется — женский род в Akk = Nominativ)
> Es gibt ein Kino. (не меняется — средний род в Akk = Nominativ)
>
> Мужской род — вот где ловушка. **Ein → einen.** Забудешь — мгновенно узнают.

## Часть 5: Языковые гаммы

### Гамма 1 — es gibt + Akkusativ (места в городе)

Прогони вслух. Следи за артиклями — это Akkusativ!

```
Es gibt einen Supermarkt in meiner Straße
Es gibt einen Bahnhof in meiner Stadt
Es gibt eine Bibliothek in meiner Nähe
Es gibt ein Krankenhaus in meinem Viertel
Es gibt keine Apotheke in meiner Straße
Es gibt keinen Park in meiner Nähe
Es gibt viele Restaurants in meiner Stadt
```

### Гамма 2 — man + глагол (правила и привычки)

```
In Deutschland trinkt man viel Kaffee
In Russland trinkt man viel Tee
Hier spricht man Deutsch
Dort spricht man Englisch
Am Wochenende schläft man lange
Im Sommer schwimmt man im See
Im Winter fährt man Ski
```

> man всегда с формой третьего лица: trink**t**, sprich**t**, schläf**t**, fähr**t**.

### Гамма 3 — es + погода (описание дня)

```
Heute ist es heiß und sonnig
Gestern war es kalt und windig
Morgen wird es regnerisch
Im Sommer ist es warm
Im Winter ist es kalt
Im Herbst regnet es oft
Im Frühling scheint die Sonne wieder
```

## Часть 6: Языковая матрица

### Матрица 1 — «Ты ↔ Я» (man-конструкции)

```
Darf man hier rauchen? – Nein, hier darf man nicht rauchen
Kann man hier parken? – Ja, hier kann man parken
Muss man hier eine Maske tragen? – Nein, man muss keine Maske tragen
Darf man hier fotografieren? – Ja, man darf hier fotografieren
Muss man das Ticket vorher kaufen? – Ja, man muss das Ticket vorher kaufen
```

### Матрица 2 — «Третьи лица» (es gibt)

```
Gibt es hier einen Supermarkt? – Ja, es gibt einen Supermarkt um die Ecke
Gibt es in deiner Stadt ein Theater? – Nein, es gibt kein Theater
Gibt es hier eine Haltestelle? – Ja, es gibt eine Haltestelle vor dem Haus
Gibt es viele Parks in Berlin? – Ja, es gibt sehr viele Parks
Gibt es hier ein Problem? – Nein, es gibt kein Problem
```

### Матрица 3 — Погода (es + Wetter)

```
Wie ist das Wetter heute? – Es ist sonnig und warm, etwa 25 Grad
Regnet es draußen? – Ja, es regnet seit einer Stunde
Schneit es im Winter oft? – Ja, im Dezember schneit es fast jeden Tag
Ist es draußen kalt? – Ja, es ist sehr kalt, minus zehn Grad
Wie wird das Wetter morgen? – Morgen wird es bewölkt und kühl
Scheint die Sonne? – Nein, heute ist es bewölkt und windig
War es gestern heiß? – Ja, gestern war es sehr heiß, fast 35 Grad
```

## Часть 7: Упражнения

### Упражнение 1. Вставь man, es или es gibt

Выбери правильную конструкцию и впиши в пропуск.

1. ___ regnet seit drei Tagen.
2. In Deutschland ___ viele Autobahnen.
3. ___ spricht hier kein Englisch.
4. Im Winter ___ sehr kalt.
5. ___ einen Fehler in deinem Text.
6. ___ muss pünktlich sein.
7. Heute ___ sonnig und warm.
8. ___ darf hier nicht rauchen.

<details>
<summary>Ключ</summary>

1. *Es* regnet seit drei Tagen.
2. In Deutschland *gibt es* viele Autobahnen.
3. *Man* spricht hier kein Englisch.
4. Im Winter *ist es* sehr kalt.
5. *Es gibt* einen Fehler in deinem Text.
6. *Man* muss pünktlich sein.
7. Heute *ist es* sonnig und warm.
8. *Man* darf hier nicht rauchen.

</details>

### Упражнение 2. es gibt + Akkusativ — вставь правильный артикль

Впиши ein/eine/einen/kein/keine/keinen.

1. Es gibt ___ Park in meiner Stadt. (m)
2. Es gibt ___ Bibliothek neben dem Rathaus. (f)
3. Es gibt ___ Kino in unserem Viertel. (n)
4. Gibt es hier ___ Supermarkt? (m)
5. Es gibt ___ Problem. (n)
6. Es gibt ___ Apotheke in der Nähe. (f, отрицание)
7. Es gibt ___ Bahnhof in diesem Dorf. (m, отрицание)
8. Es gibt ___ Gewitter heute Nacht. (n)

<details>
<summary>Ключ</summary>

1. Es gibt *einen* Park in meiner Stadt.
2. Es gibt *eine* Bibliothek neben dem Rathaus.
3. Es gibt *ein* Kino in unserem Viertel.
4. Gibt es hier *einen* Supermarkt?
5. Es gibt *ein* / *kein* Problem. (оба варианта возможны по контексту)
6. Es gibt *keine* Apotheke in der Nähe.
7. Es gibt *keinen* Bahnhof in diesem Dorf.
8. Es gibt *ein* Gewitter heute Nacht.

</details>

2. Здесь говорят по-немецки.
3. В моём городе есть большой парк.

7. Нужно много заниматься.
8. Завтра будет дождливо и ветрено.

<details>
<summary>Ключ</summary>

2. *Hier spricht man Deutsch.*
3. *In meiner Stadt gibt es einen großen Park.*
4. *Es schneit. Draußen ist es kalt.*
5. *Kann man hier ein Ticket kaufen?* / *Kann man hier eine Fahrkarte kaufen?*
6. *Gibt es hier in der Nähe eine Apotheke?*
7. *Man muss viel üben.*

</details>

### Упражнение 4. Исправь ошибку

В каждом предложении одна ошибка. Найди и исправь.

1. Es gibt ein Fehler in deinem Text.
2. Man sprechen hier Deutsch.
3. Es regnet und es kalt.
4. Es gibt nicht Supermarkt hier.
5. In Deutschland trinkt man viel Bier. Mann kann es überall kaufen.
6. Gibt es hier ein Haltestelle?

<details>
<summary>Ключ</summary>

1. Es gibt ein**en** Fehler in deinem Text. *(Akkusativ! ein → einen)*
2. Man **spricht** hier Deutsch. *(man = er-форма: spricht, не sprechen)*
3. Es regnet und es **ist** kalt. *(пропущен глагол ist)*
4. Es gibt **keinen** Supermarkt hier. *(отрицание с es gibt = kein, не nicht; + Akkusativ: keinen)*
5. In Deutschland trinkt man viel Bier. **Man** kann es überall kaufen. *(Mann = мужчина, man = безличное)*
6. Gibt es hier **eine** Haltestelle? *(die Haltestelle = f, значит eine, не ein)*

</details>

## Памятка урока

```
man + er-форма: man spricht, man muss, man kann
  man ≠ Mann!

es = заглушка: Es regnet. Es ist kalt. Es schneit.

es gibt + AKKUSATIV: einen (m!), eine (f), ein (n)
  Gibt es...? / Es gibt kein(en)...

Погода: Sonne (f), Regen (m), Schnee (m), Wind (m)
  heiß, kalt, warm, sonnig, windig, bewölkt
  es regnet, es schneit, die Sonne scheint
```
