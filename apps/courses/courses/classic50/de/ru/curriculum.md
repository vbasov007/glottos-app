# Программа уроков немецкого языка (50 разделов)
**Glottos Matrix — структурный курс**

> Принцип: 5% — понять правило, 95% — натренировать речевой аппарат. Каждую тему прогоняем через матрицу вопрос-ответ: ты → вы (мн.) → он/она/они.

## Принципы работы с каждым разделом

1. **Теория** — посмотри видео / прочитай правило (5 минут)
2. **Перевод** — переведи фразы с русского на немецкий, проверь по ключу
3. **Языковые гаммы** — прогони новое правило через все лица (ich → du → er → wir → ihr → sie)
4. **Матрица** — отработай через вопрос-ответ в трёх сценариях: ты (ед.) → вы (мн.) → третьи лица
5. **Принцип зеркала** — ответ уже заложен в вопросе, не строй с нуля

## Хаки для запоминания

- **Артикли**: 50% определяются по суффиксу (-ung → die, -chen → das). Остальные — визуализация цветом (синий = der, зелёный = das, красный = die)
- **Множественное число**: учи через контраст «Ein... → Viele...»
- **Неправильные глаголы**: группируй по паттерну гласных (bleiben-blieb-geblieben = schreiben-schrieb-geschrieben). Учи сразу с вспомогательным глаголом в предложении: *Ich bin geblieben*
