# Test do lekcji 11: Przeczenie nicht/kein + antonimy

**Jak powiedzieć po niemiecku?**
Przetłumacz zdania na język niemiecki. Wykorzystaj gramatykę i słownictwo lekcji.

---

1. Nie mam brata.
2. Książka nie jest interesująca.
3. On nie pije mleka.
4. Nie rozumiemy.
5. Ona nie ma pieniędzy.
6. Mieszkanie nie jest duże.
7. Nie kupuję roweru.
8. On nie przychodzi dzisiaj.
9. Mieszkanie jest małe, ale nie jest drogie.
10. Oni nie mają dzieci.
11. Ulica nie jest czysta. Jest brudna.
12. Kawa nie jest gorąca. Jest zimna.
13. Ona nie pracuje dzisiaj.
14. Nie mam czasu.
15. Walizka nie jest lekka. Jest ciężka.
16. Mój brat nie ma samochodu.
17. Muzyka nie jest głośna. Jest cicha.
18. To nie jest drogie. To jest tanie.
19. Ona nie kupuje torebki.
20. Nie mamy problemów.
21. Pokój nie jest jasny. Jest ciemny.
22. Zupa nie jest gorąca. Jest zimna.
23. On nie mówi głośno.
24. Ulica nie jest długa. Jest krótka.
25. On nie jest głodny. (Er hat keinen Hunger.)
26. Nie widzę żadnej różnicy.
27. Moja siostra nie ma psa.
28. Woda nie jest mokra — jest sucha. (Żart! Ręcznik nie jest mokry. Jest suchy.)
29. Pociąg nie jest szybki. Jest powolny. I nie jest nowy — jest stary.
30. Mieszkanie nie jest duże i nie jest czyste. Nie mamy garażu ani samochodu.
