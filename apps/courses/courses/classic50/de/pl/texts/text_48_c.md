# Tekst do lekcji 48 (C): Die Kunst der Rede
**Temat: Spójniki indem / so dass — wystąpienia publiczne**

---

## Tekst do słuchania i powtarzania

1. Sie gewinnt Sympathie, indem sie persönliche Geschichten erzählt.
2. Sein Ton war so monoton, dass das Publikum gähnte.
3. Man baut Spannung auf, indem man eine Frage stellt.
4. Die Rednerin sprach so lebendig, dass alle mitgerissen wurden.
5. Du vermeidest Fehler, indem du den Text vorher laut übst.
6. Das Mikrofon war so leise, dass man nichts verstand.
7. Er hält Blickkontakt, indem er verschiedene Personen anschaut.
8. Die Struktur war so klar, dass jeder dem Vortrag folgte.
9. Man überzeugt Skeptiker, indem man Zahlen und Daten nennt.
10. Sie gestikulierte so stark, dass es vom Inhalt ablenkte.
11. Ich beende den Vortrag, indem ich die Kernaussage wiederhole.
12. Das Thema war so aktuell, dass eine lebhafte Debatte entstand.
13. Er lockert die Atmosphäre, indem er mit Humor beginnt.
14. Die Stimme war so kraftvoll, dass sie den Saal füllte.
15. Man wirkt kompetent, indem man Fachbegriffe gezielt einsetzt.
16. Der Applaus war so laut, dass der Redner gerührt war.
17. Du stärkst deine Glaubwürdigkeit, indem du Quellen angibst.
18. Die Pause war so lang, dass Unruhe im Saal entstand.
19. Sie motiviert das Team, indem sie eine Vision beschreibt.
20. Das Feedback war so positiv, dass sie sofort eingeladen wurde.
21. Man vermeidet Lampenfieber, indem man ausreichend probt.
22. Er sprach so schnell, dass die Hälfte verloren ging.
23. Ich strukturiere Ideen, indem ich eine Mindmap erstelle.
24. Die Botschaft war so kraftvoll, dass sie viral ging.
25. Du gewinnst Zeit, indem du eine rhetorische Frage stellst.
26. Der Saal war so voll, dass Leute stehen mussten.
27. Man schließt professionell ab, indem man sich beim Publikum bedankt.
28. Seine Worte waren so treffend, dass alle nachdenklich wurden.
29. Sie hält die Aufmerksamkeit, indem sie das Tempo variiert.
30. Das Beispiel war so anschaulich, dass es sofort einleuchtete.
