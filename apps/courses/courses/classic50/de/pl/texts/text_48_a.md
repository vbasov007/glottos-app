# Tekst do lekcji 48: Spójniki indem / so dass
**Temat: Wystąpienia publiczne**

---

## Tekst do słuchania i powtarzania

1. Man wird ein guter Redner, indem man regelmäßig übt.
2. Er sprach so klar, dass alle ihn verstanden haben.
3. Ich überzeuge das Publikum, indem ich Beweise liefere.
4. Die Präsentation war gut strukturiert, sodass alle folgten.
5. Sie verbessert ihre Aussprache, indem sie täglich laut liest.
6. Der Vortrag war so lang, dass einige eingeschlafen sind.
7. Man lernt argumentieren, indem man an Diskussionen teilnimmt.
8. Er begründete seinen Standpunkt, sodass niemand widersprach.
9. Ich fasse die Ergebnisse zusammen, indem ich die Hauptpunkte wiederhole.
10. Sie sprach so überzeugend, dass alle applaudierten.
11. Du stärkst dein Argument, indem du konkrete Beispiele nennst.
12. Die Einleitung war so spannend, dass alle aufmerksam zuhörten.
13. Er bereitet sich vor, indem er jede Folie sorgfältig prüft.
14. Die Rednerin fasste alles zusammen, sodass die Diskussion beginnen konnte.
15. Man gewinnt Vertrauen, indem man Augenkontakt hält.
16. Der Beweis war so stark, dass die These bestätigt wurde.
17. Ich bleibe ruhig bei der Präsentation, indem ich tief durchatme.
18. Das Handout war so klar formuliert, dass keine Fragen blieben.
19. Er schlussfolgerte überzeugend, indem er alle Fakten nannte.
20. Die Gliederung war logisch, sodass der Vortrag leicht verständlich war.
21. Du kannst widersprechen, indem du ein Gegenargument bringst.
22. Sie stellte ihre These so dar, dass alle überzeugt waren.
23. Meiner Meinung nach überzeugt man durch klare Argumente.
24. Er verbesserte seinen Vortrag durch ständiges Üben.
25. Die Studenten übten so viel, dass die Präsentation perfekt gelang.
26. Ich nehme Stellung zu dem Thema, indem ich Fakten präsentiere.
27. Der Redner kam zum Schluss, sodass die Fragerunde begann.
28. Man lernt eine Sprache, indem man jeden Tag spricht.
29. Das Ergebnis war so gut, dass der Professor beeindruckt war.
30. Sie widersprach dem Redner, indem sie einen Gegenbeweis zeigte.
