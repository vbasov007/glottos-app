# Tekst do lekcji 48 (B): Vor dem Publikum
**Temat: Spójniki indem / so dass — wystąpienia publiczne**

---

## Tekst do słuchania i powtarzania

1. Man baut Spannung auf, indem man eine Frage stellt.
2. Sie sprach so lebendig, dass das Publikum begeistert war.
3. Er strukturiert seinen Vortrag, indem er drei Punkte nennt.
4. Die Stimme war so leise, dass man sie kaum hörte.
5. Du wirkst professionell, indem du frei und ruhig sprichst.
6. Das Thema war so aktuell, dass der Saal voll war.
7. Man vermeidet Nervosität, indem man vorher gut probt.
8. Er sprach so schnell, dass viele nicht mitkamen.
9. Ich gewinne Aufmerksamkeit, indem ich mit einer Geschichte beginne.
10. Die Folien waren so überladen, dass niemand sie lesen konnte.
11. Man hält Blickkontakt, indem man verschiedene Personen anschaut.
12. Der Schluss war so stark, dass alle sofort klatschten.
13. Du verbesserst deine Rhetorik, indem du dich selbst aufnimmst.
14. Er war so nervös, dass seine Hände gezittert haben.
15. Man fesselt Zuhörer, indem man persönliche Beispiele einbaut.
16. Die Argumentation war so logisch, dass niemand widersprach.
17. Sie beendet den Vortrag, indem sie die Kernaussage wiederholt.
18. Das Mikrofon war so laut, dass es im Raum hallte.
19. Man wirkt glaubwürdig, indem man Quellen transparent nennt.
20. Der Redner war so charismatisch, dass alle gebannt zuhörten.
21. Du sparst Zeit, indem du dich auf das Wesentliche konzentrierst.
22. Die Diskussion war so hitzig, dass der Moderator eingriff.
23. Man zeigt Kompetenz, indem man auf Fragen souverän antwortet.
24. Er redete so lange, dass die Hälfte den Raum verließ.
25. Ich halte die Aufmerksamkeit, indem ich die Stimme variiere.
26. Das Ergebnis war so überraschend, dass alle still wurden.
27. Man schließt die Rede ab, indem man zum Handeln aufruft.
28. Der Applaus war so laut, dass er noch eine Zugabe gab.
29. Du gewinnst das Publikum, indem du Humor gezielt einsetzt.
30. Die Botschaft war so klar, dass jeder sie verstand.
