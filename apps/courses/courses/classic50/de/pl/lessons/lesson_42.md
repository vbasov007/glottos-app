# Lekcja 42: Passiv. Passiv Perfekt
**Słownictwo: Nauka i medycyna**

---

## Jak pracować z tą lekcją

1. **Przeczytaj** regułę — zrozum logikę (5 minut)
2. **Przetłumacz** ćwiczenia pisemnie — sprawdź klucz
3. **Powiedz na głos** — powoli, świadomie, analizując każdą formę
4. **Przyspieszaj** — przeganiaj matrycę, aż wylatuje automatycznie

> Z lekcji 41 wiesz, jak mówić o rzeczach nierzeczywistych. Teraz uczymy się mówić o czynnościach, kiedy nieważne **kto** robi, tylko **co** się robi. To strona bierna — koń roboczy tekstów naukowych, biurokratycznych i newsów. Polak ma silny grunt: polska strona bierna („Samochód jest naprawiany / został naprawiony") to bezpośredni strukturalny ekwiwalent niemieckiego pasywu. Jedna różnica koncepcyjna: niemiecki używa „werden" (stawać się), tam gdzie polski używa „być/zostać".

---

## Część 1: Co to jest Passiv i po co

W stronie czynnej (Aktiv) skupiamy się na tym, **kto** robi: Der Arzt operiert den Patienten.

W stronie biernej (Passiv) — na tym, **co** się dzieje: Der Patient **wird** operiert.

Po polsku to samo: „Lekarz operuje pacjenta" vs. „Pacjent **jest operowany** / **zostaje operowany**". Lek **został opracowany**. Eksperyment **jest przeprowadzany**. To właśnie strona bierna.

> **Strukturalny kognat:** polski „być/zostać + imiesłów bierny" = niemiecki „werden + Partizip II". Polski wybiera między „być" (proces) i „zostać" (rezultat). Niemiecki ma jeden czasownik pomocniczy — **werden** — i czasem tłumaczy się go jako „być", czasem jako „zostać". Kontekst decyduje.

---

## Część 2: Główny hack — trzy filary pasywu

> **werden = pomocnik pasywu. Partizip II = na końcu. Kto robi = von + Dativ.**

Wbij tę formułę — i zbudujesz każde zdanie w stronie biernej. Reszta to detale.

Pamiętaj: werden znasz już jako „stawać się" i jako pomocnik czasu przyszłego. Nie myl! werden + Infinitiv = przyszłość. werden + Partizip II = strona bierna.

## Część 3: Passiv Präsens — formuła i wykonawca

**Formuła:** Subjekt + **werden** (odmienione) + ... + **Partizip II** (na końcu)

| Osoba | werden |
|-------|--------|
| ich | **werde** |
| du | **wirst** |
| er/sie/es | **wird** |
| wir | **werden** |
| ihr | **werdet** |
| sie/Sie | **werden** |

| Aktiv | Passiv | Polski |
|-------|--------|--------|
| Der Forscher untersucht das Problem | Das Problem **wird** untersucht | Problem jest badany |
| Die Ärztin verschreibt die Therapie | Die Therapie **wird** verschrieben | Terapia jest przepisywana |
| Man testet das Medikament | Das Medikament **wird** getestet | Lek jest testowany |

1. **Obiekt w Akkusativ ze strony czynnej → podmiot w Nominativ w stronie biernej.**
2. **Podmiot strony czynnej znika** — albo idzie w **von + Dativ**: Das Experiment **wird** von **dem Wissenschaftler** durchgeführt. (Eksperyment jest przeprowadzany przez naukowca — polskie „przez" = niemieckie „von".)
3. **Partizip II zawsze na końcu.** Bez wyjątków.

> **Pułapka!** von + Dativ = **kto** robi. durch + Akkusativ = **czym / za pomocą czego**.
> Die Krankheit wird von **dem Arzt** behandelt. (Przez kogo? Przez lekarza.)
> Die Krankheit wird durch **eine neue Therapie** geheilt. (Czym? Nową terapią.)
> Polski jednym „przez" obejmuje obie sytuacje. Niemiecki rozdziela.

## Część 4: Passiv Perfekt — formuła

**Formuła:** Subjekt + **sein** (odmienione) + ... + **Partizip II** + **worden**

Nie „geworden", tylko **worden** — bez „ge-". To specjalna forma, używana wyłącznie w Passiv Perfekt.

| Aktiv Perfekt | Passiv Perfekt | Polski |
|---------------|----------------|--------|
| Der Forscher **hat** das Gen **entdeckt** | Das Gen **ist** entdeckt **worden** | Gen został odkryty |
| Die Firma **hat** das Medikament **entwickelt** | Das Medikament **ist** entwickelt **worden** | Lek został opracowany |
| Man **hat** den Patienten **operiert** | Der Patient **ist** operiert **worden** | Pacjent został zoperowany |

> **Kotwica:** sein + Partizip II + **worden** (bez ge-!). sein się odmienia, reszta idzie na koniec.

| Osoba | Passiv Perfekt |
|-------|----------------|
| ich / du / er | bin / bist / ist ... operiert **worden** |
| wir / ihr / sie | sind / seid / sind ... operiert **worden** |

> Kognat strukturalny: niemieckie „ist entdeckt worden" = polskie „został odkryty". Polskie „zostać" pełni dokładnie tę samą rolę co niemieckie „worden" — markera rezultatu.

## Część 5: Aktiv → Passiv w 5 krokach

1. Znajdź **obiekt w Akkusativ** w zdaniu czynnym.
2. Zrób go **Nominativem** (podmiotem pasywu).
3. Wstaw **werden** (Präsens) albo **sein ... worden** (Perfekt).
4. Czasownik → **Partizip II** → na koniec.
5. Wykonawca → **von + Dativ** (jeśli potrzebny).

**Aktiv:** Die Wissenschaftlerin entwickelt **einen Impfstoff**.
**Passiv:** **Ein Impfstoff** wird von der Wissenschaftlerin entwickelt.

> **Pułapka!** Tylko czasowniki przechodnie (z obiektem w Akkusativ) tworzą pasyw osobowy. *~~Der Patient wird geholfen~~* — nie wolno! helfen wymaga Dativu → **Dem Patienten wird geholfen** (pasyw bezosobowy). Polski też zna tę różnicę: „pomagać komuś" — nie da się utworzyć normalnej strony biernej.

## Część 6: Słownictwo — Nauka i medycyna

### Rzeczowniki

| die Forschung | f | badanie naukowe |
| die Therapie | f | terapia |
| die Wissenschaft | f | nauka |
| die Entdeckung | f | odkrycie |
| die Behandlung | f | leczenie |
| die Wirkung | f | działanie, efekt |
| das Experiment | n | eksperyment |
| das Ergebnis | n | wynik |
| das Medikament | n | lek |
| das Labor | n | laboratorium |
| das Gen | n | gen |

### Czasowniki

| Niemiecki | Polski |
|-----------|--------|
| entdecken | odkrywać |
| entwickeln | opracowywać, rozwijać |
| untersuchen | badać, obserwować |

Polski robi to samo, słowo w słowo. Cała ta lekcja słownikowo jest dla Polaka łatwa.

## Część 7: Gamy językowe

### Gama 1 — Passiv Präsens (werden + Partizip II)

Przejedź na głos. Pilnuj formy werden:

```
Ich werde von dem Arzt untersucht
Du wirst von dem Arzt untersucht
Er wird von dem Arzt untersucht
Sie wird von dem Arzt untersucht
Wir werden von dem Arzt untersucht
Ihr werdet von dem Arzt untersucht
Sie werden von dem Arzt untersucht
```

> Wszystkie osoby — różne formy werden. Partizip II „untersucht" się NIE zmienia. Nigdy.

### Gama 2 — Passiv Perfekt (sein + Partizip II + worden)

```
Ich bin gestern operiert worden
Du bist gestern operiert worden
Er ist gestern operiert worden
Sie ist gestern operiert worden
Wir sind gestern operiert worden
Ihr seid gestern operiert worden
Sie sind gestern operiert worden
```

> sein odmienia się normalnie. „operiert worden" — niezmienialny blok na końcu. Polski przekład: „zostałem zoperowany / zostałeś zoperowany..." — ta sama struktura.

### Gama 3 — Transformacja aktyw → pasyw (nauka)

Czytaj pary na głos — najpierw aktyw, potem pasyw:

```
Der Forscher entdeckt ein neues Gen – Ein neues Gen wird entdeckt
Die Firma entwickelt einen Impfstoff – Ein Impfstoff wird entwickelt
Die Ärztin behandelt den Patienten – Der Patient wird behandelt
Das Team führt das Experiment durch – Das Experiment wird durchgeführt
Man testet das Medikament im Labor – Das Medikament wird im Labor getestet
```

> durchführen — przedrostek rozdzielny. W Partizip II wraca w całość: durch**geführt**.

## Część 8: Matryca językowa

### Matryca 1 — „Ty / Ja" (Passiv Präsens)

```
Wirst du oft von deinem Arzt untersucht? – Ja, ich werde regelmäßig untersucht
Wirst du von der Professorin betreut? – Ja, ich werde von ihr betreut
Wirst du gut behandelt? – Ja, ich werde sehr gut behandelt
Wirst du oft zu Experimenten eingeladen? – Nein, ich werde selten eingeladen
Wirst du über die Ergebnisse informiert? – Ja, ich werde immer sofort informiert
```

### Matryca 2 — „Trzecie osoby" (Passiv Perfekt)

```
Ist der Patient schon operiert worden? – Ja, er ist gestern operiert worden
Ist das Medikament getestet worden? – Ja, es ist erfolgreich getestet worden
Sind die Ergebnisse veröffentlicht worden? – Nein, sie sind noch nicht veröffentlicht worden
Ist das neue Gen entdeckt worden? – Ja, es ist von einem deutschen Forscher entdeckt worden
Ist der Impfstoff schon entwickelt worden? – Ja, er ist letztes Jahr entwickelt worden
```

### Matryca 3 — Nauka i medycyna (mieszany pasyw)

```
Wie wird diese Krankheit behandelt? – Sie wird mit einer neuen Therapie behandelt
Wo wird das Experiment durchgeführt? – Es wird im Labor der Universität durchgeführt
Wann ist der Impfstoff entwickelt worden? – Er ist vor zwei Jahren entwickelt worden
Von wem ist das Gen entdeckt worden? – Es ist von einem internationalen Team entdeckt worden
Werden die Patienten regelmäßig untersucht? – Ja, sie werden alle drei Monate untersucht
```

## Część 9: Ćwiczenia

### Ćwiczenie 1. Określ — Aktiv czy Passiv?

Napisz A (aktyw) albo P (pasyw) przy każdym zdaniu:

1. Der Forscher untersucht die Probe.
2. Die Probe wird untersucht.
3. Das Medikament ist entwickelt worden.
4. Die Ärztin hat den Patienten behandelt.
5. Der Patient wird von der Ärztin behandelt.
6. Man führt ein Experiment durch.

<details>
<summary>Klucz</summary>

1. *A* (podmiot — der Forscher — sam działa)
2. *P* (wird + Partizip II)
3. *P* (ist + Partizip II + worden)
4. *A* (hat + Partizip II = Perfekt Aktiv)
5. *P* (wird + von + Partizip II)
6. *A* (man + czasownik = aktyw)

</details>

### Ćwiczenie 2. Transformacja: Aktiv → Passiv Präsens

Przebuduj zdania z aktywu na pasyw. Wykonawcę pomiń.

1. Der Wissenschaftler entdeckt ein neues Virus.
2. Die Ärztin verschreibt eine Therapie.
3. Man testet das Medikament an Freiwilligen.
4. Der Forscher veröffentlicht die Studie.
5. Die Firma entwickelt einen neuen Impfstoff.
6. Das Team führt drei Experimente durch.

<details>
<summary>Klucz</summary>

1. *Ein neues Virus wird entdeckt.*
2. *Eine Therapie wird verschrieben.*
3. *Das Medikament wird an Freiwilligen getestet.*
4. *Die Studie wird veröffentlicht.*
5. *Ein neuer Impfstoff wird entwickelt.*
6. *Drei Experimente werden durchgeführt.*

</details>

### Ćwiczenie 3. Transformacja: Aktiv → Passiv Perfekt

Przebuduj na Passiv Perfekt. Wykonawcę wskaż przez von + Dativ.

1. Der Wissenschaftler hat ein Gen entdeckt.
2. Die Forscherin hat das Experiment durchgeführt.
3. Die Firma hat das Medikament entwickelt.
4. Der Arzt hat den Patienten operiert.
5. Das Team hat die Ergebnisse veröffentlicht.
6. Die Ärztin hat die Wirkung bewiesen.

<details>
<summary>Klucz</summary>

1. *Ein Gen ist von dem Wissenschaftler entdeckt worden.*
2. *Das Experiment ist von der Forscherin durchgeführt worden.*
3. *Das Medikament ist von der Firma entwickelt worden.*
4. *Der Patient ist von dem Arzt operiert worden.*
5. *Die Ergebnisse sind von dem Team veröffentlicht worden.*
6. *Die Wirkung ist von der Ärztin bewiesen worden.*

</details>

### Ćwiczenie 4. Przetłumacz z polskiego na niemiecki

1. Pacjent jest badany co miesiąc.
2. Nowy lek został opracowany w zeszłym roku.
3. Eksperyment jest przeprowadzany w laboratorium.
4. Wyniki zostały opublikowane przez naukowca.
5. Choroba jest leczona nową terapią.
6. Szczepionka została przetestowana na ochotnikach.

<details>
<summary>Klucz</summary>

1. *Der Patient wird jeden Monat untersucht.* (polskie „jest badany" → wird + Partizip II)
2. *Ein neues Medikament ist letztes Jahr entwickelt worden.* (polskie „został opracowany" → ist + Part.II + worden)
3. *Das Experiment wird im Labor durchgeführt.*
4. *Die Ergebnisse sind von dem Wissenschaftler veröffentlicht worden.* („przez" = von + Dat)
5. *Die Krankheit wird mit einer neuen Therapie behandelt.* („nową terapią" = środek → mit/durch)
6. *Der Impfstoff ist an Freiwilligen getestet worden.*

</details>

### Ćwiczenie 5. Matryca — odpowiadaj według wzorca

Przeczytaj pytanie i wpisz odpowiedź. Jeśli odpowiedź nie przychodzi natychmiast — otwórz klucz i wypowiedz ją na głos. W drugiej części odpowiadaj w Passiv Perfekt.

1. Wirst du gut behandelt? (odpowiedz: tak)
2. Ist der Patient operiert worden? (odpowiedz: tak)
3. Wie wird die Krankheit behandelt? (odpowiedz: nową terapią)
4. Ist das Medikament getestet worden? (odpowiedz: tak)
5. Wo wird das Experiment durchgeführt? (odpowiedz: w laboratorium)
6. Von wem ist das Gen entdeckt worden? (odpowiedz: niemiecki naukowiec)

**Odpowiadaj w Passiv Perfekt:**

7. Wurde der Patient untersucht? (odpowiedz: tak)
8. Wurde die Therapie verschrieben? (odpowiedz: tak)
9. Wurde das Experiment durchgeführt? (odpowiedz: tak)
10. Wurde der Impfstoff entwickelt? (odpowiedz: tak)

<details>
<summary>Klucz</summary>

1. *Ja, ich werde gut behandelt.*
2. *Ja, der Patient ist operiert worden.*
3. *Die Krankheit wird mit einer neuen Therapie behandelt.*
4. *Ja, das Medikament ist getestet worden.*
5. *Das Experiment wird im Labor durchgeführt.*
6. *Das Gen ist von einem deutschen Wissenschaftler entdeckt worden.*
7. *Ja, er ist untersucht worden.*
8. *Ja, sie ist verschrieben worden.*
9. *Ja, es ist durchgeführt worden.*
10. *Ja, er ist entwickelt worden.*

</details>

## Ściąga z lekcji

```
PASSIV PRÄSENS = werden + Partizip II
PASSIV PERFEKT = sein + Partizip II + worden (BEZ ge-!)

werden: werde / wirst / wird / werden / werdet / werden

Kto robi?  → von + Dativ      (= polskie „przez kogo")
Czym?      → durch + Akkusativ (= polskie „czym, za pomocą czego")

Aktiv → Passiv: Akk-obiekt → Nominativ, czasownik → Partizip II na koniec
Tylko czasowniki przechodnie (z Akk) → osobowy Passiv!

POLSKA ANALOGIA:
 Aktiv:        Lekarz operuje pacjenta.
 Passiv Präs:  Pacjent JEST operowany.       → Der Patient WIRD operiert.
 Passiv Perf:  Pacjent ZOSTAŁ zoperowany.    → Der Patient IST operiert WORDEN.

 Polskie „być + imiesłów"   = niemieckie „werden + Partizip II"
 Polskie „zostać + imiesłów" = niemieckie „sein + Partizip II + worden"

Słownictwo: Forschung (f), Therapie (f), Experiment (n), Medikament (n),
Labor (n), Impfstoff (m), entdecken, entwickeln, untersuchen, behandeln,
durchführen, testen, beweisen, veröffentlichen
```
