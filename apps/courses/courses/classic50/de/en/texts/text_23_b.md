# Lesson 23 text (B): Future tense. The verb werden
**Topic: Plans and dreams**

---

## Listening and repetition text

1. Nächstes Jahr werde ich Deutsch besser sprechen.
2. Mein Freund wird im Juli heiraten.
3. Wirst du morgen früh aufstehen?
4. Wir werden im Sommer ein Auto kaufen.
5. Sie wird nächste Woche einen neuen Job anfangen.
6. Ich werde eines Tages ein Haus am See bauen.
7. Er wird bald seine Prüfung bestehen.
8. Werdet ihr nächsten Monat umziehen?
9. Meine Eltern werden im Herbst nach Wien reisen.
10. Ich werde morgen einen Kuchen backen.
11. Du wirst das bestimmt gut schaffen.
12. Sie wird im September ein Studium beginnen.
13. Wir werden am Wochenende Freunde einladen.
14. Er wird wohl den ganzen Tag lernen.
15. Ich verspreche, ich werde pünktlich sein.
16. Meine Schwester wird nächstes Jahr Ärztin.
17. Wirst du im Winter Ski fahren lernen?
18. Wir werden zusammen ein Geschäft eröffnen.
19. Mein Traum ist, dass ich Pilot werde.
20. Im Sommer wird es wieder heiß und sonnig.
21. Sie werden nächste Woche in eine neue Wohnung ziehen.
22. Ich werde jeden Tag eine Stunde Sport machen.
23. Er wird wahrscheinlich im Ausland arbeiten.
24. Wann wirst du dein Studium abschließen?
25. Wir werden im Dezember Weihnachten bei Oma feiern.
26. Mein Sohn wird bald acht Jahre alt.
27. Alles wird besser, du musst nur warten.
28. Sie wird sich für einen Sprachkurs anmelden.
29. Ich werde diesen Sommer schwimmen lernen.
30. Eines Tages werden wir die Welt bereisen.

---

## Vocabulary and phrases

| German | English |
|----------|---------|
| werden + Infinitiv | future tense |
| heiraten | to get married |
| umziehen | to move (house) |
| bestehen | to pass (an exam) |
| schaffen | to manage, to accomplish |
| eröffnen | to open (a business) |
| abschließen | to complete, to finish |
| sich anmelden | to register, to sign up |
| bereisen | to travel around |
| der Sprachkurs | language course |
| die Prüfung | exam |
| das Geschäft | business, shop |
| der Pilot | pilot |
| das Ausland | abroad |
| pünktlich | on time, punctual |
| wahrscheinlich | probably |
| bald | soon |
