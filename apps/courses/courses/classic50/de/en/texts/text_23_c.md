# Lesson 23 text (C): Future tense werden — Plans for life

---

## Listening and repetition text

1. Ich werde nächstes Jahr Deutsch studieren.
2. Mein Freund wird eine eigene Firma gründen.
3. Wirst du im Herbst einen Kurs besuchen?
4. Wir werden zusammen ein Haus bauen.
5. Sie wird bald ihren Führerschein machen.
6. Im Juli werden wir nach Kanada fliegen.
7. Er wird wohl einen neuen Job finden.
8. Ich werde jeden Tag eine Stunde lesen.
9. Werdet ihr nächstes Jahr heiraten?
10. Sie werden ein Baby bekommen.
11. Ich werde mehr Geld für Bücher sparen.
12. Mein Bruder wird Medizin studieren.
13. Du wirst das Examen bestimmt bestehen.
14. Wir werden im Mai in eine neue Stadt ziehen.
15. Sie wird ihren Traumjob sicher bekommen.
16. Ich werde am Wochenende nichts tun.
17. Er wird morgen den Vertrag unterschreiben.
18. Wir werden uns öfter bei Oma treffen.
19. Wirst du dein altes Auto verkaufen?
20. Meine Schwester wird nach Wien umziehen.
21. Es wird nächste Woche viel regnen.
22. Ich werde weniger Zucker essen.
23. Sie wird im Sommer einen Sprachkurs belegen.
24. Wir werden endlich den Garten neu gestalten.
25. Er wird wohl bei der Bank arbeiten.
26. Alles wird sich zum Guten ändern.
27. Ich werde mir ein neues Fahrrad kaufen.
28. Im Winter werden wir Ski fahren lernen.
29. Meine Eltern werden bald in Rente gehen.
30. Du wirst sehen, es wird alles klappen!

| German | English |

| werden + Infinitiv | future tense |
| gründen | to found, to establish |
| der Führerschein | driving licence |
| heiraten | to get married |
| sparen | to save (money) |
| bestehen | to pass (an exam) |
| der Vertrag | contract |
| unterschreiben | to sign |
| umziehen | to move (house) |
| belegen | to sign up for (a course) |
| gestalten | to design, to arrange |
| sich ändern | to change |
| in Rente gehen | to retire |
| klappen | to work out, to succeed |
| der Traumjob | dream job |
| das Examen | exam |
| der Sprachkurs | language course |
