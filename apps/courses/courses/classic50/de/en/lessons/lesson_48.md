# Lesson 48: Conjunctions indem and so dass
**Vocabulary: Public speaking**

---

## How to work with this lesson

1. **Read** the rule — understand the logic (5 minutes)
2. **Translate** the exercises in writing — check against the key
3. **Say it out loud** — slowly, consciously, watching the word order in every subclause
4. **Speed up** — drill the matrix until it flies out on autopilot

> You now own Plusquamperfekt and can chain past events in order. Time to add two conjunctions without which coherent speech is impossible: **indem** (in what way?) and **so dass** (with what result?). These are the tools of argument and presentation.

---

## Part 1: indem — "by doing X"

In English you say: "I improved my German **by reading** every day." German has a dedicated conjunction for this: **indem**.

> Ich habe mein Deutsch verbessert, **indem** ich jeden Tag **gelesen habe**.
> I improved my German by reading every day.

indem answers the question **Wie? Auf welche Weise?** — how? in what way?

The main rule: in the indem-clause, the verb goes **to the end**.

> Man lernt eine Sprache, **indem** man jeden Tag **übt**.
> You learn a language by practicing every day.

---

## Part 2: The hack — indem = "by + -ing"

If you're an English speaker, here's your anchor:

> **indem = by (doing something)**
> Man überzeugt andere, indem man gute Argumente bringt.
> You convince others **by** bringing good arguments.

That's it. Wherever English uses "by + -ing", German uses indem. Direct mapping.

indem always describes the **method**. Not the goal, not the result, but **how you do it**.

---

## Part 3: indem vs. durch + Akkusativ

The same idea can be expressed two ways:

| Construction | Example | English |
|-------------|--------|---------|
| **indem** + clause | Man lernt, **indem** man viel **liest** | You learn by reading a lot |
| **durch** + Akk (noun) | Man lernt **durch** viel**es** Lesen | You learn through a lot of reading |

Three takeaways:

1. **indem** needs a full clause with subject and verb at the end.
2. **durch + Akkusativ** needs a noun (often a nominalized infinitive: das Lesen, das Üben, das Arbeiten).
3. In speech **indem** sounds more natural. **Durch** is the bookish version.

> **Trap!** Don't confuse indem with damit. **Indem** = method (how?). **Damit** = purpose (why?).
> Er spricht langsam, **indem** er jedes Wort betont. (Method: how exactly does he speak slowly?)
> Er spricht langsam, **damit** alle ihn verstehen. (Purpose: why is he speaking slowly?)

---

## Part 4: so dass / sodass — "so that" (result)

Now the second construction. In English: "He spoke loudly, **so that** everyone heard him."

> Er hat laut gesprochen, **so dass** alle ihn gehört haben.

so dass (also written as one word: **sodass**) introduces the **result** of the previous action. Direct cognate of English "so that" — same word, same job.

> Die Präsentation war gut strukturiert, **sodass** das Publikum alles **verstanden hat**.
> The presentation was well-structured, so that the audience understood everything.

---

## Part 5: Triple contrast — damit / so dass / indem

This is the key distinction. Drill it into your head:

| Conjunction | Meaning | Question | Example |
|------|----------|--------|--------|
| **damit** | purpose, intention (so that, in order to) | Wozu? | Er übt, **damit** er besser **wird** |
| **so dass** | actual result (so that, with the result that) | Mit welchem Ergebnis? | Er hat viel geübt, **so dass** er besser **geworden ist** |
| **indem** | method, means (by doing) | Wie? | Er wird besser, **indem** er jeden Tag **übt** |

> **Anchor:** damit = **I want** (purpose). so dass = **it happened** (result). indem = **I do it this way** (method).

The same situation, three angles:

> Ich bereite mich vor, **damit** mein Vortrag gelingt. (Purpose: so it succeeds.)
> Ich habe mich vorbereitet, **so dass** mein Vortrag gelungen ist. (Result: so it succeeded.)
> Ich bereite mich vor, **indem** ich jeden Punkt durchgehe. (Method: by going through each point.)

## Part 7: Language scales

### Scale 1 — indem + überzeugen (method of convincing)

```
Ich überzeuge das Publikum, indem ich klare Argumente bringe
Du überzeugst das Publikum, indem du klare Argumente bringst
Er überzeugt das Publikum, indem er klare Argumente bringt
Sie überzeugt das Publikum, indem sie klare Argumente bringt
Wir überzeugen das Publikum, indem wir klare Argumente bringen
Ihr überzeugt das Publikum, indem ihr klare Argumente bringt
Sie überzeugen das Publikum, indem sie klare Argumente bringen
```

> Note: the subject of the indem-clause **matches** the subject of the main clause. This is the typical setup.

### Scale 2 — so dass + Ergebnis (result of the talk)

```
Ich habe laut gesprochen, so dass alle mich gehört haben
Du hast laut gesprochen, so dass alle dich gehört haben
Er hat laut gesprochen, so dass alle ihn gehört haben
Sie hat laut gesprochen, so dass alle sie gehört haben
Wir haben laut gesprochen, so dass alle uns gehört haben
Ihr habt laut gesprochen, so dass alle euch gehört haben
Sie haben laut gesprochen, so dass alle sie gehört haben
```

### Scale 3 — durch + Akkusativ (the bookish "method" variant)

```
Ich verbessere meinen Vortrag durch ständiges Üben
Du verbesserst deinen Vortrag durch ständiges Üben
Er verbessert seinen Vortrag durch ständiges Üben
Sie verbessert ihren Vortrag durch ständiges Üben
Wir verbessern unseren Vortrag durch ständiges Üben
Ihr verbessert euren Vortrag durch ständiges Üben
Sie verbessern ihren Vortrag durch ständiges Üben
```

> Here durch + Akkusativ: ständig**es** Üben — a nominalized infinitive, neuter (das Üben).

## Part 8: Language matrix

### Matrix 1 — "You ↔ I" (indem)

```
Wie bereitest du dich auf den Vortrag vor? – Ich bereite mich vor, indem ich die Gliederung schreibe
Wie fasst du die Ergebnisse zusammen? – Ich fasse die Ergebnisse zusammen, indem ich die Hauptpunkte wiederhole
Wie überzeugst du die Zuhörer? – Ich überzeuge sie, indem ich Beweise liefere
Wie verbesserst du deine Aussprache? – Ich verbessere sie, indem ich jeden Tag laut lese
Wie lernst du neue Vokabeln? – Ich lerne sie, indem ich Karteikarten benutze
Wie bleibst du ruhig bei der Präsentation? – Ich bleibe ruhig, indem ich tief durchatme
```

### Matrix 2 — Third persons (so dass + result)

```
Hat der Redner gut gesprochen? – Ja, er hat so klar gesprochen, dass alle ihn verstanden haben
Hat sie ihre These gut begründet? – Ja, sie hat sie so gut begründet, dass niemand widersprochen hat
Haben die Studenten viel geübt? – Ja, sie haben so viel geübt, dass der Vortrag perfekt war
Hat der Professor laut genug gesprochen? – Ja, er hat laut gesprochen, so dass auch die letzte Reihe alles gehört hat
Hat die Rednerin das Publikum überzeugt? – Ja, sie hat es überzeugt, so dass alle applaudiert haben
Haben die Teilnehmer gute Argumente gebracht? – Ja, so dass die Diskussion sehr produktiv war
```

### Matrix 3 — Debate: argument and counterargument

```
Meiner Meinung nach brauchen wir mehr Übung. – Da widerspreche ich. Wir brauchen bessere Methoden
Ich bin der Meinung, dass Theorie wichtiger ist. – Im Gegenteil. Man lernt besser, indem man praktiziert
Können Sie Ihren Standpunkt begründen? – Ja. Studien zeigen, dass Praxis effektiver ist
Wie kommen Sie zu diesem Schluss? – Ich schlussfolgere das aus den Ergebnissen der Forschung
Lassen Sie mich zusammenfassen. – Bitte, fassen Sie zusammen
Haben Sie noch ein Argument? – Ja. Ich möchte noch einen Beweis anführen
```

## Part 9: Exercises

### Exercise 1. Insert the right conjunction: indem, so dass, or damit

Pick the conjunction that fits the meaning.

1. Er hat den Vortrag gut strukturiert, ___ das Publikum alles verstanden hat.
2. Man wird ein guter Redner, ___ man regelmäßig vor Leuten spricht.
3. Sie spricht langsam, ___ alle folgen können.
4. Ich bereite Folien vor, ___ meine Präsentation anschaulich wird.
5. Der Student hat seine These überzeugend dargestellt, ___ der Professor beeindruckt war.
6. Du kannst dein Argument stärken, ___ du konkrete Beispiele nennst.
7. Sie fasste alles zusammen, ___ sie die Hauptpunkte noch einmal nannte.

<details>
<summary>Key</summary>

1. *so dass* (result: so that the audience understood everything)
2. *indem* (method: by speaking in front of people regularly)
3. *damit* (purpose: so everyone can follow)
4. *damit* (purpose: so the presentation is vivid)
5. *so dass* (result: so the professor was impressed)
6. *indem* (method: by giving concrete examples)
7. *indem* (method: by repeating the main points)

</details>

### Exercise 2. Translate to German

1. You learn German by listening to podcasts every day.
2. She spoke so convincingly that everyone agreed with her.
3. He prepares for the talk by reading scientific articles. (use indem)
4. The speaker summarized so that the discussion could begin.
5. I improve my speech through constant practice. (use durch)
6. We can conclude that this argument is stronger.
7. In my opinion, more evidence needs to be presented.
8. She contradicted the speaker by bringing a counter-argument. (use indem)

<details>
<summary>Key</summary>

1. *Du lernst Deutsch, indem du jeden Tag Podcasts hörst.*
2. *Sie hat so überzeugend gesprochen, dass alle ihr zugestimmt haben.* (or: *…, so dass alle ihr zugestimmt haben.*)
3. *Er bereitet sich auf den Vortrag vor, indem er wissenschaftliche Artikel liest.*
4. *Der Redner hat zusammengefasst, so dass die Diskussion beginnen konnte.*
5. *Ich verbessere meine Rede durch ständige Praxis.*
6. *Wir können schlussfolgern, dass dieses Argument stärker ist.*
7. *Meiner Meinung nach muss man mehr Beweise anführen.*

</details>

### Exercise 3. Transformation: indem → durch + Akkusativ (and back)

Rephrase the sentences, keeping the meaning.

1. Man überzeugt andere, **indem** man Fakten präsentiert. → Man überzeugt andere **durch**…
2. Sie verbessert ihre Aussprache **durch** tägliches Vorlesen. → Sie verbessert ihre Aussprache, **indem**…
3. Er stärkt sein Argument, **indem** er Statistiken zitiert. → Er stärkt sein Argument **durch**…
4. Wir lernen **durch** regelmäßige Wiederholung. → Wir lernen, **indem**…

<details>
<summary>Key</summary>

1. *Man überzeugt andere durch die Präsentation von Fakten.* (or: *durch das Präsentieren von Fakten*)
2. *Sie verbessert ihre Aussprache, indem sie jeden Tag laut vorliest.*
3. *Er stärkt sein Argument durch das Zitieren von Statistiken.*
4. *Wir lernen, indem wir regelmäßig wiederholen.*

</details>

### Exercise 4. Contrast damit / so dass — translate the pairs

1. He speaks slowly so that everyone understands. — He spoke slowly, so that everyone understood.
2. She prepares a handout so it's more vivid. — She prepared a handout, so that it became more vivid.

<details>
<summary>Key</summary>

1. *Er spricht langsam, **damit** alle verstehen. — Er hat langsam gesprochen, **so dass** alle verstanden haben.*
2. *Sie bereitet ein Handout vor, **damit** es anschaulicher wird. — Sie hat ein Handout vorbereitet, **so dass** es anschaulicher geworden ist.*

</details>

## Lesson cheat sheet

```
INDEM = METHOD (wie?) → verb AT THE END
  = English "by + -ing"
  Man lernt, indem man übt
  (You learn by practicing)

SO DASS / SODASS = RESULT (mit welchem Ergebnis?) → verb AT THE END
  = English "so that" (result)
  Er sprach laut, so dass alle ihn hörten

DAMIT = PURPOSE (wozu?) → verb AT THE END
  = English "so that" (intention)
  Er spricht laut, damit alle ihn hören

INDEM vs DURCH:
  indem + clause: ..., indem man viel liest
  durch + Akk (noun): ...durch vieles Lesen

Vocabulary:
der Vortrag (m) — talk           das Argument (n) — argument
die Zusammenfassung (f) — summary  schlussfolgern — to conclude
überzeugen — to convince          widersprechen (+D) — to contradict
begründen — to justify            darstellen — to present
```
