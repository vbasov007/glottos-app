# Lesson 4: Regular verbs (machen, lernen, wohnen, arbeiten, spielen)
**Vocabulary: Professions**

---

## How to work with this lesson

1. **Read** — understand the rule (5 minutes, no more!)
2. **Say it out loud** — slowly, consciously, every ending
3. **Speed up** — repeat until the phrases fly out on their own

> You already know sein and haben. Now — regular verbs. Good news: 80% of German verbs conjugate the same way. Learn one pattern — and you've got 80% of all verbs in your pocket.

---

## Part 1: The conjugation pattern

In English, verbs barely change: I have, you have, **he has**, we have, they have. Only the 3rd-person singular gets an "-s". Easy.

German is more honest about it. Each person gets its own ending. But there are only **three** endings to memorize. The rest of the forms = the infinitive itself.

Any regular verb in the infinitive ends in **-en**: <say>mach**en**</say>, <say>lern**en**</say>, <say>wohn**en**</say>.

To conjugate:

1. Drop **-en** → get the stem: <say>**mach-**</say>, <say>**lern-**</say>, <say>**wohn-**</say>
2. Add the ending for the person.

| Pronoun | Ending | machen (to make/do) |
|---|---|---|
| <say>**ich**</say> | **-e** | <say>mach**e**</say> |
| <say>**du**</say> | **-st** | <say>mach**st**</say> |
| <say>**er / sie / es**</say> | **-t** | <say>mach**t**</say> |
| <say>**wir**</say> | **-en** | <say>mach**en**</say> |
| <say>**ihr**</say> | **-t** | <say>mach**t**</say> |
| <say>**Sie / sie**</say> (formal / they) | **-en** | <say>mach**en**</say> |

This is the only table you have to memorize. That's it.

Cognate: <say>**machen**</say> ≈ "to make". German **m-ch** ↔ English **m-k**. Listen for the resemblance.

---

## Part 2: The main hack — three endings

> **ich = -e, du = -st, er = -t. Everything else = infinitive.**

Unpacked:

- <say>**ich**</say> → **-e** (the only unique one)
- <say>**du**</say> → **-st** (always sounds like "-st") ← cognate alert: thou shal**st**, thou ha**st**. Old English used "-st" for "thou" too!
- <say>**er / sie / es**</say> → **-t** (one letter, third person)
- <say>**wir**</say> → **-en** (= infinitive)
- <say>**ihr**</say> → **-t** (same as <say>er</say>)
- <say>**Sie / sie**</say> → **-en** (= infinitive)

So you really only need three endings: **-e**, **-st**, **-t**. The rest = infinitive.

The "thou-st" connection is real. English had "thou hast / thou makest / thou learnst". Then English ditched "thou" and lost the system. German kept it.

---

## Part 3: The trap — stems ending in -t, -d, -n

Some verbs have stems ending in **-t**, **-d**, or **-n** (after a consonant): <say>**arbeit-en**</say>, <say>**find-en**</say>, <say>**öffn-en**</say>.

Try saying "<say>du arbeitst</say>" — you'll break your tongue. So Germans slip in an **-e-** before the **-st** and **-t** endings.

| Person | machen (normal) | arbeiten (-t stem) |
|--------|------------------|---------------------|
| <say>ich</say> | <say>mach**e**</say> | <say>arbeit**e**</say> |
| <say>du</say> | <say>mach**st**</say> | <say>arbeit**e**st</say> |
| <say>er/sie/es</say> | <say>mach**t**</say> | <say>arbeit**e**t</say> |
| <say>wir</say> | <say>mach**en**</say> | <say>arbeit**en**</say> |
| <say>ihr</say> | <say>mach**t**</say> | <say>arbeit**e**t</say> |
| <say>sie/Sie</say> | <say>mach**en**</say> | <say>arbeit**en**</say> |

> **Trap!** <say>du mach**st**</say> — but <say>du arbeit**e**st</say>. <say>Er mach**t**</say> — but <say>er arbeit**e**t</say>.
> The little **-e-** rescues you from a consonant pile-up. Same instinct as why English says "wash**es**" instead of "washs".

Same trick with <say>**finden**</say> (to find — cognate: find!): <say>du find**e**st</say>, <say>er find**e**t</say>. And <say>**öffnen**</say> (to open — cognate: open): <say>du öffn**e**st</say>, <say>er öffn**e**t</say>.

## Part 4: Key verbs for this lesson

Most of these have English cousins. Spot them.

| Verb | English | Cognate hint | Example |
|------|---------|--------------|---------|
| <say>machen</say> | to make / to do | ≈ "make" | <say>Ich mache die Hausaufgabe.</say> |
| <say>lernen</say> | to learn / to study | ≈ "learn" | <say>Du lernst Deutsch.</say> |
| <say>wohnen</say> | to live (reside) | ≈ "to dwell" (old "wonen") | <say>Er wohnt in Hamburg.</say> |
| <say>arbeiten</say> | to work | "<say>Arbeit</say>" → English "arbeitsmacht-frei", but no direct cognate. Just learn it. | <say>Sie arbeitet als Ärztin.</say> |
| <say>spielen</say> | to play | ≈ "play" (verb form. <say>**spiel**</say> = play/game) | <say>Wir spielen Fußball.</say> |
| <say>trinken</say> | to drink | ≈ "drink" | <say>Ihr trinkt Kaffee.</say> |
| <say>kaufen</say> | to buy | ≈ "shop" (no, but "<say>Kauf</say>" = purchase) | <say>Sie kaufen Brot.</say> |
| <say>hören</say> | to hear / listen | ≈ "hear" | <say>Ich höre Musik.</say> |
| <say>kochen</say> | to cook | ≈ "cook" | <say>Du kochst gut.</say> |
| <say>fragen</say> | to ask | (no English cognate) | <say>Er fragt den Lehrer.</say> |

## Part 6: Language scales

### Scale 1 — wohnen

```
Ich wohne in Berlin
Du wohnst in Berlin
Er wohnt in Berlin
Sie wohnt in Berlin
Wir wohnen in Berlin
Ihr wohnt in Berlin
Sie wohnen in Berlin
```

Repeat 3 times. On the fourth — swap "<say>Berlin</say>" for "<say>München</say>", "<say>Hamburg</say>", "<say>Köln</say>".

### Scale 2 — arbeiten (mind the -e-)

```
Ich arbeite als Lehrer
Du arbeitest als Lehrer
Er arbeitet als Lehrer
Sie arbeitet als Ärztin
Wir arbeiten als Ingenieure
Ihr arbeitet als Kellner
Sie arbeiten als Polizisten
```

> **The trap returns:** <say>du arbeit**e**st</say>, <say>er arbeit**e**t</say>, <say>ihr arbeit**e**t</say>. Don't skip that **-e-**.

### Scale 3 — spielen + object

```
Ich spiele jeden Tag Fußball
Du spielst jeden Tag Fußball
Er spielt jeden Tag Fußball
Sie spielt jeden Tag Gitarre
Wir spielen jeden Tag Tennis
Ihr spielt jeden Tag Klavier
Sie spielen jeden Tag Volleyball
```

## Part 7: Language matrix

### Matrix 1 — "You ↔ I"

```
Wo wohnst du? – Ich wohne in München
Was machst du? – Ich lerne Deutsch
Was trinkst du? – Ich trinke Tee
Was spielst du? – Ich spiele Gitarre
Was kochst du heute? – Ich koche Suppe
Wo arbeitest du? – Ich arbeite in einem Büro
```

### Matrix 2 — Third persons

```
Was macht er beruflich? – Er arbeitet als Ingenieur
Wo wohnt sie? – Sie wohnt in Frankfurt
Was lernt er? – Er lernt Spanisch
Was kauft sie? – Sie kauft Milch und Brot
Was hört er? – Er hört Musik
Wo arbeiten sie? – Sie arbeiten in einem Krankenhaus
```

### Matrix 3 — Situational (profession)

```
Was sind Sie von Beruf? – Ich bin Programmierer. Ich arbeite bei Siemens
Was machst du beruflich? – Ich bin Student. Ich lerne Medizin
Wo arbeitet dein Vater? – Er arbeitet als Mechaniker
Was macht deine Mutter? – Sie arbeitet als Krankenschwester
Wo wohnt deine Familie? – Meine Familie wohnt in Berlin
Was kochst du am Wochenende? – Ich koche Pasta und Pizza
```

## Lesson cheat sheet

```
REGULAR VERBS = STEM + ENDING

Infinitive = stem + -en (mach + en)

Endings:
  ich    → -e
  du     → -st     (echoes old English "thou hast / makest")
  er/sie → -t
  wir    → -en
  ihr    → -t
  sie/Sie→ -en

Stem in -t / -d / -n (after consonant) → slip in -e-:
  du arbeitest, er arbeitet, ihr arbeitet
  du findest, er findet
  du öffnest, er öffnet

Verbs this lesson:
  machen (make), lernen (learn), wohnen (dwell), arbeiten (work),
  spielen (play), trinken (drink), kaufen (buy), hören (hear),
  kochen (cook), fragen (ask)

Professions (-er → -erin for female):
  Lehrer, Arzt, Ingenieur, Fahrer, Kellner, Verkäufer,
  Koch, Programmierer, Polizist, Mechaniker, Student,
  Musiker, Friseur, Krankenschwester

Key phrases:
  Was sind Sie von Beruf?
  Was machst du beruflich?
  Ich bin ... von Beruf.
```
