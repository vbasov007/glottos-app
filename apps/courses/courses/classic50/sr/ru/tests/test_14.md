# Тест к уроку 14: Творительный падеж — средство, сопровождение

**Как сказать по-сербски?**
Переведи фразы на сербский язык. Пиши **только кириллицей**. Используй лексику и грамматику урока.

> Главное: **средство** — без предлога (*пишем оловком*), **сопровождение** — с *са* (*идем са братом*). Транспорт — это средство, без предлога!

---

1. Я пишу карандашом.
2. Режу ножом.
3. Ем ложкой.
4. Открываю дверь ключом.
5. Чищу тряпкой.
6. Стираю ластиком.
7. Работаю на компьютере. (как инструментом)
8. Звоню по телефону. (= телефоном)
9. Я еду поездом в Белград.
10. Мы едем автобусом.
11. Лечу самолётом.
12. Приезжаю на машине. (*кола*)
13. Катаюсь на велосипеде.
14. Иду с братом в школу.
15. Говорю с другом.
16. Живу с родителями.
17. Пью кофе с молоком.
18. Хлеб с сыром.
19. Еду с сестрой в Нови-Сад.
20. Учусь с подругой.
21. Дети играют с собакой. (*пас* → *псом*)
22. Я работаю с коллегой. (спутник)
23. Бабушка пьёт чай с лимоном. (*лимун*)
24. Занимаюсь спортом.
25. Стал инженером. (*постати* + твор.)
26. Считаю его другом. (*сматрати*)
27. Кот спит под столом.
28. Сад за домом.
29. Стою перед школой.
30. Сижу среди друзей.

---

1. *Пишем оловком.* (или *Ја пишем оловком*)
2. *Сечем ножем.* (мягк. → -ем)
3. *Једем кашиком.*
4. *Отварам врата кључем.* (мягк. → -ем)
5. *Чистим крпом.*
6. *Бришем гумицом.*
7. *Радим компјутером.* (средство — без *са*!)
8. *Зовем телефоном.* (средство)
9. *Путујем возом у Београд.* (транспорт = средство)
10. *Идемо аутобусом.*
11. *Летим авионом.*
12. *Долазим колима.* (*кола* — только мн. → -има)
13. *Возим се бициклом.*
14. *Идем са братом у школу.* (сопровождение → *са*)
15. *Причам са пријатељем.* (мягк. → -ем)
16. *Живим са родитељима.* (мн. → -има)
17. *Пијем кафу са млеком.* (сопровождение, добавка)
18. *Хлеб са сиром.*
19. *Путујем са сестром у Нови Сад.*
20. *Учим са другарицом.*
21. *Деца се играју са псом.* (*пас* → твор. *псом*)
22. *Радим са колегом.* (спутник, не инструмент!)
23. *Бака пије чај са лимуном.*
24. *Бавим се спортом.* (*бавити се* + твор. без предлога)
25. *Постао је инжењером.* (мягк. → -ем)
26. *Сматрам га пријатељем.*
27. *Мачак спава под столом.*
28. *Башта је за кућом.*
29. *Стојим пред школом.*
30. *Седим међу пријатељима.*
