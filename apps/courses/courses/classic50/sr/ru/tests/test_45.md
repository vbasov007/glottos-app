# Тест к уроку 45: Продукция сложных многоклаузных предложений

**Как сказать по-сербски?**
Переведи на сербский, сохраняя длину и структуру оригинала. Пиши **только кириллицей**. Следи за порядком клитик (вторая позиция в каждой клаузе), согласованием падежей через всё предложение и правильной формой *који*.

29. Спорный вопрос, который обсуждался на конференции в течение трёх дней, остался без окончательного ответа.
30. Считаю, что, несмотря на сложность темы, студенты, посещавшие мои лекции, смогут самостоятельно проанализировать предложенный текст.
