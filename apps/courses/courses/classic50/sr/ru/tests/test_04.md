# Тест к уроку 4: Личные местоимения. Глагол *бити* — полные и краткие формы

**Как сказать по-сербски?**
Переведи фразы на сербский язык. Пиши **только кириллицей**. Используй полные и краткие формы глагола *бити*; помни про правило клитики на второй позиции.

---

1. Я студент.
2. Я студентка.
3. Ты из Белграда.
4. Она врач.
5. Мы туристы.
6. Они сербы.
7. Вы профессор. (вежливо, к одному мужчине)
8. Ана — моя сестра.
9. Никола — врач, а Ана — учительница.
10. Я из России, из Москвы.
11. Она из Нови-Сада.
12. Они из Сербии.
13. Сегодня я устал.
14. Мы счастливые.
15. Я не из Белграда, я из Нови-Сада.
16. Её здесь нет.
17. Я не студент. Я профессор.
18. — Ты студент? — Да. *(одиночный ответ)*
19. — Вы из России? — Да. *(вежливо, к одной женщине; одиночный ответ)*
20. — Они из Белграда? — Да. *(одиночный ответ)*
21. Я **и есть** студент! *(эмфаза)*
22. Они **действительно** из Сербии. *(эмфаза)*
23. Ты Ана? *(вопрос с *ли*)*
24. Вы из Белграда? *(вежливо, к одному; вопрос с *ли*)*
25. Я русский. (мужчина)
26. Я русская.
27. Они немцы.
28. Она хорватка.
29. Мы из Москвы.
30. — Здравствуйте! Откуда Вы? — Я из Сербии, из Белграда.

<summary>Ключи</summary>

1. *Ја сам студент.*
2. *Ја сам студенткиња.*
3. *Ти си из Београда.*
4. *Она је лекарка.* (или: *Она је лекар.*)
5. *Ми смо туристи.*
6. *Они су Срби.*
7. *Ви сте професор.* (вежливое *Ви* — с большой буквы, глагол во множественном)
8. *Ана је моја сестра.* (клитика *је* — на 2-й позиции после именной группы *Ана*)
9. *Никола је лекар, а Ана је учитељица.* (клитика *је* — на 2-й позиции в **каждой** клаузе)
10. *Ја сам из Русије, из Москве.*
11. *Она је из Новог Сада.*
12. *Они су из Србије.*
13. *Данас сам уморан.* (1-я позиция: *данас*; клитика *сам* — на 2-й)
14. *Ми смо срећни.*
15. *Ја нисам из Београда, ја сам из Новог Сада.* (отрицание *нисам* — не клитика)
16. *Она није овде.*
17. *Ја нисам студент. Ја сам професор.*
18. — *Јесам.* (одиночный утвердительный ответ → полная форма)
19. — *Јесам.* (одиночный ответ; *Ви* вежливое к одной, но отвечающий говорит о себе → 1 ед.)
20. — *Јесу.* (одиночный ответ → полная форма)
21. *Ја јесам студент!* (эмфаза → полная форма)
22. *Они јесу из Србије.* (эмфаза → полная форма)
23. *Јеси ли ти Ана?* (вопрос с *ли* → полная форма *јеси*)
24. *Јесте ли Ви из Београда?* (вежливое *Ви* — с большой буквы, глагол во мн. → *јесте*)
25. *Ја сам Рус.*
26. *Ја сам Рускиња.*
27. *Они су Немци.*
28. *Она је Хрватица.*
29. *Ми смо из Москве.*
30. *Добар дан! Одакле сте Ви? — Ја сам из Србије, из Београда.*
