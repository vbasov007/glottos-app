# Тест к уроку 6: Сербская латиница. Отображение на кириллицу. Диграфы

**Как написать по-сербски?**
Часть заданий — перевод с русского на сербский (пиши **кириллицей**, если не указано иначе).
Часть заданий — *напиши на латинице*: перепиши кириллическое слово сербской **латиницей** с правильными диграфами (*lj, nj, dž, đ*) и диакритикой.
