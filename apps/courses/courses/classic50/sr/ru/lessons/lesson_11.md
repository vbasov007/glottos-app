# Урок 11: Глагольный вид (аспект) — несовершенный и совершенный
**Словарный запас: видовые пары — повседневные глаголы**

---

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — каждую видовую пару, оба партнёра подряд
3. **Учи парами** — отныне ни один новый глагол не входит в твой словарь в одиночку. Только парой: несовершенный + совершенный.

> Знать правило = 5%. Натренировать рот = 95%.
> Это **главное открытие сербского** после кириллицы и падежей. После Урока 11 ты будешь смотреть на глагол и спрашивать: «а где его пара?»

## Часть 1: Главное открытие сербского после падежей

> **Каждый сербский глагол существует ПАРОЙ.** У каждого глагола есть **несовершенный** партнёр (длительность, привычка, процесс) и **совершенный** партнёр (завершённость, граница, результат).

Тебе это уже знакомо — по русскому:

| Русский несов. | Русский сов. |
|----------------|--------------|
| читать | прочитать |
| писать | написать |
| делать | сделать |
| говорить | сказать |

Сербский устроен **точно так же** — и это **отличная новость**. Но есть три важных уточнения, которые делают сербский аспект отдельной темой:

1. **Это ЖЕЛЕЗНОЕ правило**: пара существует у **каждого** глагола, без исключений. В русском ты можешь не помнить вид — в сербском без вида ты не построишь нормальное прошедшее, будущее или повеление.
2. **Вид — свойство САМОГО глагола**, а не времени. *Читати* — несовершенный во всех временах. *Прочитати* — совершенный во всех временах.
3. **Один словарный вход — это половина глагола.** Если ты выучил *читати*, ты выучил половину. Без *прочитати* ты не сможешь сказать «я прочитал книгу» с акцентом на результат.

> **Правило урока 11:** с этого момента в твой словарь не входит ни один глагол в одиночку. Только парой: *читати (нсв) / прочитати (св)*.

## Часть 2: Что значит «несовершенный» и «совершенный»

### Несовершенный (несвршени вид) — НСВ

Несовершенный глагол отвечает на вопросы:
- **«Что делал? Что делаю? Что буду делать?»** (процесс, без указания на конец)
- **«Делал ли вообще?»** (привычка, опыт)
- **«Делал долго / часто / регулярно?»** (длительность, повтор)

Несовершенный глагол **рассказывает о действии как о процессе** — без интереса к тому, дошло ли оно до конца.

| Сербский НСВ | Русский эквивалент | Что описывает |
|--------------|---------------------|---------------|
| **читати** | читать | процесс чтения |
| **писати** | писать | процесс письма |
| **говорити** | говорить | процесс говорения |
| **видети** | видеть | состояние, восприятие |
| **купати (се)** | купаться | процесс купания |
| **радити** | работать, делать | процесс работы |

### Совершенный (свршени вид) — СВ

Совершенный глагол отвечает на вопросы:
- **«Что сделал? Что сделаю?»** (один раз, целиком, до конца)
- **«Дошло ли до результата?»** (граница, точка)

Совершенный глагол **рассказывает о действии как о точке** — действие либо случилось целиком, либо случится целиком. Процесса нет — есть факт.

| Сербский СВ | Русский эквивалент | Что описывает |
|-------------|---------------------|---------------|
| **прочитати** | прочитать | прочёл до конца |
| **написати** | написать | написал и закончил |
| **рећи** | сказать | сказал — точка |
| **угледати** | увидеть | момент, когда увидел |
| **окупати (се)** | искупаться | один раз, целиком |
| **урадити** | сделать | сделал и готово |

### Метафора: процесс vs точка

> Представь видеокамеру.
> **Несовершенный** = камера снимает действие как ленту: ты видишь, как оно идёт.
> **Совершенный** = камера делает один кадр: ты видишь только результат, без процесса.

| Несовершенный (лента) | Совершенный (кадр) |
|------------------------|----------------------|
| читао сам књигу | прочитао сам књигу |
| «я читал книгу» (процесс) | «я прочитал книгу» (до конца, есть результат) |
| писао је писмо | написао је писмо |
| «он писал письмо» | «он написал письмо» |

## Часть 3: Как образуются видовые пары — два механизма

В сербском видовая пара строится двумя способами, и оба знакомы по русскому.

### Механизм 1: ПРЕФИКСАЦИЯ (приставка делает несов. → сов.)

Берём несовершенный глагол и добавляем **приставку** — получаем совершенный.

| Приставка | НСВ | + приставка → СВ | Перевод |
|-----------|-----|-------------------|---------|
| **про-** | читати | **про**читати | прочитать |
| **на-** | писати | **на**писати | написать |
| **по-** | гледати | **по**гледати | посмотреть |
| **на-** | учити | **на**учити | выучить |
| **по-** | јести | **по**јести | съесть |
| **по-** | пити | **по**пити | выпить |
| **с-** | радити | **у**радити | сделать |
| **о-** | купати се | **о**купати се | искупаться |

> **Знакомо по русскому:** *писать → написать*, *читать → прочитать*, *учить → выучить*. В сербском работает **точно тот же** механизм. Префиксы — почти те же буквы, почти те же значения.

**Основные сербские видовые префиксы:**

| Префикс | Сербский пример | Русский аналог |
|---------|------------------|------------------|
| **про-** | прочитати | про-читать |
| **на-** | написати, научити | на-писать, на-учить |
| **по-** | погледати, попити, појести | по-смотреть, по-пить, по-есть |
| **у-** | урадити, ући | сделать, войти |
| **с-** | сачекати | подождать |
| **за-** | запамтити | запомнить |
| **из-** | изговорити | произнести |

### Механизм 2: СУФФИКСАЦИЯ (суффикс возвращает сов. → несов.)

Иногда наоборот: есть **простой совершенный** глагол, а несовершенный делается **удлинением основы** через суффикс (часто **-вати, -ивати, -авати**).

| СВ (короче) | → суффикс → НСВ (длиннее) | Перевод |
|-------------|-----------------------------|----------|
| **дати** | да**вати** | дать → давать |
| **купити** | купо**вати** | купить → покупать |
| **пасти** | пад**ати** | упасть → падать |
| **рећи** | гово**рити** / каз**ивати** | сказать → говорить |
| **бацити** | бац**ати** | бросить → бросать |
| **скочити** | скак**ати** | прыгнуть → прыгать |

> **Запомни направление:**
> Короткий глагол **+ префикс** = совершенный (*читати → прочитати*).
> Простой совершенный **+ суффикс -вати/-авати/-ивати** = несовершенный (*дати → давати*).

### А бывает третий случай — разные корни

У небольшого числа очень частотных глаголов партнёры имеют **разные корни**. Это «супплетивные» пары — их просто учат как факт.

| НСВ | СВ | Перевод |
|-----|-----|---------|
| **говорити** | **рећи** | говорить / сказать |
| **узимати** | **узети** | брать / взять |
| **долазити** | **доћи** | приходить / прийти |
| **стављати** | **ставити** | ставить / поставить |
| **тражити** | **наћи** | искать / найти |

> Сравни с русским: *говорить — сказать*, *брать — взять*, *приходить — прийти*. **Точно такой же** список «странных» пар. Не паникуй — учи как есть.

## Часть 4: Пять видовых пар, которые надо знать прямо сейчас

Это твой стартовый набор. Каждую — **парой**, вслух, оба партнёра подряд.

### Пара 1: читати / прочитати — читать / прочитать

| Сторона | Глагол | Что значит |
|---------|--------|------------|
| НСВ | **читати** | читать (процесс, привычка) |
| СВ | **прочитати** | прочитать (до конца, один раз) |

- *Сваки дан читам новине.* — Я каждый день читаю газеты. (привычка → НСВ)
- *Јуче сам прочитао књигу.* — Я вчера прочитал книгу. (до конца → СВ)

### Пара 2: писати / написати — писать / написать

| Сторона | Глагол | Что значит |
|---------|--------|------------|
| НСВ | **писати** | писать (процесс) |
| СВ | **написати** | написать (готово) |

- *Пишем писмо мајци.* — Я пишу письмо маме. (сейчас, процесс → НСВ)
- *Написала сам писмо.* — Я написала письмо. (готово, результат → СВ)

### Пара 3: говорити / рећи — говорить / сказать

| Сторона | Глагол | Что значит |
|---------|--------|------------|
| НСВ | **говорити** | говорить (вообще, владеть языком, повторять) |
| СВ | **рећи** | сказать (один раз, конкретную фразу) |

- *Говорим српски.* — Я говорю по-сербски. (вообще, владение → НСВ)
- *Рекла сам „хвала“.* — Я сказала «спасибо». (одна фраза → СВ)

> **Внимание:** разные корни! Это супплетивная пара, как русское «говорить — сказать».

### Пара 4: видети / угледати — видеть / увидеть

| Сторона | Глагол | Что значит |
|---------|--------|------------|
| НСВ | **видети** | видеть (длительно, вообще) |
| СВ | **угледати** | увидеть (момент, заметить) |

- *Видим планину сваког дана.* — Я вижу гору каждый день. (постоянно → НСВ)
- *Угледао сам пријатеља на улици.* — Я увидел друга на улице. (момент → СВ)

> Также частая пара: **видети** может работать и как НСВ, и как СВ в значении «увидеть» (двувидовый глагол — тонкости пойдут в L32). Сейчас держим: *видети* — процесс, *угледати* — момент.

### Пара 5: купати (се) / окупати (се) — купаться / искупаться

| Сторона | Глагол | Что значит |
|---------|--------|------------|
| НСВ | **купати се** | купаться (процесс, привычка) |
| СВ | **окупати се** | искупаться (один раз, целиком) |

- *Лети се купам у мору.* — Летом я купаюсь в море. (привычка → НСВ)
- *Окупала сам се и легла.* — Я искупалась и легла. (один раз → СВ)

> *Се* — возвратная частица, как русское «-ся». Она прикрепляется к глаголу в обоих видах.

## Часть 5: Когда брать НСВ, а когда СВ — три правила выбора

### Правило 1: ПРОЦЕСС или ПРИВЫЧКА → НСВ

Если речь о длительности, процессе, повторе, привычке, общем факте — **несовершенный**.

| Сербский | Русский | Маркер |
|----------|---------|---------|
| **Сваки дан читам.** | Каждый день читаю. | привычка |
| **Сада пишем писмо.** | Сейчас пишу письмо. | процесс «сейчас» |
| **Често путујемо.** | Мы часто путешествуем. | повтор |
| **Деца се играју.** | Дети играют. | длительность |

### Правило 2: ОДИН РАЗ, ДО КОНЦА → СВ

Если действие случилось/случится **один раз, целиком, с результатом** — **совершенный**.

| Сербский | Русский | Маркер |
|----------|---------|---------|
| **Прочитала сам књигу.** | Я прочитала книгу. | до конца |
| **Купио сам нов ауто.** | Я купил новую машину. | один раз, есть результат |
| **Рекао је „здраво“.** | Он сказал «привет». | одна фраза |
| **Угледали смо море.** | Мы увидели море. | момент |

### Правило 3: ЦЕПОЧКА событий в прошлом → СВ за СВ

Когда ты рассказываешь, **что случилось** одно за другим — все глаголы **совершенные**.

> **Дошао сам кући, скувао сам кафу, попио сам је и сео да читам.**
> Я пришёл домой, сварил кофе, выпил его и сел читать.
> (Все четыре завершённых действия по очереди → все СВ. Только «читать» — НСВ, потому что это процесс, который только начинается.)

И наоборот — когда ты рассказываешь **фон, обстановку**, где всё длилось — все глаголы **несовершенные**.

> **Седео сам, читао новине и пио кафу.**
> Я сидел, читал газеты и пил кофе. (фон, всё длилось → НСВ)

## Часть 6: Вид — свойство глагола, не времени

Это самое важное концептуальное правило урока.

> **Вид прикреплён к самому глаголу**, а не ко времени. Ты выбираешь вид — а потом вид «прокручивается» через все времена.

Возьми пару *читати / прочитати*:

| Время | НСВ (читати) | СВ (прочитати) |
|-------|---------------|------------------|
| Прошлое | *читао сам* — я читал (процесс) | *прочитао сам* — я прочитал (до конца) |
| Настоящее | *читам* — я читаю (сейчас, регулярно) | — *(СВ почти не имеет «настоящего» в обычном смысле)* |
| Будущее | *читаћу* — буду читать | *прочитаћу* — прочитаю (до конца) |

> **Важный нюанс:** совершенный вид в настоящем времени **в самостоятельном предложении** не работает в значении «происходит сейчас». *Прочитам* нельзя сказать про «читаю сейчас». Если СВ стоит в форме настоящего, это обычно означает **будущее** или работает внутри придаточного (после *кад, ако, да* — это пойдёт в Уроке 22).

Это та же логика, что и в русском:
- «Я читаю» — нсв (процесс сейчас).
- «Я прочитаю» — это уже **будущее**, потому что у совершенного нет «настоящего».

## Часть 7: Как видеть пару — практические подсказки

### Подсказка 1: длина слова

Часто видовые партнёры **отличаются длиной**:
- **Короче = совершеннее**, длиннее = несовершеннее (если есть -вати, -ивати).
  - *дати* (СВ, 4 буквы) — *давати* (НСВ, 6 букв)
  - *купити* (СВ) — *куповати* (НСВ)
- **Без приставки = несовершеннее**, с приставкой = совершеннее.
  - *читати* (НСВ) — *прочитати* (СВ)
  - *писати* (НСВ) — *написати* (СВ)

### Подсказка 2: приставка-сигнал

Видишь сербский глагол с приставкой **про-, на-, по-, у-, с-, за-, из-, о-** — высока вероятность, что это **совершенный** партнёр.

### Подсказка 3: -вати/-ивати/-авати в конце

Видишь длинное окончание **-вати, -ивати, -авати** — почти наверняка **несовершенный**.

### Подсказка 4: маркер в предложении

| Маркер | → выбор |
|--------|---------|
| *сваки дан*, *често*, *увек*, *обично* | НСВ (привычка) |
| *сада*, *тренутно* | НСВ (процесс сейчас) |
| *јуче*, *прошле недеље* (но был один раз) | часто СВ |
| *одједном*, *изненада* | СВ (момент) |
| *цео дан*, *три сата* (длительность) | НСВ |
| *до краја*, *све* | СВ (до конца) |

## Часть 9: Упражнения

### Упражнение 1. НСВ или СВ — выбери вид

Прочитай предложение и определи, какой вид нужен. Объясни (одним словом: процесс, привычка, до конца, момент, цепочка).

1. Сваког јутра _______ кафу. (пити / попити)
2. Јуче сам _______ целу књигу. (читати / прочитати)
3. Сада _______ писмо. (писати / написати)
4. Често _______ музику. (слушати / послушати)
5. _______ сам пријатеља на улици. (видети / угледати)
6. Дошао сам кући, _______ кафу и сео. (кувати / скувати)

<details>
<summary>Ключи</summary>

1. *пијем* — НСВ (привычка: «каждое утро»)
2. *прочитао сам* — СВ (до конца: «целую книгу»)
3. *пишем* — НСВ (процесс «сейчас»)
4. *слушам* — НСВ (привычка: «часто»)
5. *угледао сам* — СВ (момент: заметил)
6. *скувао сам* — СВ (цепочка завершённых действий: пришёл → сварил → сел)

</details>

### Упражнение 2. Найди пару

Для каждого глагола дай его видового партнёра и пометь, кто НСВ, кто СВ.

1. писати
2. прочитати
3. дати
4. купити
5. рећи
6. видети
7. ући
8. отварати

<details>
<summary>Ключи</summary>

1. *писати* (НСВ) — **написати** (СВ)
2. *прочитати* (СВ) — **читати** (НСВ)
3. *дати* (СВ) — **давати** (НСВ)
4. *купити* (СВ) — **куповати** (НСВ)
5. *рећи* (СВ) — **говорити** (НСВ) *(супплетивная пара — разные корни)*
6. *видети* (НСВ) — **угледати** (СВ)
7. *ући* (СВ) — **улазити** (НСВ)
8. *отварати* (НСВ) — **отворити** (СВ)

</details>

### Упражнение 3. Префикс или суффикс?

Для каждой пары определи, как образован совершенный (или несовершенный) партнёр: **префиксацией** (несов. + приставка → сов.) или **суффиксацией** (сов. + суффикс → несов.). Подчеркни префикс/суффикс.

1. читати / **про**читати
2. дати / да**вати**
3. писати / **на**писати
4. купити / купо**вати**
5. учити / **на**учити
6. узети / узи**мати**
7. погледати / гледати
8. бацати / бацити

<details>
<summary>Ключи</summary>

1. **Префиксация**: *читати* (НСВ) + про- → *прочитати* (СВ).
2. **Суффиксация**: *дати* (СВ) + -ва- → *давати* (НСВ).
3. **Префиксация**: *писати* (НСВ) + на- → *написати* (СВ).
4. **Суффиксация**: *купити* (СВ) + -ова- → *куповати* (НСВ).
5. **Префиксация**: *учити* (НСВ) + на- → *научити* (СВ).
6. **Суффиксация**: *узети* (СВ) + -има- → *узимати* (НСВ).
7. **Префиксация**: *гледати* (НСВ) + по- → *погледати* (СВ).
8. **Суффиксация**: *бацити* (СВ) + -а- → *бацати* (НСВ). *(Здесь суффикс -а- удлиняет основу.)*

</details>

### Упражнение 4. Переведи на сербский (кириллицей)

Решай: процесс или результат? Привычка или один раз? Выбирай вид соответственно.

1. Я каждый день читаю газету.
2. Я прочитал эту книгу за два дня.
3. Сейчас она пишет письмо.
4. Она написала длинное письмо.
5. Мы часто покупаем хлеб в этом магазине.
6. Вчера мы купили новый стол.
7. Они говорят по-сербски очень хорошо.
8. Он сказал «спасибо» и ушёл.
9. Дети купаются в реке.
10. Дети искупались и пошли спать.

<details>
<summary>Ключи</summary>

1. *Сваког дана читам новине.* (НСВ — привычка)
2. *Прочитао сам ову књигу за два дана.* (СВ — до конца)
3. *Сада пише писмо.* (НСВ — процесс сейчас)
4. *Написала је дугачко писмо.* (СВ — есть результат)
5. *Често купујемо хлеб у овој продавници.* (НСВ — привычка)
6. *Јуче смо купили нов сто.* (СВ — один раз, результат)
7. *Они говоре српски веома добро.* (НСВ — общее владение)
8. *Рекао је „хвала“ и отишао.* (СВ + СВ — цепочка завершённых)
9. *Деца се купају у реци.* (НСВ — процесс)
10. *Деца су се окупала и отишла да спавају.* (СВ — один раз, до конца)

</details>

### Упражнение 5. Матрица видовой пары через лица — *читати / прочитати*

Запиши форму для каждой клетки. **A** — несовершенный в настоящем (процесс/привычка), **Б** — совершенный в прошедшем (один раз, до конца).

**A. НСВ настоящее — *читати*:**

1. ја
2. ти
3. он / она
4. ми
5. ви
6. они / оне

**Б. СВ прошедшее — *прочитати*:**

7. ја (м / ж)
8. ти (м / ж)
9. он / она
10. ми
11. ви
12. они / оне

<details>
<summary>Ключи</summary>

1. *читам*
2. *читаш*
3. *чита*
4. *читамо*
5. *читате*
6. *читају*
7. *прочитао сам / прочитала сам*
8. *прочитао си / прочитала си*
9. *прочитао је / прочитала је*
10. *прочитали смо*
11. *прочитали сте*
12. *прочитали су / прочитале су*

</details>

## Памятка урока

```
ГЛАВНОЕ ОТКРЫТИЕ:
  Каждый сербский глагол существует ПАРОЙ.
  НСВ = процесс, привычка, длительность («что делал/делаю/буду делать»)
  СВ  = один раз, до конца, результат   («что сделал/сделаю»)

ВИД — СВОЙСТВО ГЛАГОЛА, НЕ ВРЕМЕНИ.
  Ты выбираешь вид → он прокручивается через все времена.

ДВА МЕХАНИЗМА ОБРАЗОВАНИЯ ПАРЫ:

  1. ПРЕФИКСАЦИЯ:  НСВ + приставка → СВ
        читати   → ПРО-читати
        писати   → НА-писати
        гледати  → ПО-гледати
        радити   → У-радити
        купати се → О-купати се
     Сербские видовые префиксы: про-, на-, по-, у-, с-, за-, из-, о-

  2. СУФФИКСАЦИЯ:  СВ + -вати/-ивати/-авати → НСВ
        дати    → даВАТИ
        купити  → купоВАТИ
        бацити  → бацАТИ
        пасти   → падАТИ

  3. РАЗНЫЕ КОРНИ (супплетивные пары):
        говорити (НСВ) / рећи (СВ)
        узимати  (НСВ) / узети (СВ)
        долазити (НСВ) / доћи (СВ)

ПЯТЬ ВИДОВЫХ ПАР НА ЗАПОМИНАНИЕ:
  читати   / прочитати   — читать / прочитать
  писати   / написати    — писать / написать
  говорити / рећи         — говорить / сказать
  видети   / угледати     — видеть / увидеть
  купати се / окупати се  — купаться / искупаться

ПРАВИЛА ВЫБОРА:
  «сваки дан, често, увек, сада, цео дан» → НСВ
  «јуче, одједном, до краја, све, један пут» → СВ
  Цепочка завершённых событий в прошлом → все СВ.
  Длящийся фон, обстановка → все НСВ.

ОСОБЕННОСТЬ:
  СВ почти не имеет «настоящего» в значении «сейчас».
  Если СВ стоит в настоящем — обычно это будущее или придаточное (L22).

КАРТОЧНАЯ СТРАТЕГИЯ:
  Одна карточка = ОДНА ПАРА. Никогда не разделяй.
  читати / прочитати  —  читать / прочитать
```
