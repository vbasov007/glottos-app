# Урок 14: Творительный падеж (инструменталь) — чем? с кем?
**Словарный запас: Инструменты, средства, сопровождение, транспорт**

---

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — медленно, осознанно, прогоняя через падежные карточки
3. **Ускоряйся** — повторяй, пока окончание -ом/-ем не будет вылетать само

> Знать правило = 5%. Натренировать рот = 95%.
> Сербский **инструменталь** = русский **творительный**. Имя совпадает, идея совпадает. Но есть одно тонкое различие, которое для русскоязычного — самое важное во всём уроке: **средство — БЕЗ предлога, сопровождение — С предлогом *са***.
> *Пишем оловком* = «пишу карандашом» (средство, без предлога).
> *Идем са братом* = «иду с братом» (сопровождение, с *са*).
> В русском оба случая выражаются одним творительным без предлога («пишу карандашом», «иду с братом»). Сербский же режет это пополам.

---

## Часть 1: Что делает творительный

> **Творительный (инструменталь)** отвечает на два главных вопроса:
> — **чем?** (средство, орудие) — *Чиме?*
> — **с кем? с чем?** (сопровождение) — *С ким? С чим?*

Это шестой падеж, который мы вводим (после именительного — L3, винительного — L8, родительного — L9, дательного и местного — L12). Остался один — звательный (L15).

Сербский творительный делает примерно то же, что и русский, плюс несколько частных употреблений. Главные функции:

| Функция | Пример | Перевод |
|---------|--------|---------|
| **Средство, орудие** (без предлога) | Пишем **оловком**. | Пишу карандашом. |
| **Сопровождение** (с предлогом *са*) | Идем **са братом**. | Иду с братом. |
| **Транспорт** (без предлога) | Путујемо **возом**. | Едем поездом. |
| **Пространственные предлоги** | Птица је **над кућом**. | Птица над домом. |
| **Именной предикат с *бити*** (старая норма) | Био је **лекаром**. | Был врачом. |
| **Временные обороты** | **Ноћу** спавам. | Ночью сплю. |

> **Главное различие с русским:** один русский творительный сербский разрезает на два употребления:
> — **средство → БЕЗ предлога**
> — **сопровождение → С предлогом *са***

## Часть 2: Окончания творительного — словарная карточка

### Единственное число

| Род | Окончание | Базовая форма | Творительный |
|-----|-----------|---------------|--------------|
| **М.р.** (тв. основа) | **-ом** | *брат* | *братом* |
| **М.р.** (мягкая основа: ј, љ, њ, ћ, ђ, ч, ш, ж, џ) | **-ем** | *мужем* | *мужем* (от *муж*) |
| **Ср.р.** (тв.) | **-ом** | *село* | *селом* |
| **Ср.р.** (мягк.: -е) | **-ем** | *поље* | *пољем* |
| **Ж.р.** на -а | **-ом** | *жена* | *женом* |
| **Ж.р.** на согласную | **-и** или **-ју** | *ноћ* | *ноћи / ноћу* |

### Множественное число

| Род | Окончание | Базовая форма | Творительный мн. |
|-----|-----------|---------------|------------------|
| **М.р.** | **-има** | *братови* | *братовима* |
| **Ср.р.** | **-има** | *села* | *селима* |
| **Ж.р.** на -а | **-ама** | *жене* | *женама* |
| **Ж.р.** на согл. | **-има** | *ноћи* | *ноћима* |

> **Запомни ритм:** ед.ч. **-ом/-ем** (трёх родов через одно окончание), мн.ч. **-има/-ама**. Это редкая для славянского падежа простота — сербский тут сжал систему.

### Правило «тв. vs мягкая основа» — деталь

Окончание **-ем** (вместо **-ом**) приходит после **мягких согласных**: ј, љ, њ, ћ, ђ, ч, ш, ж, џ, ц.

| Слово | Конечная согл. | Творительный |
|-------|---------------|--------------|
| *брат* | т (тв.) | *братом* |
| *град* | д (тв.) | *градом* |
| *муж* | ж (мягк.) | *мужем* |
| *кључ* | ч (мягк.) | *кључем* |
| *нож* | ж (мягк.) | *ножем* |
| *чај* | ј (мягк.) | *чајем* |
| *оловка* | а → -ом | *оловком* |
| *поље* | е → -ем | *пољем* |

Это то же самое правило, что мы видели в L8 (винительный одушевлённого) и L9 (родительный): мягкие основы получают **-ем/-ев-**, твёрдые — **-ом/-ов-**.

## Часть 3: Средство — БЕЗ предлога

Самое важное для русскоговорящего: когда речь о **средстве, инструменте, способе** — творительный идёт **голый**, без всяких предлогов.

| Сербский | Русский |
|----------|---------|
| Пишем **оловком**. | Пишу карандашом. |
| Сечем **ножем**. | Режу ножом. |
| Једем **кашиком**. | Ем ложкой. |
| Отварам врата **кључем**. | Открываю дверь ключом. |
| Радим **рукама**. | Работаю руками. |
| Чистим **крпом**. | Чищу тряпкой. |
| Бришем **гумицом**. | Стираю ластиком. |

> **Ловушка для русскоязычного:** русский «с» в выражениях типа «работаю с компьютером» соблазняет добавить *са*. Но если это **средство** (а не **спутник**), предлог не нужен:
> *Радим компјутером.* — «Работаю на компьютере» (как инструментом).
> *Радим са колегом.* — «Работаю с коллегой» (спутник).

### Транспорт — тоже средство

Способ передвижения — это средство. Поэтому без предлога:

| Сербский | Русский |
|----------|---------|
| Путујем **возом**. | Едет поездом. |
| Идем **аутобусом**. | Еду автобусом. |
| Долазим **колима**. | Приезжаю на машине. (*кола* — только мн.) |
| Летим **авионом**. | Лечу самолётом. |
| Возим се **бициклом**. | Катаюсь на велосипеде. |
| Иду **пешке**. | Иду пешком. (наречие, не творительный) |

## Часть 4: Сопровождение — С предлогом *са*

Когда творительный значит **спутника, компанию** — обязателен предлог **са** (часто пишется и произносится как **с** перед удобными согласными).

| Сербский | Русский |
|----------|---------|
| Идем **са братом**. | Иду с братом. |
| Причам **са пријатељем**. | Говорю с другом. |
| Живим **са родитељима**. | Живу с родителями. |
| Пијем кафу **са млеком**. | Пью кофе с молоком. |
| Хлеб **са сиром**. | Хлеб с сыром. |
| Путујем **са сестром**. | Еду с сестрой. |
| Учим **са другарицом**. | Учусь с подругой. |

> **Запомни тест:** замени «с кем/чем» на «вместе с кем/чем» — если смысл сохраняется, нужен *са*. Если в русском «с» можно убрать (заменить на чистый творительный), то в сербском его и не было.

### Когда *са* становится *с*

В живой речи и в письме *са* часто сокращается до *с* перед словами, начинающимися с гласной или удобной согласной:

| Полная форма | Сокращённая |
|--------------|-------------|
| *са братом* | *с братом* |
| *са Аном* | *с Аном* |
| *са пријатељем* | *с пријатељем* |

Но **перед свистящими и шипящими** (с, з, ш, ж) почти всегда оставляют полное *са*, чтобы не было *с с-* / *с з-*:

| | |
|---|---|
| *са сином* (не *с сином*) | с сыном |
| *са школом* | со школой |
| *са женом* | с женой |

> **Хак:** русский тоже различает «с» / «со» по фонетике («с братом» / «со мной»). Сербский *са/с* работает похоже.

## Часть 5: Предлоги с творительным

Кроме *са*, есть ещё **5 пространственных предлогов**, которые управляют творительным. Все они отвечают на вопрос **где?** (статика, положение в пространстве):

| Предлог | Значение | Пример | Перевод |
|---------|----------|--------|---------|
| **са** | с (сопровождение) | *Идем са братом.* | Иду с братом. |
| **под** | под | *Мачка је под столом.* | Кот под столом. |
| **над** | над | *Лампа је над креветом.* | Лампа над кроватью. |
| **пред** | перед | *Стојим пред кућом.* | Стою перед домом. |
| **за** | за (позади) | *Башта је за кућом.* | Сад за домом. |
| **међу** | между, среди | *Седи међу пријатељима.* | Сидит среди друзей. |

> **Хак на запоминание:** «**за**, **под**, **над**, **пред**, **међу** — где?» — четыре пары пространственных отношений и одно «между». Все требуют творительный, если означают **место**.

### Те же предлоги с винительным = движение

Эти же предлоги (кроме *са* и *међу*) с **винительным** (L8) дают **движение**:

| Творительный (где?) | Винительный (куда?) |
|---------------------|---------------------|
| *Мачка је **под столом**.* (где?) | *Мачка иде **под сто**.* (куда?) |
| *Слика је **над креветом**.* | *Стави слику **над кревет**.* |
| *Стојим **пред кућом**.* | *Идем **пред кућу**.* |
| *Башта је **за кућом**.* | *Иде **за кућу**.* |

> **Систематика:** падеж = тип отношения. Творительный = **статика, место**. Винительный = **направление, цель движения**. Это та же логика, что в русском (*перед домом / за дом*).

## Часть 6: Творительный с *бити* — именной предикат

Старая литературная норма (и сейчас встречается в книгах, прессе, формальном стиле):

| Сербский | Перевод | Регистр |
|----------|---------|---------|
| Био је **лекаром**. | Был врачом. | формал./литер. |
| Постао је **директором**. | Стал директором. | формал. |
| Сматрају га **геније**м. | Считают его гением. | формал. |

В **современной разговорной речи** обычно номинатив:

| Современная норма | Перевод |
|-------------------|---------|
| Био је **лекар**. | Был врачом. |
| Постао је **директор**. | Стал директором. |

> **Что учить активно:** для продукции — номинатив (*Био је лекар*). Творительный с *бити* — **на узнавание**, чтобы не споткнуться в тексте.

### Где творительный с *бити* остаётся обязательным

После некоторых глаголов значения **становления, считания, называния** творительный жив и в современной норме:

| Глагол | Пример | Перевод |
|--------|--------|---------|
| **постати** (стать) | Постао је **инжењером**. | Стал инженером. |
| **сматрати** (считать) | Сматрам га **пријатељем**. | Считаю его другом. |
| **звати** (звать) | Зову га **газдом**. | Зовут его хозяином. |
| **бавити се** (заниматься) | Бавим се **спортом**. | Занимаюсь спортом. |

> **Запомни возвратный глагол:** **бавити се чим** — «заниматься чем». Он всегда требует творительный без предлога: *Бавим се музиком, спортом, политиком*.

## Часть 7: Творительный в выражениях времени

Некоторые временные обороты выражаются голым творительным, без предлога:

| Сербский | Перевод |
|----------|---------|
| **Ноћу** спавам. | Ночью сплю. |
| **Даном** радим. | Днём работаю. |
| **Јутром** трчим. | Утром бегаю. |
| **Зимом** иде снег. | Зимой идёт снег. |
| **Летом** идемо на море. | Летом ездим на море. |

Это окаменевшие наречные формы — учи их **как наречия**, по слову. Альтернатива — конструкции с предлогами (*у току ноћи, током зиме*), которые мы пройдём позже.

## Часть 8: Прогонка через лица — *писати/написати* + творительный

Видовая пара: **писати (нсв) / написати (св)** — «писать / написать». Прогони с инструментом:

| Лицо | Сербский | Перевод |
|------|----------|---------|
| **ја** | Ја пишем писмо **оловком**. | Я пишу письмо карандашом. |
| **ти** | Ти пишеш писмо **оловком**. | Ты пишешь письмо карандашом. |
| **он/она** | Он пише писмо **оловком**. | Он пишет письмо карандашом. |
| **ми** | Ми пишемо писмо **оловком**. | Мы пишем письмо карандашом. |
| **ви** | Ви пишете писмо **оловком**. | Вы пишете письмо карандашом. |
| **они** | Они пишу писмо **оловком**. | Они пишут письмо карандашом. |

Замени *оловком* на: *хемијском (ручкой), фломастером (фломастером), кредом (мелом)*. Прогони все шесть лиц с каждым.

## Часть 10: Упражнения

### Упражнение 1. Поставь существительное в творительный (без предлога)

Это **средство** — предлог не нужен. Образец: *Пишем (оловка) → Пишем оловком.*

1. Сечем (нож).
2. Једем (кашика).
3. Отварам врата (кључ).
4. Бришем (гумица).
5. Радим (компјутер).
6. Куцам (телефон).
7. Чистим (крпа).
8. Цртам (фломастер).
9. Путујем (воз).
10. Идем (аутобус).

### Упражнение 2. Поставь предлог *са* (если нужно) и форму творительного

Реши: **средство** (без предлога) или **сопровождение** (с *са*)?

1. Идем (брат) у школу.
2. Пишем (оловка).
3. Пијем кафу (млеко).
4. Сечем (нож).
5. Причам (пријатељ).

7. Путујем (воз) у Београд.
8. Путујем (сестра) у Београд.

10. Једем хлеб (виљушка).

10. *Једем хлеб **виљушком**.* (средство, инструмент)

</details>

### Упражнение 3. Выбери падеж — творительный или винительный?

Те же предлоги (*под, над, пред, за*) дают **где? (твор.)** или **куда? (вин.)**. Образец: *Мачка је (под/сто). → Мачка је под столом.* vs *Мачка иде (под/сто). → Мачка иде под сто.*

1. Слика је (над/кревет).
2. Стави слику (над/кревет).
3. Стојим (пред/кућа).
4. Идем (пред/кућа).
5. Башта је (за/кућа).
6. Седим (међу/пријатељи).
7. Пас спава (под/сто).
8. Пас иде (под/сто).

<details>
<summary>Ключи</summary>

1. *Слика је **над креветом**.* (где? — творительный)
2. *Стави слику **над кревет**.* (куда? — винительный)
3. *Стојим **пред кућом**.* (где?)
4. *Идем **пред кућу**.* (куда?)
5. *Башта је **за кућом**.* (где?)
6. *Седим **међу пријатељима**.* (где? — *међу* всегда твор.)
7. *Пас спава **под столом**.* (где?)
8. *Пас иде **под сто**.* (куда?)

</details>

### Упражнение 4. Переведи на сербский (кириллицей)

1. Я пишу карандашом.
2. Я иду с братом в школу.
3. Мы едем поездом в Белград.
4. Дети играют с собакой.
5. Он работает на компьютере. (= компьютером, как инструментом)
6. Сад за домом.
7. Кот спит под столом.
8. Бабушка пьёт чай с молоком.
9. Стою перед школой.
10. Занимаюсь спортом.

<details>
<summary>Ключи</summary>

1. *Пишем оловком.* (или *Ја пишем оловком*)
2. *Идем са братом у школу.*
3. *Путујемо возом у Београд.*
4. *Деца се играју са псом.* (*пас* → твор. *псом*)
5. *Он ради компјутером.* (без *са* — это средство!)
6. *Башта је за кућом.*
7. *Мачак спава под столом.*
8. *Бака пије чај са млеком.*
9. *Стојим пред школом.* (или *испред школе* — но то родительный, L9)
10. *Бавим се спортом.*

</details>

### Упражнение 5. Матрица «Идем у школу са …» через лица

Запиши фразу **«Идем у школу са (X)»** для каждой пары: подлежащее + спутник. Спутника поставь в творительный.

1. ја + брат
2. ти + сестра
3. он + мајка
4. ми + пријатељи (мн.)
5. ви + колега
6. они + деца

<details>
<summary>Ключи</summary>

1. *Идем у школу са братом.*
2. *Идеш у школу са сестром.*
3. *Иде у школу са мајком.*
4. *Идемо у школу са пријатељима.*
5. *Идете у школу са колегом.*
6. *Иду у школу са децом.*

> **Принцип зеркала:** глагол *ићи* — через все лица; спутник стабильно в творительном с *са*. Бонус: повтори, заменив *школа* на *биоскоп* и *парк*.

</details>

### Упражнение 6. Заверши предложения (творительный или номинатив с *бити*)

1. У школу долазим (бицикл).
2. Радим (рука) цео дан.
3. Био је (лекар). — *современная норма*
4. Постао је (директор). — *с творительным*
5. Сматрам га (пријатељ).
6. Бавим се (музика) и (спорт).
7. Пише (хемијска оловка).
8. Игра шах (деда).

<details>
<summary>Ключи</summary>

1. *У школу долазим **бициклом**.* (транспорт = средство)
2. *Радим **рукама** цео дан.* (мн.ч. → -ама)
3. *Био је **лекар**.* (современная норма → номинатив!)
4. *Постао је **директором**.* (после *постати* творительный остался)
5. *Сматрам га **пријатељем**.* (мягк. основа → -ем)
6. *Бавим се **музиком** и **спортом**.* (*бавити се* → твор. без предлога)
7. *Пише **хемијском оловком**.* (прилагательное согласовано: ж.р. твор. → -ом)
8. *Игра шах **са дедом**.* (сопровождение!)

</details>

## Памятка урока

```
ТВОРИТЕЛЬНЫЙ (ИНСТРУМЕНТАЛЬ): ЧЕМ? С КЕМ?

ГЛАВНОЕ РАЗЛИЧИЕ С РУССКИМ:
  СРЕДСТВО     → БЕЗ предлога:   Пишем оловком.       (рус. «карандашом»)
  СОПРОВОЖДЕНИЕ → С предлогом СА:  Идем са братом.     (рус. «с братом»)
  ТРАНСПОРТ    → БЕЗ предлога:   Путујем возом.       (рус. «поездом»)

ОКОНЧАНИЯ ЕД.Ч.:
  м.р. / ср.р. тв. основа  → -ОМ   (брат → братом, село → селом)
  м.р. / ср.р. мягк. осн.  → -ЕМ   (муж → мужем, поље → пољем)
  ж.р. на -а              → -ОМ   (жена → женом, оловка → оловком)
  ж.р. на согл.            → -И / -ЈУ  (ноћ → ноћи / ноћу)

ОКОНЧАНИЯ МН.Ч.:
  м.р. / ср.р. / ж.р. на согл. → -ИМА   (братовима, селима, ноћима)
  ж.р. на -а                  → -АМА   (женама, оловкама)

ПРЕДЛОГИ С ТВОРИТЕЛЬНЫМ (где? — статика):
  СА    — с (сопровождение):     са братом, са кафом
  ПОД   — под:                   под столом
  НАД   — над:                   над креветом
  ПРЕД  — перед:                 пред кућом
  ЗА    — за (позади):           за кућом
  МЕЂУ  — между, среди:          међу пријатељима

ТЕ ЖЕ ПРЕДЛОГИ С ВИНИТЕЛЬНЫМ = ДВИЖЕНИЕ:
  Мачка је под столом. (где?, твор.)  vs.  Мачка иде под сто. (куда?, вин.)

ТВОРИТЕЛЬНЫЙ С *БИТИ*:
  Современная норма: Био је лекар.        (номинатив)
  Старая/литературная: Био је лекаром.   (творительный)
  После постати/сматрати/бавити се — творительный жив:
    Постао је инжењером. Бавим се спортом.

ВРЕМЯ В ТВОРИТЕЛЬНОМ (наречные формы):
  ноћу, даном, јутром, зимом, летом — учи как наречия.

КЛЮЧЕВОЙ ТЕСТ ДЛЯ РУССКОЯЗЫЧНОГО:
  «вместе с кем/чем?» → нужен СА.
  «при помощи чего?» → БЕЗ предлога.
  Радим са колегом (с коллегой). Радим компјутером (на компьютере).
```
