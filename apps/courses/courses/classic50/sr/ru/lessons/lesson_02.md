# Урок 2: Беглое чтение кириллицей. Бытовая лексика. Числа 20–100
**Словарный запас: Дом, еда, животные; числа от 20 до 100**

---

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — медленно, осознанно, без редукции
3. **Ускоряйся** — повторяй, пока длинные слова не будут вылетать без запинки

> Знать правило = 5%. Натренировать рот = 95%.
> Урок 1 поставил азбуку. Урок 2 ставит **скорость**: длинные слова, стечения согласных, рукописную кириллицу (на узнавание) и числа до 100.
> Помни: **никакой редукции**. *<say>Београд</say>* = «бэ-о-град», не «биагрът». *<say>Молоко</say>* = «мо-ло-ко», не «мълако».
> Этот блок — **только кириллицей**. Латиница придёт на Уроке 6.

## Часть 1: От буквы к беглости

На Уроке 1 ты прошёл алфавит по одной букве. Теперь нужно превратить узнавание букв в **узнавание целых слов**. Когда взрослый русскоязычный читает русский текст, он не читает буквы — он узнаёт **силуэт слова**. Цель Урока 2 — добиться того же по-сербски.

Главное препятствие — три привычки, которые мозг тянет из русского:

1. **Редукция.** Безударные «а» и «о» в сербском **звучат чисто**, как ударные. Мозг по умолчанию глотает их — нужно сознательно не давать ему это делать.
2. **Боязнь стечения согласных.** В русском «страшно» мы прочитываем — а вот *<say>прст</say>, <say>врх</say>, <say>мрак</say>* мозг хочет «разбавить» гласной. Не давай.
3. **Поиск «ё» / «й» / «ь».** Этих букв нет. Их функции взяли *ј, љ, њ, ћ, ђ*. Узнавай их сразу как самостоятельные знаки, без перевода через русский.

> **Тренировочный принцип на этом уроке:** каждое слово прочитывай **одним выдохом**, не паузами по буквам. Если запнулся — повтори 3 раза, пока не пойдёт слитно.

## Часть 2: Длинные слова без редукции

Безударное «а» / «о» в сербском **не меняется**. Это самое неестественное для русского уха. Прогоняй вслух:

| Слово | Как читать (по слогам) | Перевод | Типичная русская ошибка |
|-------|------------------------|---------|-------------------------|
| **<say>Београд</say>** | бе-о-град | Белград | «биагрът» |
| **<say>молоко</say>** | мо-ло-ко | молоко | «мълако» |
| **<say>добровечер</say>** *(в разг.)* | до-бро-ве-чер | добрый вечер | «дабравечер» |
| **<say>поморанџа</say>** | по-мо-ран-џа | апельсин | «памаранджа» |
| **<say>парадајз</say>** | па-ра-дајз | помидор | «парадайс» с редукцией |
| **<say>родитељи</say>** | ро-ди-те-љи | родители | «радители» |
| **<say>аутомобил</say>** | а-у-то-мо-бил | автомобиль | «автамабиль» |
| **<say>телевизија</say>** | те-ле-ви-зи-ја | телевизор / ТВ | «тиливизия» |
| **<say>географија</say>** | ге-о-гра-фи-ја | география | «гиаграфия» |
| **<say>универзитет</say>** | у-ни-вер-зи-тет | университет | «унивэрсити́т» с редукциями |

**Дрилл:** прочитай столбец «как читать» 3 раза подряд, чувствуя каждую гласную. На четвёртый раз закрой колонку «как читать» — должно вылетать само.

> **Принцип:** в сербском количество слогов = количеству гласных букв. *<say>Београд</say>* — три гласные (е, о, а) → три слога. Точка.

## Часть 3: Стечения согласных и слоговое «р»

Сербский **не вставляет гласную** между согласными, как бы много их ни шло подряд. Русский мозг хочет вставить — пресекай.

### 3.1. Слоговое «р» — повторение Урока 1 с расширением

Между двумя согласными «р» **поёт сам**, как гласный. Не «тэрст», а *<say>трст</say>*.

| Слово | Перевод |
|-------|---------|
| **<say>Трст</say>** | Триест |
| **<say>прст</say>** | палец |
| **<say>врт</say>** | сад |
| **<say>црн</say>** | чёрный |
| **<say>врх</say>** | вершина |
| **<say>први</say>** | первый |
| **<say>срце</say>** | сердце |
| **<say>црвен</say>** | красный |
| **<say>трг</say>** | площадь |
| **<say>смрт</say>** | смерть |
| **<say>крв</say>** | кровь |
| **<say>брдо</say>** | холм |
| **<say>грм</say>** | куст |

**Дрилл-цепочка:** *<say>прст</say> — <say>врт</say> — <say>црн</say> — <say>врх</say> — <say>први</say> — <say>срце</say> — <say>црвен</say> — <say>трг</say> — <say>смрт</say> — <say>крв</say>*. На одном выдохе, без пауз.

### 3.2. Длинные слова со стечениями согласных — без вставочной гласной

Самая частая ошибка русского — вставить гласную туда, где сербу хватает чистого согласного.

| Слово | Перевод | Заметка по чтению |
|-------|---------|-------------------|
| **<say>страшно</say>** | страшно | «**стра**-шно», слитно. Не «строшно». |
| **<say>радост</say>** | радость | «**ра**-дост». Не «радысть». |
| **<say>живот</say>** | жизнь *(не «живот»!)* | «**жи**-вот». Все звуки чистые. |
| **<say>здравље</say>** | здоровье | «**здра**-вље». «вље» = в + ль. |
| **<say>пријатељ</say>** | друг | «при-ја-тељ» = «при-я-тель». |
| **<say>четвртак</say>** | четверг | «че-**твр**-так». «твр» — слоговое «р». |
| **<say>среда</say>** | среда | «**сре**-да». |
| **<say>месец</say>** | месяц / луна | «**ме**-сец», «е» = «э». |
| **<say>прозор</say>** | окно | «**про**-зор», без редукции. |
| **<say>доручак</say>** | завтрак | «**до**-ру-чак», твёрдое ч. |
| **<say>вечера</say>** | ужин | «**ве**-че-ра», «е» = «э». |
| **<say>поводом</say>** | по поводу | «по-**во**-дом», все «о» чистые. |

**Принцип:** если в сербском слове ты пишешь четыре согласные подряд (*<say>страшно</say>*, *<say>здравље</say>*) — читай их **слитно**, как одну фонетическую группу. Гласных там нет, и не нужно.

### 3.3. Минимальные пары «русский vs сербский»

Где русский съедает гласные, серб их сохраняет — и наоборот.

| Сербский | Чем отличается от русского ожидания |
|----------|--------------------------------------|
| **<say>месо</say>** «мясо» | пишется и читается *<say>месо</say>* (не «мясо»), е = «э» |
| **<say>зуб</say>** | без редукции окончания (русск. «зуп») |
| **<say>рука</say>** | «**ру**-ка», без «а»-редукции |
| **<say>глава</say>** | «**гла**-ва», оба слога чёткие |
| **<say>нога</say>** | «**но**-га», без редукции |
| **<say>око</say>** | «**о**-ко», оба «о» чистые |
| **<say>уво</say>** | ухо. Конечное «о» — не «а»! |

## Часть 4: Рукописная кириллица — узнавание

Сербы в записках, на досках, в открытках используют **рукописную кириллицу**. Тебе её **не нужно уметь писать** (печатной кириллицы достаточно), но **узнавать — обязательно**.

### Что нужно знать

В печатной и рукописной формах большинство букв совпадают с русским. Но **четыре буквы выглядят непривычно**:

| Печатная | Рукописная (как выглядит) | Заметка |
|----------|---------------------------|---------|
| **б** | как русское рукописное «б», но **с горизонтальной чертой сверху**, не петлёй | Серб: *б* с прямой шляпкой. Рус.: *б* с петлёй вверху. |
| **д** | как латинское *g* с прямой ножкой, **не как русское «д»** | Похоже на *g*. Не путай с печатной *д*. |
| **п** | пишется как русское рукописное **«и»** | Да — *п* выглядит как *и*. Контекст спасает. |
| **т** | пишется как русское рукописное **«ш»** (три ножки) | Да — *т* выглядит как *ш*. Опять контекст. |

> **Хак:** *д* — горбик внизу, *п* = «и» сверху, *т* = «ш» снизу, *б* — со шляпкой. Запомни четвёрку **д-п-т-б**.

### Узнавание на примере

Представь записку: *<say>иду у школу</say>*. Если бы её записали сербской рукописной кириллицей, буква **п** в слове *<say>школу</say>* — нет, тут «п» нет; но в слове **<say>поводом</say>** рукописное «п» будет выглядеть как русская «и». А слово **<say>дом</say>** начнётся с буквы, похожей на *g*.

> **Что тренируем сейчас:** не **писать** рукописно, а **читать** рукописные записки и подписи. Если живёшь в Сербии — увидишь это на досках в кафе, на открытках, в рецептах от руки. Печатной кириллицы для тебя пока достаточно — рукописную можно постепенно осваивать после Урока 10.

Особые буквы — **љ, њ, ћ, ђ, џ, ј** — в рукописи в целом сохраняют свои надстрочные элементы (чёрточку у *ћ/ђ*, дугу у *ј*, лигатурные «хвосты» у *љ/њ/џ*).

## Часть 5: Свободное ударение — как фоновая черта

Сербское ударение **свободное**: может падать на любой слог, и в словаре его обычно **не отмечают**. Хорошая новость для русского — **одно надёжное правило**:

> **Ударение никогда не падает на последний слог.** В исконно сербских словах. Заимствования (вроде *хотел* «отель») — исключения.

Это значит: в двусложных словах ударение **всегда на первом** слоге (*<say>кућа</say>, <say>вода</say>, <say>рука</say>, <say>мама</say>, <say>тата</say>*). В трёхсложных — на первом или втором, но **не на последнем**.

| Слово | Слогов | Ударение |
|-------|--------|----------|
| *<say>кућа</say>* | 2 | **ку**-ћа |
| *<say>вода</say>* | 2 | **во**-да |
| *<say>Београд</say>* | 3 | **Бе**-о-град |
| *<say>молоко</say>* | 3 | **мо**-ло-ко |
| *<say>поморанџа</say>* | 4 | по-**мо**-ран-џа |
| *<say>универзитет</say>* | 5 | у-ни-**вер**-зи-тет |

Тонкие детали (восходящее/нисходящее, долгое/краткое) — это материал Урока 44. Сейчас просто следуй правилу: **слушай носителя и не клади ударение на последний слог**.

> **Контраст с русским:** русский систематически кладёт ударение на последний слог (*окно́, стена́, сестра́*). Сербский — нет. *Окнó по-русски → **про**зор по-сербски* — ударение перепрыгнуло вперёд.

## Часть 9: Числа 20–100

### 9.1. Десятки

Десятки строятся как **«[цифра] + десет»**, но с фонетическими стяжениями. Запомни таблицу — она исключительно регулярна.

| 20 | 30 | 40 | 50 |
|----|----|----|-----|
| **<say>двадесет</say>** | **<say>тридесет</say>** | **<say>четрдесет</say>** | **<say>педесет</say>** |

| 60 | 70 | 80 | 90 | 100 |
|----|----|----|----|-----|
| **<say>шездесет</say>** | **<say>седамдесет</say>** | **<say>осамдесет</say>** | **<say>деведесет</say>** | **<say>сто</say>** |

**Заметки:**
- *<say>четрдесет</say>* — со слоговым «р»: «че-**тр**-де-сет». Без «и» между «т» и «р».
- *<say>педесет</say>* — стяжение из *<say>пет+десет</say>*, теряет «т».
- *<say>шездесет</say>* — стяжение из *<say>шест+десет</say>*, «ст» → «з».
- *<say>сто</say>* — короткое самостоятельное слово, никакого «-десет».

### 9.2. Составные числа 21–99

Принцип: **десяток + и + единица**. Соединительный союз *и* обязателен.

| Число | По-сербски |
|-------|-----------|
| 21 | <say>двадесет и један</say> |
| 22 | <say>двадесет и два</say> |
| 25 | <say>двадесет и пет</say> |
| 33 | <say>тридесет и три</say> |
| 47 | <say>четрдесет и седам</say> |
| 58 | <say>педесет и осам</say> |
| 64 | <say>шездесет и четири</say> |
| 79 | <say>седамдесет и девет</say> |
| 86 | <say>осамдесет и шест</say> |
| 99 | <say>деведесет и девет</say> |

> **Заметка:** союз *и* в речи часто проглатывается — носители говорят быстро «двадесет-један». В письменной форме лучше его сохранить (особенно на этом этапе). В Уроке 18 пройдём падежное поведение числительных полноценно.

### 9.3. 100

*<say>Сто</say>* — самостоятельное слово, склоняется, но в счёте остаётся коротким:
- *<say>сто један</say>* — 101
- *<say>двеста</say>* — 200 *(не «два сто»!)*
- *<say>триста</say>* — 300
- *<say>четиристо</say>* — 400, *<say>петсто</say>* — 500…

На Уроке 2 запомни только **100 = <say>сто</say>**. Сотни выше — позже.

## Часть 10: Упражнения

### Упражнение 1. Прочитай вслух — длинные слова без редукции

Прочитай каждое слово **одним выдохом**, не редуцируя гласных:

1. Београд
2. молоко
3. поморанџа
4. парадајз
5. родитељи
6. аутомобил
7. телевизија
8. универзитет
9. географија
10. доручак

<details>
<summary>Ключи</summary>

1. *Београд* → «**Бе**-о-град» (3 слога, ударение на «Бе», все гласные чисты)
2. *молоко* → «**мо**-ло-ко» (никаких «а»!)
3. *поморанџа* → «по-**мо**-ран-џа» (4 слога, џ = твёрдое «дж»)
4. *парадајз* → «па-ра-**дајз**» (4 слога, ј = «й»)
5. *родитељи* → «ро-ди-**те**-љи» (4 слога, љи = «ли мягко»)
6. *аутомобил* → «**а**-у-то-мо-бил» (5 слогов)
7. *телевизија* → «те-ле-**ви**-зи-ја» (5 слогов, все «е» = «э»)
8. *универзитет* → «у-ни-**вер**-зи-тет» (5 слогов)
9. *географија* → «ге-о-**гра**-фи-ја» (5 слогов, «ге» = «гэ», не «ги»)
10. *доручак* → «**до**-ру-чак» (3 слога, твёрдое ч)

</details>

### Упражнение 2. Стечения согласных и слоговое «р»

Прочитай вслух **без вставочной гласной**:

1. прст
2. четвртак
3. страшно
4. здравље
5. срце
6. црвен
7. трг
8. смрт
9. крв
10. првих

<details>
<summary>Ключи</summary>

1. *прст* → «прст» (один слог; «р» поёт само)
2. *четвртак* → «че-**твр**-так» (3 слога, средний = слоговое «р»)
3. *страшно* → «**стра**-шно» (2 слога, без «о» между ш и н)
4. *здравље* → «**здра**-вље» (2 слога, «вље» = в + ль)
5. *срце* → «**ср**-це» (2 слога, первый — слоговое «р»)
6. *црвен* → «**цр**-вен» (2 слога, первый — слоговое «р»)
7. *трг* → «трг» (один слог)
8. *смрт* → «смрт» (один слог; «м» и «р» сливаются)
9. *крв* → «крв» (один слог)
10. *првих* → «**пр**-вих» (2 слога, первый — слоговое «р»)

</details>

### Упражнение 3. Узнавание форм букв

Эти буквы рукописной кириллицы похожи на другие. Восстанови, что это за печатная буква:

1. Рукописная буква выглядит как русская печатная **«и»** → это сербская буква …
2. Рукописная буква выглядит как русская печатная **«ш»** (три ножки внизу) → это сербская буква …
3. Рукописная буква напоминает латинскую **g** с прямой ножкой → это сербская буква …
4. Рукописная буква как русская печатная «б», но **с прямой шляпкой сверху** → это сербская буква …
5. Сколько всего букв в сербской азбуке?

<details>
<summary>Ключи</summary>

1. **п** (выглядит как «и»)
2. **т** (выглядит как «ш»)
3. **д** (выглядит как латинское *g*)
4. **б** (со шляпкой сверху, без петли)
5. **30 букв.**

Хак: четвёрка **д-п-т-б** — это четыре рукописные буквы, которые отличаются от русских и сбивают читателя.

</details>

### Упражнение 4. Числа от 20 до 100

Запиши числа словами (кириллицей):

1. 20
2. 35
3. 47
4. 50
5. 68
6. 73
7. 81
8. 90
9. 99
10. 100

<details>
<summary>Ключи</summary>

1. 20 — *двадесет*
2. 35 — *тридесет и пет*
3. 47 — *четрдесет и седам*
4. 50 — *педесет*
5. 68 — *шездесет и осам*
6. 73 — *седамдесет и три*
7. 81 — *осамдесет и један*
8. 90 — *деведесет*
9. 99 — *деведесет и девет*
10. 100 — *сто*

Произнеси каждое вслух. Особое внимание: *четрдесет* — слоговое «р», *шездесет* — стяжение из *шест-десет*.

</details>

### Упражнение 5. Перевод — быт, еда, числа

Переведи фразы на сербский (только кириллица). Для перевода используй лексику Уроков 1–2.

1. Доброе утро! Хлеб и молоко, пожалуйста.
2. У меня (мой) дом — на (через) сорок пять окон.
3. В кухне — стол, четыре стула и большой шкаф.
4. Кофе горячий, а чай холодный.
5. У нас собака, кошка и две курицы.
6. Девяносто девять плюс один — сто.

<details>
<summary>Ключи</summary>

1. *Добро јутро! Хлеб и млеко, молим.*
2. *Моја кућа има четрдесет и пет прозора.* (Букв.: «Мой дом имеет 45 окон». Падежи — позже, Уроки 8–9.)
3. *У кухињи су сто, четири столице и велики орман.*
4. *Кафа је врућа, а чај је хладан.* *(врућ = горячий, хладан = холодный)*
5. *Имамо пса, мачку и две кокошке.* (Винительный — Урок 8.)
6. *Деведесет и девет плус један — сто.*

На этом этапе не парься, если падежи получились не идеально. Цель упражнения — **узнавать слова на письме и читать их вслух без редукции**.

</details>

### Упражнение 6. Матрица — диалог в магазине

Прочитай вслух оба голоса, обращая внимание на длинные слова и числа:

> — Добар дан! Молим Вас, један хлеб, млеко и две јабуке.
> — Изволите. То је двадесет и пет динара.
> — Извините, имате ли поморанџе?
> — Имамо. Колико?
> — Четири, молим. И један парадајз.
> — То је четрдесет и осам динара укупно.
> — Хвала! Довиђења.
> — Довиђења!

Повтори диалог 3 раза. Особое внимание:
- *поморанџе* — «по-мо-**ран**-џе», без редукции
- *четрдесет* — слоговое «р»
- *парадајз* — все четыре гласные чистые
- *двадесет, четрдесет* — ударение **на «два»**, **на «че»**, не на последнем слоге

  Превратить узнавание букв (Урок 1) в узнавание слов.

  Београд = «Бе-о-град» (НЕ «биагрът»)

СТЕЧЕНИЯ СОГЛАСНЫХ — БЕЗ ВСТАВОЧНОЙ ГЛАСНОЙ:
  страшно, радост, живот, здравље, четвртак — слитно.
  Не «строшно», не «здаравье», не «чет-и-вертак».

СЛОГОВОЕ «Р» (расширенный список):
  прст, врт, црн, врх, први, срце, црвен, трг, смрт, крв, брдо, грм, четвртак
  Между двумя согласными «р» САМО становится гласным.

  Свободное, не маркируется. ПРАВИЛО: никогда не на последнем слоге.
  кућа → КУ-ћа    Београд → БЕ-о-град    универзитет → у-ни-ВЕР-зи-тет

РУКОПИСНАЯ КИРИЛЛИЦА — узнавание (не продукция!):
  д ≈ латинская g     п ≈ печатная «и»
