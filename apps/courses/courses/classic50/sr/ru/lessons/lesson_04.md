1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — медленно, осознанно, через все шесть лиц

> Сегодня — **опорная точка всего сербского**. Глагол *бити* («быть») в настоящем времени и привычка ставить клитику на второе место — фундамент, на котором стоят будущее время (Урок 21), прошедшее (Урок 13), пассив (Урок 29), вообще вся архитектура сербского предложения.

## Часть 1: Зачем сегодня — опорная точка сербского

В русском глагол «быть» в настоящем времени **выпал**: *Я студент*, *Он дома*. Связки нет.

**В сербском связка обязательна.** Глагол *бити* в настоящем времени стоит почти в каждом предложении. И главный сюрприз:

> У глагола *бити* в настоящем времени **ДВЕ параллельные формы** для каждого лица:
> – **ПОЛНАЯ** (*<say>јесам</say>, <say>јеси</say>, <say>јесте</say>, <say>јесмо</say>, <say>јесте</say>, <say>јесу</say>*) — ударная, самостоятельная: для эмфазы, для одиночного утвердительного ответа, для начала вопроса.
> – **КРАТКАЯ КЛИТИКА** (*<say>сам</say>, <say>си</say>, <say>је</say>, <say>смо</say>, <say>сте</say>, <say>су</say>*) — безударная, **всегда на ВТОРОЙ позиции** в предложении. Это обычная, повседневная форма.

Это фундамент: **правило клитик на второй позиции**, которое к Уроку 23 станет полноценным синтаксическим законом. Привычку строим сейчас.

## Часть 2: Личные местоимения

Шесть лиц — стандартная матрица сербского глагола.

| Лицо | Сербский | Русский |
|------|----------|---------|
| 1 ед. | **<say>ја</say>** | я |
| 2 ед. | **<say>ти</say>** | ты |
| 3 ед. м.р. | **<say>он</say>** | он |
| 3 ед. ж.р. | **<say>она</say>** | она |
| 3 ед. ср.р. | **<say>оно</say>** | оно |
| 1 мн. | **<say>ми</say>** | мы |
| 2 мн. | **<say>ви</say>** | вы |
| 3 мн. м.р. | **<say>они</say>** | они (м. или смешанная группа) |
| 3 мн. ж.р. | **<say>оне</say>** | они (только женщины) |
| 3 мн. ср.р. | **<say>она</say>** | они (ср. род) |

**Что бросается в глаза:**
- *<say>ја</say>* пишется **через ј** (Урок 1: «й» — это буква Ј).
- *<say>он</say> / <say>она</say> / <say>оно</say>* — как в русском.
- **Множественное в третьем лице различает род**: *<say>они</say>* (м.р./смешанные), *<say>оне</say>* (только женщины), *<say>она</say>* (ср.р.). В русском «они» одно на всех — в сербском три формы.
- *<say>ви</say>* в обычной речи — «вы» (множественное). С большой буквы — *<say>Ви</say>* — это **вежливое обращение к одному человеку**.

### Вежливое *Ви*

> Как в русском, **к одному незнакомому/старшему/официальному собеседнику** обращаются на «вы». Пишется с большой буквы: ***<say>Ви</say>***. Глагол — **во множественном числе**, даже если человек один.

Сравни:
- *<say>Како си?</say>* — «Как ты?» (неформально, *<say>си</say>* — 2 ед.)
- *<say>Како сте?</say>* — «Как Вы?» (вежливо к одному) или «Как вы?» (к группе)

### «Pro-drop» — местоимения опциональны

Личное местоимение в сербском **можно опустить**, потому что окончание глагола само показывает лицо. *<say>Ја сам студент</say>* и *<say>Сам студент</say>* — оба грамматичны; первый — с лёгкой эмфазой («**я** студент»). Подробно — Урок 7.

## Часть 3: Глагол *бити* — две параллельные формы

### Таблица: полная и краткая

| Лицо | Местоимение | ПОЛНАЯ (ударная) | КРАТКАЯ (клитика) | Русский |
|------|-------------|------------------|--------------------|---------|
| 1 ед. | <say>ја</say> | **<say>јесам</say>** | **<say>сам</say>** | я есть |
| 2 ед. | <say>ти</say> | **<say>јеси</say>** | **<say>си</say>** | ты есть |
| 3 ед. | <say>он</say> / <say>она</say> / <say>оно</say> | **<say>јесте</say>** | **<say>је</say>** | он/она/оно есть |
| 1 мн. | <say>ми</say> | **<say>јесмо</say>** | **<say>смо</say>** | мы есть |
| 2 мн. | <say>ви</say> / <say>Ви</say> | **<say>јесте</say>** | **<say>сте</say>** | вы есть / Вы есть |
| 3 мн. | <say>они</say> / <say>оне</say> / <say>она</say> | **<say>јесу</say>** | **<say>су</say>** | они есть |

> **Заметь:** полная форма для 3 ед. (*<say>јесте</say>*) и для 2 мн. (*<say>јесте</say>*) совпадает, но клитика разная: *<say>је</say>* vs *<say>сте</say>*. В реальной речи почти всегда клитика — так что путаницы нет.

### Чтение и ловушки

- *<say>јесам</say>* — «**йэ-сам**», обе гласные чисто, без редукции.
- *<say>је</say>* — «**йэ**», один слог. *<say>сам</say>* — «**сам**», безударно в предложении.
- Не путай *<say>сам</say>* (= «(я) есть») с русским «сам» (= «один»). В сербском *<say>сам</say>* — базовая клитика 1 ед.

## Часть 4: Когда полная форма, а когда клитика

Это главное правило сегодняшнего урока.

### КЛИТИКА (*<say>сам</say>, <say>си</say>, <say>је</say>, <say>смо</say>, <say>сте</say>, <say>су</say>*) — это **норма**

Используется почти всегда, когда есть хотя бы одно слово перед глаголом *<say>бити</say>*. **Клитика безударна** и встаёт на **второе место** в предложении.

| Сербский | Что на 1-й позиции | Клитика на 2-й |
|----------|---------------------|----------------|
| **<say>Ја сам студент.</say>** | *<say>Ја</say>* | *<say>сам</say>* |
| **<say>Ми смо овде.</say>** | *<say>Ми</say>* | *<say>смо</say>* |
| **<say>Ана је лекарка.</say>** | *<say>Ана</say>* | *<say>је</say>* |
| **<say>Они су из Србије.</say>** | *<say>Они</say>* | *<say>су</say>* |
| **<say>Добро сам.</say>** | *<say>Добро</say>* | *<say>сам</say>* |
| **<say>Уморан си.</say>** | *<say>Уморан</say>* | *<say>си</say>* |

### ПОЛНАЯ форма (*<say>јесам ... јесу</say>*) — в трёх случаях

**1. Утвердительный одиночный ответ** (глагол сам по себе):

| Вопрос | Ответ |
|--------|-------|
| <say>Јеси ли ти студент?</say> | **<say>Јесам.</say>** |
| <say>Да ли је Ана овде?</say> | **<say>Јесте.</say>** |
| <say>Јесу ли они из Београда?</say> | **<say>Јесу.</say>** |

Клитика *<say>сам</say> / <say>је</say> / <say>су</say>* не может стоять одна — у неё нет ударения. В коротком ответе — полная форма.

**2. Эмфаза «таки, действительно»:**
- **<say>Ја јесам студент.</say>** — «Я **и есть** студент.»
- **<say>Они јесу из Србије.</say>** — «Они **действительно** из Сербии.»

**3. Начало вопроса с *<say>ли</say>*** (альтернатива конструкции *<say>Да ли...</say>* — Урок 10):
- **<say>Јеси ли ти Ана?</say>** — «Ты Ана?»
- **<say>Јесте ли Ви из Београда?</say>** — «Вы из Белграда?»

Здесь полная форма нужна, потому что **сама частица *<say>ли</say>* — клитика**: ей нужно ударное слово впереди.

## Часть 5: Правило второй позиции — зачем это сейчас

> Клитика **никогда не первое слово** в предложении. Она встаёт **сразу после первого ударного элемента**.

Это закон сербского синтаксиса. Сегодня ты применяешь его только к *<say>сам/си/је/смо/сте/су</say>*. К Уроку 23 ты применишь его сразу к большому кластеру клитик (*<say>ми, ти, му, га, се, ће, би, ли</say>*). Привычка строится сейчас.

| Неправильно | Правильно | Почему |
|-------------|-----------|--------|
| ✗ *<say>Сам ја студент.</say>* | ✓ **<say>Ја сам студент.</say>** | клитика *<say>сам</say>* не может стоять первой |
| ✗ *<say>Је Ана лекарка.</say>* | ✓ **<say>Ана је лекарка.</say>** | клитика *<say>је</say>* не может стоять первой |
| ✗ *<say>Су они из Србије.</say>* | ✓ **<say>Они су из Србије.</say>** | клитика *<say>су</say>* не может стоять первой |

Если первое слово — длинная группа, клитика всё равно идёт **после неё** целиком:

- *<say>Моја сестра је лекарка.</say>* (после именной группы «моя сестра»)
- *<say>Из Београда сам.</say>* (после предложной группы)
- *<say>Данас сам уморан.</say>* (после наречия «данас» = «сегодня»)

## Часть 6: Простые именные предложения

Сегодня — три типа: **X есть Y (сущ.)**, **X есть Y (прил.)**, **X есть откуда**.

| Тип | Сербский | Русский |
|-----|----------|---------|
| сущ. | **<say>Ја сам студент.</say>** | Я студент. |
| сущ. | **<say>Он је лекар.</say>** | Он врач. |
| сущ. | **<say>Она је учитељица.</say>** | Она учительница. |
| сущ. | **<say>Ми смо туристи.</say>** | Мы туристы. |
| сущ. | **<say>Они су Срби.</say>** | Они сербы. |
| прил. | **<say>Ја сам добар / добра.</say>** | Я хороший / хорошая. |
| прил. | **<say>Ти си уморан / уморна.</say>** | Ты усталый / усталая. |
| прил. | **<say>Он је млад.</say>** / **<say>Она је млада.</say>** | Он молодой. / Она молодая. |
| прил. | **<say>Ми смо срећни.</say>** | Мы счастливые. |
| прил. | **<say>Они су гладни.</say>** | Они голодные. |
| откуда | **<say>Ја сам из Русије.</say>** | Я из России. |
| откуда | **<say>Она је из Новог Сада.</say>** | Она из Нови-Сада. |
| откуда | **<say>Они су из Србије.</say>** | Они из Сербии. |

> **Заметь:** там, где в русском мы сказали бы просто «Я устал», в сербском **обязательна связка**: *<say>Ја сам уморан</say>*. Не выбрасывай *<say>сам</say>*.
>
> Прилагательное согласуется с подлежащим в роде и числе (Урок 5). Конструкция «из + родительный» — Урок 9; пока запомни шаблон целиком.

## Часть 7: Отрицательная форма *бити* — маленький бонус

Полностью отрицание разберём в Уроке 10, но базу проще запомнить сразу — она частотная и устроена просто.

> Отрицание = слитная форма **ни-** + клитика *<say>бити</say>*. Всегда ударная, не клитика, может стоять где угодно.

| Лицо | Отрицание | Перевод |
|------|-----------|---------|
| 1 ед. | **<say>нисам</say>** | я не (есть) |
| 2 ед. | **<say>ниси</say>** | ты не |
| 3 ед. | **<say>није</say>** | он/она/оно не |
| 1 мн. | **<say>нисмо</say>** | мы не |
| 2 мн. | **<say>нисте</say>** | вы не / Вы не |
| 3 мн. | **<say>нису</say>** | они не |

Примеры:
- *<say>Ја нисам студент. Ја сам професор.</say>* — Я не студент. Я профессор.
- *<say>Она није овде.</say>* — Её здесь нет.
- *<say>Ми нисмо из Београда, ми смо из Новог Сада.</say>* — Мы не из Белграда, мы из Нови-Сада.

## Часть 9: Упражнения

### Упражнение 1. Подставь правильную клитику *бити*

Заполни пропуск нужной краткой формой (*сам / си / је / смо / сте / су*):

1. Ја ____ студент.
2. Ти ____ из Београда.
3. Она ____ лекарка.
4. Ми ____ туристи.
5. Они ____ Срби.
6. Ви ____ професор. (вежливое к одному мужчине)
7. Марко и Ана ____ из Новог Сада.
8. Оно ____ мало.
9. Ја ____ из Москве.
10. Ти и Ана ____ овде.

<summary>Ключи</summary>

1. Ја **сам** студент.
2. Ти **си** из Београда.

4. Ми **смо** туристи.
5. Они **су** Срби.

</details>

### Упражнение 2. Полная или краткая? Выбери нужную форму

Подумай: предложение целое (тогда клитика на 2-й позиции) или одиночный ответ / эмфаза (тогда полная)?

1. — Јеси ли ти из Србије? — ____. (да)
2. Ана ____ моја сестра.
3. — Да ли сте Ви професор? — ____. (да)
4. Ја ____ студент, не професор!  (с эмфазой: «я **таки** студент»)
5. Они ____ овде.
6. — Јесу ли деца гладна? — ____. (да)
7. Ми ____ из Москве.
8. Никола ____ лекар.

<details>
<summary>Ключи</summary>

1. — **Јесам.** (одиночный ответ — нужна полная форма)
2. Ана **је** моја сестра. (клитика на 2-й позиции)
3. — **Јесам.** (полная — одиночный ответ; *Ви* = вежливое к одному, отвечающий говорит о себе → 1 ед.)
4. Ја **јесам** студент, не професор! (эмфаза — полная форма)
5. Они **су** овде. (клитика)
6. — **Јесу.** (полная — одиночный ответ)
7. Ми **смо** из Москве. (клитика)
8. Никола **је** лекар. (клитика)

8. *из Русије / сам / ја*

### Упражнение 5. Матрица — клитика и согласование через шесть лиц

Впиши клитику *бити* (а в группе Б — и нужную форму прилагательного).

**A. «… из Београда»** — подставь правильную клитику.

1. Ја … из Београда.
2. Ти … из Београда.
3. Он … из Београда.
4. Она … из Београда.
5. Ми … из Београда.
6. Ви (вежл.) … из Београда.
7. Они … из Београда.
8. Оне … из Београда.

**Б. «… уморан»** — клитика **+** окончание прилагательного по роду/числу.

9. Ја (м/ж) … …
10. Ти (м/ж) … …
11. Он … …
12. Она … …
13. Ми … …
14. Ви (вежл.) … …
15. Они … …
16. Оне … …

<details>
<summary>Ключи</summary>

**A.**
1. *Ја **сам** из Београда.*
2. *Ти **си** из Београда.*
3. *Он **је** из Београда.*
4. *Она **је** из Београда.*
5. *Ми **смо** из Београда.*
6. *Ви **сте** из Београда.*
7. *Они **су** из Београда.*
8. *Оне **су** из Београда.*

**Б.**
9. *Ја **сам уморан / уморна**.*
10. *Ти **си уморан / уморна**.*
11. *Он **је уморан**.*
12. *Она **је уморна**.*
13. *Ми **смо уморни**.*
14. *Ви **сте уморни**.*
15. *Они **су уморни**.*
16. *Оне **су уморне**.*

> **Принцип зеркала:** окончание прилагательного следует за родом/числом подлежащего: *он → уморан*, *она → уморна*, *оно → уморно*, *ми/они → уморни*, *оне → уморне*. Полная система — Урок 5.

</details>

### Упражнение 6. Диалог знакомства (проговори вслух оба голоса)

> — Добар дан! Како се зовете?
> — Зовем се Иван. А Ви?
> — Ја сам Ана. Драго ми је. Одакле сте Ви?
> — Ја сам из Русије, из Москве. А Ви?
> — Ја сам из Србије, из Београда. Јесте ли Ви турист?
> — Јесам. А Ви? Шта сте Ви?
> — Ја сам професор.

**Заметки:** *одакле* = «откуда»; *Шта сте Ви?* = «Кто Вы (по профессии)?» (буквально «что»). В каждом предложении найди клитику *сам / сте / је* и проверь, что она **на второй позиции**. Повтори диалог 3 раза, потом по памяти.

---

## Памятка урока

```
ЛИЧНЫЕ МЕСТОИМЕНИЯ:
  ед.:  ја      ти      он / она / оно
  мн.:  ми      ви      они (м.) / оне (ж.) / она (ср.)

  ВЕЖЛИВОЕ Ви (с большой буквы) = к одному человеку,
  глагол ВО МНОЖЕСТВЕННОМ числе.

ГЛАГОЛ БИТИ В НАСТОЯЩЕМ — ДВЕ ФОРМЫ:

  Лицо         ПОЛНАЯ     КЛИТИКА
  ──────────────────────────────────
  ја           јесам      сам
  ти           јеси       си
  он/она/оно   јесте      је
  ми           јесмо      смо
  ви / Ви      јесте      сте
  они/оне/она  јесу       су

КОГДА КАКАЯ:
  КЛИТИКА (сам, си, је, смо, сте, су) — НОРМА.
    Безударна. ВСЕГДА на ВТОРОЙ позиции в предложении.
    Никогда не первая, никогда не одна.

  ПОЛНАЯ (јесам ... јесу):
    1) одиночный утвердительный ответ:
         — Јеси ли ти Ана? — Јесам.
    2) эмфаза «таки, действительно»:
         Ја ЈЕСАМ студент.
    3) начало вопроса с ли:
         Јеси ли ти из Београда?

ПРАВИЛО ВТОРОЙ ПОЗИЦИИ:
  Клитика — на месте №2 после первого ударного элемента.
    ✓ Ја сам студент.       ✗ Сам ја студент.
    ✓ Ана је лекарка.       ✗ Је Ана лекарка.
    ✓ Данас сам уморан.     ✗ Уморан данас сам.
  (Это станет полноценным правилом в Уроке 23.)

ОТРИЦАНИЕ — не клитика, может стоять где угодно:
  нисам, ниси, није, нисмо, нисте, нису
  Ја нисам из Београда. Она није овде.

ИМЕННЫЕ ПРЕДЛОЖЕНИЯ (связка обязательна!):
  X je Y (сущ.)        Ја сам студент. / Она је лекарка.
  X je Y (прил.)       Ја сам уморан. / Они су срећни.
  X je из Y (откуда)   Ја сам из Русије.

НАЦИОНАЛЬНОСТИ (м. / ж. / страна):
  Рус / Рускиња / Русија
  Србин / Српкиња / Србија
  Хрват / Хрватица / Хрватска
  Немац / Немица / Немачка
  Француз / Францускиња / Француска
  Енглез / Енглескиња / Енглеска
  Италијан / Италијанка / Италија
  Американац / Американка / Америка
```

> **Следующий урок:** Урок 5 — Прилагательные: согласование в роде и числе (именительный падеж), место прилагательного перед существительным, первое касание определённой и неопределённой формы. Сегодня ты заметил, что *уморан / уморна / уморно / уморни / уморне* меняются по роду и числу — на Уроке 5 это станет системой.
