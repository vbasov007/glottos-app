# Урок 5: Прилагательные — согласование в роде и числе. Описательная лексика
**Словарный запас: Цвета, размер, качество, характер**

---

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — медленно, осознанно, согласовывая прилагательное с существительным
3. **Ускоряйся** — повторяй, пока согласование не пойдёт автоматом

> Знать правило = 5%. Натренировать рот = 95%.
> Хорошая новость: сербское прилагательное ведёт себя **почти как русское**. Согласуется в роде и числе, стоит перед существительным. Если у тебя есть русский в голове — у тебя уже есть 80% сербских прилагательных.

---

## Часть 1: Главное правило

> **Прилагательное в сербском согласуется с существительным в роде и числе** (а позже — и в падеже). Как в русском.
> **Место — перед существительным**: *<say>добар студент</say>*, *<say>велика кућа</say>*, *<say>плаво небо</say>*.
> Из именительного падежа собираем 6 форм: **3 рода × 2 числа**.

Это твой рефлекс на Уроке 5. Падежное согласование (всех семь падежей) — материал Урока 16; здесь — только именительный, но **во всех родах и числах**.

---

## Часть 2: Шесть форм прилагательного (именительный)

Возьмём прилагательное *<say>нов</say>* «новый». Вот вся таблица:

| | Ед. число | Мн. число |
|---|---|---|
| **м.р.** | <say>нов студент</say> | <say>нови студенти</say> |
| **ж.р.** | <say>нова студенткиња</say> | <say>нове студенткиње</say> |
| **ср.р.** | <say>ново село</say> | <say>нова села</say> |

**Окончания именительного — запомни как костяк:**

| Род | Ед.ч. | Мн.ч. |
|---|---|---|
| мужской | **–** (нулевое) | **–и** |
| женский | **–а** | **–е** |
| средний | **–о** / **–е** | **–а** |

> **Зеркало:** окончания прилагательного **точно те же**, что и окончания существительных в L3. *<say>Студент → нов студент</say>*; *<say>кућа → нова кућа</say>*; *<say>село → ново село</say>*. Та же мелодия.

---

## Часть 3: Прогонка через все шесть форм

Тренируй **полную сетку**, не отдельные формы. Возьмём *<say>добар</say>* «добрый, хороший».

| | Ед.ч. | Мн.ч. |
|---|---|---|
| **м.р.** | <say>добар човек</say> (хороший человек) | <say>добри људи</say> (хорошие люди) |
| **ж.р.** | <say>добра жена</say> (хорошая женщина) | <say>добре жене</say> (хорошие женщины) |
| **ср.р.** | <say>добро дете</say> (хороший ребёнок) | <say>добра деца</say> (хорошие дети) |

> **Беглое «а» в м.р.:** *<say>добар</say>, <say>кратак</say>, <say>паметан</say>, <say>гладан</say>* — в краткой м.р. форме есть «беглый» гласный *а*, который **выпадает** при склонении: *<say>добар → добра жена</say>*, *<say>кратак → кратка реч</say>*, *<say>паметан → паметна девојка</say>*. Это как русское «умён → умна».

### Прогонка *<say>велик</say>* «большой»

| | Ед.ч. | Мн.ч. |
|---|---|---|
| **м.р.** | <say>велик град</say> | <say>велики градови</say> |
| **ж.р.** | <say>велика кућа</say> | <say>велике куће</say> |
| **ср.р.** | <say>велико село</say> | <say>велика села</say> |

### Прогонка *<say>мали</say>* «маленький»

| | Ед.ч. | Мн.ч. |
|---|---|---|
| **м.р.** | <say>мали пас</say> | <say>мали пси</say> |
| **ж.р.** | <say>мала мачка</say> | <say>мале мачке</say> |
| **ср.р.** | <say>мало дете</say> | <say>мала деца</say> |

*<say>мали</say>* — особый: его словарная м.р. форма уже на *-и* (это **определённая** форма, см. Часть 5; неопределённой *<say>мал</say>* в живом сербском почти не бывает).

## Часть 4: Место прилагательного — ПЕРЕД существительным

> Сербский прилагательное **ставит перед существительным**, как русский. *<say>Добар студент</say>*, не *<say>студент добар</say>*.

| Сербский | Русский |
|---|---|
| **<say>црвена јабука</say>** | красное яблоко |
| **<say>велика кућа</say>** | большой дом |
| **<say>нови ауто</say>** | новая машина |
| **<say>паметна девојка</say>** | умная девушка |
| **<say>плаво небо</say>** | синее небо |

**Постпозиция** (прилагательное после существительного) возможна — но это либо предикат («<say>Јабука је црвена</say>» — «Яблоко красное»), либо стилистический приём. **На Уроке 5 — только препозиция.**

### Предикативное употребление с *бити* (повторяем L4)

Если прилагательное — сказуемое (через *бити*), оно по-прежнему **согласуется в роде и числе** с подлежащим:

| Сербский | Русский |
|---|---|
| **<say>Студент је добар.</say>** | Студент хороший. |
| **<say>Студенткиња је добра.</say>** | Студентка хорошая. |
| **<say>Дете је добро.</say>** | Ребёнок хороший. |
| **<say>Студенти су добри.</say>** | Студенты хорошие. |
| **<say>Жене су добре.</say>** | Женщины хорошие. |
| **<say>Деца су добра.</say>** | Дети хорошие. (Внимание: *<say>деца</say>* грамматически ж.р. ед.ч.! Но смысл — мн.; см. ниже.) |

> **Ловушка *<say>деца</say>*:** слово *<say>деца</say>* «дети» по форме — ж.р. ед.ч. (как *<say>жена</say>*), но по смыслу — мн.ч. Согласуется **по форме**: *<say>добра деца</say>, <say>мала деца</say>, <say>паметна деца</say>*. С глаголом тоже: *<say>Деца је овде</say>* в книжной речи, но в живом — чаще *<say>Деца су овде</say>* (согласование по смыслу). На Уроке 5 запомни: пиши *<say>добра деца</say>*.

## Часть 5: Первое касание определённой и неопределённой формы

> В сербском у прилагательного **м.р. ед.ч.** есть **две формы**: краткая (неопределённая) и длинная (определённая). Это уникальная черта сербского — её нет в русском.

| Краткая (неопр.) | Длинная (опр.) | Значение |
|---|---|---|
| **<say>нов студент</say>** | **<say>нови студент</say>** | новый студент |
| **<say>добар човек</say>** | **<say>добри човек</say>** | хороший человек |
| **<say>црн ауто</say>** | **<say>црни ауто</say>** | чёрный автомобиль |
| **<say>велик град</say>** | **<say>велики град</say>** | большой город |

### Семантическое различие

| Форма | Когда | Перевод-подсказка |
|---|---|---|
| **краткая** (<say>нов</say>) | впервые упоминается, «какой-то», неизвестный | «**один новый** студент» / a new student |
| **длинная** (<say>нови</say>) | известный, конкретный, тот самый | «**тот самый новый** студент» / the new student |

**Сравни:**

| Сербский | Тонкость |
|---|---|
| **<say>Имамо новог студента.</say>** | У нас есть **(один) новый** студент. (неопр.) |
| **<say>То је нови студент.</say>** | Это **(тот) новый** студент. (опр., конкретный) |
| **<say>Видим црног пса.</say>** | Вижу **(какого-то) чёрного** пса. |
| **<say>Видим црног пса (нашег комшије).</say>** | Вижу **того чёрного** пса (нашего соседа). |

> **На Уроке 5 — только узнавание.** Запомни: если у *<say>нов</say> / <say>добар</say> / <say>црн</say>* есть два варианта (*<say>нов</say>* и *<say>нови</say>*), это не ошибка. Краткая — «какой-то», длинная — «тот самый».

### Что важно сразу

- В **ж.р.** и **ср.р.** различие почти стёрто в современном сербском: *<say>нова кућа</say>* и *<say>ново село</say>* используются и как опр., и как неопр.
- В **м.р. мн.ч.** различия нет: *<say>нови студенти</say>* — и опр., и неопр.
- **Различие живёт в м.р. ед.ч.** — там оно реально, и его слышно.

### Подсказки по узнаванию

| Указатель | Часто требует | Пример |
|---|---|---|
| **<say>један</say>** (один, какой-то) | краткую | *<say>један нов студент</say>* |
| **<say>овај, тај, онај</say>** (этот, тот) | длинную | *<say>овај нови студент</say>* |
| **<say>мој, твој, наш</say>** (мой, твой, наш) | длинную | *<say>мој стари другар</say>* |
| предикат с *<say>је</say>* | краткую | *<say>Студент је нов.</say>* |

> **Хак:** В словарях сербских прилагательных словарная форма — **краткая м.р.**: *<say>нов</say>, <say>добар</say>, <say>црн</say>, <say>велик</say>*. Но в современной разговорной речи многие прилагательные **в м.р. ед.ч. чаще встречаются в длинной форме** (*<say>нови</say>, <say>добри</say>, <say>црни</say>, <say>велики</say>*). Не пугайся, если слышишь *<say>Видим црни ауто</say>* там, где «по правилу» ждал *<say>црн</say>*.

## Часть 6: Словарь — Цвета

- *<say>браон</say>* и *<say>роза</say>* — несклоняемые заимствования: *<say>браон ципеле</say>* (коричневые туфли), *<say>роза хаљина</say>* (розовое платье).
- *<say>наранџаст</say>, <say>љубичаст</say>* — от *<say>наранџа</say>* (апельсин) и *<say>љубичица</say>* (фиалка). Конечный *-ст* как русское «-ист».

### Упражнение 4. Краткая vs длинная м.р. — выбери

Выбери правильную форму прилагательного в м.р. ед.ч.:

1. *Имамо (нов / нови) студента.* — У нас есть один новый студент.
2. *Овај (нов / нови) студент је из Београда.* — Этот новый студент из Белграда.
3. *Видим (црн / црни) ауто.* — Вижу (тот самый) чёрный автомобиль.
4. *Један (црн / црни) пас стоји испред куће.* — Один чёрный пёс стоит перед домом.
5. *Студент је (добар / добри).* — Студент хороший. (предикат)
6. *Мој (стар / стари) другар живи у Новом Саду.* — Мой старый друг живёт в Нови-Саде.
7. *(Велик / Велики) град воли младе људе.* — Большой город любит молодых людей. (известный, конкретный город)
8. *Ово је (велик / велики) проблем.* — Это большая проблема. (новая, неопределённая)

<details>
<summary>Ключ</summary>

1. *Имамо **нов** студента.* (неопр., впервые упомянут)
2. *Овај **нови** студент…* (опр., после *овај*)
3. *Видим **црни** ауто.* (опр., конкретный)
4. *Један **црн** пас…* (неопр., после *један*)
5. *Студент је **добар**.* (предикат → краткая)
6. *Мој **стари** другар…* (опр., после *мој*)
7. *…**Велики** град воли…* (опр., конкретный)
8. *Ово је **велик** проблем.* (неопр., «какая-то» проблема). *Велики* тут тоже встречается, особенно в разговоре; для тренировки на Уроке 5 — *велик*.

</details>

### Упражнение 5. Переведи на сербский (кириллицей)

7. Красные яблоки лежат на столе.

2. *Ово је **нова** књига.*
3. *Студенти су **добри**.*
4. *Ана је **паметна** девојка.*
5. *Црна мачка спава.*
6. *Деца су **мала** и **весела**.* (деца — ж.р. ед.ч. по форме → ср.р. мн.ч. в согл.; в стандарте *мала, весела*)
7. *Црвене јабуке леже на столу.*

## Памятка урока

```
ГЛАВНОЕ ПРАВИЛО:
  Прилагательное согласуется с существительным в роде и числе.
  Место — ПЕРЕД существительным: добар студент, велика кућа, плаво небо.

ОКОНЧАНИЯ ИМЕНИТЕЛЬНОГО (НОВ):
              ед.ч.     мн.ч.
   м.р.       нов  (–)   нови  (–и)
   ж.р.       нова (–а)  нове  (–е)
   ср.р.      ново (–о)  нова  (–а)

ЗЕРКАЛО ОКОНЧАНИЙ СУЩЕСТВИТЕЛЬНОГО (L3):
   студент → нов студент; студенти → нови студенти
   кућа    → нова кућа;   куће    → нове куће
   село    → ново село;   села    → нова села

БЕГЛОЕ «А» В М.Р. КРАТКОЙ:
   добар → добра, кратак → кратка, паметан → паметна,
   гладан → гладна, тужан → тужна, болестан → болесна.

КРАТКАЯ vs ДЛИННАЯ М.Р. ЕД.Ч.:
   нов (какой-то новый)   ↔   нови (тот самый новый)
   После један, један-то новый → НОВ.
   После овај, тај, онај, мој, твој, наш → НОВИ.
   Предикат (Студент је нов.) → НОВ.

ЦВЕТА (ВЫСОКОЧАСТОТНЫЕ):
   бео — бела / црн — црна / црвен — црвена / плав — плава
   зелен — зелена / жут — жута
   (+) сив — сива, смеђ — смеђа, наранџаст — наранџаста,
       љубичаст — љубичаста, златан — златна, сребрн — сребрна
   несклоняемые: браон, роза.
   ЛОВУШКА: плава девојка = блондинка, не «синяя».

РАЗМЕР: велик — велика, мали — мала, дугачак — дугачка, кратак — кратка,
        висок — висока, низак — ниска, широк — широка, узак — уска,
        дебео — дебела, танак — танка, тежак — тешка, лак — лака.

КАЧЕСТВО И ХАРАКТЕР:
  добар — лош, леп — ружан, нов — стар, млад — стар,
  паметан — глуп, вредан — лењ, весео — тужан,
  гладан, жедан, уморан, здрав — болестан,
  богат — сиромашан, скуп — јефтин.

ПАДЕЖНАЯ КАРТОЧКА (с Урока 5):
  Каждое прилагательное запоминай в двух формах:
       краткая м.р. — ж.р.    (нов — нова, добар — добра, кратак — кратка)
```
