# Урок 33: Словообразование глаголов — префиксы и вторичный имперфектив
**Словарный запас: Глагольные семьи по префиксации (~34 единицы)**

---

## Как работать с этим уроком

1. **Прочитай** — пойми правило (5 минут, не больше!)
2. **Проговори вслух** — каждое глагольное гнездо целиком, парами вид-вид
3. **Ускоряйся** — пока префикс не начнёт «звучать» сам, как слышимый смысл

> Знать правило = 5%. Натренировать рот = 95%.
> Сегодня — самый продуктивный механизм сербской лексики. **Один корень + 10 префиксов = 10 разных глаголов.** Освоив систему, ты сразу узнаёшь сотни новых слов, ни разу не открывая словарь.

## Часть 1: Что мы уже знаем — и что добавляем

В **Уроке 11** ты узнал про вид. В **Уроке 25** — что глаголы движения берут префикс, и префикс делает сразу **две вещи**: даёт направление (вошёл, вышел, перешёл) **и** превращает несов. вид в сов. вид.

Сегодня обобщаем это правило **на весь глагол**:

> **Префикс в сербском почти всегда делает двойную работу:**
> 1. Образует **видового партнёра** (несов. → сов.).
> 2. Добавляет **новое лексическое значение** (направление, степень, образ действия).
>
> Поэтому от одного простого глагола разрастается целая **семья** — десять-пятнадцать дериватов, каждый со своим оттенком.

И второй механизм — обратный:

> Когда сов. префиксальный глагол хочется снова сделать несов. (для процесса, привычки, повторения), к нему добавляют **вторичный имперфективный суффикс**: **-ива-**, **-ава-**, **-ва-**.

Эта пара — префиксация + вторичная имперфективация — и есть мотор сербского глагольного словаря.

## Часть 2: Префикс делает двойную работу

Возьмём базовый глагол **писати** (нсв) — «писать».

| Префикс | Глагол | Вид | Значение |
|---------|--------|-----|----------|
| (нет) | **писати** | нсв | писать (процесс) |
| на- | **написати** | св | написать (до конца, один раз) |
| пре- | **преписати** | св | переписать; списать |
| за- | **записати** | св | записать (заметку) |
| по- | **потписати** | св | подписать (под чем-то) |
| ис- | **исписати** | св | исписать (полностью, до конца) |
| у- | **уписати** | св | вписать; зачислить (в школу) |
| до- | **дописати** | св | дописать (добавить) |
| о- | **описати** | св | описать (вокруг — словесно) |
| раз- | **расписати** | св | расписать (распределить) |

Видишь? **Один корень — десять разных слов.** Каждое — сов. вид (это первая работа префикса) и каждое — со своим значением (это вторая работа).

> **Вывод:** учить надо не отдельные глаголы, а **гнёзда**. Запомнил один корень — открыл доступ ко всей семье.

## Часть 3: Систематическая семантика главных префиксов

Префиксы — не случайный набор. У каждого есть **базовое значение**, перенесённое из пространственной метафоры на абстрактные смыслы. Вот карта.

### на- — «на (поверхность)», наполнение, набор количества

| Глагол | Значение |
|--------|----------|
| **напунити** | наполнить (на + пунити «полнить») |
| **набрати** | набрать (цветов, ягод) |
| **написати** | написать (= довести до конца) |
| **научити** | научить(ся), выучить |
| **навикнути се** | привыкнуть |

**Метафора:** «на поверхность» → «достаточное количество» → «полное завершение».

### про- — «сквозь, через, мимо», процесс целиком

| Глагол | Значение |
|--------|----------|
| **проћи** | пройти |
| **прочитати** | прочитать (= пройти весь текст) |
| **провести** | провести (время) |
| **проговорити** | заговорить (сквозь молчание) |
| **продати** | продать (= передать сквозь) |

**Метафора:** «сквозь объект» → «весь процесс целиком» → «исчерпание».

### за- — «за (объект)», заход, начало действия

| Глагол | Значение |
|--------|----------|
| **зайти** *(в сербском: ући)* — но: **заћи** | зайти (за угол, за солнце) |
| **запитати** | спросить (начать спрашивать) |
| **запевати** | запеть (начать петь) |
| **заплакати** | заплакать |
| **заборавити** | забыть |

**Две метафоры:** (1) пространственно — «за чем-то», (2) фазисно — **начало** действия (запеть, заплакать, заговорить).

### по- — «на немного», смягчение, дистрибутивность

| Глагол | Значение |
|--------|----------|
| **посетити** | посетить |
| **попити** | выпить (целиком) |
| **поразговарати** | поговорить (немного) |
| **поспати** | поспать (немного) |
| **потражити** | поискать |

**Метафора:** «слегка, ненадолго» — или, наоборот, **результативное завершение** (попити = «выпить до дна»).

### у- — «внутрь», окончательное вхождение в состояние

| Глагол | Значение |
|--------|----------|
| **ући** | войти |
| **унети** | внести |
| **умрети** | умереть |
| **уплашити се** | испугаться |
| **уморити се** | устать |

**Метафора:** «внутрь, в новое состояние» → переход в состояние (испугаться = войти в страх).

### из- (ис-) — «из, наружу», до конца, исчерпание

| Глагол | Значение |
|--------|----------|
| **изаћи** *(или* **изићи** *)* | выйти |
| **изнети** | вынести |
| **испити** | выпить (до дна) |
| **испричати** | рассказать (целиком) |
| **исписати** | исписать (весь лист) |

**Метафора:** «изнутри наружу» → «до конца, до исчерпания».

> **Фонетика:** перед глухим согласным **из- → ис-** (*испричати, исписати, испити*).

### пре- — «через, поверх», заново, чрезмерно

| Глагол | Значение |
|--------|----------|
| **прећи** | перейти |
| **превести** | перевести |
| **преписати** | переписать; списать |
| **прочитати** *vs* **препрочитати** | прочитать vs перечитать |
| **препунити** | переполнить (= слишком сильно) |

**Метафора:** (1) «через границу» → пере**ход**; (2) «снова поверх старого» → **заново**; (3) «слишком много» → **чрезмерно**.

### о- (об-) — «вокруг», превращение, охват

| Глагол | Значение |
|--------|----------|
| **обући** | надеть (= одеть вокруг тела) |
| **окружити** | окружить |
| **описати** | описать (словесно — обвести смыслом) |
| **обрисати** | вытереть (вокруг — целиком) |
| **остарити** | состариться (= превратиться в старого) |

**Метафора:** «вокруг объекта» → «охватить целиком» → «привести в новое состояние».

### под- — «под», тайно, дополнительно

| Глагол | Значение |
|--------|----------|
| **подићи** | поднять (= под низ, вверх) |
| **поднети** | подать; вынести (нагрузку) |
| **потписати** | подписать (= написать под) |
| **подсетити** | напомнить (= подвести к мысли) |
| **подметнути** | подсунуть (тайно) |

**Метафора:** «снизу, под объектом» → «незаметно», «дополнительно».

### раз- (рас-) — «в разные стороны», рассеять, развить

| Глагол | Значение |
|--------|----------|
| **разумети** | понять (= раз+умеjати) |
| **разговарати** | разговаривать (= речь в разные стороны) |
| **разделити** | разделить |
| **разбити** | разбить |
| **рашчистити** | расчистить |

**Метафора:** «в стороны, врозь» → «разъединить, рассеять, развить».

> **Фонетика:** перед глухим **раз- → рас-**, перед *ч/ш* → **раш-** (*рашчистити*).

### с- (са-) — «с (поверхности)», вниз, вместе

| Глагол | Значение |
|--------|----------|
| **сићи** | сойти (вниз) |
| **снети** | снести (вниз) |
| **сабрати** | собрать (вместе); сложить |
| **сачувати** | сохранить |
| **срушити** | разрушить (= снести вниз) |

**Метафора:** (1) «вниз с поверхности», (2) «вместе, в одно место».

### вз- / уз- — «вверх» (нерегулярный, ограниченный)

В сербском «вверх» обычно — **уз-** или редкие реликты на **вз-**:

| Глагол | Значение |
|--------|----------|
| **узети** | взять (= под-нять к себе) |
| **успети се** | подняться (вверх) |
| **устати** | встать |
| **узнемирити** | разволновать (= возмутить вверх) |

> Этот префикс продуктивен меньше других — но ты узнаешь его в **узети, устати, успети**.

## Часть 4: Вторичный имперфективный суффикс

Когда префикс делает глагол совершенным, **смысл уже не нейтральный** — он совершённый, точечный. Но в живой речи нам часто нужно сказать: «я уже **подписываю** этот документ» (процесс!), «он **переписывает** письма по часу» (привычка!), «она **записывает** лекции каждый день» (повторение).

Для этого сербский (как и русский) добавляет **вторичный имперфективный суффикс** — и сов. глагол снова становится несов.

### Главные суффиксы: -ива-, -ава-, -ва-

| Базовый (нсв) | Префиксальный (св) | Вторичный (нсв) |
|---------------|---------------------|------------------|
| **писати** «писать» | **написати** «написать» | **написива*ти*** «писать (постоянно)» |
| **читати** «читать» | **прочитати** «прочесть» | **прочитавати** «прочитывать» |
| **дати** «дать» | **продати** «продать» | **продавати** «продавать» |
| **звати** «звать» | **позвати** «позвать» | **позивати** «приглашать, звать (постоянно)» |
| **бити** «бить» | **разбити** «разбить» | **разбијати** «разбивать» |
| **рећи** «сказать» | **прорећи** «предсказать» | **прорицати** «прорицать» |
| **бацити** «бросить» | **избацити** «выбросить» | **избацивати** «выбрасывать» |
| **купити** «купить» | **откупити** «выкупить» | **откупљивати** «выкупать» |
| **видети** «видеть» | **превидети** «не заметить» | **превиђати** «не замечать» |

### Как это работает в трёхуровневой системе

Сербский глагол часто живёт в **трёх формах**:

```
ПРОСТОЙ нсв  →  ПРЕФИКСАЛЬНЫЙ св  →  ВТОРИЧНЫЙ нсв
   писати    →    написати           →    написивати
   читати    →    прочитати          →    прочитавати
   дати      →    продати            →    продавати
```

- Первая форма (простой нсв) — нейтральный процесс.
- Вторая (префикс + св) — единичное завершённое действие с новым лексическим оттенком.
- Третья (вторичный нсв) — процесс/привычка/повтор с этим **новым** лексическим значением.

> **Зачем тебе третья форма?** Потому что часто **только** она передаёт нужный смысл. Сказать «он переписывает книгу каждый день» через простой *писати* нельзя — там нет приставки «пере-». Нужен **преписивати**.

### Морфология суффикса — что меняется

При добавлении -ива-/-ава-/-ва- часто происходят чередования:

- **писати → написива-ти**: корень *пис-* остаётся, добавляется -ива-.
- **дати → продавати**: к корню *да-* добавляется -ва-.
- **бацити → избацивати**: *ц* остаётся, тематическое *-и-* меняется на *-ива-*.
- **купити → откупљивати**: появляется *љ* (йотация: п+ј → пљ).
- **писати → исписивати** *(не \*исписавати)*: суффикс выбирает форму -ива-.

> **Правило для уха:** если глагол **четырёхсложный** в новой форме и в нём слышится **«-ива/-ова/-ава/-ва-»** перед *-ти*, ты слышишь **вторичный имперфектив**.

## Памятка урока

```
ДВОЙНАЯ РАБОТА ПРЕФИКСА:
  Префикс  =  видовой партнёр (нсв → св)  +  новое лексическое значение
              (написати = «дописать до конца»)  (написати ≠ просто «писать»)

СИСТЕМАТИЧЕСКАЯ СЕМАНТИКА ГЛАВНЫХ ПРЕФИКСОВ:
  на-   = на поверхность; наполнить, набрать; завершить (написати, научити)
  про-  = сквозь, мимо; целиком (проћи, прочитати, провести)
  за-   = за; НАЧАЛО действия (запевати, заплакати, заборавити)
  по-   = немного; результативно (попити, посетити, поразговарати)
  у-    = внутрь, в состояние (ући, унети, умрети, уплашити се)
  из-/ис- = наружу; до конца (изаћи, испити, испричати, исписати)
  пре-  = через; заново; чрезмерно (прећи, превести, преписати)
  о-/об- = вокруг; превращение (обући, описати, окружити, остарити)
  под-  = под, тайно, дополнительно (подићи, потписати, подсетити)
  раз-/рас- = в стороны, рассеять (разделити, разговарати, разбити)
  с-/са- = вниз; вместе (сићи, сабрати, сачувати)
  уз-/вз- = вверх (узети, устати, успети)

ВТОРИЧНЫЙ ИМПЕРФЕКТИВ:  -ИВА- / -АВА- / -ВА-
  Возвращает несов. вид префиксальному сов. — для процесса/привычки/повтора.
  писати → написати → написивати
  читати → прочитати → прочитавати
  дати   → продати   → продавати
  бирати → изабрати  → (изабирати / бирати)
  ставити → поставити → постављати   (с йотацией: ви → вљ)
  творити → отворити  → отварати

ТРЁХУРОВНЕВАЯ СИСТЕМА:
  ПРОСТОЙ нсв   →   ПРЕФИКСАЛЬНЫЙ св   →   ВТОРИЧНЫЙ нсв
  нейтральный      завершённый            процесс/привычка
  процесс          + новое значение       с этим новым значением

ГЛАГОЛЬНЫЕ СЕМЬИ:
  -пис-: писати, написати, преписати, потписати, исписати, уписати, описати…
  -ићи / -лазити: ући/улазити, изаћи/излазити, прећи/прелазити, проћи/пролазити…
  -дати / -давати: продати/продавати, предати/предавати, издати/издавати…

ЗАПОМИНАНИЕ:
  Учи не отдельные глаголы — учи ГНЁЗДА.
  Один корень + 10 префиксов = 10 слов с предсказуемым смыслом.
  В тетради карточка: НСВ — СВ — (если есть) ВТОРИЧНЫЙ НСВ.
```
