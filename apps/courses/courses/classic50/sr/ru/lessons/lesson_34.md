# Урок 34: Словообразование существительных и прилагательных
**Словарный запас: Производные сущ. и прил., абстрактная лексика, словарные семьи (~36 единиц)**

---

## Как работать с этим уроком

1. **Прочитай** — пойми, как из одного корня вырастает целое семейство слов (10 минут).
2. **Разбери** — на каждое новое слово найди в голове **корень + суффикс**.
3. **Используй** — собирай свои производные по моделям. Половина «новых» слов — это знакомый корень + знакомый суффикс.

> Знать корень + знать главные суффиксы = понимать в 3 раза больше слов, чем формально учил.
> Сербское словообразование **прозрачнее русского**: суффиксы те же или близкие, но работают регулярнее. *Учити → учитељ → учитељски → учитељство.* Целая семья из одного глагола.

---

## Часть 1: Зачем учить суффиксы отдельно

> **Главный принцип:** в сербском, как и в русском, **новые слова чаще всего делаются не из воздуха, а из старых корней через суффикс**. Знаешь корень *пев-* (петь) + суффикс **-ач** (агенс) — знаешь *певач* (певец), не заучивая отдельно.

Для русскоязычного это **подарок**: большинство сербских суффиксов имеют прямые русские пары. *-ост = -ость, -ство = -ство, -ица = -ица, -ски = -ский, -ов = -ов*. Узнаваемо до прозрачности.

В этом уроке:
- **четыре главные группы** именных суффиксов (агенс, абстракт, диминутив, аугментатив);
- **три суффикса** для образования прилагательных из сущ. (*-ски, -ин, -ов*);
- **сложные слова** и **словарные семьи**.

## Часть 2: Группа 1 — АГЕНС (тот, кто делает действие)

Суффиксы, превращающие глагол или существительное в **деятеля**.

### -ач (от глаголов)

| Глагол | Агенс | Перевод |
|--------|-------|---------|
| **певати** (петь) | **певач** | певец |
| **играти** (играть) | **играч** | игрок |
| **возити** (везти) | **возач** | водитель |
| **трчати** (бежать) | **тркач** | бегун |
| **ковати** (ковать) | **ковач** | кузнец |

### -ник (от сущ. и прил.)

| Корень | Агенс | Перевод |
|--------|-------|---------|
| **рад** | **радник** | рабочий |
| **наука** | **научник** | учёный |
| **путовати** | **путник** | путник |
| **болест** | **болесник** | больной |
| **уметност** | **уметник** | художник |

### -тељ (от глаголов; профессия / роль)

| Глагол | Агенс | Перевод |
|--------|-------|---------|
| **учити** | **учитељ** | учитель |
| **родити** | **родитељ** | родитель |
| **читати** | **читатељ** | читатель |
| **спасити** | **спаситељ** | спаситель |
| **слушати** | **слушатељ** | слушатель |

### -ар (от сущ.; «работающий с X»)

| Корень | Агенс | Перевод |
|--------|-------|---------|
| **риба** | **рибар** | рыбак |
| **пчела** | **пчелар** | пчеловод |
| **новине** | **новинар** | журналист |
| **зуб** | **зубар** | стоматолог |
| **(пећи)** | **пекар** | пекарь |

> **Женский род агенсов:** *-ица / -ница / -ка* — *певачица, радница, учитељица, новинарка*.

## Часть 3: Группа 2 — АБСТРАКТ (свойство, действие, состояние)

Превращают прилагательное или глагол в **абстрактное существительное**.

### -ост (от прил.) = русское -ость

| Прил. | Сущ. на -ост | Перевод |
|-------|--------------|---------|
| **млад** | **младост** | молодость |
| **храбар** | **храброст** | храбрость |
| **мудар** | **мудрост** | мудрость |
| **радостан** | **радост** | радость |
| **жалостан** | **жалост** | печаль |
| **могућ** | **могућност** | возможность |
| **љубазан** | **љубазност** | любезность |

> Близкий суффикс **-ота**: *леп → лепота / љепота* (красота). Оба образуют абстрактные сущ. ж.р.

### -ство (= русское -ство)

| Корень | Сущ. на -ство | Перевод |
|--------|---------------|---------|
| **друг** | **друштво** | общество, компания |
| **пријатељ** | **пријатељство** | дружба |
| **брат** | **братство** | братство |
| **дете** | **детињство** | детство |
| **човек** | **човечанство** | человечество |
| **богат** | **богатство** | богатство |
| **царь** | **царство** | царство |

### -ње (от глаголов; «процесс делания»)

| Глагол | Сущ. на -ње | Перевод |
|--------|-------------|---------|
| **читати** | **читање** | чтение |
| **певати** | **певање** | пение |
| **писати** | **писање** | письмо (процесс) |
| **учити** | **учење** | учение, учёба |
| **спавати** | **спавање** | сон (процесс) |
| **играти** | **играње** | игра (процесс) |
| **гледати** | **гледање** | просмотр |

> **Модель:** инфинитив минус *-ти*, плюс *-ње* (часто с йотированием: *носити → ношење, видети → виђење*). Все они **среднего рода**.

## Часть 4: Группа 3 — ДИМИНУТИВ (уменьшительно-ласкательное)

В сербском диминутивы **в активном употреблении** — даже там, где русский уже не уменьшает.

### -ић (м.р.)

| Сущ. | Диминутив | Перевод |
|------|-----------|---------|
| **дом** | **домић** | домик |
| **град** | **градић** | городок |
| **сто** | **столић** | столик |
| **прозор** | **прозорчић** | окошко |
| **нос** | **носић** | носик |

> **Ловушка:** *братић* в современном сербском = **племянник**, а не «братик». Уменьш. от *брат* — *братац*.

### -ица (ж.р.)

| Сущ. | Диминутив | Перевод |
|------|-----------|---------|
| **сестра** | **сестрица** | сестричка |
| **рука** | **ручица** | ручка |
| **вода** | **водица** | водичка |
| **птица** | **птичица** | птичка |
| **књига** | **књижица** | книжечка |

> **Двойная роль -ица:** (а) диминутив для ж.р., (б) женский эквивалент мужского деятеля (*радник → радница, певач → певачица*).

### -че (ср.р.; молодое существо / детёныш)

| Сущ. | Производное | Перевод |
|------|-------------|---------|
| **момак** | **момче** | паренёк, юноша |
| **мачка** | **маче** | котёнок |
| **пас** | **куче** | щенок |
| **унук** | **унуче** | внучек (нейтр.) |

> *-че* — **средний род**, склоняется с расширением *-ет-*: *момче — момчета — момчету*.

---

## Часть 5: Группа 4 — АУГМЕНТАТИВ (увеличительно-пренебрежительное)

Противоположность диминутива: «большой, грубый, пренебрежительный». В разговорной речи и фольклоре — активно.

### -ина / -етина / -чина

| Сущ. | Аугментатив | Перевод |
|------|-------------|---------|
| **човек** | **човечина** | мужичище, здоровяк |
| **рука** | **ручетина / ручурда** | ручища |
| **пас** | **псетина** | псина (грубо) |
| **кућа** | **кућетина / кућерина** | домище |
| **жена** | **женетина** | бабища (грубо) |
| **момак** | **момчина** | здоровый парень |

> **Стиль:** аугментативы — почти всегда **разговорные, с эмоцией** (восхищение или пренебрежение). В нейтральном тексте не используй.
> **Заметка:** **-ина** также образует абстрактные сущ. от прил.: *висок → висина, широк → ширина.* Контекст различает.

## Часть 6: Прилагательные из существительных

### -ски (= русское -ский)

| Сущ. | Прил. | Перевод |
|------|-------|---------|
| **Срб** | **српски** | сербский |
| **жена** | **женски** | женский |
| **човек** | **људски** | человеческий |
| **град** | **градски** | городской |
| **село** | **сеоски** | деревенский |
| **школа** | **школски** | школьный |
| **Београд** | **београдски** | белградский |
| **дете** | **дечји** | детский |
| **студент** | **студентски** | студенческий |

> **Чередования:** перед *-ски* срабатывают фонетические сдвиги: *Француз → француски* (з+с → с), *Енглез → енглески*, *Турчин → турски*.

### -ин (притяжательное от ж.р. имён и родства)

| Сущ. | Прил. | Перевод |
|------|-------|---------|
| **мама** | **мамин** | мамин |
| **сестра** | **сестрин** | сестрин |
| **тетка** | **теткин** | тётин |
| **Ана** | **Анин** | Анин |
| **бака** | **бакин** | бабушкин |

> Согласуется по родам: *мамин шал, мамина торба, мамино име*.

### -ов / -ев (притяжательное от м.р. имён и родства)

| Сущ. | Прил. | Перевод |
|------|-------|---------|
| **брат** | **братов** | братов |
| **отац** | **очев / татин** | отцов / папин |
| **Марко** | **Марков** | Марков |
| **Петар** | **Петров** | Петров |
| **краљ** | **краљев** | королевский |
| **муж** | **мужевљев** | мужний |

> **-ев** появляется после **мягких согласных** (*ј, љ, њ, ћ, ђ, ж, ш, ч, ц, р*): *краљ → краљев, цар → царев*.

> **Важно:** *-ов* в сербском — **полноценное прилагательное**, склоняется по родам, числам, падежам: *Марков ауто, Маркова кућа, Марково име, Маркови пријатељи*.

## Часть 7: Сложные слова (композиты)

Сербский склеивает корни через **соединительную гласную -о- / -е-**.

| Сложное слово | Разбор | Перевод |
|---------------|--------|---------|
| **Београд** | бео + град | «белый город» |
| **родољуб** | род + о + љуб- | патриот |
| **дрводеља** | дрво + о + дељ- | плотник |
| **виноград** | вино + град(-) | виноградник |
| **поштоноша** | пошта + о + нос- | почтальон |
| **месојед** | месо + јед- | плотоядный |
| **староседелац** | стар + о + седел + ац | коренной житель |
| **црногорац** | црн + о + гора + ц | черногорец |

> **Модель:** **корень + о/е + корень (+ суффикс)**.

## Часть 8: Словарные семьи

Один корень даёт **5–10 родственных слов**. Учи их пачкой.

**уч-:** *учити → научити → учитељ/учитељица → учитељски → ученик/ученица → учење → наука → научник/научница → научни.*

**пис-:** *писати → писмо → писац → писар → писање → писмени → рукопис → потпис.*

**род-:** *род → родити → родитељ → родитељски → родбина → рођак/рођака → родни → родољуб → природа.*
