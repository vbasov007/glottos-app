# Программа уроков сербского языка (50 разделов)
**Glottos Matrix — структурный курс (екавская норма, Белград/Сербия)**

> Принцип: 5% — понять правило, 95% — натренировать речевой аппарат. Каждую тему прогоняем через матрицу вопрос-ответ: ја → ти → он/она → ми → ви → они.
> Курс ведётся на **екавском** литературном стандарте Сербии. Иjекавица упоминается только один раз (Урок 48) — на узнавание, не на продукцию.
> Курс ведётся **кириллицей** с Урока 1; латиница вводится позже (Урок 6) как зеркальная замена. До L6 — только кириллица.

## Принципы работы с каждым разделом

1. **Теория** — посмотри видео / прочитай правило (5 минут)
2. **Перевод** — переведи фразы с русского на сербский (кириллицей), проверь по ключу
3. **Языковые гаммы** — прогони правило через все лица (ја → ти → он/она → ми → ви → они)
4. **Матрица** — отработай через вопрос-ответ в трёх сценариях: ты (ед.) → Ви (формал./мн.) → третьи лица
5. **Принцип зеркала** — ответ уже заложен в вопросе, не строй с нуля
6. **Видовая пара** — каждый новый глагол учи **парой**: несовершенный + совершенный. Помечай: *читати (нсв) / прочитати (св)*
7. **Падежная карточка** — каждое новое существительное запоминай с **родом и парой нмн./мн.** (*кућа ж, куће*)

## Хаки для запоминания

- **Кириллица первой**: с L1 по L5 всё только кириллицей. Латиница — на L6, как зеркало. До L6 латиница не используется, чтобы кириллица легла как первая ассоциация.
- **Род**: сербский ≈ русский по интуиции окончаний. Согласная → м.р. (*град, мост*), -а → ж.р. (*кућа, жена*), -о/-е → ср.р. (*село, поље*). Сюрпризы — учи отдельно.
- **Падежи учим парой ед.+мн.**: с самого начала. *кућа → у кући → куће → у кућама*.
- **Аспект — рефлекс, не правило**: с L11 каждое новое глагольное действие — это видовая пара. Несовершенный = «делал/делаю» (процесс); совершенный = «сделал» (один раз, до конца). На L13 это станет автоматическим различием в перфекте.
- **Звуки без русского аналога**: ћ vs ч — оба «ч», но **ћ мягче** («тьч»), **ч твёрже» («тш»). ђ vs џ — то же для «дж»: **ђ мягче**, **џ твёрже**. љ = «ль», њ = «нь», ј = «й». Прогоняй парами: *чика — ћирилица, џем — ђак*.
- **Слоговое р**: *Трст, прст, црн* — между двумя согласными «р» **сам поёт**, как гласная. Не вставляй гласную (НЕ «тэрст»). Просто рычи и отпускай.
- **Без редукции**: безударное «а» в сербском звучит так же чисто, как ударное. *Београд* = «бэ-о-град», не «биагрът». Откажись от русской редукции с первого слова.
- **Клитики на второй позиции**: с L4 знай, что *сам, си, је, ми, ти, му, га, се* — всегда на втором месте в предложении. Это станет правилом L23, но привычку строй сразу.
- **Два алфавита, один язык**: кириллица и латиница — два **облика одного и того же**. После L6 переключайся осознанно: новостной сайт может быть в любом, паспорт — кириллицей, чат — часто латиницей.
