# Текст Б к уроку 4: Откуда я, кто по национальности
**Тема: Страны, национальности, конструкция *из + …***

---

## Текст для аудирования и повторения

1. Ја сам из Русије.
2. Ја сам Рус.
3. Ја сам Рускиња.
4. Ти си из Србије.
5. Ти си Србин.
6. Он је из Београда.
7. Она је из Новог Сада.
8. Ми смо из Москве.
9. Они су из Хрватске.
10. Они су Хрвати.
11. Она је Хрватица.
12. Марко је Црногорац, из Подгорице.
13. Ана је Босанка, из Сарајева.
14. Они су Немци, из Берлина.
15. Она је Немица.
16. Ви сте Француз? *(вежливо, к одному)*
17. Не, ја сам Енглез, из Лондона.
18. Она је Францускиња, из Париза.
19. Они су Американци.
20. Ми смо Италијани, из Рима.
21. Он је Шпанац.
22. Она је Кинескиња.
23. Они су Турци.
24. Она је Гркиња, из Атине.
25. Он је Мађар, из Будимпеште.
26. Они су Македонци, из Скопља.
27. Ја нисам из Београда, ја сам из Новог Сада.
28. — Јесте ли Ви из Србије? — Јесам.
29. — Одакле си ти? — Ја сам из Загреба.
30. Ми смо туристи из Русије, а они су туристи из Кине.
