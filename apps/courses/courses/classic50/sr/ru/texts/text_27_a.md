# Текст A к уроку 27: Описательные относительные предложения
**Тема: Описание людей и вещей через *који* (мужчина, который…; женщина, чей…)**

---

## Текст для аудирования и повторения

1. Човек који седи поред прозора је мој деда.
2. Жена која носи црвену хаљину је моја тетка.
3. Девојка која се смеје је моја најбоља другарица.
4. Дечак који трчи по парку је мој млађи брат.
5. Студент кога видиш тамо учи медицину.
6. Девојка коју познајем из школе сада живи у Берлину.
7. Човек кога сам срео јуче је познати писац.
8. Жена коју слушам на радију је новинарка.
9. Старац коме помажем сваки дан живи преко пута.
10. Девојка којој сам поклонио цвеће је моја вереница.
11. Пријатељ с којим сам одрастао живи у иностранству.
12. Колегиница с којом радим веома је вредна.
13. Професор о коме сви причају држи занимљива предавања.
14. Песникиња о којој се пише у новинама је из Београда.
15. Дете које се игра на улици веома је живахно.
16. Деца која уче српски брзо напредују.
17. Људи који раде овде су веома љубазни.
18. Жене које шетају у парку су наше комшинице.
19. Студенти којима предајем долазе из разних земаља.
20. Девојке с којима учим језик су веома паметне.
21. Човек кога свако поштује је наш директор.
22. Пријатељица коју сам упознала на путу зове ме често.
23. Старија госпођа која живи поред нас увек је насмејана.
24. Млад човек кога сам јуче упознао свира гитару.
25. Лекар коме верујем ради у градској болници.
26. Студенткиња с којом делим собу је из Сарајева.
27. Дечаци који играју фудбал на терену су из мог разреда.
28. Девојчица којој причам бајке је моја сестричина.
29. Уметник о коме сви говоре излаже у галерији.
30. Особа на коју рачунам је моја мајка.
