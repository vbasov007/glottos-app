# Словарь курса сербского языка
**50 уроков + 150 текстов для аудирования (A1 → B2, екавская норма)**

> Сортировка по сербской кириллице. Род указан только для существительных: **м** (мужской), **ж** (женский), **с** (средний), **мн** (только множественное). В сербском нет артиклей, поэтому слово даётся без них.
> Для глаголов вид помечается в скобках после русского перевода (нсв = несовершенный, св = совершенный); видовая пара даётся вместе.
