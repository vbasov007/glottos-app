# Listening text for Lesson 1 — Variant A: A scuola di italiano

**Theme: first day at Italian school**

---

## Listening text

1. Ciao!
2. Buongiorno!
3. Buongiorno, signora.
4. Come stai?
5. Bene, grazie. E tu?
6. Bene.
7. Io sono Anna.
8. Tu come ti chiami?
9. Mi chiamo Marco.
10. Piacere, Marco.
11. Piacere, Anna.
12. Sono studente.
13. Anche io sono studente.
14. Questo è un libro.
15. Questa è una penna.
16. Questa è una sedia.
17. Questo è un tavolo.
18. Per favore, ripeti.
19. Sì, certo.
20. Grazie mille.
21. Prego.
22. Uno, due, tre.
23. Quattro, cinque, sei.
24. Sette, otto, nove, dieci.
25. Undici, dodici, tredici.
26. Quattordici, quindici, sedici.
27. Diciassette, diciotto.
28. Diciannove, venti.
29. Arrivederci, Anna.
30. Ciao, Marco. A domani!

---

| grazie | thank you |
| io sono | I am |
| tu | you |
| come ti chiami | what's your name |
| mi chiamo | my name is |
| piacere | nice (to meet you) |
| studente | student |
| anche | also, too |
| questo / questa | this (m. / f.) |
| libro | book |
| penna | pen |
| sedia | chair |
| tavolo | table |
| per favore | please (when asking) |
| ripeti | repeat |
| sì | yes |
| certo | of course |
| mille | a thousand; "mille grazie" — thanks a lot |
