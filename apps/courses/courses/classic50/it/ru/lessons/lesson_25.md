# Урок 25: Futuro semplice. Будущее и догадка
**Словарный запас: маркеры будущего, планы, предсказания, вероятность**

---

## Как работать с этим уроком

1. **Прочитай Часть 1** — будущее в итальянском похоже на испанское по структуре, но **проще** в окончаниях. Главная новость: то, что русский говорит «буду делать», итальянец чаще говорит **простым** настоящим.
2. **Запомни регулярные окончания** — одни на все три спряжения: *-ò, -ai, -à, -emo, -ete, -anno*.
3. **Выучи 9 нерегулярных основ** — sarò, avrò, farò, andrò, verrò, potrò, dovrò, saprò, vorrò. Эти **корни вернутся** в conditional (L26), поэтому они load-bearing.
4. **Обрати особое внимание на догадку:** *Saranno le otto* — «Должно быть, восемь». Это **уникальная функция** итальянского будущего, которой нет в русском.

> Это **последний** урок Блока 3. После него все базовые времена итальянского у тебя в руках: настоящее, два прошедших и будущее. С урока 26 — система ветвится в conditional и subjunctive.

## Часть 1: Что такое futuro semplice

**Futuro semplice** — простое будущее. Используется для:

1. **Предсказаний** — что произойдёт.
2. **Планов на отдалённое будущее** — на следующий год, через 10 лет.
3. **Обещаний и решений** — «я сделаю это».
4. **Догадки / вероятности в настоящем** — «должно быть, что-то».

> **Главное предупреждение:** для **близких планов** и **уверенных намерений** итальянцы чаще ставят **простое настоящее** или **stare per + инфинитив**:
> <say>Domani vado al cinema.</say> — Завтра иду в кино (planированно, уверенно).
> <say>Sto per uscire.</say> — Я собираюсь выходить (прямо сейчас).
> *Futuro semplice* в таких контекстах звучит **более формально, отдалённо или менее уверенно**.

Сравни:

| Простое настоящее (близкое, уверенное) | Futuro semplice (отдалённое / предсказание / обещание) |
|----------------------------------------|--------------------------------------------------------|
| <say>Domani lavoro a Milano.</say> | <say>L'anno prossimo lavorerò a Milano.</say> |
| <say>Stasera mangio a casa.</say> | <say>Domani penso a quello che mangerò.</say> |
| <say>Vengo subito.</say> | <say>Un giorno verrò a trovarti.</say> |

## Часть 2: Форма — регулярные окончания

В отличие от других времён, futuro имеет **одни окончания для всех трёх спряжений** (с одной маленькой поправкой для -are).

### Спряжения -are и -ere — окончания идентичны (важно!)

Глаголы на **-are** в futuro меняют **a → e** в основе: *parlare → parler-* (как глаголы на -ere). Дальше — общие окончания.

| Лицо | Окончание |
|------|-----------|
| io | **-ò** |
| tu | **-ai** |
| lui / lei | **-à** |
| noi | **-emo** |
| voi | **-ete** |
| loro | **-anno** |

| parlare (-are) | credere (-ere) | dormire (-ire) |
|----------------|----------------|----------------|
| parler**ò** | creder**ò** | dormir**ò** |
| parler**ai** | creder**ai** | dormir**ai** |
| parler**à** | creder**à** | dormir**à** |
| parler**emo** | creder**emo** | dormir**emo** |
| parler**ete** | creder**ete** | dormir**ete** |
| parler**anno** | creder**anno** | dormir**anno** |

> **Ключевая мнемоника:** окончания futuro — это **«ò, ai, à, emo, ete, anno»**. По сути ты слышишь в них слова **«ho, hai, ha, abbiamo, avete, hanno»** (формы avere)! Это не совпадение — futuro исторически произошёл из «инфинитив + avere».

### Ударения

> Все формы 1-го и 3-го лица единственного числа — с **ударным значком**: **parler**ò**, parler**à**. Не забудь графический акцент — он различает: *parlerà* (он будет говорить) vs *parlera* (несуществующая форма).

### Орфографические нюансы

- **-care, -gare** → добавляют **h** для сохранения твёрдого звука: cercare → **cercherò**, pagare → **pagherò**.
- **-ciare, -giare** → теряют **i** перед e: cominciare → **comincerò**, mangiare → **mangerò**.

## Часть 3: Девять нерегулярных основ — load-bearing

Эти основы вернутся **слово в слово** в conditional (L26). Запомни их сейчас — экономишь два урока.

| Инфинитив | Основа будущего | io-форма | Перевод |
|-----------|-----------------|----------|---------|
| <say>essere</say> | sar- | **sarò** | буду |
| <say>avere</say> | avr- | **avrò** | буду иметь |
| <say>fare</say> | far- | **farò** | сделаю |
| <say>andare</say> | andr- | **andrò** | пойду |
| <say>venire</say> | verr- | **verrò** | приду |
| <say>potere</say> | potr- | **potrò** | смогу |
| <say>dovere</say> | dovr- | **dovrò** | буду должен |
| <say>sapere</say> | sapr- | **saprò** | узнаю |
| <say>volere</say> | vorr- | **vorrò** | захочу |

### Полные парадигмы — самых частотных

#### essere → sarò

| Лицо | Форма |
|------|-------|
| io | **sarò** |
| tu | **sarai** |
| lui / lei | **sarà** |
| noi | **saremo** |
| voi | **sarete** |
| loro | **saranno** |

#### avere → avrò

| Лицо | Форма |
|------|-------|
| io | **avrò** |
| tu | **avrai** |
| lui / lei | **avrà** |
| noi | **avremo** |
| voi | **avrete** |
| loro | **avranno** |

#### andare → andrò

| Лицо | Форма |
|------|-------|
| io | **andrò** |
| tu | **andrai** |
| lui / lei | **andrà** |
| noi | **andremo** |
| voi | **andrete** |
| loro | **andranno** |

> **Хак запоминания:** essere и avere — самые частотные, выучи их твёрдо. Остальные семь — заметь паттерн **«потеря гласной из инфинитива»**: andare → andr-, dovere → dovr-, potere → potr-, sapere → sapr-, avere → avr-. *Venire* и *volere* — особые: **двойное r** (verr-, vorr-).

### Несколько ещё нерегулярных (вспомогательные)

| Инфинитив | Основа | Перевод |
|-----------|--------|---------|
| <say>vedere</say> | vedr- | увидеть |
| <say>rimanere</say> | rimarr- | остаться |
| <say>tenere</say> | terr- | держать |
| <say>vivere</say> | vivr- | жить |
| <say>bere</say> | berr- | пить |

## Часть 4: Использования

### Использование 1: Предсказание

<say>Domani **pioverà**.</say> — Завтра пойдёт дождь.
<say>Nei prossimi anni il clima **cambierà**.</say> — В ближайшие годы климат изменится.
<say>L'anno prossimo i prezzi **aumenteranno**.</say> — В следующем году цены вырастут.

### Использование 2: Отдалённые планы

<say>Tra dieci anni **vivrò** in Italia.</say> — Через десять лет я буду жить в Италии.
<say>Quando avrò trent'anni, **comprerò** una casa.</say> — Когда мне будет тридцать, я куплю дом.
<say>Dopo la laurea **cercherò** lavoro all'estero.</say> — После выпуска я буду искать работу за границей.

### Использование 3: Обещание и решение

<say>**Ti chiamerò** stasera.</say> — Я тебе позвоню вечером.
<say>**Non lo dirò** a nessuno.</say> — Я никому об этом не скажу.
<say>**Ti aiuterò**, promesso.</say> — Я помогу тебе, обещаю.

### Использование 4: Догадка / вероятность в настоящем

Это **уникальная функция** futuro, которой нет в русском напрямую. Когда итальянец говорит **о настоящем** в futuro — он выражает **догадку**.

<say>**Saranno** le otto.</say> — Должно быть, восемь часов. (= наверное, восемь)
<say>Maria non risponde. **Sarà** occupata.</say> — Мария не отвечает. Должно быть, она занята.
<say>Quanti anni avrà? **Avrà** quarant'anni.</say> — Сколько ей лет? Наверное, сорок.
<say>Marco non c'è. **Sarà** ancora a casa.</say> — Марко нет. Наверное, он ещё дома.

> **Запомни мост к русскому:** *Sarà…* / *Saranno…* = «должно быть» / «наверное». Если можешь вставить русское «наверное», итальянец почти всегда поставит futuro.

### Использование 5: Уступительная фраза «может, и…»

<say>**Sarà** ricco, ma non è felice.</say> — Может, он и богат, но не счастлив.
<say>**Avrà** ragione, ma non lo ascolto.</say> — Может, он и прав, но я его не слушаю.

> Это книжная функция, но встречается в живой речи.

### Использование 6: После se / quando / appena в придаточном будущего

Когда главное предложение в futuro, придаточное с *se / quando / appena* **тоже** часто идёт в futuro (отличие от английского и испанского).

<say>**Quando arriverò** a Roma, **ti chiamerò**.</say> — Когда я приеду в Рим, я тебе позвоню.
<say>**Se piove** domani, **non usciremo**.</say> — Если завтра пойдёт дождь, мы не пойдём гулять. (с *se* — обычно настоящее)
<say>**Se pioverà** sabato, andremo al cinema.</say> — Если будет дождь в субботу, пойдём в кино. (futuro допустим)
<say>**Appena vedrò** Marco, **gli dirò** tutto.</say> — Как только увижу Марко, скажу ему всё.

> **Правило различия:** с **quando** и **appena** в будущем смысле — **только futuro** в обеих частях. С **se** — обычно настоящее в условии (это «реальное» условие, L42).

## Часть 5: Маркеры будущего

Эти слова сигналят будущее. С ними futuro semplice уместен — или, для близкого будущего, простое настоящее.

| Italiano | Русский |
|----------|---------|
| <say>domani</say> | завтра |
| <say>dopodomani</say> | послезавтра |
| <say>la prossima settimana</say> | на следующей неделе |
| <say>il mese prossimo</say> | в следующем месяце |
| <say>l'anno prossimo</say> | в следующем году |
| <say>fra / tra un'ora</say> | через час |
| <say>fra dieci giorni</say> | через десять дней |
| <say>fra poco</say> | скоро |
| <say>presto</say> | скоро, рано |
| <say>più tardi</say> | позже |
| <say>fra cinque anni</say> | через пять лет |
| <say>in futuro</say> | в будущем |
| <say>un giorno</say> | однажды (в будущем) |
| <say>quando avrò X anni</say> | когда мне будет X лет |
| <say>dopo</say> | после |
| <say>nel duemilatrenta</say> | в 2030 году |

> **Заметь:** *fra* и *tra* — одно и то же ("через сколько"), выбор по благозвучию. *Fra dieci anni* / *Tra dieci anni* — оба норма.

## Language scales

### Scale 1 — parlare (регулярный -are)

> <say>Domani io parlerò con Marco.</say>
> <say>Domani tu parlerai con Marco.</say>
> <say>Domani lui parlerà con Marco.</say>
> <say>Domani noi parleremo con Marco.</say>
> <say>Domani voi parlerete con Marco.</say>
> <say>Domani loro parleranno con Marco.</say>

### Scale 2 — essere (нерегулярный)

> <say>L'anno prossimo io sarò in Italia.</say>
> <say>L'anno prossimo tu sarai in Italia.</say>
> <say>L'anno prossimo lui sarà in Italia.</say>
> <say>L'anno prossimo noi saremo in Italia.</say>
> <say>L'anno prossimo voi sarete in Italia.</say>
> <say>L'anno prossimo loro saranno in Italia.</say>

### Scale 3 — догадка (sarà)

> <say>Chi è? Sarà Marco.</say>
> <say>Quanti anni ha? Avrà trent'anni.</say>
> <say>Dove sono? Saranno a casa.</say>
> <say>Che ora è? Saranno le dieci.</say>
> <say>Perché non risponde? Sarà occupata.</say>

### Scale 4 — quando + futuro

> <say>Quando arriverò a Roma, ti chiamerò.</say>
> <say>Quando arriverai a Roma, mi chiamerai.</say>
> <say>Quando arriverà a Roma, ci chiamerà.</say>
> <say>Quando arriveremo a Roma, ti chiameremo.</say>
> <say>Quando arriverete a Roma, mi chiamerete.</say>
> <say>Quando arriveranno a Roma, ci chiameranno.</say>

## Упражнения

### Упражнение 1. Поставь в futuro semplice

Поставь глагол в правильную форму:

1. (io / parlare) ___ con il direttore domani.
2. (tu / venire) ___ alla festa stasera?
3. (lui / arrivare) ___ alle sette.
4. (noi / mangiare) ___ in un ristorante italiano.
5. (voi / studiare) ___ insieme per l'esame.
6. (loro / partire) ___ per Roma la prossima settimana.
7. (io / essere) ___ in vacanza in agosto.
8. (lei / avere) ___ trent'anni a marzo.
9. (noi / fare) ___ una grande festa.
10. (loro / andare) ___ al mare.
11. (io / sapere) ___ la verità presto.
12. (tu / dovere) ___ aspettare un po'.

<details><summary>Ключи</summary>

1. *Parlerò con il direttore domani.* (заметь: -a- → -e-)
2. *Verrai alla festa stasera?* (нерегулярный — verr-)
3. *Arriverà alle sette.* (заметь: arriver-, не arrivar-)
4. *Mangeremo in un ristorante italiano.* (mangiare → mangeremo, -i- выпадает)
5. *Studierete insieme per l'esame.*
6. *Partiranno per Roma la prossima settimana.*
7. *Sarò in vacanza in agosto.*
8. *Avrà trent'anni a marzo.*
9. *Faremo una grande festa.*
10. *Andranno al mare.*
11. *Saprò la verità presto.*
12. *Dovrai aspettare un po'.*

</details>

### Упражнение 2. Догадка о настоящем

Переведи на итальянский, используя futuro для догадки:

1. Должно быть, восемь часов.
2. Наверное, Марко занят.
3. Кто это? Наверное, почтальон.
4. Ему, должно быть, лет сорок.
5. Они, наверное, дома.
6. Где Анна? Наверное, на работе.
7. Сколько это будет стоить? Наверное, сто евро.
8. Почему она не звонит? Должно быть, забыла.

<details><summary>Ключи</summary>

1. *Saranno le otto.*
2. *Marco sarà occupato.* (или: *Sarà occupato Marco.*)
3. *Chi è? Sarà il postino.*
4. *Avrà quarant'anni.* (или: *Avrà circa quarant'anni.*)
5. *Saranno a casa.*
6. *Dov'è Anna? Sarà al lavoro.*
7. *Quanto costerà? Costerà circa cento euro.*
8. *Perché non chiama? Avrà dimenticato.* (или: *…si sarà dimenticata.*; в простом futuro — *dimenticherà*, но логичнее «forse ha dimenticato»)

</details>

### Упражнение 3. Переведи на итальянский

1. Завтра я буду работать дома.
2. На следующей неделе мы поедем в Рим.
3. Через год я закончу университет.
4. Когда я приеду в Москву, я тебе позвоню.
5. Если завтра будет дождь, мы останемся дома. (используй настоящее в условии)
6. Они приедут в восемь часов.
7. Что ты будешь делать в субботу?
8. Я думаю, что Марко не придёт.
9. Когда мне будет тридцать, я куплю дом.
10. Скоро ты узнаешь правду.
11. Должно быть, мне сейчас полночь.
12. В будущем мы будем жить дольше.

1. *Domani lavorerò a casa.* (или, ближе к разговорному: *Domani lavoro a casa.*)

5. *Se piove domani, resteremo a casa.* (или: *…rimarremo a casa.*)

9. *Quando avrò trent'anni, comprerò una casa.*
10. *Presto saprai la verità.* (или: *Saprai presto la verità.*)
11. *Sarà mezzanotte ormai.* (или: *Sarà già mezzanotte.*)
12. *In futuro vivremo più a lungo.*

</details>

### Упражнение 4. Найди и исправь ошибку

1. <say>Domani io parlarò con Marco.</say>
2. <say>L'anno prossimo verrerò in Italia.</say>
3. <say>Andrà al cinema sabato.</say> (без ошибки? проверь)
4. <say>Quando io arrivo a Roma, ti chiamerò.</say>
5. <say>Sara le otto.</say>
6. <say>Mangerò la pizza.</say> (проверь корректность)
7. <say>Pagaranno il conto loro.</say>

<details><summary>Ключи</summary>

1. *Domani parlerò con Marco.* — parlare → parler- (a меняется на e!).
2. *L'anno prossimo verrò in Italia.* — нерегулярный venire → verr-, не verrer-.
3. Без ошибки. *Andrà al cinema sabato.* — корректно.
4. *Quando arriverò a Roma, ti chiamerò.* — с *quando* в будущем смысле — futuro в обеих частях.
5. *Saranno le otto.* — должно быть мн. ч. (le otto — мн.), + графический акцент на *à*. Или *Sarà l'una.*
6. Корректно. *Mangerò la pizza.* — mangiare теряет i перед e: mang+erò.
7. *Pagheranno il conto loro.* — pagare добавляет h перед e: pagh+eranno.

</details>

## Памятка урока

```
ФОРМА — ОДНИ ОКОНЧАНИЯ ДЛЯ ВСЕХ:
  -ò, -ai, -à, -emo, -ete, -anno

ОСНОВЫ ДЛЯ ТРЁХ СПРЯЖЕНИЙ:
  -are: parlare → parler-   (a → e!)
  -ere: credere → creder-
  -ire: dormire → dormir-

  parlerò, parlerai, parlerà, parleremo, parlerete, parleranno

ОРФОГРАФИЯ:
  -care, -gare → +h:  cercare → cercherò, pagare → pagherò
  -ciare, -giare → -i: cominciare → comincerò, mangiare → mangerò

9 НЕРЕГУЛЯРНЫХ ОСНОВ (повторятся в conditional L26!):
  essere  → sar-     sarò, sarai, sarà…
  avere   → avr-     avrò, avrai, avrà…
  fare    → far-     farò, farai, farà…
  andare  → andr-    andrò, andrai…
  venire  → verr-    verrò, verrai…    (двойное r!)
  potere  → potr-    potrò, potrai…
  dovere  → dovr-    dovrò, dovrai…
  sapere  → sapr-    saprò, saprai…
  volere  → vorr-    vorrò, vorrai…    (двойное r!)

Ещё: vedere → vedrò, rimanere → rimarrò, tenere → terrò,
     vivere → vivrò, bere → berrò.

ИСПОЛЬЗОВАНИЯ:
  1. Предсказание:      Domani pioverà.
  2. Отдалённый план:   Tra dieci anni vivrò a Roma.
  3. Обещание:          Ti chiamerò stasera.
  4. Догадка о НАСТОЯЩЕМ:  Saranno le otto. (= наверное, восемь)
  5. Уступка:           Sarà ricco, ma non è felice.
  6. После quando/appena: Quando arriverò, ti chiamerò.

ОЧЕНЬ ВАЖНО — РАЗГОВОРНОЕ:
  Близкое и уверенное будущее — чаще ПРОСТОЕ НАСТОЯЩЕЕ:
    Domani vado al cinema.  (не obbligatoriamente "andrò")
  Stare per + инфинитив — «вот-вот»:
    Sto per uscire.  (я собираюсь выходить)
  Futuro semplice звучит более формально / отдалённо.

ДОГАДКА — УНИКАЛЬНАЯ ФУНКЦИЯ:
  Saranno le otto.        — Должно быть, восемь.
  Avrà quarant'anni.      — Ему, наверное, сорок.
  Sarà occupata.          — Должно быть, занята.
  Мост к русскому: «наверное» → futuro в итальянском.

С QUANDO / APPENA:
  Quando + futuro, futuro:  Quando arriverò, ti chiamerò.
  Appena + futuro, futuro:  Appena lo vedrò, glielo dirò.
  С SE — обычно настоящее в условии: Se piove, restiamo.
```

> **Следующий шаг:** Урок 26 — **condizionale**. Та же девятка нерегулярных основ, но с другими окончаниями: *sarei, avrei, farei, andrei*. Это «бы»-форма — вежливость, гипотезы, советы, мечты. Самое полезное время для повседневной вежливой речи: *vorrei un caffè* — «я бы хотел кофе».
