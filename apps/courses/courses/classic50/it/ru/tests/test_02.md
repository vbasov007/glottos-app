# Тест к уроку 2: Род и число существительных; числа 20–100

**Как сказать это по-итальянски?**
Переведи фразы. Указывай род артиклем там, где он напрашивается, и не забывай менять окончание во множественном числе (-o → -i, -a → -e, -e → -i).
