## Sefardyjska (= współcz. izrael.) vs. aszkenazyjska wymowa

| Hebrajski | Sefardyjska (standard) | Aszkenazyjska | Polski |
|-------|------------------------|-------------|---------|
| שבת שלום | shabbat shalom | **shabbos sholom** | pokojowej soboty |
| שבת קודש | shabbat kodesh | **shabbos kóydesh** | święta sobota |
| תורה | tora | **tóyrah** | Tora |
| שלום עליכם | shalom aleichem | **shólom aléykhem** | pokój wam |
| ישראל | yisra'el | **yísroel / yisrúel** | Izrael |
| משפחה | mishpacha | **mishpókhe** | rodzina |
| בית כנסת | beit knesset | **béyz knéses** | synagoga |
| אבא ואמא | abba ve-imma | **tate un mame** (często w jidysz) / **ábba un ímma** | tata i mama |
| סבא וסבתא | saba ve-savta | **zéyde un bóbe** (jidysz) / **sábe un sávte** | dziadek i babcia |
| שמע ישראל | shma yisra'el | **shma yísroel** | słuchaj, Izraelu |
| שנה טובה | shana tova | **shóne tóyve** | dobrego roku |
| גמר חתימה טובה | gmar chatima tova | **gmar khasíme tóyve** | dobrego podpisu (w Jom Kipur) |
| מזל טוב | mazal tov | **mázel tov** | gratulacje |
| ברוך השם | barukh hashem | **bórekh hashém / bórikh ashém** | błogosławione Imię (= dzięki Bogu) |
| תודה לאל | toda la'el | **tóyde lóel** | dziękowanie Bogu |
| ספר תורה | sefer tora | **séyfer tóyrah** | zwój Tory |
| פסח | pesach | **péysakh** | Pesach |
| סוכות | sukkot | **súkkes** | Sukkot |
| חנוכה | chanuka | **khánuke / khánike** | Chanuka |
| ראש השנה | rosh hashana | **róysh hashóne** | Nowy Rok |
| יום כיפור | yom kippur | **yóm kípper** | Jom Kipur |
| ארץ ישראל | eretz yisra'el | **érets yísroel** | Ziemia Izraela |
| עם ישראל חי | am yisra'el chai | **am yísroel khái** | naród Izraela żyje |
| אדון עולם | adon olam | **ádoyn óylom** | Władca świata |
| אבינו שבשמיים | avinu shebashamayim | **óvinu shebashomáyim** | Ojciec nasz w niebiosach |
| רחמנא ליצלן | rachmana litzlan | **rakhmóne litslán** | Miłosierny niech ocali |
| אם ירצה השם | im yirtse hashem | **im yírtse hashém** | jeśli zechce Imię |
| בעזרת השם | be'ezrat hashem | **b'ezras hashém** | z pomocą Imienia |
| כל הכבוד | kol hakavod | **kol hakóvod** | wszelki szacunek / brawo |
| לחיים | lechayim | **l'kháyim** | za życie! (toast) |

> **Co się zmienia:** kamac (a → o), cholam (o → oj), ת bez dagesz (t → s), ה na końcu ż.r. (-ah → -e), akcent ciągnie na przedostatnią sylabę.
