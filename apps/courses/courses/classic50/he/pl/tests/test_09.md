# Test do lekcji 9: Rodzajnik ha-, zgodność przymiotnika, prefiksy-przyimki ב/ל/מ

**Jak to powiedzieć po hebrajsku?**
Przetłumacz frazy na hebrajski z nikudem. Pilnuj: rodzajnik ה- łączny, przepływa na przymiotnik; prefiksy ב-, ל-, מ- pisze się w jednym słowie; *be- + ha- = ba-*, *le- + ha- = la-*, *mi- + ha- = me-ha-*.

1. *סֵפֶר גָּדוֹל* — **sefer gadol** (oba bez ha-)
2. *הַסֵּפֶר הַגָּדוֹל* — **ha-sefer ha-gadol** (rodzajnik na obu)
3. *יַלְדָּה קְטַנָּה* — **yalda ktana** (ż.l.poj. + ż.l.poj.)
4. *הַיַּלְדָּה הַקְּטַנָּה* — **ha-yalda ha-ktana**
5. *בַּיִת חָדָשׁ* — **bayit chadash** (m.l.poj. + m.l.poj.)
6. *הַבַּיִת הֶחָדָשׁ* — **ha-bayit he-chadash** (*he-* przed ח — typowy wariant)
7. *יְלָדִים טוֹבִים* — **yeladim tovim** (m.l.mn. + m.l.mn.)
8. *הַיְלָדִים הַטּוֹבִים* — **ha-yeladim ha-tovim**
9. *נָשִׁים חֲכָמוֹת* — **nashim chakhamot** (ż.l.mn. + ż.l.mn.)
10. *הַנָּשִׁים הַחֲכָמוֹת* — **ha-nashim ha-chakhamot**
11. *מְכוֹנִית אֲדֻמָּה* — **mekhonit aduma** (ż.l.poj. + ż.l.poj.; *mekhonit* — ż.r.)
12. *הַמְּכוֹנִית הָאֲדֻמָּה* — **ha-mekhonit ha-aduma**
13. *דֶּלֶת לְבָנָה* — **delet levana** (ż.l.poj. + ż.l.poj.; *delet* — ż.r.)
14. *הַדֶּלֶת הַלְּבָנָה* — **ha-delet ha-levana**
15. *אִישׁ גָּבוֹהַּ* — **ish gavoha** (m.l.poj. + m.l.poj.)
16. *הָאִישׁ הַגָּבוֹהַּ* — **ha-ish ha-gavoha** (przed א rodzajnik wydłuża się w *ha-* z kamacem)
17. *הַבַּיִת גָּדוֹל* — **ha-bayit gadol** (rodzajnik tylko na rzecz. → zdanie „dom duży")
18. *הַמּוֹרָה טוֹבָה* — **ha-mora tova** („nauczycielka — dobra")
19. *הַסֵּפֶר חָדָשׁ* — **ha-sefer chadash** („książka — nowa")
20. *הַיְלָדִים שְׂמֵחִים* — **ha-yeladim smechim** („dzieci — wesołe")
21. *בְּבַיִת* — **be-vayit** (ב- + bayit, bez rodzajnika)
22. *בַּבַּיִת* — **ba-bayit** (ב- + ha-bayit; sklejenie *be- + ha- = ba-*)
23. *לְמוֹרֶה* — **le-more**
24. *לַמּוֹרֶה* — **la-more** (sklejenie *le- + ha- = la-*)
25. *מֵעִיר* — **me-ir** (מ- przed gardłową ע = *me-*)
26. *מֵהָעִיר* — **me-ha-ir** (מ- NIE łączy się z rodzajnikiem; rodzajnik ה- zostaje; przed ע — *ha-* z kamacem)
27. *לְבֵית הַסֵּפֶר* — **le-vet ha-sefer** (dosłownie „do domu-książki"; „szkoła" — *bet-sefer*)
28. *מֵהַיְלָדִים* — **me-ha-yeladim** (od dzieci — określonych)
29. *יַלְדָּה קְטַנָּה בַּבַּיִת הַגָּדוֹל* — **Yalda ktana ba-bayit ha-gadol.** (dziewczynka nieokreślona → bez ha-; dom określony → *ba-* + ha-gadol)
30. *הָאֲנָשִׁים הָעֲיֵפִים הָאֵלֶּה מֵהָעִיר* — **Ha-anashim ha-ayefim ha-ele me-ha-ir.** (rodzajnik na rzecz. + dwóch przym.; *me-ha-* — z określonego miasta)
