# Test zu Lektion 34: Verbalsubstantive (shem pe'ula) nach Binjanim

**Wie sagt man es auf Hebräisch?**
Übersetze die Wendungen ins Hebräische. Nutze die Grammatik und den Wortschatz dieser Lektion.

---

1. Schreiben (Niederschrift) ist eine Kunst. (Pa'al → ktiva)
2. Das Lesen von Büchern ist wichtig. (Pa'al → kri'a)
3. Das Lernen (als Prozess) ist ein langer Weg. (Pa'al → lemida)
4. Die Wahl war schwer. (Pa'al → bchira)
5. Die Eröffnung des Ladens war gestern. (Pa'al → ptikha)
6. Die Erzählung ist interessant. (Pi'el → sipur)
7. Bildung (Erziehung) in Israel ist wichtig. (Pi'el → chinukh)
8. Die Leitung der Firma ist anspruchsvoll. (Pi'el → nihul)
9. Die Suche nach Arbeit dauert an. (Pi'el → chipus)
10. Die Veränderung ist schnell. (Pi'el → shinui)
11. Das Erhalten des Passes dauert einen Monat. (Pi'el → kabala)
12. Das Einkommen des Staates ist gefallen. (Hif'il → haknasa)
13. Die Ausgaben der Firma sind gestiegen. (Hif'il → hotsa'a)
14. Die Entscheidung des Lehrers war richtig. (Hif'il → hachlata)
15. Die Anmeldung zum Kurs ist offen. (Hif'il → harshama)
16. Die Einladung (Bestellung) ist fertig. (Hif'il → hazmana)
17. Erfolg im Studium ist wichtig. (Hif'il → hatslacha)
18. Die Investition ist groß. (Hif'il → hashka'a)
19. Die Bekanntmachung war in den Nachrichten. (Hif'il → hakhraza)
20. Die Entwicklung des Kindes ist schnell. (Hitpa'el → hitpatchut)
21. Das Erwachen war früh. (Hitpa'el → hit'orerut)
22. Ihre Aufregung war groß. (Hitpa'el → hitragshut)
23. Das Interesse am Thema wächst. (Hitpa'el → hit'anyenut)
24. Die Korrespondenz mit dem Freund dauert seit Jahren. (Hitpa'el → hitkatvut)
25. Der Fortschritt ist sichtbar. (Hitpa'el → hitkadmut)
26. Seine Einstellung zur Arbeit ist gut. (Hitpa'el → hityachasut)
27. Der Eintritt in den Saal war leise. (Nif'al → hikansut)
28. Vorsicht ist notwendig. (Nif'al → hizaharut)
29. Die Abwesenheit des Lehrers wurde bemerkt. (Nif'al → he'adrut)
30. Die Entscheidung über die Schließung der Firma wurde getroffen. (hachlata + sgira — zwei shem pe'ula)

---

<details>
<summary>Lösung</summary>

1. *הכתיבה היא אומנות.* — **ha-ktiva hi omanut** (ktiva, Pa'al-shem-pe'ula von likhtov)
2. *קריאת ספרים חשובה.* — **kri'at sfarim chashuva** (kri'a in Smichut → kri'at, Pa'al von likro)
3. *הלמידה היא דרך ארוכה.* — **ha-lemida hi derekh aruka** (lemida, Pa'al von lilmod)
4. *הבחירה הייתה קשה.* — **ha-bchira hayta kasha** (bchira, Pa'al von livchor)
5. *פתיחת החנות הייתה אתמול.* — **ptichat ha-chanut hayta etmol** (ptikha → ptichat in Smichut, Pa'al von liftoach)
6. *הסיפור מעניין.* — **ha-sipur me'anyen** (sipur, Pi'el von lesaper)
7. *החינוך בישראל חשוב.* — **ha-chinukh be-Yisra'el chashuv** (chinukh, Pi'el von lechanekh)
8. *ניהול החברה מסובך.* — **nihul ha-chevra mesubakh** (nihul, Pi'el von lenahel; Smichut)
9. *החיפוש אחר עבודה נמשך.* — **ha-chipus achar avoda nimshakh** (chipus, Pi'el von lechapes)
10. *השינוי מהיר.* — **ha-shinui mahir** (shinui, Pi'el von leshanot)
11. *קבלת הדרכון אורכת חודש.* — **kabalat ha-darkon orekhet chodesh** (kabala → kabalat in Smichut, Pi'el von lekabel)
12. *הכנסות המדינה ירדו.* — **haknasot ha-medina yardu** (haknasa, Pl. haknasot, Hif'il von lehakhnis)
13. *הוצאות החברה גדלו.* — **hotsa'ot ha-chevra gadlu** (hotsa'a, Pl., Hif'il von lehotsi)
14. *ההחלטה של המורה הייתה נכונה.* — **ha-hachlata shel ha-more hayta nekhona** (hachlata, Hif'il von lehachlit)
15. *ההרשמה לקורס פתוחה.* — **ha-harshama la-kurs ptucha** (harshama, Hif'il von leharshim)
16. *ההזמנה מוכנה.* — **ha-hazmana mukhana** (hazmana, Hif'il von lehazmin)
17. *ההצלחה בלימודים חשובה.* — **ha-hatslacha ba-limudim chashuva** (hatslacha, Hif'il von lehatsliach)
18. *ההשקעה גדולה.* — **ha-hashka'a gdola** (hashka'a, Hif'il von lehashki'a)
19. *ההכרזה הייתה בחדשות.* — **ha-hakhraza hayta ba-chadashot** (hakhraza, Hif'il von lehakhriz)
20. *ההתפתחות של הילד מהירה.* — **ha-hitpatchut shel ha-yeled mehira** (hitpatchut, Hitpa'el von lehitpate'ach)
21. *ההתעוררות הייתה מוקדמת.* — **ha-hit'orerut hayta mukdemet** (hit'orerut, Hitpa'el von lehit'orer)
22. *ההתרגשות שלה הייתה גדולה.* — **ha-hitragshut shela hayta gdola** (hitragshut, Hitpa'el von lehitragesh)
23. *ההתעניינות בנושא גדלה.* — **ha-hit'anyenut ba-nose gdela** (hit'anyenut, Hitpa'el von lehit'anyen)
24. *ההתכתבות עם החבר נמשכת שנים.* — **ha-hitkatvut im ha-chaver nimshekhet shanim** (hitkatvut, Hitpa'el von lehitkatev)
25. *ההתקדמות ניכרת.* — **ha-hitkadmut nikeret** (hitkadmut, Hitpa'el von lehitkadem)
26. *ההתייחסות שלו לעבודה טובה.* — **ha-hityachasut shelo la-avoda tova** (hityachasut, Hitpa'el von lehityaches)
27. *הכניסה לאולם הייתה שקטה.* — **ha-knisa la-ulam hayta sheketa** (knisa — alltägliche Pa'al-Form; Variante Nif'al: ha-hikansut)
28. *ההיזהרות הכרחית.* — **ha-hizaharut hekhrechit** (hizaharut, Nif'al von lehizaher)
29. *ההיעדרות של המורה נרשמה.* — **ha-he'adrut shel ha-more nirshema** (he'adrut, Nif'al von lehe'ader)
30. *ההחלטה על סגירת החברה התקבלה.* — **ha-hachlata al sgirat ha-chevra hitkabla** (zwei shem pe'ula: hachlata von Hif'il + sgira → sgirat in Smichut, Pa'al von lisgor)

</details>
