# Test zu Lektion 9: Artikel ha-, Adjektivkongruenz, Präpositionspräfixe ב/ל/מ

**Wie sagt man es auf Hebräisch?**
Übersetze die Wendungen ins Hebräische mit Nikkud. Achte: der Artikel ה- ist direkt geschrieben, fließt auf das Adjektiv; die Präfixe ב-, ל-, מ- werden in einem Wort geschrieben; *be- + ha- = ba-*, *le- + ha- = la-*, *mi- + ha- = me-ha-*.

<summary>Lösung</summary>

1. *סֵפֶר גָּדוֹל* — **sefer gadol** (beide ohne ha-)
2. *הַסֵּפֶר הַגָּדוֹל* — **ha-sefer ha-gadol** (Artikel auf beiden)
3. *יַלְדָּה קְטַנָּה* — **yalda ktana** (f. Sg. + f. Sg.)
4. *הַיַּלְדָּה הַקְּטַנָּה* — **ha-yalda ha-ktana**
5. *בַּיִת חָדָשׁ* — **bayit chadash** (m. Sg. + m. Sg.)
6. *הַבַּיִת הֶחָדָשׁ* — **ha-bayit he-chadash** (*he-* vor ח — typische Variante)
7. *יְלָדִים טוֹבִים* — **yeladim tovim** (m. Pl. + m. Pl.)
8. *הַיְלָדִים הַטּוֹבִים* — **ha-yeladim ha-tovim**
9. *נָשִׁים חֲכָמוֹת* — **nashim chakhamot** (f. Pl. + f. Pl.)
10. *הַנָּשִׁים הַחֲכָמוֹת* — **ha-nashim ha-chakhamot**
11. *מְכוֹנִית אֲדֻמָּה* — **mekhonit aduma** (f. Sg. + f. Sg.; *mekhonit* — f.)
12. *הַמְּכוֹנִית הָאֲדֻמָּה* — **ha-mekhonit ha-aduma**
13. *דֶּלֶת לְבָנָה* — **delet levana** (f. Sg. + f. Sg.; *delet* — f.)
14. *הַדֶּלֶת הַלְּבָנָה* — **ha-delet ha-levana**
15. *אִישׁ גָּבוֹהַּ* — **ish gavoha** (m. Sg. + m. Sg.)
16. *הָאִישׁ הַגָּבוֹהַּ* — **ha-ish ha-gavoha** (vor א wird der Artikel zu *ha-* mit Kamatz verlängert)
17. *הַבַּיִת גָּדוֹל* — **ha-bayit gadol** (Artikel nur auf Subst. → Satz „das Haus ist groß")
18. *הַמּוֹרָה טוֹבָה* — **ha-mora tova** („die Lehrerin ist gut")
19. *הַסֵּפֶר חָדָשׁ* — **ha-sefer chadash** („das Buch ist neu")
20. *הַיְלָדִים שְׂמֵחִים* — **ha-yeladim smechim** („die Kinder sind fröhlich")
21. *בְּבַיִת* — **be-vayit** (ב- + bayit, ohne Artikel)
22. *בַּבַּיִת* — **ba-bayit** (ב- + ha-bayit; Verschmelzung *be- + ha- = ba-*)
23. *לְמוֹרֶה* — **le-more**
24. *לַמּוֹרֶה* — **la-more** (Verschmelzung *le- + ha- = la-*)
25. *מֵעִיר* — **me-ir** (מ- vor Kehllaut ע = *me-*)
26. *מֵהָעִיר* — **me-ha-ir** (מ- verschmilzt NICHT mit dem Artikel; ה- bleibt; vor ע — *ha-* mit Kamatz)
27. *לְבֵית הַסֵּפֶר* — **le-vet ha-sefer** (wörtlich „zum Haus-des-Buchs"; „Schule" — *bet-sefer*)
28. *מֵהַיְלָדִים* — **me-ha-yeladim** (von den Kindern — bestimmt)
29. *יַלְדָּה קְטַנָּה בַּבַּיִת הַגָּדוֹל* — **Yalda ktana ba-bayit ha-gadol.** (Mädchen unbestimmt → ohne ha-; Haus bestimmt → *ba-* + ha-gadol)
30. *הָאֲנָשִׁים הָעֲיֵפִים הָאֵלֶּה מֵהָעִיר* — **Ha-anashim ha-ayefim ha-ele me-ha-ir.** (Artikel auf Subst. + zwei Adj.; *me-ha-* — aus der bestimmten Stadt)
