# Test zu Lektion 23: Futur in Nif'al/Hitpa'el. Imperativ in allen Binjanim. Verbot mit אל

**Wie sagt man es auf Hebräisch?**
Übersetze die Wendungen ins Hebräische. Nutze die Grammatik und den Wortschatz dieser Lektion. Drill Nif'al/Hitpa'el-Futur, klassischen und umgangssprachlichen Imperativ, negativen Imperativ mit אל. Ohne Vokalzeichen.

---

1. Ich werde ins Zimmer eintreten. (Nif'al, ani)
2. Er wird zu Hause bleiben. (Nif'al, hu)
3. Wir werden uns morgen treffen. (Nif'al, anachnu)
4. Sei vorsichtig! (Nif'al, umgangssprachlich, an einen Mann)
5. Sie werden sich schnell anziehen. (Hitpa'el, hem)
6. Ich rufe dich abends an. (Hitpa'el, ani)
7. Sie wird sich gewöhnen. (Hitpa'el, hi)
8. Geh duschen! (Hitpa'el, umgangssprachlich, an eine Frau)
9. Schreib! (klassischer Imperativ, an einen Mann)
10. Schreib! (klassischer Imperativ, an eine Frau)
11. Schreib mir einen Brief! (umgangssprachlich, an einen Mann)
12. Sprich Hebräisch! (klassisch, an einen Mann)
13. Sprich mit mir! (umgangssprachlich, an eine Frau)
14. Vollende die Aufgabe! (Hif'il, klassisch, an einen Mann)
15. Erkläre mir! (Hif'il, umgangssprachlich, an eine Frau)
16. Tritt ein! (Nif'al, klassisch, an einen Mann)
17. Zieh dich an! (Hitpa'el, klassisch, an einen Mann)
18. Ruf mich an! (Hitpa'el, umgangssprachlich, an einen Mann)
19. Schreib das nicht! (Verbot, an einen Mann)
20. Sprich nicht so! (Verbot, an eine Frau)
21. Geh nicht dorthin! (Verbot, an eine Frau)
22. Komm morgen nicht! (Verbot, an einen Mann)
23. Vergiss nicht! (Verbot, an einen Mann)
24. Tretet dort nicht ein! (Verbot, an Pl.)
25. Reg dich nicht auf! (Verbot, an eine Frau, Hitpa'el)
26. Rauch hier nicht! (Verbot, an einen Mann)
27. Setz dich, bitte. (umgangssprachlich, an einen Mann)
28. Geh geradeaus, dann nach rechts. (an eine Frau)
29. Warte hier und geh nicht weg. (an eine Frau)
30. Ruft mich morgen an, vergesst nicht! (an Pl.)
