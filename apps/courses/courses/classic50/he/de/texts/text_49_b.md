## Sefardisch (= modernes Israelisch) vs. aschkenasische Aussprache

| Hebräisch | Sefardisch (Standard) | Aschkenasisch | Deutsch |
|-------|------------------------|-------------|---------|
| שבת שלום | shabbat shalom | **shabbos sholom** | Friedlicher Schabbat |
| שבת קודש | shabbat kodesh | **shabbos kóydesh** | heiliger Schabbat |
| תורה | tora | **tóyrah** | Tora |
| שלום עליכם | shalom aleichem | **shólom aléykhem** | Friede mit euch |
| ישראל | yisra'el | **yísroel / yisrúel** | Israel |
| משפחה | mishpacha | **mishpókhe** | Familie |
| בית כנסת | beit knesset | **béyz knéses** | Synagoge |
| אבא ואמא | abba ve-imma | **tate un mame** (oft auf Jiddisch) / **ábba un ímma** | Papa und Mama |
| סבא וסבתא | saba ve-savta | **zéyde un bóbe** (Jiddisch) / **sábe un sávte** | Opa und Oma |
| שמע ישראל | shma yisra'el | **shma yísroel** | Höre, Israel |
| שנה טובה | shana tova | **shóne tóyve** | gutes Jahr |
| גמר חתימה טובה | gmar chatima tova | **gmar khasíme tóyve** | gute Endbesiegelung (zu Jom Kippur) |
| מזל טוב | mazal tov | **mázel tov** | Glückwunsch |
| ברוך השם | barukh hashem | **bórekh hashém / bórikh ashém** | gesegnet sei der Name (= G-tt sei Dank) |
| תודה לאל | toda la'el | **tóyde lóel** | Dank an G-tt |
| ספר תורה | sefer tora | **séyfer tóyrah** | Tora-Rolle |
| פסח | pesach | **péysakh** | Pessach |
| סוכות | sukkot | **súkkes** | Sukkot |
| חנוכה | chanuka | **khánuke / khánike** | Chanukka |
| ראש השנה | rosh hashana | **róysh hashóne** | Neujahr |
| יום כיפור | yom kippur | **yóm kípper** | Jom Kippur |
| ארץ ישראל | eretz yisra'el | **érets yísroel** | Land Israel |
| עם ישראל חי | am yisra'el chai | **am yísroel khái** | Das Volk Israel lebt |
| אדון עולם | adon olam | **ádoyn óylom** | Herr der Welt |
| אבינו שבשמיים | avinu shebashamayim | **óvinu shebashomáyim** | Unser Vater im Himmel |
| רחמנא ליצלן | rachmana litzlan | **rakhmóne litslán** | der Barmherzige bewahre |
| אם ירצה השם | im yirtse hashem | **im yírtse hashém** | wenn der Name will |
| בעזרת השם | be'ezrat hashem | **b'ezras hashém** | mit Hilfe des Namens |
| כל הכבוד | kol hakavod | **kol hakóvod** | alle Achtung / Bravo |
| לחיים | lechayim | **l'kháyim** | aufs Leben! (Toast) |

> **Was sich ändert:** Kamatz (a → o), Cholam (o → oy), ת ohne Dagesch (t → s), ה am Ende f. (-ah → -e), Betonung zieht zur vorletzten Silbe.
