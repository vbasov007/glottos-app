# Тест к уроку 3: Места, предметы, беглость чтения с никкудом

**Как сказать на иврите?**
Переведи фразы на иврит с никкудом. Используй лексику урока: места в городе, повседневные предметы, конструкции *yesh li / ein li*, предлоги *ba- / la- / ʿal*. Помни про род (ʿir — ж.р., dira — ж.р., chanut — ж.р., delet — ж.р., mita — ж.р.).
