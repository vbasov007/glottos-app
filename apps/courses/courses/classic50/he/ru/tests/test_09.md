# Тест к уроку 9: Артикль ha-, согласование прилагательного, префиксы-предлоги ב/ל/מ

**Как сказать на иврите?**
Переведи фразы на иврит с никкудом. Следи: артикль ה- слитный, протекает на прилагательное; префиксы ב-, ל-, מ- пишутся в одно слово; *be- + ha- = ba-*, *le- + ha- = la-*, *mi- + ha- = me-ha-*.

<summary>Ключи</summary>

1. *סֵפֶר גָּדוֹל* — **sefer gadol** (оба без ha-)
2. *הַסֵּפֶר הַגָּדוֹל* — **ha-sefer ha-gadol** (артикль на обоих)
3. *יַלְדָּה קְטַנָּה* — **yalda ktana** (ж.ед. + ж.ед.)
4. *הַיַּלְדָּה הַקְּטַנָּה* — **ha-yalda ha-ktana**
5. *בַּיִת חָדָשׁ* — **bayit chadash** (м.ед. + м.ед.)
6. *הַבַּיִת הֶחָדָשׁ* — **ha-bayit he-chadash** (*he-* перед ח — типичный вариант)
7. *יְלָדִים טוֹבִים* — **yeladim tovim** (м.мн. + м.мн.)
8. *הַיְלָדִים הַטּוֹבִים* — **ha-yeladim ha-tovim**
9. *נָשִׁים חֲכָמוֹת* — **nashim chakhamot** (ж.мн. + ж.мн.)
10. *הַנָּשִׁים הַחֲכָמוֹת* — **ha-nashim ha-chakhamot**
11. *מְכוֹנִית אֲדֻמָּה* — **mekhonit aduma** (ж.ед. + ж.ед.; *mekhonit* — ж.р.)
12. *הַמְּכוֹנִית הָאֲדֻמָּה* — **ha-mekhonit ha-aduma**
13. *דֶּלֶת לְבָנָה* — **delet levana** (ж.ед. + ж.ед.; *delet* — ж.р.)
14. *הַדֶּלֶת הַלְּבָנָה* — **ha-delet ha-levana**
15. *אִישׁ גָּבוֹהַּ* — **ish gavoha** (м.ед. + м.ед.)
16. *הָאִישׁ הַגָּבוֹהַּ* — **ha-ish ha-gavoha** (перед א артикль удлиняется в *ha-* с камацем)
17. *הַבַּיִת גָּדוֹל* — **ha-bayit gadol** (артикль только на сущ. → предложение «дом большой»)
18. *הַמּוֹרָה טוֹבָה* — **ha-mora tova** («учительница — хорошая»)
19. *הַסֵּפֶר חָדָשׁ* — **ha-sefer chadash** («книга — новая»)
20. *הַיְלָדִים שְׂמֵחִים* — **ha-yeladim smechim** («дети — весёлые»)
21. *בְּבַיִת* — **be-vayit** (ב- + bayit, без артикля)
22. *בַּבַּיִת* — **ba-bayit** (ב- + ha-bayit; слияние *be- + ha- = ba-*)
23. *לְמוֹרֶה* — **le-more**
24. *לַמּוֹרֶה* — **la-more** (слияние *le- + ha- = la-*)
25. *מֵעִיר* — **me-ir** (מ- перед гортанной ע = *me-*)
26. *מֵהָעִיר* — **me-ha-ir** (מ- НЕ сливается с артиклем; артикль ה- остаётся; перед ע — *ha-* с камацем)
27. *לְבֵית הַסֵּפֶר* — **le-vet ha-sefer** (буквально «к дому-книги»; «школа» — *bet-sefer*)
28. *מֵהַיְלָדִים* — **me-ha-yeladim** (от детей — определённых)
29. *יַלְדָּה קְטַנָּה בַּבַּיִת הַגָּדוֹל* — **Yalda ktana ba-bayit ha-gadol.** (девочка не определена → без ha-; дом определён → *ba-* + ha-gadol)
30. *הָאֲנָשִׁים הָעֲיֵפִים הָאֵלֶּה מֵהָעִיר* — **Ha-anashim ha-ayefim ha-ele me-ha-ir.** (артикль на сущ. + двух прил.; *me-ha-* — из определённого города)
