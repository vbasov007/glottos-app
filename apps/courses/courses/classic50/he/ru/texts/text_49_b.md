## Сефардское (= совр. израильское) vs. ашкеназское произношение

| Иврит | Сефардское (стандарт) | Ашкеназское | Русский |
|-------|------------------------|-------------|---------|
| שבת שלום | shabbat shalom | **shabbos sholom** | мирной субботы |
| שבת קודש | shabbat kodesh | **shabbos kóydesh** | святая суббота |
| תורה | tora | **tóyrah** | Тора |
| שלום עליכם | shalom aleichem | **shólom aléykhem** | мир вам |
| ישראל | yisra'el | **yísroel / yisrúel** | Израиль |
| משפחה | mishpacha | **mishpókhe** | семья |
| בית כנסת | beit knesset | **béyz knéses** | синагога |
| אבא ואמא | abba ve-imma | **tate un mame** (часто на идиш) / **ábba un ímma** | папа и мама |
| סבא וסבתא | saba ve-savta | **zéyde un bóbe** (идиш) / **sábe un sávte** | дедушка и бабушка |
| שמע ישראל | shma yisra'el | **shma yísroel** | слушай, Израиль |
| שנה טובה | shana tova | **shóne tóyve** | хорошего года |
| גמר חתימה טובה | gmar chatima tova | **gmar khasíme tóyve** | хорошей подписи (в Йом Кипур) |
| מזל טוב | mazal tov | **mázel tov** | поздравляю |
| ברוך השם | barukh hashem | **bórekh hashém / bórikh ashém** | благословен Имя (= слава Б-гу) |
| תודה לאל | toda la'el | **tóyde lóel** | благодарение Б-гу |
| ספר תורה | sefer tora | **séyfer tóyrah** | свиток Торы |
| פסח | pesach | **péysakh** | Песах |
| סוכות | sukkot | **súkkes** | Суккот |
| חנוכה | chanuka | **khánuke / khánike** | Ханука |
| ראש השנה | rosh hashana | **róysh hashóne** | Новый год |
| יום כיפור | yom kippur | **yóm kípper** | Йом Кипур |
| ארץ ישראל | eretz yisra'el | **érets yísroel** | Земля Израиля |
| עם ישראל חי | am yisra'el chai | **am yísroel khái** | народ Израиля жив |
| אדון עולם | adon olam | **ádoyn óylom** | Владыка мира |
| אבינו שבשמיים | avinu shebashamayim | **óvinu shebashomáyim** | Отец наш на небесах |
| רחמנא ליצלן | rachmana litzlan | **rakhmóne litslán** | Милосердный да избавит |
| אם ירצה השם | im yirtse hashem | **im yírtse hashém** | если пожелает Имя |
| בעזרת השם | be'ezrat hashem | **b'ezras hashém** | с помощью Имени |
| כל הכבוד | kol hakavod | **kol hakóvod** | всё уважение / молодец |
| לחיים | lechayim | **l'kháyim** | за жизнь! (тост) |

> **Что меняется:** камац (a → o), холам (o → oy), ת без дагеша (t → s), ה на конце ж.р. (-ah → -e), ударение тянет на предпоследний слог.
