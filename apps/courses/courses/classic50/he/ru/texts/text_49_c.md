**Тема: Сленг и разговорные клише, узнаваемые «на слух»**

> Тут не учебный, а живой иврит молодёжи, тель-авивских кафе, армии и интернета. Многие выражения — заимствования из арабского, английского, идиша. Без огласовок.

## Сленг — словарь поколений

| Иврит | Транслит / звучание | Русский / контекст |
|-------|---------------------|--------------------|
| סוף הדרך | sof haderech | высший класс (буквально «конец пути») |
| חבל על הזמן | chaval al hazman → chaval'azman / chazaman | офигенно (буквально «жалко времени») |
| גבר | gever | мужик, чувак (обращение) |
| לא קפצתי | lo kafatzti | мне не зашло (буквально «не прыгнул») |
| אין מצב | ein matzav → en matsav | без шансов / ни за что |
| אחותי | achoti | сестра моя (обращение к подруге) |
| יש מצב | yesh matsav | возможно / есть шанс |
| וואלה | walla | реально / правда (из арабского) |
| כאפרה | kapara | дорогая/ой (из мизрахи, любовное) |
| מותק | motek | сладкий/ая, дорогая |
| אל תפיל | al tapil (oti) | не подведи, не разочаруй |
| דאחקה | dakhka | прикол, шутка (арабское) |
| נשבע | nishba | клянусь |
| סבבה אגוזים | sababa egozim | супер-окей (буквально «класс орехи») |
| די נו | dai nu | хватит уже |
| נמאס | nim'as | надоело |
| תכלס | takhles | по сути, реально (из идиш) |
| בקטנה | b'ktana | без проблем (буквально «по-маленькому») |
| בגדול | b'gadol | в целом, в общем (буквально «по-большому») |
| אחלה | achla | классно (из арабского) |
| סטלה | stella | кайф, заторможенное состояние |
| מקשקש | m'kashkesh | болтаешь чушь |
| עזוב שטויות | azov shtuyot | брось ерунду |
| בלאגן | balagan | хаос, беспорядок |
| פצצה | ptzatza | бомба / огонь (о месте, человеке) |
| שיגעת אותי | shigata oti | ты меня свёл с ума |
| הלך עליי | halakh alay | мне капец (буквально «прошло на мне») |
| בפיגוע | b'pigua | в раздрае (буквально «в теракте» — сильный сленг) |
| מת מעייפות | met me'ayefut | умираю от усталости |
| שיכניס | shechnis | мощно (буквально «пусть введёт») |
| אין על זה | ein al ze | нет ничего лучше |
| ענק | anak | гигант / огромный (= круто) |
| נפלתי מהכיסא | nafalti mehakisse | упал со стула (от смеха) |
| מסטלל | mistalel | торможу, втыкаю (сленг от «стелла») |
| תרגיע | targia | расслабься, успокойся |
| לא שווה | lo shave | не стоит того |
| יאללה ביי | yalla bye | давай пока (арабское + английское) |
| אחי | achi | бро (универсальное обращение к мужчине) |

> **Поколенческое расслоение:**
> - **Старшее поколение** (60+): редко скажут *walla, kapara, achla, dakhka* — это «арабско-мизрахи» сленг.
> - **Средние** (30–50): свободно используют *sababa, chaval al hazman, sof haderech, takhles, balagan*.
> - **Молодёжь** (15–30): добавляет *b'pigua, mistalel, shechnis, ein al ze, ptzatza* — самый текучий слой, меняется каждые 3–5 лет.
> - **Религиозная молодёжь**: использует свой пласт сленга на смеси иврита и идиша — *takhles, b'ezras hashem, im yirtse hashem*.
