# Текст C к уроку 22: Смешанное повторение L1–L21 — в ктив мале
**Тема: Семья, город, учёба, числа, прилагательные — БЕЗ никкуда**

> Сводный обзор лексики из первых 21 уроков, переписанный **в ктив мале**. Цель — натренировать глаз быстро узнавать корень и силуэт знакомых слов без подпорок никкуда.

16. אנחנו שותים קפה ומדברים.
17. אנחנו מדברים על ספרים.
18. אני אוהב לקרוא ספרים.

28. החיים יפים.
29. תל אביב היא עיר יפה ליד הים.
30. שלום, תודה, ולהתראות מחר!
