# Text 10c for Lesson 10: Questions and answers — dialogue
**Theme: All seven question words in a live conversation + negation lo**

> Pointed text. Every sentence is a question or an answer. We pass through mi, ma, eifo, matai, lama, eikh, kama, plus the extended set me-eifo, le'an, eize. We connect things with ve- / aval / o / ki.

---

## Listening text

1. שָׁלוֹם! מָה שְׁלוֹמְךָ?
2. טוֹב, תּוֹדָה. וְאַתְּ?
3. גַּם כֵּן טוֹב.
4. אֵיךְ קוֹרְאִים לָךְ?
5. קוֹרְאִים לִי דָּנָה. וּלְךָ?
6. קוֹרְאִים לִי יוֹסִי.
7. מֵאֵיפֹה אַתָּה?
8. אֲנִי מִתֵּל אָבִיב. וְאַתְּ?
9. אֲנִי מִיְּרוּשָׁלַיִם.
10. מָה זֶה?
11. זֶה סֵפֶר חָדָשׁ.
12. מִי זֶה?
13. זֶה הֶחָבֵר שֶׁלִּי.
14. אֵיפֹה הַסֵּפֶר?
15. הַסֵּפֶר עַל הַשּׁוּלְחָן.
16. מָתַי הַשִּׁעוּר?
17. הַשִּׁעוּר מָחָר בַּבֹּקֶר.
18. לָמָּה אַתָּה לֹא בָּא?
19. כִּי אֵין לִי זְמַן הַיּוֹם.
20. כַּמָּה סְפָרִים יֵשׁ לְךָ?
21. יֵשׁ לִי חֲמִשָּׁה סְפָרִים.
22. לְאָן אַתְּ הוֹלֶכֶת?
23. אֲנִי הוֹלֶכֶת לַקָּפֶה.
24. אֵיזֶה קָפֶה אַתָּה רוֹצֶה?
25. אֲנִי רוֹצֶה קָפֶה שָׁחֹר, בְּבַקָּשָׁה.
26. תֵּה אוֹ קָפֶה?
27. קָפֶה, תּוֹדָה. אֲבָל בְּלִי סֻכָּר.
28. יֵשׁ לָךְ זְמַן מָחָר?
29. לֹא, מָחָר אֵין לִי זְמַן, כִּי יֵשׁ לִי עֲבוֹדָה.
30. אֵין בְּעָיָה. לְהִתְרָאוֹת!

## Word and phrase list

| Hebrew | Translit | English |
|--------|----------|---------|
| מִי | mi | who |
| מָה | ma | what |
| אֵיפֹה | eifo | where |
| מָתַי | matai | when |
| לָמָּה | lama | why |
| אֵיךְ | eikh | how |
| כַּמָּה | kama | how much / how many |
| מֵאֵיפֹה | me-eifo | from where |
| לְאָן | le'an | to where, whither |
| אֵיזֶה / אֵיזוֹ | eize / eizo | which (m./f.) |
| מָה שְׁלוֹמְךָ? | ma shlomkha? | how are you (to m.) |
| גַּם כֵּן | gam ken | likewise, also |
| קוֹרְאִים לִי… | kor'im li… | my name is… |
| מִתֵּל אָבִיב | mi-Tel Aviv | from Tel Aviv |
| מִירוּשָׁלַיִם | mi-Yerushalayim | from Jerusalem |
| חָדָשׁ | chadash | new |
| הֶחָבֵר שֶׁלִּי | he-chaver sheli | my friend |
| עַל | al | on |
| שׁוּלְחָן | shulchan | table |
| בַּבֹּקֶר | ba-boker | in the morning |
| בָּא / בָּאָה | ba / ba'a | I'm coming (m. / f.) |

| זְמַן | zman | time |
| חֲמִשָּׁה | chamisha | five (m., before a m. noun) |

| רוֹצֶה / רוֹצָה | rotse / rotsa | I want (m. / f.) |
| שָׁחֹר | shachor | black |
| בְּבַקָּשָׁה | bevakasha | please |
| תֵּה / קָפֶה | te / kafe | tea / coffee |
| אוֹ | o | or |
| אֲבָל | aval | but |
| בְּלִי | bli | without |
| סֻכָּר | sukar | sugar |
| עֲבוֹדָה | avoda | work |
| אֵין בְּעָיָה | ein be'aya | no problem |
| לְהִתְרָאוֹת | lehitra'ot | goodbye |

> **Notice:** in 19 and 29 — the pair **lama? — ki…** ("why? — because…"). In 27 — **bli sukar** ("without sugar"): the preposition *bli* — "without", the counterpart of *im* ("with"). In 24 — *eize* agrees with *kafe* (m.); for an f. noun it would be *eizo mora* ("which (female) teacher").
