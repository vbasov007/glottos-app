# Text A for Lesson 22: Greetings and Introductions — Now Without Nikkud
**Theme: Familiar greeting and introduction lexicon — in ktiv male**

> This is the same material as in L1, but **WITHOUT vowels** — in ktiv male. Read each sentence aloud, catch the root and the silhouette. If you stumble — mentally "add" the nikkud back, as in L1.

## Word and phrase list (in ktiv male)

| Hebrew (ktiv male) | Translit | English |
|--------------------|----------|---------|
| שלום | shalom | hello / hi / peace |
| בוקר טוב | boker tov | good morning |
| ערב טוב | erev tov | good evening |
| לילה טוב | layla tov | good night |
| תודה | toda | thank you |
| תודה רבה | toda raba | thank you very much |
| בבקשה | bevakasha | please / you're welcome |
| סליחה | slicha | excuse me |
| כן | ken | yes |
| לא | lo | no |
| מה שלומך? | ma shlomkha? (m.) / ma shlomekh? (f.) | how are you? |
| טוב | tov | good / well |
| ומה שלומך? | u-ma shlomkha? | and how are you? |
| גם כן | gam ken | also, too |
| איך קוראים לך? | eikh kor'im lekha? (m.) / lakh? (f.) | what's your name? |
| קוראים לי… | kor'im li… | my name is… |
| נעים מאוד | na'im me'od | very nice (to meet you) |
| אני | ani | I |
| את / אתה | at (f.) / ata (m.) | you |
| הוא / היא | hu / hi | he / she |
| תלמיד / תלמידה | talmid / talmida | student (m./f.) |
| מורה | more (m.) / mora (f.) | teacher (m./f.) |
| חבר / חברה | chaver / chavera | friend (m./f.) |
| להתראות | lehitra'ot | goodbye |
| יום טוב | yom tov | good day, have a good day |

> **Notes on ktiv male:**
> - שלום: ו in the middle → vowel "o" (shal**o**m).
> - תודה: ו → "o" (t**o**da).
> - בוקר: ו → "o" (b**o**ker). Compare with pointed בֹּקֶר from L1.
> - לילה: י → "i/y" (la-yla).
> - דוד: ו → "a," but this is a **proper name** (David), written without a vav-hint. The same spelling = "uncle" (dod) — context distinguishes.
