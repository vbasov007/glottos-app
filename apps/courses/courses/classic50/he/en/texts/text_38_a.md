# Text 38a for Lesson 38: Word order and topicalization — one meaning, different "outfits"
**Theme: SVO by default vs. fronting topic/focus to the beginning. et is preserved when an object is fronted; resumptive pronoun for topicalization.**

> Text without vowel points. Thirty sentences: first neutral SVO, then various versions with a topic or focus fronted. Listen to **what is highlighted** in each order.

---

## Listening text

1. דנה קוראת ספר.
2. הילדים אכלו עוגה.
3. המורה הסביר את החומר.
4. אני אוהב קפה.
5. יוסי קנה מכונית חדשה אתמול.
6. את הספר הזה אני אוהב.
7. את הספר הזה — קראתי אותו כבר.
8. את יוסי דנה פגשה בקפה.
9. את דנה — ראיתי אותה אתמול ברחוב.
10. את המתנה הזאת קיבלתי מאמא.
11. אתמול קניתי את הספר.
12. אתמול דנה קראה את הספר הזה.
13. בבוקר אני שותה קפה, בערב אני שותה תה.
14. בקיץ אנחנו נוסעים לים.
15. אצלי בבית תמיד יש אוכל.
16. על השולחן הזה ההורים שלי אכלו שלושים שנה.
17. לדנה נתתי את המתנה.
18. ליוסי אמרתי את האמת.
19. למורה הסברנו את הבעיה.
20. גלידה — הילדים אוהבים.
21. ספרים — אני קונה רק בחנות הזאת.
22. אנגלית אני מדבר, צרפתית אני לא מבין.
23. את החדר הזה אני אוהב יותר מכל החדרים.
24. דנה — אמא שלה מורה.
25. הילד הזה — אבא שלו רופא.
26. הספר הזה — הכריכה שלו קרועה.
27. השכן שלנו — את האישה שלו לא ראיתי שנים.
28. את התשובה הנכונה רק דנה ידעה.
29. את כל הסיפור הזה שמעתי אתמול ברדיו.
30. את עצמי אני מכיר טוב.

## Word and phrase list

| Hebrew | Translit | English |
|--------|----------|---------|
| דנה קוראת ספר | Dana koret sefer | Dana reads a book (neutral SVO) |
| את הספר הזה אני אוהב | et ha-sefer ha-ze ani ohev | this book I love (object topicalization; et preserved) |
| קראתי אותו כבר | karati oto kvar | I already read it |
| פגשה | pagsha | (she) met |
| ברחוב | ba-rechov | on the street |
| מתנה | matana | gift |
| קיבלתי | kibalti | I received |
| בבוקר / בערב | ba-boker / ba-erev | in the morning / in the evening |
| בקיץ | ba-kayitz | in summer |
| נוסעים לים | nos'im la-yam | we go to the sea |
| אצלי בבית | etsli ba-bayit | at my home |
| על השולחן הזה | al ha-shulchan ha-ze | at this table |
| הורים | horim | parents |
| נתתי | natati | I gave |
| הסברנו | hisbarnu | we explained |
| גלידה | glida | ice cream |
| ספרים | sfarim | books |
| חנות | chanut | shop |
| צרפתית | tsarfatit | French |
| הכריכה | ha-krikha | the cover |
| קרוע / קרועה | karu'a / kru'a | torn (m./f.) |
| השכן | ha-shakhen | the neighbor |
| לא ראיתי שנים | lo ra'iti shanim | I haven't seen for years |

| מכיר | makir | knows, am acquainted with |

> **Notice №6, 7, 8, 9, 10:** **et** is preserved when a definite object is fronted. This is not "double et" — it's just the same et that has moved with the object.

> **Notice №7, 9:** the resumptive pronoun (**oto, ota**) is the norm of conversational Hebrew in topicalization.

> **Notice №20, 21, 29:** "glida", "sfarim", "et kol ha-sipur" — fronted objects; in №20-21 the object is indefinite (no article) → **no et**.

> **Notice №24, 25, 26, 27:** "hanging topic" — a fronted noun that is not syntactically linked to what follows, only by a thread of meaning ("Dana — her mother…"). This is a very frequent conversational structure.
