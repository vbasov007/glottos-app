## Slang — a generational glossary

| Hebrew | Translit / sound | English / context |
|-------|---------------------|--------------------|
| סוף הדרך | sof haderech | top tier (literally "end of the road") |
| חבל על הזמן | chaval al hazman → chaval'azman / chazaman | awesome (literally "waste of time") |
| גבר | gever | man, dude (address) |
| לא קפצתי | lo kafatzti | wasn't impressed (literally "didn't jump") |
| אין מצב | ein matzav → en matsav | no way / out of the question |
| אחותי | achoti | sis (address to a female friend) |
| יש מצב | yesh matsav | maybe, there's a chance |
| וואלה | walla | really, true (from Arabic) |
| כאפרה | kapara | dear (from Mizrahi, affectionate) |
| מותק | motek | sweetie, dear |
| אל תפיל | al tapil (oti) | don't let me down, don't disappoint |
| דאחקה | dakhka | joke, prank (Arabic) |
| נשבע | nishba | I swear |
| סבבה אגוזים | sababa egozim | super okay (literally "cool nuts") |
| די נו | dai nu | enough already |
| נמאס | nim'as | fed up |
| תכלס | takhles | bottom line, really (from Yiddish) |
| בקטנה | b'ktana | no problem (literally "in the small way") |
| בגדול | b'gadol | by and large (literally "in the big way") |
| אחלה | achla | great (from Arabic) |
| סטלה | stella | high, dazed state |
| מקשקש | m'kashkesh | talking nonsense |
| עזוב שטויות | azov shtuyot | drop the nonsense |
| בלאגן | balagan | chaos, mess |
| פצצה | ptzatza | bomb / fire (of a place, person) |
| שיגעת אותי | shigata oti | you drove me crazy |
| הלך עליי | halakh alay | I'm done for (literally "it went on me") |
| בפיגוע | b'pigua | a wreck (literally "in a terror attack" — heavy slang) |
| מת מעייפות | met me'ayefut | dying of tiredness |
| שיכניס | shechnis | hard-hitting (literally "let it come in") |
| אין על זה | ein al ze | nothing tops it |
| ענק | anak | giant / huge (= great) |
| נפלתי מהכיסא | nafalti mehakisse | fell out of my chair (laughing) |
| מסטלל | mistalel | zoning out, spaced (slang from "stella") |
| תרגיע | targia | relax, calm down |
| לא שווה | lo shave | not worth it |
| יאללה ביי | yalla bye | bye then (Arabic + English) |
| אחי | achi | bro (universal address to a man) |

> **Generational stratification:**
> - **Older generation** (60+): rarely say *walla, kapara, achla, dakhka* — that's "Arabic-Mizrahi" slang.
> - **Middle** (30–50): freely use *sababa, chaval al hazman, sof haderech, takhles, balagan*.
> - **Youth** (15–30): add *b'pigua, mistalel, shechnis, ein al ze, ptzatza* — the most fluid layer, changes every 3–5 years.
> - **Religious youth**: uses its own slang layer mixing Hebrew and Yiddish — *takhles, b'ezras hashem, im yirtse hashem*.
