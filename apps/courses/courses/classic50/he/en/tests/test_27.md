# Test for Lesson 27: Mishkalim — noun patterns

**How do you say it in Hebrew?**
Translate the words and phrases into Hebrew. Use the grammar and vocabulary from this lesson. Hebrew without vowel points. Rely on the mishkalim: miCCaC / miCCaCa (place), maCCeC (instrument/agent), CaCCan and CaCaC (profession), CCiCa and CaCeCet / CiCCa (process and abstraction).

---

1. Office (place of service; root ש-ר-ד).
2. Kitchen (place where one cooks; root ט-ב-ח).
3. Hairdresser's (place of haircutting; root ס-פ-ר).
4. Laundry (place of washing; root כ-ב-ס).
5. Clinic (place of healing; root ר-פ-א).
6. Restaurant (place where one eats; root ס-ע-ד).
7. College (place of study; root ל-מ-ד).
8. Temple (sanctuary; root ק-ד-ש).
9. Key (that with which one opens; root פ-ת-ח).
10. Airplane (that with which one flies; root ט-ו-ס).
11. Fork (root ז-ל-ג).
12. Refrigerator (that which cools; root ק-ר-ר).
13. (Photo)camera (that with which one shoots; root צ-ל-מ).
14. Leader (the one who leads; root נ-ה-ג).
15. Journalist (agent CaCaC; root כ-ת-ב).
16. Photographer (profession CaCCan; root צ-ל-מ).
17. Dancer (profession CaCCan; root ר-ק-ד).
18. Actor / player (profession CaCCan; root ש-ח-ק).
19. Liar (property CaCCan; root ש-ק-ר).
20. Lazy person (property CaCCan; root ע-צ-ל).
21. Contractor (profession CaCCan; root כ-ב-ל).
22. Writing / the process of writing (CCiCa; root כ-ת-ב).
23. Reading (CCiCa; root ק-ר-א).
24. Guarding (CCiCa; root ש-מ-ר).
25. Walking / going on foot (CCiCa; root ה-ל-כ).
26. Joy (abstraction CiCCa; root ש-מ-ח).
27. Love (abstraction CaCaCa; root א-ה-ב).
28. Hatred (abstraction CCa'a; root ש-נ-א).
29. Flu (CaCeCet; root ש-פ-ע).
30. The journalist is writing a letter in the office. (use katav, mikhtav, misrad)
