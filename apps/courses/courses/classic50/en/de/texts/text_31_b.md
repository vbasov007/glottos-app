# Text zur Lektion 31: At work
**Thema: Reale Arbeitsbedingungen — nullter und erster Konditional**

---

## Hörtext zum Wiederholen

1. If you finish the report by Friday, you'll get a bonus.
2. I always answer emails if they come from clients.
3. If the meeting starts late, we'll have less time for questions.
4. Please call me as soon as the package arrives.
5. If you press Control plus S, the file saves automatically.
6. Unless we hire more people, we won't meet the deadline.
7. If the boss is in a good mood, he'll agree to your idea.
8. The system shuts down if you don't use it for an hour.
9. If you have any questions, just send me an email.
10. I'll send you the details when I get them.
11. If the printer doesn't work, restart it.
12. We'll launch the product if the test results are positive.
13. If clients call, tell them I'll be back at three.
14. The office closes early if there's a holiday.
15. If you're late again, the manager will speak to you.
16. I'll join the call as soon as I finish this task.
17. If we work together, we'll finish in time.
18. Unless you check the numbers, mistakes happen.
19. If the server goes down, we lose money.
20. I'll write a report after the meeting ends.
21. If she accepts the offer, we'll celebrate.
22. The contract is valid provided both sides sign.
23. If you need help, ask any colleague.
24. We'll review the budget if costs go up.
25. If the project fails, we'll learn from it.
26. Things break if you don't take care of them.
27. If everything is ready, we can start.
28. I'll let you know as soon as we decide.
29. If you take this job, you'll travel a lot.
30. The team works better if the goals are clear.
