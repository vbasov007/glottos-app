# Lekcja 46: Słowotwórstwo — jak z jednego rdzenia wyrasta dwadzieścia słów

**Słownictwo: produktywne afiksy, rodziny słów, leksyka abstrakcyjna i formalna (~42)**

---

## Jak pracować z tą lekcją

1. **Przeczytaj** reguły afiksów — to nie wkuwanie, to formuły (10 minut).
2. **Przepuść znane rdzenie** przez wszystkie sufiksy — w godzinę podwoisz słownictwo.
3. **Zgaduj** znaczenia nieznanych słów w tekstach, rozkładając na prefiks + rdzeń + sufiks.
4. **Przećwicz** konwersję: to samo słowo jako rzeczownik i jako czasownik — typowe dla angielskiego.

> Uczyć się pojedynczych słów = arytmetyka. Uczyć się afiksów = algebra.
> Jeden sufiks otwiera setkę słów. Jeden prefiks — jeszcze tyle samo.

---

## Część 1: Po co Ci słowotwórstwo na C1

W tym momencie masz już duże słownictwo — może 3000–5000 słów. Żeby przejść z B2 do C1, musisz umieć **samodzielnie poszerzać** ten zasób. I jest dobra wiadomość: angielski to **najbardziej słowotwórczo produktywny** język w tej serii. Jeden czasownik *decide* daje Ci:

- <say>decide</say> — czasownik: *decydować, podejmować decyzję*
- <say>decision</say> — rzeczownik: *decyzja*
- <say>decisive</say> — przymiotnik: *decydujący, zdecydowany*
- <say>decisively</say> — przysłówek: *zdecydowanie*
- <say>indecisive</say> — przymiotnik przeczący: *niezdecydowany*
- <say>indecision</say> — rzeczownik przeczący: *niezdecydowanie*

**Sześć słów z jednego rdzenia.** Jeśli widzisz wzorzec, nie uczysz się sześciu słów — uczysz się *jednej formuły* i stosujesz ją do setek innych rdzeni.

> **Główna strategia C1:** widząc nieznane słowo, rozłóż je na **prefiks + rdzeń + sufiks**. W 80% przypadków znaczenie da się zgadnąć. *Unmistakable* = *un-* (nie) + *mistake* (błąd) + *-able* (możliwość) → „taki, że nie sposób się pomylić".

## Część 2: Główne sufiksy rzeczowników

Sufiks to to, co zamienia jedną część mowy w inną. W angielskim jest kilkanaście produktywnych sufiksów rzeczowników; weźmiemy osiem najważniejszych.

| Sufiks | Z czego | Znaczenie | Przykład |
|--------|---------|-----------|----------|
| **-tion / -sion** | czasownik | czynność lub rezultat | <say>create → creation</say>, <say>decide → decision</say> |
| **-ment** | czasownik | czynność, rezultat, stan | <say>develop → development</say>, <say>achieve → achievement</say> |
| **-er / -or** | czasownik | wykonawca (kto robi) | <say>teach → teacher</say>, <say>act → actor</say> |
| **-ee** | czasownik | adresat (z kim się robi) | <say>employ → employee</say>, <say>interview → interviewee</say> |
| **-ness** | przymiotnik | abstrakcyjna cecha | <say>happy → happiness</say>, <say>kind → kindness</say> |
| **-ity / -ty** | przymiotnik | cecha (rdzenie romańskie) | <say>able → ability</say>, <say>real → reality</say> |
| **-ism** | rzeczownik / przym. | doktryna, system poglądów | <say>real → realism</say>, <say>capital → capitalism</say> |
| **-ship** | rzeczownik | stan, relacje, status | <say>friend → friendship</say>, <say>leader → leadership</say> |

> **Hak wyboru:** przymiotniki z rdzeni germańskich (krótkie, codzienne: *happy, kind, dark, sad*) prawie zawsze dają **-ness**. Przymiotniki z rdzeni romańskich (długie, łacińskie: *able, real, similar, possible*) — **-ity**. Czasowniki na *-ate, -ize, -ify* prawie zawsze dają **-tion** (*create → creation, organize → organization, simplify → simplification*).

> **Pułapka -ee:** sufiks **-ee** oznacza tego, **na kim** wykonuje się czynność, w przeciwieństwie do *-er* (tego, który ją wykonuje). *Employer* — pracodawca; *employee* — pracownik. *Interviewer* — prowadzący rozmowę; *interviewee* — ten, z kim się rozmawia.

## Część 3: Główne sufiksy przymiotników

| Sufiks | Z czego | Znaczenie | Przykład |
|--------|---------|-----------|----------|
| **-able / -ible** | czasownik | „może być zrobione" | <say>read → readable</say>, <say>access → accessible</say> |
| **-ful** | rzeczownik | „pełen czegoś" | <say>care → careful</say>, <say>hope → hopeful</say> |
| **-less** | rzeczownik | „bez czegoś" | <say>care → careless</say>, <say>hope → hopeless</say> |
| **-ous** | rzeczownik | „posiadający" | <say>danger → dangerous</say>, <say>fame → famous</say> |
| **-y** | rzeczownik | „charakteryzowany przez" | <say>sun → sunny</say>, <say>noise → noisy</say> |
| **-al** | rzeczownik | „odnoszący się do" | <say>nature → natural</say>, <say>culture → cultural</say> |
| **-ive** | czasownik / rzecz. | „skłonny do" | <say>act → active</say>, <say>create → creative</say> |
| **-ic** | rzeczownik | „odnoszący się do" (rdzenie greckie) | <say>history → historic</say>, <say>economy → economic</say> |

> **Złota para:** **-ful** vs. **-less** — to idealne antonimy. *Useful* / *useless*, *hopeful* / *hopeless*, *helpful* / *helpless*, *careful* / *careless*. Dają Ci podwójne słownictwo.

> **Pułapka -able vs -ible:** reguła prosta w 90% przypadków — **-able** dokleja się do rdzeni angielskich (*readable, drinkable, comfortable*), **-ible** — do rdzeni łacińskich, częściej do tych, które mają już formę na *-ion* (*visible → vision, possible, edible*). Uczyć się lepiej z konkretnym słowem.

## Część 4: Prefiksy — główna broń C1

Prefiks w angielskim prawie zawsze **zmienia znaczenie, nie część mowy**. Czasownik pozostaje czasownikiem, przymiotnik przymiotnikiem. Ale sens się przekręca lub przesuwa.

### Prefiksy przeczące

To najbardziej produktywna grupa. Znać je = podwoić zasób przymiotników.

| Prefiks | Kiedy | Przykład |
|---------|-------|----------|
| **un-** | uniwersalny, rdzenie germańskie | <say>happy → unhappy</say>, <say>do → undo</say>, <say>known → unknown</say> |
| **in-** | rdzenie romańskie | <say>active → inactive</say>, <say>visible → invisible</say> |
| **im-** | przed **m, p** | <say>possible → impossible</say>, <say>mature → immature</say> |
| **il-** | przed **l** | <say>legal → illegal</say>, <say>logical → illogical</say> |
| **ir-** | przed **r** | <say>regular → irregular</say>, <say>responsible → irresponsible</say> |
| **dis-** | „przeciwieństwo" lub „rozdzielenie" | <say>agree → disagree</say>, <say>appear → disappear</say> |
| **non-** | neutralne „nie" (bez oceny) | <say>smoker → non-smoker</say>, <say>fiction → non-fiction</say> |
| **mis-** | „błędnie, nieprawidłowo" | <say>understand → misunderstand</say>, <say>spell → misspell</say> |

> **Reguła asymilacji:** *in-, im-, il-, ir-* — to **ten sam prefiks**, który zmienia kształt przed różnymi spółgłoskami. **im-** przed wargowymi (m, p, b), **il-** przed l, **ir-** przed r, **in-** w pozostałych przypadkach. Nie wkuwaj tabeli — zapamiętaj zasadę: prefiks „dopasowuje się do dźwięku", podobnie jak polskie „bez- / bes-".

> **Subtelna różnica: un- vs in- vs non-.** *Unhappy* — nieszczęśliwy (emocja, ocena). *Inactive* — bezczynny (neutralna charakterystyka). *Non-smoker* — niepalący (po prostu fakt, bez oceny). Na C1 ważne jest wyczucie różnicy: *unprofessional* = krytyka; *non-professional* = po prostu „nie zawodowiec".

> **Dis- vs un-:** w czasownikach te dwa często brzmią podobnie, ale *un-* zazwyczaj „cofnąć czynność" (*undo* — cofnąć zrobione, *unpack* — rozpakować), a *dis-* — „czynność przeciwna" (*disagree* — nie zgadzać się, *dislike* — nie lubić).

### Inne ważne prefiksy

| Prefiks | Znaczenie | Przykład |
|---------|-----------|----------|
| **re-** | znów, ponownie | <say>write → rewrite</say>, <say>build → rebuild</say> |
| **pre-** | przed, z góry | <say>view → preview</say>, <say>pay → prepay</say> |
| **post-** | po | <say>war → post-war</say>, <say>graduate → postgraduate</say> |
| **over-** | nad, nadmiernie | <say>work → overwork</say>, <say>sleep → oversleep</say> |
| **under-** | niedostatecznie, pod | <say>cook → undercook</say>, <say>estimate → underestimate</say> |
| **sub-** | pod, poniżej | <say>marine → submarine</say>, <say>conscious → subconscious</say> |
| **super-** | nad, ponad | <say>man → superman</say>, <say>natural → supernatural</say> |
| **co-** | wspólnie | <say>worker → co-worker</say>, <say>operate → cooperate</say> |
| **ex-** | były, z | <say>husband → ex-husband</say>, <say>president → ex-president</say> |
| **anti-** | przeciw | <say>virus → antivirus</say>, <say>social → antisocial</say> |

> **Hak C1:** widząc długie nieznane słowo, szukaj **znanego rdzenia w środku**. *Underestimate* = *under-* (niedo-) + *estimate* (oszacować). *Misinterpretation* = *mis-* (błędnie) + *interpret* (interpretować) + *-ation* (czynność) → „błędna interpretacja". Słowo z sześciu liter zamienia się w słowo z trzech.

## Część 5: Konwersja — jedno słowo, dwie części mowy

To **specyfika angielskiego**, którą polskojęzyczni najczęściej niedoceniają. W angielskim to samo słowo często działa i jako rzeczownik, i jako czasownik — bez żadnej zmiany formy.

| Słowo | Jako rzeczownik | Jako czasownik |
|-------|-----------------|----------------|
| <say>water</say> | woda | podlewać |
| <say>email</say> | wiadomość | napisać maila |
| <say>google</say> | (marka) | szukać w Google |
| <say>book</say> | książka | rezerwować |
| <say>plan</say> | plan | planować |
| <say>hand</say> | ręka | podawać |
| <say>face</say> | twarz | stawiać czoła, stykać się |
| <say>head</say> | głowa | przewodzić |
| <say>name</say> | imię | nazywać |
| <say>park</say> | park | parkować |

> **Kontrast z polskim:** w polskim potrzebne są różne słowa lub wyraźny sufiks (*ręka → ręczny, ręczyć*). W angielskim często wystarcza kontekst: *I will water the plants* — podlewać; *I drink water* — woda. Żadne -ny, -ować, -izować nie jest potrzebne.

> **Obserwacja C1:** we współczesnym angielskim, zwłaszcza w biznesie i technologii, **dowolny rzeczownik** może stać się czasownikiem. *Friend* → *to friend someone* (na Facebooku). *Inbox* → *to inbox someone*. *Adult* → *to adult* (żyć po dorosłemu). To naturalny proces językowy, nie błąd.

### Przesunięcie akcentu przy konwersji

Często u dwusylabowych słów zmienia się akcent między rolami. Rzeczownik — akcent na pierwszą sylabę, czasownik — na drugą:

| Słowo | Rzeczownik | Czasownik |
|-------|-----------|-----------|
| <say>record</say> | **RE**cord (zapis) | re**CORD** (nagrywać) |
| <say>present</say> | **PRE**sent (prezent) | pre**SENT** (przedstawiać) |
| <say>object</say> | **OB**ject (obiekt) | ob**JECT** (sprzeciwiać się) |
| <say>increase</say> | **IN**crease (wzrost) | in**CREASE** (wzrastać) |
| <say>contract</say> | **CON**tract (kontrakt) | con**TRACT** (kurczyć się) |
| <say>conduct</say> | **CON**duct (zachowanie) | con**DUCT** (prowadzić) |

> To reguła żywa, ale nie absolutna — są wyjątki. Jednak wzorzec jest tak częsty, że warto go rozpoznawać.

## Część 6: Rodziny słów — ucz się grupami, nie pojedynczo

Weź dowolny ważny rdzeń i **zbuduj wokół niego rodzinę**. To Twoje główne zadanie na C1.

### Przykład: rodzina *create*

| Część mowy | Słowo | Tłumaczenie |
|------------|-------|-------------|
| czasownik | <say>create</say> | tworzyć |
| rzeczownik (czynność) | <say>creation</say> | tworzenie |
| rzeczownik (wykonawca) | <say>creator</say> | twórca |
| przymiotnik | <say>creative</say> | twórczy |
| przysłówek | <say>creatively</say> | twórczo |
| abstrakcyjny rzecz. | <say>creativity</say> | kreatywność |
| przymiotnik | <say>creational</say> | związany z tworzeniem |
| przeczenie | <say>uncreative</say> | nietwórczy |
| przeczenie | <say>recreation</say> | (1) odtworzenie; (2) rekreacja |

Z jednego rdzenia — dziewięć słów. Z czego połowa to wysokoczęstotliwa leksyka C1.

### Przykład: rodzina *act*

| Część mowy | Słowo | Tłumaczenie |
|------------|-------|-------------|
| czasownik | <say>act</say> | działać |
| rzeczownik (czynność) | <say>action</say> | działanie |
| rzeczownik (wykonawca) | <say>actor / actress</say> | aktor / aktorka |
| przymiotnik | <say>active</say> | aktywny |
| przysłówek | <say>actively</say> | aktywnie |
| abstrakcyjny rzecz. | <say>activity</say> | aktywność, działalność |
| czasownik | <say>activate</say> | aktywować |
| rzeczownik | <say>activation</say> | aktywacja |
| przymiotnik | <say>inactive</say> | nieaktywny |
| abstrakcyjny rzecz. | <say>inactivity</say> | bezczynność |
| przymiotnik | <say>interactive</say> | interaktywny |
| czasownik | <say>react</say> | reagować |
| rzeczownik | <say>reaction</say> | reakcja |

Trzynaście słów. Rdzeń *act* — trzy litery.

## Część 8: Gama językowa

### Gama 1 — Rozpędzanie jednej rodziny (create)

Przeczytaj na głos, świadom(a) tego, jak zmienia się część mowy:

```
She wants to create something new.
Her creation was praised by everyone.
The creator stood quietly in the corner.
He is a very creative person.
She solved the problem creatively.
The creativity of the team is remarkable.
This approach is uncreative and outdated.
We need recreation after a long week.
```

### Gama 2 — Prefiks un- przez wszystkie osoby

```
I am unable to help you right now.
You are unable to come tomorrow.
He is unable to find the file.
She is unable to read this letter.
We are unable to predict the result.
They are unable to agree on a date.
```

### Gama 3 — Konwersja noun ↔ verb

Przepuść pary — jedno słowo w dwóch rolach:

```
I water the plants every morning.    I drink a glass of water.
I email her every week.               I got a long email yesterday.
I book a table for two.               I read a good book last night.
I plan my day in the morning.         The plan worked perfectly.
I park my car here every day.         The park is full of children.
```

## Część 9: Matryca językowa

### Matryca 1 — Od czasownika do rzeczownika

```

What did you arrange? — The arrangement is final.

- przysłówek: *successfully* (pomyślnie)
- przeczenie: *unsuccessful* (nieudany)

Zauważ: *succeed → success* — przykład tego, że nie każdy rzeczownik od czasownika dostaje sufiks. Czasem rdzeń się zmienia. Sześć słów wychodzi z jednego: *succeed, success, successful, successfully, unsuccessful, unsuccessfully*.

## Ściąga z lekcji

```
SŁOWOTWÓRSTWO = trzy narzędzia: PREFIKS + RDZEŃ + SUFIKS

GŁÓWNE SUFIKSY RZECZOWNIKÓW:
  czasownik → -tion / -sion   create → creation, decide → decision
  czasownik → -ment           develop → development
  czasownik → -er / -or       teach → teacher
  czasownik → -ee             employ → employee  (ten, na kim czynność)
  przym. → -ness              happy → happiness   (germańskie)
  przym. → -ity               able → ability      (romańskie)
  rzecz. → -ship              friend → friendship
  rzecz. → -ism               real → realism

SUFIKSY PRZYMIOTNIKÓW:
  czasownik → -able / -ible   read → readable
  rzecz. → -ful / -less       care → careful / careless
  rzecz. → -ous               danger → dangerous
  rzecz. → -al                culture → cultural
  czasownik → -ive            act → active

PREFIKSY PRZECZĄCE (wg rdzenia/dźwięku):
  un-    germańskie:        unhappy, unknown
  in-    romańskie:         inactive, invisible
  im-    przed m,p,b:       impossible, immature
  il-    przed l:           illegal, illogical
  ir-    przed r:           irregular, irresponsible
  dis-   przeciwieństwo:    disagree, disappear
  non-   neutralne:         non-smoker, non-fiction
  mis-   błędnie:           misunderstand, misspell

INNE PREFIKSY:
  re-   znów             rewrite, rebuild
  pre-  z góry           preview, prepay
  post- po               post-war
  over- nadmiernie       overwork, oversleep
  under niedostatecznie  underestimate
  co-   wspólnie         co-worker, cooperate
  ex-   były             ex-husband

KONWERSJA (jedno słowo, dwie części mowy):
  water (woda / podlewać)     email (wiadomość / pisać)
  book (książka / rezerwować) plan (plan / planować)
  hand (ręka / podawać)       face (twarz / spotkać)

  Przesunięcie akcentu: RE-cord (rzecz.) / re-CORD (czas.)
                        PRE-sent (rzecz.) / pre-SENT (czas.)

STRATEGIA C1:
  Widzisz długie nieznane słowo → rozłóż na prefiks + rdzeń + sufiks
  W 80% przypadków znaczenie da się zgadnąć bez słownika.
  Ucz się słów RODZINAMI, nie pojedynczo.
