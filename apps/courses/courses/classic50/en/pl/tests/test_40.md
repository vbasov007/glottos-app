# Test bloku 4 (Lekcje 31–40): Count — Warunkowe i zdania złożone B2

**Jak to powiedzieć po angielsku?**
Przetłumacz frazy na angielski. To **kumulatywny test Bloku 4**: warunki realne (0/1), warunki nierealne (2/3), I wish / if only, strona bierna we wszystkich czasach, mowa zależna z przesunięciem czasów, must be / can't be / might have done, gerund vs to + infinitive, although / despite / so that, przedimki czwarte podejście (in bed, at home, by car), architektura długiego zdania złożonego.
