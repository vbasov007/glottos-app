# Tekst do lekcji 32: If I were you
**Temat: Nierealne sytuacje w teraźniejszości — drugi warunkowy**

---

## Tekst do słuchania i powtarzania

1. If I had more time, I would travel more.
2. If I were rich, I would buy a small house by the sea.
3. If I knew his number, I would call him right now.
4. If she lived closer, we would meet every weekend.
5. If I were you, I would talk to her honestly.
6. What would you do if you won the lottery?
7. If I had a car, I would drive everywhere.
8. If we lived in a warmer country, life would be easier.
9. If he spoke English, he would find a better job.
10. If I were the boss, I would change a lot of things.
11. If you asked me, I would tell you the truth.
12. If I could fly, I would go to the moon.
13. If we had more money, we would help our friends.
14. If she trusted me, things would be different.
15. If I were taller, I would play basketball.
16. If you studied more, you would pass easily.
17. If I were younger, I would start a new career.
18. If we worked together, we would finish faster.
19. If I had a dog, I would walk every morning.
20. If she came tonight, the party would be perfect.
21. What would you say if I asked you to marry me?
22. If I were in your place, I would say yes.
23. If I lived alone, I would be very bored.
24. If the weather were warmer, we would swim.
25. If he loved her, he would call her every day.
26. If I had a chance, I would learn three languages.
27. If we had a garden, we would grow tomatoes.
28. If I were free tonight, I would join you.
29. If she knew how I felt, she would understand.
30. If we left now, we might catch the last bus.
