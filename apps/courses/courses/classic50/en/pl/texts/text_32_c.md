# Tekst do lekcji 32: A conversation about choices
**Temat: Hipotetyczne rozmowy o życiu — drugi i trzeci warunkowy**

---

## Tekst do słuchania i powtarzania

1. What would you do if you could start your life again?
2. I would study music if I had the chance.
3. Really? I thought you wanted to be a lawyer.
4. I did, but if I were honest, music is my real passion.
5. So why didn't you choose it the first time?
6. If my parents had supported me, I would have tried.
7. But they wanted you to have a stable career, right?
8. Yes. If I had been braver, I would have gone my own way.
9. What about now? You could still change your career.
10. If I left my job, my family would lose everything.
11. I understand. If I were in your place, I would feel the same.
12. Some people would say it's never too late to change.
13. True. If I had more time, I would take guitar lessons.
14. You could start small. If you practiced an hour a day, it would add up.
15. Maybe. If I had not given up so easily before, I would be a real musician now.
16. That's a mixed feeling, isn't it?
17. Yes. And if you had become a teacher, would you be happy now?
18. Probably yes. If I had chosen teaching, I would have helped many kids.
19. But you have helped me. If you hadn't given me that advice, I would have failed.
20. I'm glad I did. If I could give one more piece of advice, what would you want to hear?
21. Tell me what you would do if you were thirty again.
22. I would take more risks. If I had taken risks, life would be very different.
23. And what would you not change?
24. The people I love. If I lost them, nothing else would matter.
25. That's beautiful.
26. If everyone thought like that, the world would be kinder.
27. If we hadn't met today, I would have missed this conversation.
28. Me too. If you ever need to talk again, just call.
29. I will. If you had more evenings free, we would meet more often.
30. Then let's plan one. If we set a date now, we won't forget.
