# Тест блока 4 (Уроки 31–40): Count — Условные и сложные предложения B2

**Как сказать это по-английски?**
Переведи фразы на английский. Это **кумулятивный тест Блока 4**: реальные условия (0/1), нереальные условия (2/3), I wish / if only, пассив во всех временах, косвенная речь со сдвигом времён, must be / can't be / might have done, gerund vs to + infinitive, although / despite / so that, артикли 4-й заход (in bed, at home, by car), архитектура длинного сложного предложения.
