# Урок 9: There is / there are; квантификаторы
**Словарный запас: Мебель и комнаты**

---

## Как работать с этим уроком

1. **Прочитай** правило (5 минут)
2. **Проговори вслух** — каждое <say>there is</say> и <say>there are</say>. Сократи в речи: <say>**there's**</say>.
3. **Опиши свою комнату** вслух. Это лучший тренажёр для этой темы.

> Сегодня ты научишься говорить «есть» и «нет» — описывать, что находится в комнате, доме, городе. И главное — **сколько**: «много», «мало», «немного», «никаких». Все эти слова в английском разделены **по типу существительного** (вспомни L8: исчисляемое / неисчисляемое).

---

## Часть 1: <say>there is</say> и <say>there are</say> — главная идея

Когда мы хотим сказать **«где-то существует / находится»** — английский ставит формулу <say>there is</say> (для одного) или <say>there are</say> (для многих). Дословно: «там есть».

Сравни с русским:

| Русский | Английский |
|---------|------------|
| В комнате (есть) стол. | <say>**There is** a table in the room.</say> |
| В комнате (есть) два стола. | <say>**There are** two tables in the room.</say> |
| В саду (есть) вода. | <say>**There is** water in the garden.</say> |

В русском «есть» можно опустить — и часто это так и делают. В английском **нельзя**: формула обязательна.

> **Главное правило:** <say>there is</say> — для **одного** исчисляемого предмета **или** для любого неисчисляемого. <say>there are</say> — для **многих** исчисляемых.

| Что описываем | Формула |
|---------------|---------|
| Один исчисляемый | <say>**There is** a book on the table.</say> |
| Много исчисляемых | <say>**There are** books on the table.</say> |
| Неисчисляемое (всегда!) | <say>**There is** water in the glass.</say> |

### Сокращения

В речи и в неформальном письме почти всегда используется:

- <say>**there's**</say> = there is
- <say>**there're**</say> = there are (произносится «зэйа», но реже сокращается в письме)

> **Ловушка:** не путай <say>there is</say> с <say>it is</say>. <say>It is a book</say> — «это книга» (отождествление). <say>There is a book</say> — «есть книга, существует» (наличие).

## Часть 2: Вопрос и отрицание с <say>there is/are</say>

В уроке 6 ты узнал, что вопросы обычно строятся через <say>do/does</say>. Но **не здесь!** С <say>there is/are</say> работает та же логика, что и с <say>to be</say> (L3) — глагол сам прыгает на первое место.

### Вопрос

> **Is/Are there + (квантификатор) + существительное + ...?**

- <say>**Is there** a book on the table?</say> — На столе есть книга?
- <say>**Are there** any chairs in the room?</say> — В комнате есть стулья?
- <say>**Is there** water in the bottle?</say> — В бутылке есть вода?

### Короткий ответ

- <say>Yes, there is. / No, there isn't.</say>
- <say>Yes, there are. / No, there aren't.</say>

### Отрицание

> **There is no / There are no + существительное** — «нет / никаких»
> или **There isn't / aren't any** — то же самое, разговорнее

- <say>**There is no** time.</say> = <say>**There isn't any** time.</say> — Нет времени.
- <say>**There are no** books on the table.</say> = <say>**There aren't any** books on the table.</say> — Книг на столе нет.

## Часть 3: Квантификаторы — слова количества

Это слова, отвечающие на «сколько?» — но **разные** для исчисляемых и неисчисляемых.

### Универсальные таблицы

| Сколько | С исчисляемым (мн.ч.) | С неисчисляемым |
|---------|----------------------|------------------|
| много | <say>**many**</say> apples / <say>**a lot of**</say> apples | <say>**much**</say> water / <say>**a lot of**</say> water |
| немного | <say>**a few**</say> apples | <say>**a little**</say> water |
| мало / почти нет | <say>**few**</say> apples (без a) | <say>**little**</say> water (без a) |
| какие-то / немного | <say>**some**</say> apples | <say>**some**</say> water |
| (какие-то?) | <say>**any**</say> apples (в вопр. и отриц.) | <say>**any**</say> water (в вопр. и отриц.) |
| никаких | <say>**no**</say> apples | <say>**no**</say> water |

### Главные ошибки русскоязычного

> **Ловушка №1:** <say>much</say> — только с **неисчисляемым** (much water, much money). С исчисляемым — <say>many</say> (many apples, many people). НЕ <say>much apples</say>.

> **Ловушка №2:** <say>a few</say> (с «a») = «немного, но достаточно» (положительный оттенок). <say>few</say> (без «a») = «мало, почти нет» (отрицательный оттенок). То же с <say>a little</say> vs <say>little</say>.

> **Ловушка №3:** <say>a lot of</say> = «много» — работает с **обоими** типами. Это **самый безопасный** вариант. В сомнении — говори <say>a lot of</say>.

## Часть 4: <say>some</say> vs <say>any</say> — главная пара

Это очень важная пара. Запомни **простое правило**:

| | <say>some</say> | <say>any</say> |
|-|-----------------|----------------|
| **Утверждение** | ✓ <say>I have some friends.</say> | ✗ |
| **Отрицание** | ✗ | ✓ <say>I don't have any friends.</say> |
| **Вопрос** | в просьбах/предложениях | в обычных вопросах |

Примеры:

- **Утверждение** (есть): <say>There are **some** apples on the table.</say>
- **Отрицание** (нет): <say>There aren't **any** apples on the table.</say>
- **Обычный вопрос:** <say>Are there **any** apples?</say>
- **Просьба / предложение:** <say>Would you like **some** tea?</say> (предлагаешь — значит, хочешь, чтобы согласились → some)

> **Хак:** в вопросе, где ты ждёшь «да» (предлагаешь, просишь), — <say>some</say>. В вопросе, где ты не знаешь ответа («есть ли?»), — <say>any</say>.

---

## Часть 5: <say>How much</say> vs <say>How many</say>

Вспомни L6 — два «сколько»:

- <say>**How much**</say> + неисчисляемое: <say>How much water do you drink?</say>
- <say>**How many**</say> + исчисляемое (мн.ч.): <say>How many books do you have?</say>

> Это **прямое следствие** того, что мы учили в L8. <say>How much</say> = «сколько массы». <say>How many</say> = «сколько штук».

## Языковые гаммы

### Гамма 1 — there is / there are

```
There is a book on the table
There is a chair near the table
There is a lamp on the desk
There are two windows in the room
There are some pictures on the wall
There are many books on the shelf
```

### Гамма 2 — Вопросы и отрицания

```
Is there a sofa in the living room? – Yes, there is.
Is there a fridge in the kitchen? – Yes, there is.
Are there any chairs in the bedroom? – No, there aren't.
Are there any pictures on the wall? – Yes, there are some.
Is there any water in the bottle? – No, there isn't.
Are there many people in the park? – Yes, there are a lot of people.
```

### Гамма 3 — Квантификаторы

```
There are many books on the shelf.
There is much water in the bottle. (формально; разг.: a lot of)
There are a few chairs in the kitchen.
There is a little milk in the fridge.
There are some apples on the table.
There aren't any oranges on the table.
There is no time.
```

Прогоняй каждую гамму 3 раза.

## Памятка урока

```
THERE IS / THERE ARE — «есть / находится»

  There is + ед.ч. (исч.) / неисч.    There is a book / There is water
  There are + мн.ч. (исч.)            There are books

  Сокр.: there's = there is

ВОПРОС:
  Is there a book?       Yes, there is.    No, there isn't.
  Are there any books?   Yes, there are.   No, there aren't.

ОТРИЦАНИЕ:
  There is no time.   = There isn't any time.
  There are no books. = There aren't any books.

КВАНТИФИКАТОРЫ — РАЗДЕЛЕНИЕ ПО ТИПУ:

                Исчисл. (мн.ч.)     Неисчисл.
  много         many / a lot of     much / a lot of
  мало (-)      few                 little
  немного (+)   a few               a little
  какие-то      some                some
  никаких       no / not any        no / not any

  Хак: a lot of — работает с ОБОИМИ. В сомнении — a lot of.
  Ловушка: many с исчисл., much с неисчисл. Не "much books".

SOME vs ANY:
  утвержд.:  some  (I have some friends)
  отриц.:    any   (I don't have any friends)
  вопрос:    any (обычный) / some (просьба, предложение)
            Would you like some tea? — предлагаю, жду «да»

A FEW vs FEW   |   A LITTLE vs LITTLE
  a few = немного, достаточно (+)
  few   = мало, почти нет     (-)
  a little = немного (+)
  little   = мало (-)

HOW MUCH / HOW MANY:
  How much water?    (неисч.)
  How many books?    (исч.)

КЛЮЧЕВЫЕ НЕИСЧ. (вспомни L8):
  water, milk, bread, money, time, information, advice, music, furniture
  → С ними: is, much, a little, some
```
