# Урок 46: Словообразование — как из одного корня вырастают двадцать слов

**Словарный запас: продуктивные аффиксы, словарные семьи, абстрактная и формальная лексика (~42)**

---

## Как работать с этим уроком

1. **Прочитай** правила аффиксов — это не зубрёжка, это формулы (10 минут).
2. **Прогони знакомые корни** через все суффиксы — за час удвоишь словарный запас.
3. **Угадывай** значения незнакомых слов в текстах разбором на префикс + корень + суффикс.
4. **Прокачай** конверсию: одно и то же слово как существительное и как глагол — типично для английского.

> Учить отдельные слова = арифметика. Учить аффиксы = алгебра.
> Один суффикс открывает сотню слов. Один префикс — ещё столько же.

---

## Часть 1: Зачем тебе словообразование на C1

К этому моменту у тебя уже большой словарный запас — может быть, 3000–5000 слов. Чтобы перейти от B2 к C1, ты должен(а) уметь **самостоятельно расширять** этот запас. И вот хорошая новость: английский — это **самый словообразовательно продуктивный** язык в этой серии. Один глагол *decide* даёт тебе:

- <say>decide</say> — глагол: *решать, принимать решение*
- <say>decision</say> — существительное: *решение*
- <say>decisive</say> — прилагательное: *решающий, решительный*
- <say>decisively</say> — наречие: *решительно*
- <say>indecisive</say> — отрицательное прилагательное: *нерешительный*
- <say>indecision</say> — отрицательное существительное: *нерешительность*

**Шесть слов из одного корня.** Если ты видишь паттерн, ты не учишь шесть слов — ты учишь *одну формулу* и применяешь её к сотням других корней.

> **Главная C1-стратегия:** видя незнакомое слово, разложи его на **префикс + корень + суффикс**. В 80% случаев значение угадывается. *Unmistakable* = *un-* (не) + *mistake* (ошибка) + *-able* (возможность) → «такой, что ошибиться невозможно».

## Часть 2: Главные суффиксы существительных

Суффикс — это то, что превращает одну часть речи в другую. У существительных в английском есть штук пятнадцать продуктивных суффиксов; мы возьмём восемь самых важных.

| Суффикс | От чего | Значение | Пример |
|---------|---------|----------|--------|
| **-tion / -sion** | глагол | действие или результат | <say>create → creation</say>, <say>decide → decision</say> |
| **-ment** | глагол | действие, результат, состояние | <say>develop → development</say>, <say>achieve → achievement</say> |
| **-er / -or** | глагол | агент (кто делает) | <say>teach → teacher</say>, <say>act → actor</say> |
| **-ee** | глагол | пациент (с кем делают) | <say>employ → employee</say>, <say>interview → interviewee</say> |
| **-ness** | прилагательное | абстрактное качество | <say>happy → happiness</say>, <say>kind → kindness</say> |
| **-ity / -ty** | прилагательное | качество (романские основы) | <say>able → ability</say>, <say>real → reality</say> |
| **-ism** | существительное / прил. | учение, система взглядов | <say>real → realism</say>, <say>capital → capitalism</say> |
| **-ship** | существительное | состояние, отношения, статус | <say>friend → friendship</say>, <say>leader → leadership</say> |

> **Хак выбора:** прилагательные германских корней (короткие, бытовые: *happy, kind, dark, sad*) почти всегда дают **-ness**. Прилагательные романских корней (длинные, латинские: *able, real, similar, possible*) — **-ity**. Глаголы на *-ate, -ize, -ify* почти всегда дают **-tion** (*create → creation, organize → organization, simplify → simplification*).

> **Ловушка -ee:** суффикс **-ee** обозначает того, **на ком** совершается действие, в отличие от *-er* (того, кто совершает). *Employer* — работодатель; *employee* — работник. *Interviewer* — берущий интервью; *interviewee* — у кого берут.

## Часть 3: Главные суффиксы прилагательных

| Суффикс | От чего | Значение | Пример |
|---------|---------|----------|--------|
| **-able / -ible** | глагол | «может быть сделанным» | <say>read → readable</say>, <say>access → accessible</say> |
| **-ful** | существительное | «полный чего-то» | <say>care → careful</say>, <say>hope → hopeful</say> |
| **-less** | существительное | «без чего-то» | <say>care → careless</say>, <say>hope → hopeless</say> |
| **-ous** | существительное | «обладающий» | <say>danger → dangerous</say>, <say>fame → famous</say> |
| **-y** | существительное | «характеризуемый чем-то» | <say>sun → sunny</say>, <say>noise → noisy</say> |
| **-al** | существительное | «относящийся к» | <say>nature → natural</say>, <say>culture → cultural</say> |
| **-ive** | глагол / сущ. | «склонный к» | <say>act → active</say>, <say>create → creative</say> |
| **-ic** | существительное | «относящийся к» (греческие корни) | <say>history → historic</say>, <say>economy → economic</say> |

> **Золотая пара:** **-ful** vs. **-less** — это идеальные антонимы. *Useful* / *useless*, *hopeful* / *hopeless*, *helpful* / *helpless*, *careful* / *careless*. Дают тебе двойной словарь.

> **Ловушка -able vs -ible:** правило простое в 90% случаев — **-able** прибавляется к английским корням (*readable, drinkable, comfortable*), **-ible** — к латинским основам, чаще к тем, у которых уже есть форма на *-ion* (*visible → vision, possible, edible*). Учить лучше с конкретным словом.

## Часть 4: Префиксы — главное оружие C1

Префикс в английском почти всегда **меняет значение, не часть речи**. Глагол остаётся глаголом, прилагательное прилагательным. Но смысл переворачивается или сдвигается.

### Отрицательные префиксы

Это самая продуктивная группа. Знать их = удвоить корпус прилагательных.

| Префикс | Когда | Пример |
|---------|-------|--------|
| **un-** | универсальный, германские корни | <say>happy → unhappy</say>, <say>do → undo</say>, <say>known → unknown</say> |
| **in-** | романские корни | <say>active → inactive</say>, <say>visible → invisible</say> |
| **im-** | перед **m, p** | <say>possible → impossible</say>, <say>mature → immature</say> |
| **il-** | перед **l** | <say>legal → illegal</say>, <say>logical → illogical</say> |
| **ir-** | перед **r** | <say>regular → irregular</say>, <say>responsible → irresponsible</say> |
| **dis-** | «противоположность» или «отделение» | <say>agree → disagree</say>, <say>appear → disappear</say> |
| **non-** | нейтральное «не» (без оценки) | <say>smoker → non-smoker</say>, <say>fiction → non-fiction</say> |
| **mis-** | «неправильно, ошибочно» | <say>understand → misunderstand</say>, <say>spell → misspell</say> |

> **Правило ассимиляции:** *in-, im-, il-, ir-* — это **один и тот же префикс**, который меняет вид перед разными согласными. **im-** перед губными (m, p, b), **il-** перед l, **ir-** перед r, **in-** в остальных случаях. Не запоминай таблицу — запомни принцип: префикс «подгоняется под звук», как у нас «без- / бес-».

> **Тонкая разница: un- vs in- vs non-.** *Unhappy* — несчастный (эмоция, оценка). *Inactive* — бездействующий (нейтральная характеристика). *Non-smoker* — некурящий (просто факт, без оценки). На C1 важно чувствовать разницу: *unprofessional* = критика; *non-professional* = просто «не профессионал».

> **Dis- vs un-:** в глаголах эти двое часто звучат похоже, но *un-* обычно «отменить действие» (*undo* — отменить сделанное, *unpack* — распаковать), а *dis-* — «противоположное действие» (*disagree* — не соглашаться, *dislike* — не любить).

### Другие важные префиксы

| Префикс | Значение | Пример |
|---------|----------|--------|
| **re-** | снова, заново | <say>write → rewrite</say>, <say>build → rebuild</say> |
| **pre-** | до, заранее | <say>view → preview</say>, <say>pay → prepay</say> |
| **post-** | после | <say>war → post-war</say>, <say>graduate → postgraduate</say> |
| **over-** | сверх, чрезмерно | <say>work → overwork</say>, <say>sleep → oversleep</say> |
| **under-** | недостаточно, под | <say>cook → undercook</say>, <say>estimate → underestimate</say> |
| **sub-** | под, ниже | <say>marine → submarine</say>, <say>conscious → subconscious</say> |
| **super-** | сверх, выше | <say>man → superman</say>, <say>natural → supernatural</say> |
| **co-** | совместно | <say>worker → co-worker</say>, <say>operate → cooperate</say> |
| **ex-** | бывший, из | <say>husband → ex-husband</say>, <say>president → ex-president</say> |
| **anti-** | против | <say>virus → antivirus</say>, <say>social → antisocial</say> |

> **C1-хак:** видя длинное незнакомое слово, ищи **знакомый корень внутри**. *Underestimate* = *under-* (недо-) + *estimate* (оценить). *Misinterpretation* = *mis-* (ошибочно) + *interpret* (истолковать) + *-ation* (действие) → «ошибочная интерпретация». Слово в шесть букв превращается в слово в три.

## Часть 5: Конверсия — одно слово, две части речи

Это **специфика английского**, которую русскоговорящие чаще всего недооценивают. В английском одно и то же слово часто работает и как существительное, и как глагол — без всякого изменения формы.

| Слово | Как существительное | Как глагол |
|-------|---------------------|------------|
| <say>water</say> | вода | поливать |
| <say>email</say> | письмо | написать письмо |
| <say>google</say> | (бренд) | искать в Гугле |
| <say>book</say> | книга | бронировать |
| <say>plan</say> | план | планировать |
| <say>hand</say> | рука | передавать |
| <say>face</say> | лицо | стоять лицом к, столкнуться |
| <say>head</say> | голова | возглавлять |
| <say>name</say> | имя | называть |
| <say>park</say> | парк | парковаться |

> **Контраст с русским:** в русском нужны разные слова или явный суффикс (*рука → ручной, ручаться*). В английском часто хватает контекста: *I will water the plants* — поливать; *I drink water* — вода. Никакого -ой, -овать, -изиров- не нужно.

> **C1-наблюдение:** в современном английском, особенно в бизнесе и технологиях, **любое существительное** может стать глаголом. *Friend* → *to friend someone* (на фейсбуке). *Inbox* → *to inbox someone*. *Adult* → *to adult* (по-взрослому жить). Это естественный процесс языка, не ошибка.

### Сдвиг ударения при конверсии

Часто у двусложных слов меняется ударение между ролями. Существительное — ударение на первый слог, глагол — на второй:

| Слово | Существительное | Глагол |
|-------|-----------------|--------|
| <say>record</say> | **RE**cord (запись) | re**CORD** (записывать) |
| <say>present</say> | **PRE**sent (подарок) | pre**SENT** (представлять) |
| <say>object</say> | **OB**ject (объект) | ob**JECT** (возражать) |
| <say>increase</say> | **IN**crease (рост) | in**CREASE** (расти) |
| <say>contract</say> | **CON**tract (контракт) | con**TRACT** (сжиматься) |
| <say>conduct</say> | **CON**duct (поведение) | con**DUCT** (проводить) |

> Это правило живое, но не абсолютное — есть исключения. Однако паттерн настолько частый, что его стоит узнавать.

## Часть 6: Словарные семьи — учи группами, не по одному

Возьми любой важный корень и **построй вокруг него семью**. Это твоя главная задача на C1.

### Пример: семья *create*

| Часть речи | Слово | Перевод |
|------------|-------|---------|
| глагол | <say>create</say> | создавать |
| существительное (действие) | <say>creation</say> | создание |
| существительное (агент) | <say>creator</say> | создатель |
| прилагательное | <say>creative</say> | творческий |
| наречие | <say>creatively</say> | творчески |
| абстрактное сущ. | <say>creativity</say> | креативность |
| прилагательное | <say>creational</say> | связанный с творением |
| отрицание | <say>uncreative</say> | нетворческий |
| отрицание | <say>recreation</say> | (1) воссоздание; (2) отдых |

Из одного корня — девять слов. Из них половина — высокочастотная C1-лексика.

### Пример: семья *act*

| Часть речи | Слово | Перевод |
|------------|-------|---------|
| глагол | <say>act</say> | действовать |
| существительное (действие) | <say>action</say> | действие |
| существительное (агент) | <say>actor / actress</say> | актёр / актриса |
| прилагательное | <say>active</say> | активный |
| наречие | <say>actively</say> | активно |
| абстрактное сущ. | <say>activity</say> | активность, деятельность |
| глагол | <say>activate</say> | активировать |
| существительное | <say>activation</say> | активация |
| прилагательное | <say>inactive</say> | неактивный |
| абстрактное сущ. | <say>inactivity</say> | бездеятельность |
| прилагательное | <say>interactive</say> | интерактивный |
| глагол | <say>react</say> | реагировать |
| существительное | <say>reaction</say> | реакция |

Тринадцать слов. Корень *act* — три буквы.

## Часть 8: Языковые гаммы

### Гамма 1 — Прокачка одной семьи (create)

Прочитай вслух, осознавая, как меняется часть речи:

```
She wants to create something new.
Her creation was praised by everyone.
The creator stood quietly in the corner.
He is a very creative person.
She solved the problem creatively.
The creativity of the team is remarkable.
This approach is uncreative and outdated.
We need recreation after a long week.
```

### Гамма 2 — Префикс un- через все лица

```
I am unable to help you right now.
You are unable to come tomorrow.
He is unable to find the file.
She is unable to read this letter.
We are unable to predict the result.
They are unable to agree on a date.
```

### Гамма 3 — Конверсия noun ↔ verb

Прогони пары — одно слово в двух ролях:

```
I water the plants every morning.    I drink a glass of water.
I email her every week.               I got a long email yesterday.
I book a table for two.               I read a good book last night.
I plan my day in the morning.         The plan worked perfectly.
I park my car here every day.         The park is full of children.
```

## Часть 9: Языковая матрица

### Матрица 1 — От глагола к существительному

```
What did you decide? — That was a hard decision.
What did you achieve? — It was my biggest achievement.
What did you develop? — The development took two years.
What did you improve? — The improvement is obvious.
What did you arrange? — The arrangement is final.
What did you discover? — The discovery changed everything.
```

### Матрица 2 — От прилагательного к существительному

```
Is she happy? — Yes, her happiness is contagious.
Is he kind? — Yes, his kindness is well known.
Is the task possible? — Yes, the possibility is real.
Is the system flexible? — Yes, its flexibility is its strength.
Is he responsible? — Yes, he has full responsibility for this.
Is she aware? — Yes, her awareness of the issue is high.
```

### Матрица 3 — Префиксная гимнастика

```
Is it possible? — No, it is impossible.
Is it legal? — No, it is illegal.
Is it regular? — No, it is irregular.
Is it logical? — No, it is illogical.
Is it relevant? — No, it is irrelevant.
Is it polite? — No, it is impolite.
Is it predictable? — No, it is unpredictable.
Does he agree? — No, he disagrees.
Did you understand? — No, I misunderstood.
Will you do it? — No, I will undo it.
```

### Упражнение 4. Семья одного корня

Заполни таблицу для корня *succeed*:

| Часть речи | Слово |

<details>
<summary>Ключ</summary>

- существительное: *success* (успех) — нерегулярный, без -ion

## Памятка урока

```
СЛОВООБРАЗОВАНИЕ = три инструмента: ПРЕФИКС + КОРЕНЬ + СУФФИКС

ОСНОВНЫЕ СУФФИКСЫ СУЩЕСТВИТЕЛЬНЫХ:
  глагол → -tion / -sion   create → creation, decide → decision
  глагол → -ment           develop → development
  глагол → -er / -or       teach → teacher
  глагол → -ee             employ → employee  (тот, на ком действие)
  прил. → -ness           happy → happiness   (германские)
  прил. → -ity            able → ability      (романские)
  сущ. → -ship            friend → friendship
  сущ. → -ism             real → realism

СУФФИКСЫ ПРИЛАГАТЕЛЬНЫХ:

  сущ. → -ful / -less     care → careful / careless
  сущ. → -ous             danger → dangerous
  сущ. → -al              culture → cultural

ОТРИЦАТЕЛЬНЫЕ ПРЕФИКСЫ (по корню/звуку):
```
