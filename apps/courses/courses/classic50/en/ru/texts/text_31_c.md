# Текст к уроку 31: A weekend trip
**Тема: Реальные планы на выходные — нулевой и первый условный в диалоге**

---

## Текст для аудирования и повторения

1. What will we do this weekend if the weather is bad?
2. If it rains, we'll stay home and watch a film.
3. And if the sun comes out, we'll go to the lake.
4. I always pack a swimsuit in case it gets warm.
5. If we leave early, we'll avoid the traffic.
6. The road gets very busy if everyone goes at the same time.
7. Should we take the dog? Yes, if you take care of him.
8. He gets sad if we leave him alone for too long.
9. If we take him, we can't go to that restaurant.
10. Dogs aren't allowed there unless they stay outside.
11. I'll book a table somewhere else if you want.
12. Call my brother as soon as we decide.
13. If he's free, he'll join us.
14. He always says yes if there's food involved.
15. We'll need petrol if the tank is low.
16. The car uses more if you drive fast.
17. If we share the costs, the trip won't be expensive.
18. I'll pay for the petrol if you pay for lunch.
19. That works. If the kids come, we'll split everything.
20. They get bored if the drive is too long.
21. We'll stop on the way if they need a break.
22. If everyone is ready by nine, we'll leave on time.
23. If anyone is late, we leave without them.
24. I'll send a message in our group chat tonight.
25. Tell them to bring a jacket in case it gets cold.
26. If the lake is calm, we can swim safely.
27. Be careful if the water is too cold.
28. If we have time, we'll walk around the village.
29. The place is lovely when the weather is right.
30. I'll let you know as soon as I have all the details.
