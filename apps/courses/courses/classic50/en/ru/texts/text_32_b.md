# Текст к уроку 32: If I had known
**Тема: Сожаления о прошлом — третий условный**

---

## Текст для аудирования и повторения

1. If I had known, I would have come earlier.
2. If she had studied, she would have passed the exam.
3. If we had left on time, we would have caught the train.
4. If you had told me, I would have helped you.
5. If they had invited us, we would have gone to the party.
6. If I had seen the message, I would have answered.
7. If he had been more careful, the accident would not have happened.
8. If we had booked the tickets earlier, we would have got better seats.
9. If she had stayed in school, she would have become a doctor.
10. If I had eaten less, I wouldn't have felt sick.
11. If he had asked, I would have said yes.
12. If we had taken a map, we wouldn't have got lost.
13. If she had not lied, we would still be friends.
14. If I had brought my umbrella, I wouldn't have got wet.
15. If you had practiced, you would have won the match.
16. If they had listened to me, the project would have succeeded.
17. If I had had your phone, I would have called you.
18. If we had moved to London, our life would have changed.
19. If she had married him, she would have been miserable.
20. If I had saved money, I could have bought the apartment.
21. If we had warned them, the disaster would have been avoided.
22. If he had been honest, I would have forgiven him.
23. If they had checked the figures, they would have spotted the error.
24. If I had slept better, I would have done a good job.
25. If she had taken the offer, her career would have grown.
26. If we had stayed longer, we would have seen the show.
27. If I had been there, I would have helped.
28. If he had spoken up, things would be different now.
29. If we had bought the house then, we would be rich today.
30. What would you have done if you had been in my place?
