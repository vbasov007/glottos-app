# Юнит 10 — Винительный падеж + контрольный смотр Stage 1

**Словарный запас: GRAMMAR + CHECKPOINT · Функция: сказать, чего вы хотите, что берёте, покупаете — и закрепить весь Stage 1 · Грамматика: винительный падеж (аккузатив) — PRODUCE, Tier 4 по цене ошибки, но вводится как опора для частых фраз.**

**Что вы сможете после этого юнита:** правильно сказать «я хочу / беру / ищу [предмет]» — и провести связный бытовой разговор, объединяющий весь этап.

---

## Раздел 1. Объяснение темы

### 1.1. Главная идея

Когда предмет — это **то, на что направлено действие** (я беру *что?*, я хочу *что?*), он стоит в винительном падеже. Хорошая новость: видимо меняется только **один** артикль — мужской род.

| Род | Именительный (кто/что делает) | Винительный (на что направлено) |
|-----|-------------------------------|----------------------------------|
| мужской | der Kaffee | **den** Kaffee |
| женский | die Suppe | die Suppe (без изменений) |
| средний | das Buch | das Buch (без изменений) |
| множ. | die Bücher | die Bücher (без изменений) |

Неопределённый артикль: мужской **ein → einen**, женский *eine* и средний *ein* не меняются.

```
Ich nehme   den   Kaffee.      ← мужской: der → den
Ich nehme   die   Suppe.       ← женский: без изменений
Ich möchte  einen Salat.       ← мужской: ein → einen
Ich möchte  ein   Buch.        ← средний: без изменений
```

### 1.2. Ключевые фразы

Ich möchte einen Kaffee. (Я хочу кофе.) Ich nehme das Buch. (Я беру книгу.) Ich suche eine Wohnung. (Я ищу квартиру.) Ich habe einen Bruder. (У меня есть брат.)

### 1.3. Это единственный падеж этапа

В Stage 1 есть только винительный — и только в «лёгкой» версии: запомнить надо одно изменение, *der → den* и *ein → einen*. Дательный падеж — Юнит 28, родительный — вообще не для производства. Не пытайтесь учить «все падежи» сейчас. Одно изменение, мужской род.

### 1.4. Почему ошибка здесь недорога

Если вы скажете *Ich nehme der Kaffee* вместо *den Kaffee* — вас всё равно поймут мгновенно. Это Tier 4: винительный стоит знать, потому что он опирает частые фразы, но ошибка в нём не рушит общение. Поэтому он вводится в конце этапа, а не в начале.

### CHECKPOINT — контрольный смотр Stage 1

> Это контрольная часть юнита (повторение R4). Выполняется как связная цепочка — «утро в немецком городе».

### Задание C1. Постройте цепочку реплик
Переведите этот сценарий на немецкий — стройте каждое предложение сами. Сюжет дан; ваша задача — только немецкие фразы. Пройдите все семь ситуаций по порядку.
*«— Добрый день! Меня зовут [имя]. — Очень приятно. — Я из России. Сейчас я живу в Мюнхене. — Утром я работаю, днём у меня есть время. — В кафе: я бы хотел кофе и пирог. — Извините, где вокзал? — Прямо, потом налево. — В магазине: у вас есть книги? Эту я беру. — До свидания!»*
В каждой из семи частей — минимум по два предложения.

Это контрольное задание намеренно даётся без готового сюжета — проверяется, можете ли вы говорить о себе самостоятельно. Без опоры напишите рассказ о себе на 8–10 предложений, используя весь материал Stage 1.

## Заметка для автора курса (не показывается изучающему)

Stage 1 завершён. Замечания для авторства Stage 2–5:

- **Тип юнита виден по Разделу 1.** GRAMMAR-юнит: Раздел 1 объясняет новую грамматику. PRACTICE-юнит: Раздел 1 объясняет функцию и наборы фраз, новой грамматики нет — это явно указано в шапке. CHECKPOINT встроен в последний юнит этапа.
- **Раздел 2 — только порождение.** Запрещены задания «вставь слово», «выбери из скобок», «соедини половинки». Каждое упражнение даёт стимул-намерение, изучающий строит фразу с нуля. 6–8 упражнений, минимум пять разных типов. Ключ — образец + критерий проверки.
- **Упражнения на связный текст / диалог дают готовый сюжет на русском.** Если упражнение просит написать связный текст из нескольких предложений или диалог, сюжет (что именно сказать, по-русски) даётся в задании. Изучающий не тратит силы на сочинение содержания — он только порождает немецкие фразы. Исключение — одно намеренно свободное упражнение на юнит-PRACTICE и задание C2 в чекпойнте: они без сюжета, потому что проверяют речь без опоры; это явно помечается в формулировке задания.
- **Раздел 3 — ровно три аудио-текста по 30 предложений.** Только скрипты для TTS, без заданий — изучающий работает в приложении glottos. Темп TTS обычный; трудно — увеличить число повторов. RECOGNIZE-материал помечается одним примечанием в конце раздела.
- **Метки PRODUCE/RECOGNIZE соблюдаются строго.** В упражнениях на производство проверяется только PRODUCE-материал. RECOGNIZE-материал может быть во входе (реплики, аудио), но не проверяется в порождаемой фразе.
- **Интервальное повторение.** Каждый юнит в конце указывает, какой материал возвращается (R1 — следующий юнит, R2 — через три, R3 — через семь, R4 — контрольный смотр этапа, R5 — вход следующего этапа).
- Скрипты аудио — контент для polyGlottos TTS и голосовых упражнений.
- Frontmatter каждого юнита оформляется как у образца Юнита 1; при разбивке на отдельные файлы — по одному файлу на юнит в `lessons/de/ru/A1/`.
