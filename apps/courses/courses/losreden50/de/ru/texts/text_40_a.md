# Текст к Юниту 40 (вариант A): Anweisungen vom Lehrer

**Тема: Повелительное наклонение + контрольный смотр Stage 4**

---

## Текст для аудирования

1. In der Sprachschule gibt der Lehrer viele Anweisungen.
2. Setzen Sie sich, bitte.
3. Öffnen Sie Ihr Buch auf Seite zwanzig.
4. Lesen Sie den Text einmal in Ruhe.
5. Achten Sie auf die neuen Wörter.
6. Schreiben Sie die wichtigen Vokabeln auf.
7. Arbeiten Sie zu zweit.
8. Diskutieren Sie das Thema kurz.
9. Stellen Sie Fragen, wenn Sie etwas nicht verstehen.
10. Haben Sie keine Angst vor Fehlern.
11. Wiederholen Sie die Sätze laut.
12. Sprechen Sie deutlich und langsam.
13. Hören Sie sich gegenseitig zu.
14. Korrigieren Sie sich freundlich.
15. Schreiben Sie die Hausaufgabe für morgen auf.
16. Machen Sie eine kurze Pause.
17. Trinken Sie etwas Wasser.
18. Atmen Sie tief durch.
19. Konzentrieren Sie sich danach wieder.
20. Üben Sie auch zu Hause regelmäßig.
21. Schauen Sie sich neue Filme an.
22. Hören Sie deutsche Musik.
23. Sprechen Sie mit echten Menschen.
24. Habt Geduld mit euch selbst.
25. Sprachen brauchen Zeit.
26. Aber gebt nicht auf.
27. Mit jedem Tag werdet ihr besser.
28. Auf Wiedersehen!
