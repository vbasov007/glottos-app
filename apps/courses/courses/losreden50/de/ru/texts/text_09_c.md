# Текст к Юниту 9 (вариант C): Ein Tag, drei Geschäfte

**Тема: Покупки**

---

## Текст для аудирования

1. Anna hat heute frei.
2. Sie möchte einkaufen gehen.
3. Zuerst geht sie in ein Schuhgeschäft.
4. Sie braucht neue Sommerschuhe.
5. Die Verkäuferin ist sehr hilfsbereit.
6. Anna probiert mehrere Paare an.
7. Ein Paar ist bequem und schön.
8. Aber leider sind die Schuhe teuer.
9. Anna überlegt einen Moment.
10. Dann entscheidet sie sich dagegen.
11. Danach besucht sie eine Buchhandlung.
12. Dort stöbert sie eine Weile.
13. Sie kauft einen Reiseführer über Italien.
14. Sie plant nämlich einen Urlaub.
15. Im dritten Geschäft gibt es Kleidung.
16. Anna sucht ein leichtes Sommerkleid.
17. Sie findet ein hübsches Kleid in Gelb.
18. Der Preis ist günstig, nur dreißig Euro.
19. Sie kauft das Kleid sofort.
20. An der Kasse ist eine lange Schlange.
21. Anna wartet geduldig.
22. Endlich ist sie an der Reihe.
23. Sie bezahlt und packt alles ein.
24. Müde, aber zufrieden geht sie nach Hause.
25. Unterwegs trinkt sie noch einen Kaffee.
26. Sie denkt schon an den nächsten Einkauf.
27. Einkaufen kann anstrengend sein.
28. Aber heute war es ein schöner Tag!
